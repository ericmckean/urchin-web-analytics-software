Erreurs de lien

Comment assurer le suivi des sites dont les liens vers votre site sont obsolètes ou erronés ? Le rapport Erreurs de lien contient les erreurs signalées dans vos journaux et indique le nombre de tentatives de requête erronées. Cliquez sur pour voir le détail du contenu demandé sur votre site, puis sur le site référent ou la page Web d'où provenaient les requêtes avant de rencontrer l'erreur sur votre site.
