Affichage des rapports
Cet écran affiche une liste de tous les profils disponibles auxquels vous avez accès. Vous avez la possibilité de consulter les données de performance de plusieurs profils sur un même écran.

Calendrier limité
Le calendrier limité, situé dans le coin supérieur droit de l’écran, vous permet de choisir une plage de dates (jour, semaine, mois, année).

Accéder au rapport
Cliquez sur le nom du profil pour lancer la fenêtre de création de rapports pour ce profil. La fenêtre de création de rapports vous permet de consulter tous les rapports disponibles pour ce profil.

Modifier (super administrateur uniquement)
Le bouton “Modifier“ vous donne accès à tous les paramètres du profil, sous forme d’onglets.

État (lien hypertexte, super administrateur uniquement)
L’état de chaque tâche peut être : “Non planifié“, “En attente“, “Placé en file d’attente“, “Exécution“, “Annulation“, “Terminé“, “Annulé“, “Avertissement“ ou “Erreur“. Ces états sont codés par des couleurs. Pour trier la liste par état, cliquez sur la flèche située sous le mot “État”. Pour afficher les détails d’exécution d’une tâche, cliquez sur l’entrée d’état codée par une couleur à côté de cette tâche. Une nouvelle fenêtre affiche alors les détails d’exécution.

Commandes
Utilisez le menu déroulant “Nombre d’éléments à afficher“ pour modifier le nombre d’entrées répertoriées simultanément. Utilisez les boutons “Précédent“ et “Suivant“ pour consulter les entrées page par page. Pour accéder au site en lui-même, cliquez sur le lien à côté du nom du profil.

Compte (en mode Datacenter)
En mode Datacenter, le menu “Compte“ peut filtrer les entrées.

Afficher
Utilisez le menu déroulant “Afficher“ pour sélectionner l’affichage Administrateur, Standard ou Entreprises pour les pages de présentation.
