Détail des adresses IP

Le rapport Détail des adresses IP permet de naviguer entre les adresses IP pour isoler des blocs d'adresses IP spécifiques. Pour sélectionner un bloc d'adresses IP, sélectionnez d'abord le bouton situé à côté des premiers chiffres (compris entre 0 et 255) de l'adresse IP demandée et précédant le premier point (.). Cliquez sur le bouton situé à côté des chiffres correspondants suivants (0-255), précédant le point (.) suivant de l'adresse IP, pour voir encore plus de détails sur l'adresse IP. Poursuivez cette procédure jusqu'à l'isolement du bloc d'adresses IP de votre choix.
