Gestionnaire de balises AdWords

Nous vous recommandons d'adopter une stratégie globale pour mettre en œuvre le balisage de vos annonces en ligne avec Urchin. Urchin pourra ainsi vous indiquer quelles activités marketing sont efficaces. À l'aide du Gestionnaire de balises AdWords, vous pouvez baliser en un seul clic vos liens associés à des mots clés achetés. Il vous suffit de suivre les instructions suivantes :

Cible à baliser :

  • Sélectionnez le groupe d'annonces contenant les mots clés/annonces que vous souhaitez baliser.
  • Vous pouvez choisir de baliser des mots clés, des annonces ou les deux.

URL à baliser :
En spécifiant l'URL à baliser vous avez deux options :

  • Baliser les URL existantes (avec l'option "Ne pas prendre en compte les URL déjà balisées").
  • Appliquer une nouvelle URL à tous les éléments balisés à l'intérieur du groupe d'annonces. Cette option modifiera l'URL de destination de vos mots clés/annonces.

Balises UTM :
N'utilisez que les variables de campagne dont vous avez besoin.
Les fonctionnalités de balisage de liens d'Urchin permettent d'identifier de façon unique presque toutes les campagnes. Urchin effectue le suivi des campagnes en ligne en associant les cinq dimensions marketing suivantes :

  • Source
  • Support
  • Terme
  • Contenu
  • Campagne
