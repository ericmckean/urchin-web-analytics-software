Adresses IP non résolues

Le rapport Adresses IP non résolues répertorie tous les visiteurs dont l'adresse IP ne comporte pas de données de propriété (par exemple, la situation géographique ou l'hôte du domaine). Le suivi des adresses IP contenues dans le rapport Adresses IP non résolues est impossible dans les rapports Urchin relatifs au domaine ou à la situation géographique.
