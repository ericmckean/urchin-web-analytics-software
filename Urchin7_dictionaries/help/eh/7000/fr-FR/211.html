Noms d'hôtes

Si vous avez plusieurs domaines avec le même contenu de site Web, quels domaines les visiteurs consultent-ils ? Ce rapport indique le nombre total de pages vues pour chaque domaine. Si ce rapport affiche un domaine qui ne vous appartient pas, cela signifie que ce domaine dirige vers un site Web contenant votre UTM (voire d'autres contenus vous appartenant).

Cliquez sur le bouton à gauche d'une entrée pour afficher les valeurs par période.

Pour comparer deux périodes distinctes, cliquez sur l'icône de la barre de titre bleue Période.
