Chemins de navigation vers l'objectif

Quel chemin de navigation suivent les visiteurs qui atteignent mes objectifs ? Ce rapport répertorie les chemins de navigation permettant d'atteindre l'objectif sélectionné (choisissez votre objectif dans le menu déroulant dans la partie supérieure gauche) et affiche le nombre de conversions par chemin. Ce rapport est différent des rapports d'entonnoirs de conversion définis dans la mesure où il indique les différents chemins de navigation utilisés pour atteindre un objectif, au lieu d'effectuer un suivi des entrées et sorties vers/depuis ces entonnoirs de conversion définis. Utile pour les sites comportant de nombreux liens, ce rapport vous permet d'avoir un aperçu du contenu "obligatoire" dont les visiteurs doivent prendre connaissance avant de pouvoir réaliser la conversion. Ces informations peuvent servir à améliorer la navigation globale au niveau du site.
