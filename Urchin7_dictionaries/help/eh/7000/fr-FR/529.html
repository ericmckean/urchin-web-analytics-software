Type de fichier

Type de fichier : Le rapport Types de fichiers répertorie les types MIME de tous les fichiers envoyés par vos serveurs, classés par appels de fichier réussis et par taille de fichier. Vous pouvez configurer les types MIME de ce rapport sous l'onglet Création de rapports de la page Paramètres de profil.
