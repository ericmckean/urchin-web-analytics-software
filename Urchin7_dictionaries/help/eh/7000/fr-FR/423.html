Synthèse géographique

La synthèse géographique est une représentation géographique du volume de visiteurs en provenance de diverses régions à travers le monde. Les points les plus larges représentent les régions amenant le plus de visiteurs sur votre site Web. Cliquez sur un point pour voir le nombre de visites, le nombre de pages vues par visite, les taux de conversion et la valeur moyenne par visite de la région concernée. Modifiez le nombre de données en augmentant le nombre affiché dans le menu déroulant Affichage de la barre de titre. Pour effectuer un zoom avant sur une région, cliquez avec le bouton droit de la souris et sélectionnez "zoom avant" dans le menu Affichage.
