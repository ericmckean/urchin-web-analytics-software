Répartition sur la journée

À quelles heures de la journée chacun des mots clés est-il le plus efficace ? Ce rapport compare le nombre de réponses et les objectifs ou transactions par réponse pour chaque mot clé.

Cliquez sur le signe plus (+) à côté d'un mot clé pour détailler ces données et les afficher selon l'heure de la journée.
<%dic|default|507%>

Pour comparer deux périodes distinctes, cliquez sur dans la barre de titre bleue "Période".

"Réponses" correspond au nombre de clics qui ont généré une visite sur votre site. "O/R" correspond aux objectifs divisés par le nombre de réponses. "T/R" correspond au nombre de transactions divisé par le nombre de réponses.

Cliquez sur "Informations sur le mot clé" (disponibles uniquement pour les mots clés AdWords) pour consulter le compte, la campagne et le groupe d'annonces AdWords associés au mot clé.

Utilisez le lien ou le lien Générateur de mots clés AdWords pour copier les mots clés sélectionnés dans le Générateur de mots clés, où ils pourront être utilisés pour générer de nouvelles idées de mots clés. Les mots clés générés peuvent être importés dans votre compte AdWords.
