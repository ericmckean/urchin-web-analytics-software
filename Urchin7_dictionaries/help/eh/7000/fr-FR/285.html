Corrélation produit/mot clé

Quel est le niveau de popularité des produits auprès des clients qui ont effectué une recherche par mot clé ? Ce rapport compare les achats de produits sur le site et les mots clés utilisés pour la recherche. Ces informations vous permettent d'identifier quels produits promouvoir dans les pages de destination et dans le contenu des annonces pour favoriser l’activité sur le site. Les éléments indiqués sont : le nombre de produits vendus, le chiffre d'affaires total, le prix moyen et la quantité moyenne commandée pour chaque produit vendu en ligne.

Cliquez sur le signe plus (+) à côté d’une entrée pour rechercher et afficher les mots clés utilisés par les clients pour la recherche.
Cliquez sur le bouton à gauche d’une entrée pour afficher les données par période.
Pour comparer deux périodes distinctes, cliquez sur l'icône de la barre de titre bleue Période.
