Como os visitantes de diferentes regiões diferem em relação às conversões e ao valor médio da visita? Esse relatório compara o número de visitas de cada região geográfica, as exibições de página por visita, as taxas de conversão e o valor médio de cada visita.

Clique no sinal de mais próximo a cada região para comparar as regiões dentro de determinada região.

Para comparar dois períodos, clique no ícone na barra de título azul Período.

Visitas é o número de visitas do seu site a partir do país/região/cidade. P/Visita é o número de exibições de página divididos pelo número de visitas. G/Visita é o número de conversões da meta dividido pelo número de visitas. T/Visitas é o número de transações de comércio eletrônico dividido pelo número de visitas. $/Visita é a receita (do comércio eletrônico) ou o valor da meta dividido pelo número de visitas.
