Conteúdo principal

Quais páginas são mais populares e qual a importância que essas páginas têm para os meus negócios? Esse relatório mostra quantas exibições de página e visitas cada página do seu site recebeu, o tempo médio que os visitantes passaram na página, a freqüência na qual os visitantes saíram do seu site após visualizar a página e o $Índice. As páginas com $Índice mais alto são aquelas que foram visitadas normalmente antes das conversões de maior valor durante a mesma visita. Esse relatório é útil para localizar o conteúdo principal que deve ser uma parte do fluxo de navegação do site.

Clique no botão na extremidade esquerda de qualquer entrada para visualizar os dados ao longo do tempo, para visualizar o valor da vida útil até esta data ou para verificar o segmento cruzado desses dados.

Para comparar dois períodos, clique no ícone na barra de título azul Período.

Visitas é o número total de visitas na qual a página foi visualizada. Exibições de página é o número de vezes que a página foi visualizada. Tempo médio é o tempo médio durante o qual a página foi visualizada. % de saída é o percentual de tempo durante o qual a página foi a página de saída (a última página visualizada durante a visita). $Índice é a receita total ganha (originada das transações de comércio eletrônico subseqüentes ou das conversões em metas com avaliações) dividida pelo número de visitas daquela página.
