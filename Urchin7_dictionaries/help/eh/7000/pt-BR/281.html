Correlação entre país e produto

Qual a popularidade de cada produto em cada país? Este relatório mostra o número de itens vendidos, a receita total, o preço médio e a quantidade média pedida de cada produto vendido on-line.

Clique no sinal de adição próximo a qualquer entrada para obter um relatório detalhado e visualizar os países.
Clique no botão na extremidade esquerda de qualquer entrada para visualizar os dados ao longo do tempo.
Para comparar dois períodos, clique no ícone na barra de título azul Período.
