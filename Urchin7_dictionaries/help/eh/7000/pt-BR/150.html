Acompanhamento de metas de conversão

Como as taxas de conversão variam ao longo do tempo para cada uma das minhas metas? Esse relatório mostra as taxas de conversão ao longo do tempo para cada meta definida.

Para adicionar as taxas de conversão para outro período, clique no ícone na barra de título azul “Período”.

“Taxa de conversão (G/Visita)” é o número de conversões (metas) dividido pelo número de visitas.
