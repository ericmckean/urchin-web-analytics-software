Origem de referência

Como os visitantes referidos por diferentes origens (como mecanismo de pesquisa, boletins informativos ou sites de referência) diferem em relação a número de visitas, taxa de conversão e valor médio de cada visita?

Para comparar dois períodos, clique no ícone na barra de título azul Período.

Cada origem é qualificada por uma mídia, entre colchetes, como [referência]. Cada mídia é uma das mídias com a qual você marcou os links de referência ou um dos seguintes casos especiais:

  • [orgânica] indica os visitantes referidos por mecanismos de pesquisa gratuitos.
  • [referência] indica os visitantes referidos por links que não foram marcados com nenhuma variável de campanha.
  • [(não definido)] indica os visitantes referidos por links marcados com variáveis de campanha, mas cuja variável de mídia não foi definida.
  • (direta)[(nenhum)] indica os visitantes que visitaram o site digitando o URL diretamente no navegador.

Visita é o número de visitas referidas pela origem. P/Visita é a exibição de página dividida pelo número de visitas. G/Visita é o número de conversões da meta dividido pelo número de visitas. T/Visitas é o número de transações de comércio eletrônico dividido pelo número de visitas. $/Visita é a receita (do comércio eletrônico) ou o valor da meta dividido pelo número de visitas.
