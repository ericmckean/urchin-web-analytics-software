Correlação entre origem e produto

Qual a popularidade de cada produto com os clientes de cada origem? Esse relatório mostra o número de itens vendidos, a receita total, o preço médio e a quantidade média pedida de cada produto vendido on-line.

Clique no sinal de adição próximo a qualquer dado para obter um relatório detalhado e visualizar a origem.
Clique no botão na extremidade esquerda de qualquer entrada para visualizar os dados ao longo do tempo.
Para comparar dois períodos, clique no ícone na barra de título azul Período.

Dentro de cada produto, no relatório detalhado, cada origem é qualificada pela mídia, entre colchetes, tais como [referência]. Cada mídia é uma das mídias que foram marcadas com links de referência ou um dos seguintes casos especiais:

  • [orgânica] indica visitantes referidos por mecanismos de pesquisa gratuitos.
  • [referência] indica visitantes referidos pelos links que não foram marcados com nenhuma variável da campanha.
  • [(não definido)] indica os visitantes referidos por links marcados com variáveis de campanha, mas cuja variável de mídia não foi definida.
  • (direta)[(nenhuma)] indica os visitantes que visitaram o site digitando o URL diretamente no navegador.
