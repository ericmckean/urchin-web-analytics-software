Toda a navegação

Como os visitantes foram para cada página do meu site e para onde eles foram depois? Selecione uma página para analisar no lado esquerdo do relatório. O número de cliques, as taxas de conversões e a pontuação média são exibidos em cada página "Clique de e Clique para". As páginas com a pontuação média mais alta são aquelas que mais recebem visitas freqüentes antes das conversões de valor alto.

Os Cliques são a quantidade de cliques da página ou para a página. % de cliques é o percentual total de cliques para a página ou da página selecionada à esquerda. G/Cliques é o percentual de vezes em que um clique de uma página para a próxima resultou em uma conversão posterior durante a visita. T/Cliques é o percentual de vezes em que um clique de uma página para a próxima no caminho resultou em uma transação de comércio eletrônico posterior durante a visita. Leia artigos relevantes na Central de Ajuda do Urchin para obter explicações sobre Pontuação média.
