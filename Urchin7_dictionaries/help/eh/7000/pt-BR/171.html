Definido pelo usuário

Esse relatório permite que você monitore o comportamento do visitante com base em uma variável de segmentação que você define. Por exemplo, o valor selecionado no menu suspenso de um formulário de cadastro pode ser usado para identificar o visitante e utilizado para determinar métricas de comportamento e de conversão. Esse relatório compara como visitantes identificados e classificados segundo os grupos definidos diferem em relação ao número de visitas, exibições de página por visita, taxas de conversão e valor médio da visita. Leia -->Como definir um Segmento do visitante personalizado<-- para saber como preencher este relatório.

Para comparar dois períodos, clique no ícone na barra de título azul Período.

Visitas é o número de visitas para o seu site por visitantes que foram identificados com cada um dos diferentes grupos. P/Visita é o número de exibições de página dividido pelo número de visitas. G/Visita é o número de conversões da meta dividido pelo número de visitas. T/Visitas é o número de transações de comércio eletrônico dividido pelo número de visitas. $/Visita é a receita (do comércio eletrônico) ou o valor da meta dividido pelo número de visitas.
