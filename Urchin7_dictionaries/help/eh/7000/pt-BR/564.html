Detalhamento do IP

O relatório Detalhamento do IP permite que você faça um detalhamento por IP para isolar blocos específicos de IP. Para selecionar um bloco de IP, primeiro selecione o botão próximo aos primeiros números (0-255) do início do IP solicitado e antecedendo o primeiro ponto (.). Clique no botão próximo aos números seguintes correspondentes (0-255) precedendo o próximo ponto (.) do IP para obter mais detalhes sobre o endereço IP. Continue essas etapas até que você isole com êxito o bloco IP de sua escolha.
