Gerenciador de tags do AdWords

Ao usar o Urchin, é recomendável verificar se você possui uma estratégia completa para marcar seus anúncios on-line. Isso permitirá que o Urchin mostre quais atividades de marketing estão compensando. Ao usar o Gerenciador de tags do AdWords, marque os seus links de palavra-chave paga com apenas um clique. Siga estas simples etapas:

Destino de codificação:

  • Selecione o grupo de anúncios que contém as palavras-chave/anúncios que você deseja marcar.
  • Opte por marcar palavras-chave, anúncios ou ambos.

URL de codificação:
Ao especificar URL de codificação, você tem duas opções:

  • Marcar URLs existentes (com a opção de Ignorar URLs já marcados).
  • Aplicar novo URL a todos os objetos marcados dentro do grupo de anúncios. Isso modificará o URL de destino das suas palavras-chave/anúncios.

Tags UTM:
Use somente as variáveis de campanha necessárias.
Os recursos de codificação de link do Urchin permitem identificar praticamente todas as campanhas desejadas. O Urchin acompanha campanhas on-line usando uma combinação das cinco dimensões de marketing a seguir:

  • Origem
  • Mídia
  • Termo
  • Conteúdo
  • Campanha
