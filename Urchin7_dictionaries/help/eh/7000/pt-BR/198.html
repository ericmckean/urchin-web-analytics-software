Conteúdo por títulos

Quais tags de títulos de HTML são utilizadas nas páginas mais visitadas? Esse relatório compara as páginas segundo a tag de título HTML, agregando como uma única página as páginas que têm o mesmo título em HTML. As páginas são comparadas em termos de número de visitas, número de exibições de página, tempo médio gasto na(s) página(s), percentual de tempo que a(s) página(s) é(são) uma página de saída e do $Índice. As páginas com $Índice mais alto são aquelas que foram visitadas antes das conversões de valor alto durante a mesma visita.

Clique no botão na extremidade esquerda de qualquer entrada para consultar as métricas exibidas ao longo do tempo.

Para comparar dois períodos, clique no ícone na barra de título azul Período.

Visitas é o número total de visitas em que a página foi visualizada. Exibições de página é o número de vezes que a página foi visualizada. Tempo médio é o tempo médio durante o qual a página foi visualizada. %Saída é o percentual de tempo durante o qual a página foi a página de saída (a última página visualizada durante a visita). $Índice é a receita total ganha (originada das transações de comércio eletrônico subseqüentes ou das conversões em metas com avaliações) dividida pelo número de visitas daquela página.
