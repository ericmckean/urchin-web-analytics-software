Verificação de meta

Esse relatório mostra as páginas que levam a cada uma das suas metas e o número de vezes que cada uma das páginas foi visualizada (selecione uma meta no menu suspenso na barra de ferramentas na parte superior desse relatório). Esse relatório é útil para verificar quais de páginas de meta foram registradas no log, considerando que uma meta pode ser definida como um conjunto de páginas.

Se, por exemplo, você definiu uma meta como um diretório, cada uma das páginas dentro daquele diretório é incluída na meta. Neste caso, esse relatório mostraria cada uma dessas páginas, o número de visitas à página e o percentual de conversões total da meta que cada página representa.

Clique no botão na extremidade esquerda de qualquer entrada para consultar as métricas exibidas ao longo do tempo.
