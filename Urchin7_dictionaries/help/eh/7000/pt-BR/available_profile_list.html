Visualizar relatórios
Essa tela lista todos os perfis disponíveis aos quais você tem acesso. Ela permite examinar as métricas de desempenho de vários perfis em uma única tela.

O calendário limitado
O calendário limitado no canto superior direito permite escolher um período (dia, semana, mês ou ano).

Ir para relatório
Clique no nome do perfil para ativar a janela de relatórios desse perfil. A janela de relatórios permite a visualização de todos os relatórios disponíveis do perfil.

Editar (somente superadministrador)
O botão “Editar“ fornece acesso a todas as configurações do perfil, organizadas em um formato de guia.

Status (com hiperlink, somente superadministrador)
O Status de cada tarefa é “Não programada“, “Pendente“, “Em fila“, “Em exibição“, “Em cancelamento“, “Concluída“, “Cancelada“, “Aviso“ ou “Erro“. Eles são codificados por cor. Para classificar a lista por Status, clique na seta abaixo da palavra “Status”. Para visualizar detalhes do tempo de execução de uma tarefa, clique na entrada de status codificado por cor ao lado dessa tarefa. É aberta uma nova janela que exibe os detalhes do tempo de execução.

Controles
Use o menu suspenso “Nº mostrado“ para alterar a quantidade de entradas relacionadas por vez. Use os botões “Anterior“ e “Próxima“ para mudar de uma página para outra. Para clicar no site real, clique no link ao lado do nome do perfil.

Conta (somente Modo Datacenter)
No Modo Datacenter, o menu “Conta“ pode filtrar as entradas.

Visualizar
Use o menu suspenso “Visualizar“ para selecionar a visualização “Administrador“, “Empresa“ ou “Básica“ das páginas de visão geral.
