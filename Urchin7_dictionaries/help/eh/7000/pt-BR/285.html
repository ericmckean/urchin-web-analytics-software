Correlação entre palavra-chave e produto

Qual é a popularidade de cada produto para os clientes que pesquisaram cada palavra-chave? Esse relatório correlaciona as compras de produtos às palavras-chave originais utilizadas em uma pesquisa. Essa informação fornece insights sobre quais produtos que você deve promover usando anúncios e o conteúdo de páginas de destino, para assim aumentar o volume de negócios do site. É mostrado o número de itens vendidos, a receita total, o preço médio e a quantidade média pedida de cada produto vendido on-line.

Clique no sinal de adição próximo a qualquer entrada para obter um relatório detalhado e visualizar as palavras-chave que os clientes pesquisaram.
Clique no botão na extremidade esquerda de qualquer entrada para visualizar os dados ao longo do tempo.
Para comparar dois períodos, clique no ícone na barra de título azul Período.
