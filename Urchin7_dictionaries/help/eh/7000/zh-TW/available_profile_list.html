查看報表
此畫面會列出可供您使用的所有設定檔，讓您可以透過一個畫面集中查看多個設定檔的成效指標。

指定檢視日曆
右上角的指定檢視日曆可讓您選擇日期範圍 ([日]、[週]、[月]、[年])。

前往報表
按一下設定檔名稱即可開啟該設定檔的報表視窗，此視窗可讓您查看該設定檔的所有可用報表。

編輯 (僅限超級管理員)
[編輯] 按鈕以分頁格式組織，可讓您查看該設定檔的所有設定。

狀態 (超連結，僅限超級管理員)
各項工作的狀態會顯示為「未安排」、「未放送」、「排入佇列」、「放送中」、「取消中」、「已完成」、「已取消」、「警告」或「錯誤」。不同狀態將以不同顏色標示；只要按一下「狀態」下方的箭頭，即可按狀態排序清單。如果要查看特定工作執行時期的詳情，請按一下該工作旁以顏色標示的狀態項目，這時會開啟另一個視窗，讓您查看執行時期詳情。

控制
使用 [顯示數目] 下拉式選單可變更一次列出的項目數量。如果要一次捲動一整頁的項目，請使用 [前一頁] 或 [下一頁] 按鈕。如果要連進實際網站，只要按一下設定檔名稱旁的連結即可。

帳戶 (僅限 Datacenter 模式)
在 Datacenter 模式作業下，[帳戶] 功能表可用來篩選列出的項目。

檢視
使用 [檢視] 下拉式選單可為總覽網頁選取 [管理員]、[公司] 或 [隨機] 檢視。
