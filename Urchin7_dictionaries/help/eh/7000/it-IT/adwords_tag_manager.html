Gestore codifica AdWords

Quando utilizzi Urchin, devi assicurarti di avere una strategia completa per codificare i tuoi annunci online. Si tratta di un prerequisito importante che permette a Urchin di segnalarti le attività di marketing effettivamente redditizie. Con Gestore codifica AdWords puoi codificare i link delle parole chiave a pagamento con un semplice clic. Procedi nel seguente modo:

Codifica target:

  • Seleziona il gruppo di annunci contenente le parole chiave/gli annunci da codificare.
  • Scegli di codificare le parole chiave, gli annunci o entrambi.

Codifica URL:
Specificando Codifica URL hai due opzioni:

  • Codifica URL esistenti (con l'opzione Ignora URL già codificati).
  • Applica questo nuovo URL a tutti gli oggetti codificati all'interno del gruppo di annunci. Ciò modificherà l'URL di destinazione delle tue parole chiave/annunci.

Tag UTM:
Utilizza solo le variabili campagna necessarie.
La funzione di codifica dei link di Urchin consente di identificare in modo univoco ogni campagna. Urchin monitora le campagne online utilizzando una combinazione delle cinque seguenti dimensioni di marketing:

  • Sorgente
  • Mezzo
  • Termine
  • Contenuto
  • Campagna
