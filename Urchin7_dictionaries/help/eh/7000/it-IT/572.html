IP non risolti

Nel rapporto IP non risolti sono elencati tutti gli indirizzi IP per i quali non vi sono dati sulla proprietà, come l'ubicazione geografica o il nome di dominio host. Per gli indirizzi IP inclusi nel rapporto IP non risolti non è possibile eseguire il monitoraggio nei rapporti di Urchin relativi al dominio o all'ubicazione geografica di Urchin.
