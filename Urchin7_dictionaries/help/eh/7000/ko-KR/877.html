캠페인 전환

방문수, 방문당 페이지뷰, 전환율, 평균 방문값을 기준으로 모든 마케팅 캠페인(추천 링크에서 지정한 모든 캠페인)을 비교하여 가장 우수한 트래픽을 유발한 캠페인을 알려주는 보고서입니다.

특정 캠페인에서 소스(Google 스폰서 링크와 뉴스레터 등)의 실적을 자세히 비교하려면 캠페인 옆에 있는 덧셈 기호를 클릭하십시오.
<%dic|default|505%>

두 기간을 비교하려면 파란색 기간 제목 표시줄에 있는 아이콘을 클릭하십시오.

괄호 안에 있는 캠페인은 다음과 같은 특수 사례 중 하나를 나타냅니다.

  • (자연어 트래픽)은 무료 검색 엔진의 링크를 통해 방문한 사용자를 말합니다.
  • (추천)은 캠페인 변수로 태그가 지정되지 않은 링크에 의해 추천된 방문자를 나타냅니다.
  • (비설정)은 캠페인 변수가 태그로 지정되었지만 설정되지는 않은 링크에 의해 추천된 방문자를 나타냅니다.
  • (직접)은 브라우저에 URL을 직접 입력하여 사이트를 조회한 방문자를 나타냅니다.

개별 캠페인 내에 있는 상세 보고서에서 각 소스는 [추천]과 같이 대괄호로 묶어 표시되는 매체에 의해 자격이 부여됩니다. 각 매체는 추천 링크에 태그가 추가된 매체 중 하나이거나 다음 특수 사례 중 하나입니다.

  • [자연어 트래픽]은 무료 검색 엔진의 링크를 통해 방문한 사용자를 말합니다.
  • [추천]은 캠페인 변수로 태그가 지정되지 않은 링크에 의해 추천된 방문자를 나타냅니다.
  • [(비설정)]은 캠페인 변수가 태그로 지정되었지만 설정되지는 않은 링크에 의해 추천된 방문자를 나타냅니다.
  • (직접)[(없음)]은 URL을 브라우저에 직접 입력하여 사이트를 조회한 방문자를 나타냅니다.

방문수는 각 캠페인에 의해 추천된 방문수입니다. P/방문은 페이지뷰를 방문수로 나눈 값이며 G/방문은 목표 전환수를 방문수로 나눈 값입니다. T/방문은 전자상거래 거래 횟수를 방문수로 나눈 값입니다. $/방문은 전자상거래에서 발생한 수익 또는 목표값을 방문수로 나눈 값입니다.

애드워즈에 로그인하려면 '애드워즈로 이동'을 클릭하십시오.
'키워드 도구'를 사용하여 선택된 URL 또는 키워드로 키워드 생성 도구를 실행하면 새로운 추천 키워드를 생성할 수 있습니다. 생성된 키워드는 애드워즈 계정으로 가져올 수 있습니다.

애드워즈 캠페인에 대해 사용할 수 있는 기능은 다음과 같습니다.

  • 애드워즈 도구에서 캠페인 세부정보를 선택하면 애드워즈 캠페인의 정보가 표시됩니다.
  • 애드워즈 도구에서 캠페인 최적화도구를 선택하면 애드워즈 캠페인 최적화 도구를 사용할 수 있습니다.
