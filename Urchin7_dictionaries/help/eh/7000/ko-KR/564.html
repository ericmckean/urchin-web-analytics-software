IP 드릴다운

IP 드릴다운 보고서를 사용하여 특정 IP 블록을 IP별로 드릴다운하여 분리할 수 있습니다. IP 블록을 선택하려면 먼저 요청된 IP의 시작 부분의 첫 번째 마침표 바로 앞에 있는 처음 몇 개의 숫자(0-255) 옆에 있는 버튼을 선택하십시오. IP 주소를 더 자세히 드릴다운하려면 IP의 두 번째 마침표 바로 앞에 있는 다음 해당 숫자(0-255) 옆에 있는 버튼을 클릭하십시오. 선택한 IP 블록을 성공적으로 분리할 때까지 이 단계를 계속하십시오.
