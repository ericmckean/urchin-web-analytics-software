Benutzerdefiniert

Welche von Ihnen selbst festgelegten Besuchersegmente erbringen den größten Umsatz? In diesem Bericht wird der Umsatz auf Basis einer von Ihnen definierten Segmentvariablen aufgeschlüsselt. Beispielsweise kann der in einem Dropdown-Menü eines Registrierungsformulars ausgewählte Wert mit dem Besucher verknüpft und dazu verwendet werden, Messwerte für die Conversion und das Verhalten zu ermitteln. In diesem Bericht werden die Besucher, die den verschiedenen von Ihnen definierten Gruppen zugeordnet werden können, im Hinblick auf Umsatz, Anzahl der Transaktionen und durchschnittliche Transaktionswerte verglichen. Informationen zum Füllen dieses Berichts mit Daten erhalten Sie im Abschnitt zum -->Definieren von benutzerdefinierten Besuchersegmenten<--.

Klicken Sie auf die Schaltfläche links neben einem Eintrag, um Daten über einen bestimmten Zeitraum hinweg anzuzeigen.

Wenn Sie zwei Zeiträume vergleichen möchten, klicken Sie auf das Symbol in der blauen Titelleiste für den Zeitraum.

Der Umsatz ergibt sich aus dem Gesamtumsatz der E-Commerce-Transaktionen von Besuchern aus einem bestimmten benutzerdefinierten Segment. "Trans" ist die Anzahl der Transaktionen. "€/Trans" steht für den Umsatz geteilt durch die Transaktionen. "Prozent €" gibt an, wie viel Besucher aus dem benutzerdefinierten Segment prozentual zum Gesamt-E-Commerce-Umsatz Ihrer Website beitragen.
