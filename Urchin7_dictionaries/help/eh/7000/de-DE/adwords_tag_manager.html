AdWords Tag-Manager

Bei der Verwendung von Urchin sollten Sie sich unbedingt eine umfassende Strategie zur Kennzeichnung der Online-Anzeigen durch Tags überlegen. Die Kennzeichnung der Online-Anzeigen durch Tags ist eine wichtige Voraussetzung, damit Urchin die tatsächlich lukrativen Marketingaktivitäten ermitteln kann. Mit dem AdWords Tag-Manager können Sie Ihre Links mit bezahlten Keywords mit nur einem Klick kennzeichnen:

Zielkennzeichnung:

  • Wählen Sie die Anzeigengruppe mit den zu kennzeichnenden Keywords/Anzeigen aus.
  • Wählen Sie zum Kennzeichnen entweder Anzeigen oder Keywords oder beides aus.

Kennzeichnungs-URL:
Bei Angabe der Kennzeichnungs-URL haben Sie zwei Möglichkeiten:

  • Kennzeichnen vorhandener URLs mit der Option, bereits gekennzeichnete URLs zu überspringen.
  • Anwenden einer neuen URL auf alle gekennzeichneten Objekte innerhalb der Anzeigengruppe. Dadurch wird die Ziel-URL für Ihre Keywords/Anzeigen geändert.

UTM- Tags:
Verwenden Sie nur die Kampagnenvariablen, die Sie benötigen.
Mit den Funktionen der Tag-Kennzeichnung von Links in Urchin können Sie praktisch jede vorstellbare Kampagne eindeutig kennzeichnen. Urchin protokolliert Online-Kampagnen über eine Kombination aus den folgenden fünf Marketingdimensionen:

  • Quelle
  • Medium
  • Begriff
  • Content
  • Kampagne
