Berichte anzeigen
In diesem Fenster werden alle verfügbaren Profile aufgeführt, auf die Sie zugreifen können. Sie können die Leistungskennzahlen für mehrere Profile in einem einzigen Fenster überprüfen.

Begrenzter Kalender
In dem oben rechts angezeigten begrenzten Kalender können Sie einen Zeitraum (Tag, Woche, Monat, Jahr) auswählen.

Gehe zu Bericht
Klicken Sie auf den Profilnamen, um das Berichtfenster für das Profil zu öffnen. In diesem Fenster können Sie alle verfügbaren Berichte für das Profil anzeigen.

Bearbeiten (nur Superadministrator)
Über die Schaltfläche “Bearbeiten“ erhalten Sie Zugriff auf die in Tabellenformat angeordneten Einstellungen für das jeweilige Profil.

Status (als Hyperlink, nur Superadministrator)
Jede Aufgabe hat einen der folgenden Status: “Nicht geplant“, “Ausstehend“, “In Warteschlange“, “Ausführen“, “Abbrechen“, “Abgeschlossen“, “Abgebrochen“, “Warnung“ oder “Fehler“. Die Statusangaben sind farbcodiert. Klicken Sie zum Sortieren der Liste nach dem Status auf den Pfeil unterhalb des Worts “Status“. Zum Anzeigen von Ausführungsdetails für eine Aufgabe klicken Sie auf den farbcodierten Status neben der jeweiligen Aufgabe. Daraufhin wird ein neues Fenster mit den Ausführungsdetails angezeigt.

Steuerelemente
Über das Dropdown-Menü “Angezeigte Anzahl“ ändern Sie die Anzahl der gleichzeitig angezeigten Einträge. Mithilfe der Schaltflächen “Zurück“ und “Weiter“ können Sie Seite für Seite durch die Einträge blättern. Sie gelangen zur eigentlichen Seite, indem Sie auf den Link neben dem Profilnamen klicken.

Konto (nur Datacenter-Modus)
Im Datencenter-Modus können Sie im Menü “Konto“ die Einträge filtern.

Ansicht
Wählen Sie im Dropdown-Menü “Ansicht“ die Administrator- oder Unternehmensansicht bzw. die einfache Ansicht aus, um die Übersichtsseiten zu öffnen.
