Segmentübergreifende Leistung

In diesem Bericht wird mit einem sekundären Segment auf das primäre Segment in Ihrem Hauptbericht querverwiesen. Das primäre Segment wird auf den Registerkarten links oben im Bericht angezeigt. Angezeigt werden: Anzahl der Besuche, Seiten pro Besuch, Conversion-Raten pro Ziel und durchschnittlicher Besuchswert.

  • Wenn Sie zum Hauptbericht zurückkehren möchten, klicken Sie auf die Registerkarte links oben im Bericht.
  • Wenn Sie das sekundäre Segment ändern möchten, wählen Sie ein Segment aus dem Dropdown-Menü "Segment" im oberen Bereich dieses Berichts aus.
