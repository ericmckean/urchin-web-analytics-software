Keyword-Tool
Generieren Sie mithilfe des Keyword-Tools neue Ideen. Wählen Sie eine Option aus, um ein paar beschreibende Wörter oder Wortgruppen einzugeben. Alternativ können Sie die URL Ihrer Website eingeben.

Das Keyword-Tool macht es einfach, neue Keywords für Ihre Kampagnen zu finden. Leiten Sie beispielsweise ein günstiges Hotel, kann das Keyword-Tool hilfreiche und entsprechende Keywords wie "Hotel billig" oder "Motels" vorschlagen. Durch das Hinzufügen dieser alternativen Begriffe zu den Keyword-Listen Ihrer Anzeigengruppe können Sie neue Kunden gewinnen, die Ihnen andernfalls vielleicht entgangen wären.

In den drei folgenden Situationen ist das Keyword-Tool besonders nützlich:

  • Beim Erstellen einer neuen Anzeigengruppe
  • Bei einer schlechten Leistung Ihrer aktuellen Keywords und der Notwendigkeit, bessere Optionen zu finden
  • Bei guter Leistung eines Keywords, das Sie durch weitere ähnliche Keywords ergänzen möchten
Beachten Sie, dass das Keyword-Tool am besten mit kurzen Keywords mit einem oder zwei Wörtern funktioniert. Mit "Günstiges Hotel" lassen sich bessere Ergebnisse als mit "Günstige Unterkunft bei einer Reise" erzielen. Dafür gibt es folgenden Grund: Besteht Ihr Keyword aus vielen Wörtern, kann das Keyword-Tool nur schwer die wichtigsten Wörter ermitteln und geeignete Alternativen vorschlagen.

Suchen Sie nach neuen Keywords? Testen Sie das Keyword-Tool auf Basis von Suchanfragen. Hierbei handelt es sich um ein neues Tool zur Generierung von Keywords, die auf den Content Ihrer Website abgestimmt sind.
