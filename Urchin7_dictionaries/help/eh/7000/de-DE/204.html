Länge des Besuchs
Wie viel Zeit verbringen Besucher auf meiner Website? Besuche unter zehn Sekunden werden auf der linken Seite der Grafik angezeigt. Besuche über 1801 Sekunden werden auf der rechten Seite der Grafik angezeigt.

Besuchszeit auf Website
Das Erfassen der Besuchszeit auf der Website ist eine Methode zur Analyse der Besuchsqualität. Wenn Besucher lange auf Ihrer Website verweilen, findet eventuell eine ausgedehnte Interaktion statt. Die Besuchszeit auf Website kann jedoch auch ein trügerischer Messwert sein, da Besucher häufig das Browserfenster geöffnet lassen, obwohl sie Ihre Website nicht wirklich ansehen oder verwenden.

Wenn Sie zwei Zeiträume vergleichen möchten, klicken Sie in der blauen Titelleiste für den Zeitraum auf das Symbol .
