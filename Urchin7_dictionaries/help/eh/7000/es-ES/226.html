Verificación de objetivos

Para cada uno de sus objetivos (seleccione un objetivo del menú desplegable de la barra de herramientas situada en la parte superior de este informe), este informe muestra las páginas que han permitido lograr el objetivo y el número de veces que cada una de ellas se ha visualizado. Resulta útil para verificar las páginas objetivo específicas que se han registrado, dado que un objetivo puede definirse como un grupo de páginas.

Si, por ejemplo, ha definido un objetivo como un directorio, cada una de las páginas de ese directorio habrá permitido lograr el objetivo. En ese caso, este informe reflejaría cada una de esas páginas, el número de visitas que ha recibido cada una de ellas y el porcentaje de las conversiones totales obtenidas que representa cada página.

Haga clic en el botón situado a la izquierda de una de las entradas para ver las cifras que se han ido mostrando.
