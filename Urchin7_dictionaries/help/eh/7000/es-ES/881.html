Conversiones totales por palabra clave

¿Qué palabras clave generan el tráfico de mejor calidad? Este informe muestra el número de visitas, páginas vistas por visita, porcentajes de conversión y valor medio por visita tanto para palabras clave orgánicas como para palabras clave de pago.

Haga clic en el signo más de una de las palabras clave para acceder a la información y comparar las palabras clave de motores de búsqueda y pago con las orgánicas.

<%dic|default|505%>

Para comparar dos periodos, haga clic en el icono de la barra de título azul "Intervalo de tiempo".

"Visitas" indica el número de visitas provenientes de cada origen. "Páginas vistas/Visitas" es el número de páginas vistas dividido entre el número de visitas. "Objetivos/Visitas" representa el número de conversiones de objetivos dividido entre el número de visitas. "Transacciones/Visitas" se calcula dividiendo las transacciones de comercio electrónico entre el número de visitas. "Ingresos/Visitas" hace referencia a los beneficios derivados de las transacciones de comercio electrónico o el valor del objetivo dividido entre el número de visitas.

Haga clic en "Información detallada de palabra clave" (únicamente disponible para las palabras clave de AdWords) para ver la cuenta, campaña y grupo de anuncios de AdWords asociados al término.

Con los enlaces y Herramienta para palabras clave de AdWords puede copiar los términos elegidos en la Herramienta para generar palabras clave (KGT), que le servirán para crear nuevas sugerencias. Puede importar esas sugerencias a su cuenta de AdWords.
