Herramienta para palabras clave
Utilice esta herramienta para obtener sugerencias de términos. Elija una opción para introducir algunas palabras o frases descriptivas o bien utilice la URL de su sitio web.

La Herramienta para palabras clave es una forma excelente de encontrar nuevos términos para las campañas. Por ejemplo, si regenta un hotel económico, la Herramienta para palabras clave puede sugerir términos relacionados como "descuentos en hoteles" u "hostales". Añadir estos términos a las listas de palabras clave de su grupo de anuncios puede ayudarle a encontrar nuevos clientes que, de no ser por la herramienta, habría perdido.

La herramienta puede serle especialmente útil en estos casos:

  • Se dispone a crear su primer grupo de anuncios
  • El rendimiento de las palabras clave es insuficiente y necesita buscar otros términos
  • Tiene una palabra clave que ofrece muy buenos resultados y quiere encontrar otras parecidas
Recuerde que la herramienta funciona mejor con palabras clave cortas formadas por uno o dos términos. Con "hotel económico" obtendrá mejores resultados que con "sitio barato para alojarse durante los viajes". Esto se debe a que si la palabra clave incluye demasiados términos, la herramienta tiene dificultades para determinar los que son más importantes y ofrecer alternativas adecuadas.
