IP no resueltas

En el informe de IP no resueltas se incluyen todos los visitantes con IP sin información sobre el propietario, como ubicación geográfica o host del dominio. No se podrá realizar un seguimiento de estas IP en los informes relacionados de Urchin de ubicación geográfica o dominio.
