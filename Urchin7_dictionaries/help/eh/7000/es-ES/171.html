Usuario definido

Este informe le permite realizar el seguimiento del comportamiento del usuario en función de una variable de segmento arbitraria que usted defina. Por ejemplo, el valor seleccionado en un menú desplegable de un formulario de registro puede etiquetarse para el usuario y usarse para determinar los indicadores de conversión y de comportamiento. Este informe muestra las diferencias entre los usuarios que se identifican con cada uno de estos grupos respecto al número de visitas, páginas vistas por visita, tasas de conversión y valor medio de visita. Lea -->Cómo definir un segmento de usuario personalizado<-- para saber cómo rellenar este informe.

Para comparar dos intervalos de tiempo, haga clic en el icono de la barra de título azul “Intervalo de tiempo”.

"Visitas" indica el número de visitas que su sitio recibe de usuarios que se identifican con cada uno de los distintos grupos. "Páginas vistas/Visitas" son las páginas vistas divididas entre el número de visitas. "Conversiones obtenidas/Visita" es el número de conversiones obtenidas dividido entre el número de visitas. "Transacciones/Visitas" se calcula dividiendo las transacciones de comercio electrónico entre el número de visitas. "Ingresos/Visitas" hace referencia a los beneficios derivados de las transacciones de comercio electrónico o del valor del objetivo dividido entre el número de visitas.
