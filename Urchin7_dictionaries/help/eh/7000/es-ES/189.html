Conversiones totales por palabra clave

¿Qué palabras clave generan el tráfico de mejor calidad? Este informe muestra el número de visitas, páginas vistas por visita, porcentajes de conversión y valor medio por visita tanto para palabras clave orgánicas como para palabras clave de pago.

Haga clic en el signo más de una de las palabras clave para ver los detalles y comparar los motores de búsqueda, así como las palabras clave de pago frente a las orgánicas.

<%dic|default|505%>

Para comparar dos intervalos de tiempo, haga clic en el icono de la barra de título azul "Intervalo de tiempo".

"Visitas" indica el número de visitas provenientes de cada palabra clave. "Páginas vistas/Visitas" son las páginas vistas divididas entre el número de visitas. "Objetivos/Visitas" son las conversiones de objetivos dividido entre el número de visitas. "Transacciones/Visitas" se calcula dividiendo las transacciones de comercio electrónico entre el número de visitas. "Ingresos/Visitas" hace referencia a los beneficios derivados de las transacciones de comercio electrónico o el valor del objetivo dividido entre el número de visitas.
