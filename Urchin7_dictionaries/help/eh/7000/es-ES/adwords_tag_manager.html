Administrador de etiquetas de AdWords

Cuando utilice Urchin, le recomendamos que disponga de una estrategia global para etiquetar sus anuncios online. Gracias a ello, Urchin le indicará qué actividades de marketing le son rentables. Con el Administrador de etiquetas de AdWords, puede etiquetar los enlaces de palabra clave de pago con un solo clic. Para ello, siga estos sencillos pasos:

Objetivo de etiquetado:

  • seleccione el grupo de anuncios que contenga las palabras clave o los anuncios que quiera etiquetar.
  • Puede etiquetar palabras clave, anuncios o ambos elementos.

URL de etiquetado:
si etiqueta URL, tiene dos opciones:

  • Etiquetar URL existentes (con la posibilidad de omitir aquellas URL que ya han sido etiquetadas).
  • Aplicar la URL nueva a todos los objetos etiquetados del grupo de anuncios. Con esto, se modificará la URL de destino de sus palabras clave o anuncios.

Etiquetas de UTM:
utilice únicamente las variables de campaña necesarias.
Las capacidades de etiquetado de enlaces de Urchin permiten identificar prácticamente cualquier campaña que se pueda imaginar. Urchin realiza el seguimiento de campañas online mediante una combinación de los cinco siguientes parámetros de marketing:

  • Fuente
  • Medio
  • Término
  • Contenido
  • Campaña
