Page Query Terms

Which parameter names and values were used in requests to dynamic content? This report analyzes internal search and dynamic content, listing all the pages whose URL contained query parameters (generally after the ? query token).

Click the plus to the left of any entry to drill down and see the variable name of each name/value pair that came after the query token. Drill down again to see the values of each variable name.

Click the button to the far left of any entry to see any metric displayed over time.

To compare two date ranges, select the "Compare to Past" option on the Calendar tab.
