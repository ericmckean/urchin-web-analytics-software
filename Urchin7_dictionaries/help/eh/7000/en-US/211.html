Hostnames

If you have multiple domains with the same website content, which domains do visitors visit? This report shows the total number of pageviews that each domain received. If a domain appears on this report that is not one of your domains, the domain in question points to a website that has your UTM (and possibly other content owned by you).

Click the button on the far left of any entry to see metrics displayed over time.

To compare two date ranges, select the "Compare to Past" option on the Calendar tab.
