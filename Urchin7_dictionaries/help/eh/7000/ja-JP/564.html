IP の詳細

IP の詳細レポートでは、IP ごとの詳細を確認して特定の IP ブロックを分離できます。 IP ブロックを選択するには、まず際書にリクエストされた IP の最初から 1 つ目のピリオド (.) までの数字 (0 ~ 255) の横にある ボタンをクリックします。 次に、2 つ目のピリオドまでの数字 (0 ~ 255) の横にある ボタンをクリックすると、IP アドレスをより詳細に確認できます。 該当の IP ブロックを正しく分離できるまで、上記のステップを繰り返します。

<%dic|default|505%>

2 つの期間のデータを比較するには、青色の [期間] タイトル バーにある アイコンをクリックします。
