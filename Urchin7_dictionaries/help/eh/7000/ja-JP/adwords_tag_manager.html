AdWords タグ マネージャ

Urchin を使用する際は、オンライン広告のタグ設定を体系的に行うことをおすすめします。これは、Urchin でどのマーケティング活動が実際に収益をあげているかをトラッキングするためです。AdWords タグ マネージャを使用すると、キーワード広告のリンクに簡単にタグを設定できます。手順は下記のとおりです。

ターゲットのタグ設定:

  • タグを設定するキーワードや広告を含む広告グループを選択します。
  • キーワードか広告、またはその両方にタグを設定するかを選択します。

URL のタグ設定:
URL のタグ設定には 2 つのオプションがあります。

  • 既存の URL にタグを設定します（既にタグを設定している URL を除外することもできます）。
  • 広告グループ内のタグが設定されているすべてのオブジェクトに新しい URL を適用します。これにより、キーワードや広告のリンク先 URL が変更されます。

UTM タグ:
必要なキャンペーン変数のみを使用します。
Urchin のリンク タグ設定機能を使用すると、ほとんどのキャンペーンを識別することができます。Urchin では、下記の 5 つのマーケティング ディメンションを組み合わせてオンライン キャンペーンをトラッキングします。

  • ソース
  • メディア
  • キーワード
  • コンテンツ
  • キャンペーン
