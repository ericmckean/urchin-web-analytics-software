我可以將資料目錄移至其他位置嗎？

總覽

Urchin 資料目錄是 Urchin 根程式目錄的預設子目錄。您可以編輯 ~urchin/etc/urchin.conf 檔案並手動移動已存在的 ~urchin/data 目錄至您想要的位置，變更位置一點也不麻煩。請注意，這個動作可能會對現存的授權造成影響，在這個情況下，您必須與 Urchin 軟體的授權顧問聯絡，並要求重設您的序號。

移動資料目錄的程序：

您可以在 urchin.conf 檔案中變更 Urchin 儲存其報告資料的位置，此檔案位在 Urchin 主安裝目錄下的 etc 目錄中。

適用於 UNIX 類型系統：

  1. 以 Urchin 安裝使用者的身分開啟指令殼層
  2. 使用 cd 指令移至 Urchin 的安裝目錄
  3. 使用下列指令停用 Urchin 服務：bin/urchinctl stop
  4. 使用文字編輯器，在 Urchin 分佈的 etc 目錄下開啟 urchin.conf 檔案
  5. 移除開頭的 "#" 字元以取消下列行的註解：

    • #dataDirectory:./data/，並以您想要使用的目錄路徑替代預設目錄，例如 dataDirectory: /bigdisk/urchin/data/

  6. 依需求建立新的資料目錄。重要注意事項： Urchin 執行使用者必須能夠讀取和寫入此目錄。

  7. 將現有的資料複製至新位置。為了確保您正確儲存所有的權限和資料，建議您使用下列指令：
    • cd data; tar cf - .| (cd /bigdisk/urchin/data; tar xpf -)
  8. 將資料目錄重新命名為 data.old。您可以依需求將其完全移除，但在這麼做之前，建議您先確保一切都能正常運作。
  9. 為了方便管理，建議您在指向新位置 (例如 -s /bigdisk/urchin/data ./data) 的 Urchin 主目錄中建立 Symlink。
  10. 使用下列指令重新啟動 Urchin 服務：bin/urchinctl start
  11. 以管理員使用者身分登入 Urchin。您面前應該會出現授權 Urchin 畫面。只要按一下 [重新啟用授權] 連結，一切就大功告成。

適用於 Windows 系統：

  1. 停用 Urchin 服務：[開始] > [程式集] > [Urchin] > [停用 Urchin 服務]

  2. 開啟 Windows 檔案總管，並瀏覽至 Urchin 分佈的 etc 資料夾。預設路徑為 C:\Program Files\Urchin\etc
  3. 使用文字編輯器開啟 urchin.conf 檔案
  4. 移除開頭的 "#" 字元以取消下列行的註解：

    • #dataDirectory:./data/，並以您想要使用的磁碟機字母代號和完整路徑名稱替代預設資料夾，例如 dataDirectory: E:\Urchin\data

  1. 依需求建立新的資料夾。重要注意事項：這個資料夾的權限必須允許您讀取和寫入 Urchin 服務。
    1. 將資料資料夾的內容複製至新資料夾。
    2. 將資料資料夾重新命名為 data.old。您可以依需求將其完全移除，但在這麼做之前，建議您先確保一切都能正常運作。

    3. 重新啟動 Urchin 服務：[開始] > [程式集] > [Urchin] > [啟用 Urchin 服務]

注意事項

基於 UNIX 類型系統實作授權以避免竄改的方式，移動 Urchin 資料目錄時，您必須重新啟用 Urchin 授權。不過，重新授權作業過程十分簡易 - 您只需要以 Urchin 管理員使用者身分登入，並按一下 [重新啟用授權] 就可完成。
