定義自訂的訪客區段

除了以預先定義的區段類型 (例如地理區域及語言偏好設定) 來區隔使用者，您還可以使用 Urchin 6 定義自訂的區段，並分析每個區段的使用者行為。例如，請訪客在表單上勾選其工作性質 (技術 / 工程師、行銷人員 / 公關等等)，然後根據訪客勾選的工作類別資訊分析訪客的瀏覽及購買行為。

您可以從 [行銷最佳化] > [訪客區段效能] > [使用者定義] 報告中檢視每個自訂區段的轉換行為。同時，許多其他的報告都可讓您根據訪客區段來跨資料分析。若要跨資料分析，請按一下多數報告任何項目最左邊的分析選項按鈕。選取 [使用者定義] 以檢視您自訂的區段。

若要設定訪客區段，只要呼叫 JavaScript __ utmSetVar 函數即可。例如，請在網頁 Uuchin 流量監視器下方的任何位置進行下列呼叫： 

    __utmSetVar("Marketing/PR");

範例：將訪客歸類到此網頁的「行銷人員 / 公關」區段

在此範例中，任何造訪此網頁的人都會歸類到「行銷人員 / 公關」區段。 

    <body onLoad="javascript:__utmSetVar("Marketing/PR")">

範例：將訪客歸類到訪客特定區段

在此範例中，會根據訪客在表單中的選擇將他們歸類到某區段。 

    <form onSubmit="__utmSetVar(this.mymenu.options[this.mymenu.selectedIndex].value);">
    <select name=mymenu>
    <option value="Technical/Engineering">技術 / 工程師</option>
    <option value="Marketing/PR">行銷人員 / 公關</option>
    <option value="Manufacturing">製造</option>
    <option value="General Management">一般管理</option>
