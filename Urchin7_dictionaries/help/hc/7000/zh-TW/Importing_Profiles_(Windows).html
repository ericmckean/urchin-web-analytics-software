匯入設定檔 (Windows)

總覽

Urchin 的「匯入設定檔」功能可讓執行 Microsoft Internet Information Server (IIS，微軟網際網路資訊伺服器) 系統的使用者快速為每個 IIS 網站設定設定檔。Urchin 能夠讀取 IIS 組態，判斷伺服器上正在執行哪些網站，然後使用 IIS 記錄做為網站的記錄來源，為每個網站建立基本設定檔。接下來，您就可以視需要針對匯入的網站自訂設定檔或新增其他設定檔。

如何匯入設定檔

若要開始匯入設定檔，請以管理員的身分登入 Urchin 管理系統，然後按一下介面右上角的 [組態]。按一下右上角的 [匯入] 按鈕，進入「匯入設定檔」畫面。此畫面可讓您選取要匯入的網站 (如果有的話)。完成選取後，按一下 [匯入] 按鈕。完成所有匯入作業後，請按一下 [完成]。

建議

建議您至少為伺服器上的每一個網站建立一個設定檔，如此您就可以透過 Urchin 的「摘要報告」完整瞭解伺服器的流量。「摘要報告」可提供您伺服器的整體流量資訊，並根據各種流量參數提供每個網站的排名。如果您本身是代管商，並根據頻寬用量收費，這個方法便非常實用。請注意，「摘要報告」只會根據已設定的設定檔顯示資料，而不會將那些沒有有效設定檔的網站包括進來。
