如何在 RedHat 上安裝 Urchin 啟動指令碼？

Urchin 會在安裝期間建立一組啟動指令碼，並以 "urchin_daemons" 的名稱儲存在 Urchin util 目錄中。"/path/to/urchin/util/"

請將此指令碼複製到 /etc/rc.d/init.d/ directory 中。

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • 查看 /etc/inittab 檔案，並從中判斷預設的執行層級 (通常為 3)。[[請參閱下方]]
  • 然後移至 /etc/rc.d/rcX.d (其中的 X 是執行層級)。
  • cd /etc/rc.d/rc3.d
  • 建立名為 S95urchin 和 K05urchin 的符號連結，並將其指向 /etc/rc.d/init.d/urchin_daemons。
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # 預設執行層級。RHS 使用的執行層級為：

  • 0 - 停止 (不要將此值設為初始預設值)
  • 1 - 單一使用者模式
  • 2 - 多重使用者，不採用 NFS (如果沒有網路的話，則此設定與 3 相同)
  • 3 - 完整多重使用者模式
  • 4 - 未使用
  • 5 - X11
  • 6 - 重新開機 (不要將此值設為初始預設值)
