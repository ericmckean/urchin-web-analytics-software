使用 NFS 時需鎖定

Urchin 組態使用 data/conf 安裝資料夾中的資料檔案，並且會在設定和管理期間設定讀取和寫入鎖定。如果 data/conf 資料夾裝載於 NFS 上，則 NFS 伺服器必須執行正確的鎖定常駐程式來處理遠端鎖定。

如果鎖定功能未正常運作，Urchin 安裝可能會無限期當機。

技術注意事項

Urchin 在 UNIX 上使用 fcntl() 函數來執行非強制和獨佔 (讀取 / 寫入) 鎖定。某些舊系統可能不支援不同平台的所有每次點擊的收益鎖定要求。請確認您執行的包含最新修補程式的新版作業系統，並確認 NFS 伺服器正在執行 'statd' 和 'lockd' 或同等級程式。
