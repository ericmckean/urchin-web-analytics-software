Urchin 的共同品牌經營

總覽

Urchin 在管理介面、報告介面及登入畫面中加入了共同品牌經營功能。若要在介面最上方包含 HTML，您需要編輯 3 個檔案。如果您已完成入口網站整合，您的應用程式伺服器就可以在頁框組或表格內傳送 Urchin 報告。

共同品牌經營指示

若要在介面上使用共同品牌經營，您需要編輯下列位在 Urchin 安裝中的檔案： 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • 移動 ".sample" (檔案命名方式如下列範例：cobrand_admin.tpl) 並重新命名檔案
  • 依需求在這些檔案中新增 HTML，以包含您的品牌說明。

第一和第二個檔案分別控制管理介面和報告介面中的共同品牌經營功能。第三個檔案則控制登入畫面中的共同品牌經營功能。這些檔案中提供的 HTML 會置於 Urchin 介面頂端。
