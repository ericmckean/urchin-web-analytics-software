由於我使用 Flash，我的網站只有一個網址。要如何設定目標程序？

您可以追蹤自己網站上任何以瀏覽器為基礎的事件，包括 Flash 事件。透過使用 urchinTracker JavaScript 函數，您可以將網頁檔名指派給任何 Flash 動作，並在適當的目標或程序步驟中輸入該檔名。

若要追蹤事件，請使用指定該事件名稱的引數來呼叫 urchinTracker() JavaScript 函數。例如，呼叫： 

    javascript:urchinTracker('/purchase_funnel/page1.html');

會將每次呼叫 Flash 事件生都記錄成一次 /purchase_funnel/page1.html 的網頁檢視。引數的開頭必須是斜線。

事件名稱可以組織成任何您想要的目錄樣式結構。urchinTracker 路徑 / 檔名引數可以不是網站上的實際網址。

Flash 程式碼範例 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
