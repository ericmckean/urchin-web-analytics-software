為何 Urchin 要安裝 Apache 網頁伺服器？

Urchin 會安裝僅佔少許資源的 Apache 網頁伺服器，以便提供網頁式管理和動態報告傳送等功能。透過使用 Urchin 隨附的獨立版 Apache，您就不必為了支援 Urchin 而修改現有網頁伺服器的組態。此外，在現有網頁伺服器之外執行 Urchin 的 Apache 網頁伺服器也有其他優點：

  • Urchin 不會受標準網頁伺服器的變更所影響
  • 管理 Urchin 服務時不會影響到標準網頁伺服器
  • Urchin 的圖形化使用者管理介面可用來變更獨立 Apache 網頁伺服器的特定作業參數
  • Urchin 升級能夠完全獨立作業，任何必要的變更 (例如 SSL 支援) 都可以透過 Urchin 升級程序個別處理，而不需對標準網頁伺服器進行任何變更。Urchin 的 Apache 安裝應該不會影響您電腦上任何其他現有的網頁服務，也不會佔用太多電腦資源。Urchin 的設定允許透過網路遠端存取報告。
