Urchin 流量監視器 (UTM) 快速安裝 (IIS)

  • 本文由 Urchin 軟體授權顧問 EpikOne 撰寫

以下為快速安裝概要，說明如何在任何執行 Microsoft IIS 伺服器的 Windows 平台上安裝網站的 Urchin 流量監視器 (UTM)。 如需更多關於 Urchin 流量監視器 (UTM) 的詳細資訊，請參閱此節標題為《Urchin 流量監視器 (UTM)》的文章。

步驟 1： 將 Urchin 流量監視器 (UTM) 檔案複製到網站的文件根。 urchin.js 和 __utm.gif 檔案位於 Urchin 分佈的 "utils\utm" 資料夾中。 請將這兩個檔案複製到網站內容的主資料夾中。 重要： GIF 檔是以兩個劃底線的字母開頭。

步驟 2： 您網站 HTML 裡的參照 Urchin 流量監視器 (UTM)。 請將下行複製到您的所有 HTML 網頁中。 因為它可放在網頁中的任何位置，因此我們建議放在 </body> 關閉標記之前。 如果您使用常見的 [包含] 或範本，可以在此輸入。 

<script src="/urchin.js" type="text/javascript"></script>

<script type="text/javascript">

        urchinTracker();

</script>

如果您使用像是 "HTML Tidy" 之類的套件，則可以將 Javascript 指令行放在網頁的 HEAD 區域。例如： 

<html>

<head>

  <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1">

  <script src="/urchin.js" type="text/javascript"></script>

  <script type="text/javascript">

        urchinTracker();

  </script>

  ...

</head>

步驟 3： 啟用您 IIS 記錄中的 Cookie。 開啟 IIS 管理員，帶出網站的 [內容] 視窗。 確認記錄已經啟用，且設定為 W3C 延伸格式。 按一下格式旁、位於 [延伸內容] 標籤下的 [內容] 按鈕，勾選 [Cookie] 旁的方塊。

步驟 4： 設定 Urchin 設定檔使用 Urchin 流量監視器 (UTM)。 在 Urchin 管理介面中，編輯該設定檔，然後按一下 [設定] 標籤。 將 [訪客追蹤方式] 設定為 Urchin 流量監視器 (UTM)。 將 Urchin 流量監視器 (UTM) 網域設定為您的網址 (不包含 www)。 然後將 [報告集] 選擇為預設，並按一下 [更新] 按鈕。

大功告成！ 現在，您的網站會開始將 Urchin 流量監視器 (UTM) 資料記錄到一般記錄檔中，當您下次執行 Urchin 時，就會辨識這些記錄檔。

有在運作嗎？ 若要查看 Urchin 流量監視器 (UTM) 是否成功將項目記錄到您的記錄檔中，請在安裝 Urchin 流量監視器 (UTM) 並點擊網站的幾個網頁後查看記錄。 您會在記錄檔的結尾看到如下的項目： 

... "GET /__utm.gif?..." 200 ..."__utma=..."

如果沒有看到 utma 項目，請務必檢查記錄中是否有正確啟用 Cookie。 如果狀態碼不是 200，則請確定檔案已正確複製到您的文件根。
