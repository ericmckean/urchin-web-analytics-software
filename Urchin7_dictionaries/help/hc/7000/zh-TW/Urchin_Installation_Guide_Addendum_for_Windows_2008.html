本文中的附註適用於在 Windows Server 2008 及 Windows Vista 上安裝 Urchin 的情況。 雖然 Urchin 安裝指南 (Windows) 中的步驟適用於 Windows Server 2008 及 Windows Vista，但仍請留意下列事項。

Windows 2008 包含新的「使用者帳戶控制 (UAC)」安全性功能，若需瞭解詳情，請參閱 http://technet.microsoft.com/en-us/library/cc709691.aspx 。

UAC 啟用時的已知問題包括：

  1. 啟用 UAC 時，應用程式就無法正確寫入至 "Program files" 資料夾。 詳情請參閱以下網址： http://msdn.microsoft.com/en-us/library/aa905330.aspx

  2. 按照預設，Urchin 的執行程式無法從命令提示字元執行。 您必須以管理員的身份使用命令提示字元，或將執行程式設定為以下內容： 點選右鍵然後選取 [內容] > [相容性] > [以管理員的身份執行此程式] > [確定]。 請注意，Urchin Windows 服務應該會自行運作，您不需進行其他動作。

解決方案：

  1. 停用 UAC 並按照您的需要設定 Urchin。 使用二進位或 Windows 服務使用 Urchin。
  2. 如果 UAC 為啟用狀態，「切勿」在 "Program files" 資料夾中安裝 Urchin。 例如，改安裝在您的「使用者」目錄中。 修改內容，以採用二進位資料在管理模式中執行 (如上文第二點所述)，或在不變更設定的情況下使用 Windows 服務。
