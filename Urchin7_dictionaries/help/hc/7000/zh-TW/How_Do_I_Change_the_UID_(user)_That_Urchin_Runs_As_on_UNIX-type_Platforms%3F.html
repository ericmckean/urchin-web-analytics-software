如何在 UNIX 類型平台上變更 Urchin 執行使用者的 UID (使用者)？

變更 Urchin 執行使用者的「UID / 使用者」和 / 或「GID / 群組」非常簡單。您可能必須擁有「根權限」才能執行下列作業，因此在大部分情況下，只有系統管理員可以完成這項作業。

  1. 如果您的系統上並沒有新的使用者和群組，請在密碼和群組檔案中建立適當的項目。
  2. 使用 "urchinctl stop" 停止 urchin 服務。"urchinctl" 二進位格式檔位於 Urchin 分佈的 "bin" 目錄中。
  3. 在 Urchin 分佈的根目錄中執行 "chown -R newuser:newgroup"。如果您已經在 "urchin.conf" 檔案中為 "var" 或 "data" 目錄指定替代位置，則也必須針對這些目錄執行這個 "chown" 指令。
  4. 在 Urchin 發行軟體的 "var" 目錄中，編輯 "urchinwebd.conf.template" 檔案，變更下列幾行程式碼：
    • User newuser Group newgroup
  5. 使用 "urchinctl start" 重新啟動 Urchin 服務。
