在設定好 Urchin 後，如何將它改成資料中心模式？

  • 本文由 Urchin 軟體授權顧問 ROI Revolution 撰寫

使用資料中心模式，您可以在一個群組中的一或多個設定檔內新增管理權限層級。 如果您已經完成了初始設定程序，並且想要啟用資料中心模式，只要透過 MySQL 或 PostgreSQL 就能輕鬆完成設定。

透過 MySQL 進行： 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

透過 PostgreSQL 進行： 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
