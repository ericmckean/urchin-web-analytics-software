如何追蹤 Flash 事件？

Urchin 可讓您追蹤任何以瀏覽器為基礎的事件，包括 Flash 和 Javascript 事件。若要追蹤事件，請使用指定該事件名稱的引數來呼叫 urchinTracker JavaScript 函數。例如，呼叫： 

    javascript:urchinTracker('/homepage/flashbuttons/button1');

會將每次 Flash 事件記錄成一次 /homepage/flashbuttons/button1 的網頁檢視。引數的開頭必須是斜線。事件名稱可以組織成任何您想要的目錄樣式結構，就算與這些檔名相對應的實際網頁不存在也無妨。

例如，如果您想要按網頁或事件類型來組織 flash 事件，可以根據下列幾行程式碼來組織階層： 

    /homepage/flashbuttons/button1
    /homepage/clips/clip1

Flash 程式碼範例 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file');");
    }
