記錄檔大小有限制嗎？

Urchin 的記錄處理引擎可以處理大於 2GB 的網頁伺服器記錄，無論是利用 bzip (.bz2)、gzip (.gz) 或 zip (.zip) 壓縮的檔案都可以處理，因為 Urchin 是利用 zlib 程式庫解壓縮檔案。

不過，Urchin 的記錄處理引擎在 util 目錄中是利用解壓縮 / 壓縮二進位資料，來建立各設定檔報告資料的備份與封存資料。如果月份的報告資料大到檔案壓縮後仍超過 2GB，Urchin 就無法建立備份或封存，而且除非停用自動備份與封存功能，否則記錄處理程序也會失敗。在這樣的情況下，您就需利用其他外部公用程式手動備份檔案。
