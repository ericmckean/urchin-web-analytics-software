追蹤聯外連結和廣告

Urchin 提供簡單的方式讓您追蹤橫幅廣告或其他聯外連結上的點擊。由於這些連結不會引導訪客到您網站中包含 Urchin 流量監視器 JavaScript 的網頁，因此如果您想要追蹤這些連結，必須以 urchinTracker JavaScript 來標記連結本身。這段 JavaScript 會將網頁檢視指派給針對連結的點擊，而網頁檢視來自於您指定的檔名。

對於動畫 GIF 或其他類型的靜態橫幅廣告，請將 urchinTracker JavaScript 添加到橫幅廣告的 <a> 標記中： 

    <a href="http://www.example.com/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
