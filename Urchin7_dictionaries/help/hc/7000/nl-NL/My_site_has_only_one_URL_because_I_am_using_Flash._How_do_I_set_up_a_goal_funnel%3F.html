Mijn site heeft slechts één URL omdat ik Flash gebruik. Hoe stel ik een doeltrechter in?

Op uw website kunt u conversies voor elke browsergebeurtenis bijhouden, waaronder Flash-gebeurtenissen. Met behulp van de JavaScript-functie urchinTracker kunt u een paginabestandsnaam toewijzen aan elke willekeurige Flash-actie en die bestandsnaam in de juiste doel- of trechterstap invoeren.

Om conversies voor een gebeurtenis bij te houden, roept u de JavaScript-functie urchinTracker() aan met een argument waarin de naam van de gebeurtenis wordt aangegeven. Als u bijvoorbeeld het volgende aanroept: 

    javascript:urchinTracker('/purchase_funnel/page1.html');

wordt elke instantie van die aangeroepen Flash-gebeurtenis vastgelegd als paginaweergave met de naam /purchase_funnel/page1.html. Het argument moet beginnen met een slash.

De namen van de gebeurtenissen kunnen worden geordend in elke gewenste directory-stijl. Het argument pad/bestandsnaam voor urchinTracker hoeft geen bestaande URL op uw website te zijn.

Voorbeelden van Flash-code 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
