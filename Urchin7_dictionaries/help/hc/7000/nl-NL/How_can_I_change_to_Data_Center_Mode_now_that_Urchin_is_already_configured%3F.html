Hoe kan ik omschakelen naar de modus 'Datacentrum' nu Urchin al is geconfigureerd?

  • Dit artikel is geschreven door ROI Revolution, geautoriseerd softwareconsultant voor Urchin

In de modus 'Datacentrum' kunt u niveaus met beheerdersrechten toevoegen aan een of meer profielen in een groep. Als u de eerste configuratieprocedure al heeft voltooid en u de modus 'Datacentrum' wilt inschakelen, kunt u dit eenvoudig doen via MySQL of PostgreSQL.

Voor MySQL: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

Voor PostgreSQL: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
