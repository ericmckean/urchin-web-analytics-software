Hoe vind ik mijn serienummer?

Er zijn twee manieren om uw huidige serienummer van Urchin weer te geven.

  1. Beheerinterface
    • Start de beheerinterface van Urchin vanuit een webbrowser.

      Klik op 'Configuratie --> Instellingen --> Licentie'.

  2. Het hulpprogramma inspector uitvoeren
    • Open een opdrachtvenster en gebruik de opdracht cd om naar de directory basedir/urchin/util/ te gaan. Voer het hulpprogramma inspector uit. Het serienummer wordt samen met andere informatie van Urchin weergegeven. Houd er rekening mee dat in Unix/Linux alle opdrachten moeten worden voorafgegaan door './'.
