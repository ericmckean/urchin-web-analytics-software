Cobranding van Urchin

Overzicht

Cobranding is mogelijk in de beheerinterface, de rapportage-interface en het aanmeldingsvenster van Urchin. U moet drie bestanden bewerken voordat u HTML boven aan de interface kunt toevoegen. Bij een volledige portalintegratie kan de rapportage van Urchin worden weergegeven in een frameset of tabel via uw toepassingsserver.

Instructies voor cobranding

Als u cobranding wilt toevoegen aan de interface, moet u de volgende bestanden uit de Urchin-installatie bewerken: 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • Hernoem de bestanden door de extensie '.sample' te verwijderen (de bestandsnaam moet bijvoorbeeld 'cobrand_admin.tpl' zijn).
  • Voeg de benodigde HTML-inhoud voor cobranding toe aan deze bestanden.

Het eerste bestand beheert de cobranding in de beheerinterface en het tweede bestand de cobranding in de rapportage-interface. Het derde bestand beheert de cobranding in het aanmeldingsvenster. De HTML-code in deze bestanden wordt boven op de Urchin-interface geplaatst.
