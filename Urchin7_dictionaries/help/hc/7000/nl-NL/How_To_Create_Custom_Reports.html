Aangepaste rapporten maken

Een aangepast rapport maken:

  1. Kies een tabel voor gegevensoverzichten of definieer een nieuwe tabel (raadpleeg Aangepaste gegevensoverzichten]);

  2. Maak een nieuw rapportbestand ('.rs2') door <Urchin>/lib/custom/profiletypes/Default/default.rs2.sample te kopiëren.

  3. Definieer het nieuwe rapport door een nieuwe rij toe te voegen aan het rapportbestand.

De indeling van het rapportbestand wordt hieronder beschreven:

  • ID - unieke rapport-id:
    • C=Categorie
    • D=Gegevens
    • M=Menu
    • R=Rapport
    • S=Subrapport
    • P=Paginarapport
    • B=Dashboard
  • RS - Dashboard-/rapportagesecties:
    • 1=leidinggevende
    • 2=marketeer
    • 4=webmaster
  • RR - Rapportvereisten:
    • 1=Software
    • 2=Geavanceerd
    • 4=E-commerce
    • 8=UTM
  • NTITLE - navigatietitel (nummer woordenlijstitem)
  • ONCOLOR - kleur bij navigatie over
  • OFFCOLOR - kleur bij navigatie buiten
  • HELP - Help bij rapport (nummer woordenlijstitem)
  • NDATA - aantal gegevenselementen (items moeten direct hierna volgen)
    • Dit veld kan worden gevolgd door aanvullende subrapporten met type D (gegevens)
  • CTITLES - kolomtitels (nummer woordenlijstitem)
  • CTYPES - kolomtype gebruikt voor lay-out
    • 10=tekenreeks
    • 11=datum/jaarmaand
    • 12=datum
    • 20=nummer
    • 21=opbrengst
    • 22=bytes
    • 23=tijdverhouding
    • 24=verhouding
    • 25=pct
    • 26=tijd
    • 30=totaal pct
    • 40=opbrengst
    • 41=opbrengst/num
    • 42=num/opbrengst
  • TABLE - gegevenstabelnummer:
    • T=totaal
    • TR=transactie
    • H=histogram
    • V=bezoeker
    • 1-n-tabel
  • DXFORMAT - indeling voor gegevensextractie
  • RF - verplicht veld
  • RT - rapporttype:
    • 1=grafiektotalen
    • 2=histogram
    • 3=stack
    • 4=top
    • 5=verkenner
    • 6=directory
    • 7=grafiek-beperkt
    • 8=punt
    • 9=doorklik
    • 10=gedefinieerde trechter
    • 11=overlay voor startpagina
    • 12=kaartoverlay
    • 13=doelverificatie
    • 14=doelpad
    • 20=kpi-datumrapport
    • 21=kpi top tien
    • 22=trendoverzicht
    • 23=conversieoverzicht
    • 24=kaartminiatuur
  • GTYPES - typen grafieken:
    • 1=balk
    • 2=lijn
    • 3=cirkel
    • 4=horizontale balk
    • 5=trechter
    • 6=meerdere lijnen
  • FSEP - veldscheidingsteken
  • SP - speciale opties (bitmap-nummers toevoegen):
    • 1=totalen
    • 2=gemiddelden
    • 4=datum vergelijken
    • 8=naar kleine letters
    • 16=gegevens voor een bepaalde periode
    • 32=overlay
    • 64=taal opzoeken
    • 128=segment
    • 256=waarde tot op heden
    • 512=transactiedetail
    • 1024=bezoekersspecificatie
    • 2048=term opzoeken
  • SR - markering voor speciaal rapport:
    • 1=suggesties voor zoekwoorden
    • 2=doorklik
    • 3=gespecificeerd naar dagdeel
    • 4=bezoekerslijst
    • 5=kaartoverlay
    • 6=aangepaste woordenlijst voor aangepast gebruik
    • 7=datumvergelijking
  • FNT - filternummer filtertype:
    • 1: =

    • 2: >=

    • 3: <=

    • 4: >

    • 5: <

  • FNF - nummerveld filter
  • FNV - filterwaarde
  • FST - type filtertekenreeks:
    • 1=uitsluiten
    • 2=opnemen
  • FSF - tekenreeksveld filter
  • FSX - tekenreeks van reguliere expressie
