Ik gebruik Safari onder Mac OS X en kan geen rapporten weergeven.

  • Oudere versies van Safari (v. 1.3.1 voor Mac OS X Panther en v. 2.0.1 voor Mac OS X Tiger) bieden geen ondersteuning voor Urchin. Op dit moment worden alleen Mozilla-browsers (Firefox, Mozilla en Netscape Navigator) ondersteund voor Macs. U kunt het beste een van deze ondersteunde browsers gebruiken om Urchin-rapporten weer te geven.
