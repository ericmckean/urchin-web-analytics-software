Hoe houd ik klikken op uitgaande links bij?

Als u uw site uitgaande links bevat, is er een eenvoudige manier om bij te houden op welke links bezoekers klikken om uw site te verlaten.

Om uitgaande klikken bij te houden, voegt u de volgende code toe binnen de <a>-tag: 

    <a href="http://www.site.com" onClick="javascript:urchinTracker('/outboundlinks/sitename/pagename');">

Met deze code worden klikken op de uitgaande link geregistreerd als paginaweergave met de naam /outboundlinks/sitename/pagename. We raden u aan om alle uitgaande links in een logische directorystructuur te registreren, zoals in het voorbeeld wordt weergegeven. Op deze manier kunt u eenvoudig herkennen op welke pagina's gebruikers klikken om uw site te verlaten.
