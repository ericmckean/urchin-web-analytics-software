Waarom wordt de Microsoft Data Access Internet Publishing Provider weergegeven in het rapport Browsers?

Dit browsertype wordt meestal weergegeven als uw site downloadbare bestanden bevat die worden geopend met een Microsoft Office-product (bijvoorbeeld Word .doc-bestanden en Excel .xls-bestanden). De invoegtoepassingen die door Internet Explorer worden gebruikt om deze bestandstypen weer te geven, melden zichzelf als Microsoft Data Access Internet Publishing Provider wanneer deze pagina wordt bekeken.
