Hoe kan ik de volledige URI, inclusief zoektermen, in één rapport weergeven?

In Urchin 6 worden zoekopdrachtpatronen weergegeven in een speciaal gedetailleerd rapport 'Inhoudsoptimalisatie -> Inhoudsprestatie -> Paginazoektermen'. Als u de volledige pagina en het zoekopdrachtpatroon wilt weergeven in het rapport 'Belangrijkste inhoud', kunt u het onderstaande filter gebruiken om de weergave van deze gegevens te wijzigen.

Opmerking: als u dit doet, neemt de verwerkingstijd toe voor elk profiel waarop het filter wordt toegepast. Daarnaast kan de limiet van uw database worden overschreden door de toename in unieke paginanamen.

Maak een nieuw 'Profielfilter' met de volgende instellingen:

  • Filtertype: Geavanceerd
  • Filter A: request_stem (AUTO) - A extraheren (.*)
  • Filter B: request_query (AUTO) - B extraheren (.*)
  • Uitvoeren naar: request_stem (AUTO) - Samenstellen $A1/$B1
  • Uitvoerveld negeren: 'Ja'
  • Verplichte velden: 'Beide velden'
  • Hoofdlettergevoelig: 'Nee'
