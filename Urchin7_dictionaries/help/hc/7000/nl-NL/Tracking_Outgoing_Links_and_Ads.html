Het bijhouden van uitgaande links en advertenties

Urchin biedt een eenvoudige manier om klikken op banneradvertenties of andere uitgaande links bij te houden. Omdat deze links niet naar een pagina met de UTM JavaScript leiden, moet u de link zelf coderen met urchinTracker JavaScript als u deze klikken wilt bijhouden. Deze JavaScript-code wijst een paginaweergave toe aan elke klik. De paginaweergave wordt toegevoegd aan een bestand met een door u opgegeven naam.

Voor een GIF-animatieadvertentie of een andere statische banneradvertentie voegt u de urchinTracker JavaScript toe aan de code <a> van uw banner: 

    <a href="http://www.example.com/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
