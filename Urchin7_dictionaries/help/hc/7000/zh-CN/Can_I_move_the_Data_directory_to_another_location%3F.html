能否将数据目录转移到其他位置？

概述

Urchin 数据目录是 Urchin 程序根目录的默认子目录。更改位置很方便：可修改 ~urchin/etc/urchin.conf 文件并手动将现有 ~urchin/data 目录移到所需位置。请注意，这个操作可能会中断现有许可；这种情况下，您需要联系 Urchin 软件授权顾问，并请求重置您的序列号。

移动数据目录的步骤：

可在 urchin.conf 文件中更改 Urchin 保存其报告数据的位置，此文件位于 Urchin 主安装目录的 etc 目录中。

对于 UNIX 类型的系统：

  1. 以 Urchin 安装时所用的用户身份打开 command shell
  2. 通过 cd 命令进入安装 Urchin 的目录
  3. 通过以下命令终止 Urchin 服务：bin/urchinctl stop
  4. 用文本编辑器打开 Urchin 发行套装 etc 目录中的 urchin.conf 文件
  5. 删除开头的"#"字符以取消对以下行的注释：

    • #dataDirectory:./data/，并将默认目录替换为希望使用的完整目录路径，例如 dataDirectory: /bigdisk/urchin/data/

  6. 根据需要，还可创建新的数据目录。重要说明： 运行 Urchin 的用户必须能够读取和写入此目录。

  7. 将现有的数据复制到新位置。为确保正确保存所有的权限和数据，建议您使用下列命令：
    • cd data; tar cf - . | (cd /bigdisk/urchin/data; tar xpf -)
  8. 将数据目录重命名为 data.old。可根据需要将其完全删除，但在此之前，最好能确保一切均正常运行。
  9. 为便于管理，您可以在 Urchin 主目录中创建指向新位置的 Symlink，例如
    • ln -s /bigdisk/urchin/data ./data
  10. 通过以下命令重新启动 Urchin 服务：bin/urchinctl start
  11. 以管理员用户身份登录 Urchin。您将看到 Urchin 许可屏幕。只要点击"重新激活许可"链接，一切就大功告成。

对于 Windows 系统：

  1. 停止 Urchin 服务：开始->程序->Urchin->禁用 Urchin 服务

  2. 打开 Windows 资源管理器，并转到 Urchin 发行套装的 etc 文件夹。默认路径为 C:\Program Files\Urchin\etc
  3. 使用文本编辑器打开 urchin.conf 文件
  4. 删除开头的"#"字符以取消对以下行的注释：

    • #dataDirectory:./data/，并将默认文件夹替换为希望使用的驱动器号和完整路径名，例如 dataDirectory: E:\Urchin\data

  5. 根据需要，还可创建新的数据文件夹。重要说明：这个文件夹的权限必须允许您读取和写入 Urchin 服务。
  6. 将数据文件夹的内容复制到新文件夹。
  7. 将数据文件夹重命名为 data.old。可根据需要将其完全删除，但在此之前，最好能确保一切均正常运行。

  8. 重新启动 Urchin 服务：开始->程序->Urchin->启用 Urchin 服务

注意事项

UNIX 类型的系统采用可避免篡改的许可发放方式，因此移动 Urchin 数据目录后，必须重新激活 Urchin 许可。不过，重新授予许可的操作非常简单 - 只需以 Urchin 管理员用户身份登录，并点击"重新激活许可"即可。
