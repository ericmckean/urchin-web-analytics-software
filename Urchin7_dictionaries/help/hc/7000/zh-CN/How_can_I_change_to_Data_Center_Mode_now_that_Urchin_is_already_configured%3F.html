在配置好 Urchin 后，如何将其更改为数据中心模式？

  • 本文由 Urchin 软件授权顾问 ROI Revolution 编写

在数据中心模式下，您可以向一个组中的一个或多个配置文件添加各种管理权限级别。 如果您已经完成了初始配置过程，并想启用数据中心模式，可以通过 MySQL 或 PostgreSQL 轻松实现此操作。

对于 MySQL： 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

对于 PostgreSQL： 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
