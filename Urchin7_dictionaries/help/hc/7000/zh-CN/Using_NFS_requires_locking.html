使用 NFS 需要锁定

Urchin 配置采用安装目录 data/conf 文件夹中的数据文件，将在安装和管理过程中设置读写锁定。如果此 data/conf 文件夹是通过 NFS 安装的，则要求 NFS 服务器运行合适的锁定后台程序来处理远程锁定。

如果该锁定没有正常运作，则 Urchin 安装可能会无限期地挂起。

技术注意事项

Urchin 在 UNIX 中使用 fcntl() 函数执行建议和独占（读/写）锁定。有些旧版的系统可能不支持不同平台的所有 rpc 锁定请求。请确保您运行的是包含最新补丁程序的最新版本的 OS。确保 NFS 服务器正在运行 'statd' 和 'lockd'（或等同项目）。
