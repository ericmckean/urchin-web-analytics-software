本文中的说明适用于在 Windows Server 2008 和 Windows Vista 上安装 Urchin 的情况。 尽管 Urchin 安装指南 (Windows) 中介绍的步骤适用于 Windows Server 2008 和 Windows Vista，但需注意以下事项。

Windows 2008 包含一项名为用户帐户控制 (UAC) 的新增安全功能，详情请参阅 http://technet.microsoft.com/en-us/library/cc709691.aspx 。

启用 UAC 时，存在以下已知问题：

  1. 如果启用了 UAC，则应用程序无法正确地写入“Program files”文件夹。 有关详情，请访问： http://msdn.microsoft.com/en-us/library/aa905330.aspx

  2. 默认情况下不能通过命令提示符运行 Urchin 的可执行文件。 您必须以管理员身份使用命令提示符，或者按照如下方法来设置可执行文件的属性： 右键单击该文件，然后依次选择“属性”->“兼容性”->“以管理员身份运行此程序”->“确定”。 请注意，Urchin Windows 服务可按预期运行，而不必执行任何其他操作。

解决方法：

  1. 停用 UAC 并将 Urchin 安装到所需位置。 通过二进制程序或 Windows 服务运行 Urchin。
  2. 如果启用了 UAC，则不要将 Urchin 安装到 Program files 文件夹。 您可以将它安装到您的用户目录。 修改属性以在管理员模式下运行二进制程序（参见上面第 2 条说明），或者不做任何更改而直接使用 Windows 服务。
