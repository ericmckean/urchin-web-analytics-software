指向我网站的这些链接已包含我自己的广告系列跟踪变量，我能使用这些变量替代 Urchin 的变量吗？

如果您已使用自己的跟踪变量标记了在线链接，则可将这些现有变量映射到 Urchin 的广告系列跟踪变量中。您标记的链接需包含至少 3 个变量。这些变量将映射到：

  1. utm_source: 您的广告系列来源（如 Google、Yahoo）
  2. utm_medium: 您的广告系列媒介（每次点击费用、横幅、电子邮件）
  3. utm_campaign: 您的广告系列名称（春季特价）

在这些说明中，我们假设您的网站包含以下网址： 

            www.example.com/index.html?camp=GADS&kwd=memory&version=47&type=ppc&adver=google

第 1 步 将您的变量与 Urchin 的变量进行匹配。创建采用以下示例所示格式的表： 

               广告系列     广告来源    广告类型     关键字   A/B 内容  主要
用户代码=    camp       adver       type        kwd       version      
Urchin 代码=  utm_campaign utm_source     utm_medium  utm_term  utm_content  utm_id

第 2 步 修改 UTM 文件。

  • 在文本编辑器中打开 urchin.js 文件。在文件顶部有一个用户定义的部分，其中包括以下 utm 变量： 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="utm_campaign";
                var _ucmd="utm_medium";
                var _ucsr="utm_source";
                var _uctr="utm_term";
                var _ucct="utm_content";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";
  • 利用上述标记的链接示例，可将此部分编辑如下： 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="camp";
                var _ucmd="type";
                var _ucsr="adver";
                var _uctr="kwd";
                var _ucct="version";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";

第 3 步 为这些新跟踪变量创建过滤器。

  • 以下是您应用于各过滤器的基本过滤器配置： 

                过滤器类型：高级
                字段 A：utm_campaign_uri (raw)
                提取 A：new_variable =([^&]*)
                字段 B：（无）
                提取 B：（空）
                输出至：utm _( old_variable )
                构造器：$A1
                必填字段 A：是
                必填字段 B：否
                覆盖输出字段：是
                区分大小写：否
  • old_variable 指默认的 Urchin 跟踪变量。new_variable 指您自己的跟踪变量。请记住，每个修改的变量都需要有一个过滤器。
