Urchin 为什么要安装 Apache 网络服务器？

Urchin 会安装一个小型 Apache 网络服务器，目的是提供基于网络的管理和动态报告传递等功能。通过将独立版 Apache 与 Urchin 相捆绑，不需要修改网络服务器的现有配置即可支持 Urchin。此外，在现有网络服务器之外单独运行 Urchin 的 Apache 网络服务器还有其他一些优点：

  • 对标准网络服务器的更改不会影响 Urchin
  • 管理 Urchin 服务时不会影响到标准网络服务器
  • 可使用 Urchin 的管理 GUI 更改独立 Apache 网络服务器的特定操作参数
  • Urchin 升级完全能够独立进行，可通过 Urchin 升级过程独立处理任何必要的更改（例如 SSL 支持），而无需更改标准网络服务器。Urchin 的 Apache 安装应该不会影响您计算机上任何其他现有的网络服务，也不会占用太多计算机资源。Urchin 的配置允许通过网络远程访问报告。
