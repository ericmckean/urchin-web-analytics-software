Urchin 服务控制实用程序 (urchinctl)

概述

  • Urchin Services Control utility (urchinctl) 检验 Urchin Scheduler 和 Urchin Webserver 服务的状态，并允许您启动和停止这两种服务。 对于 UNIX 类型的系统，通常从系统的启动时间脚本之一调用 urchinctl 以自动启动或关闭 Urchin 服务。

urchinctl 可执行的操作有：

  • 验证 Urchin6 主/从调度程序的状态
  • 启动、停止或重启调度程序或网络服务器（或二者）
  • 在替代端口启动网络服务器
  • 启动经过 SSL 加密的网络服务器

用法

urchinctl 位于 Urchin 发行套装的 bin 目录下。

该实用程序的用法如下： 

  urchinctl -h（显示用法消息并退出）

  urchinctl -v（显示版本并退出）

  urchinctl [-e] [-p 端口] [[-s] | [-w]] 操作

其中， 

  -e  在网络服务器中，启用加密 (SSL)

  -p  指定网络服务器的监听端口

  -s  仅在 Urchin 调度程序上执行操作

  -w  仅在 Urchin 网络服务器上执行操作

此操作是下列操作之一： 

  start   （启动服务）

  stop   （停止服务）

  restart   （停止服务，然后再启动服务）

  status  （显示网络服务器/调度程序的运行时间状态）

除非已指定 "-s"（调度程序）或 "-w"（网络服务器）选项，否则在默认情况下，此操作将在网络服务器和调度程序中执行。 请注意，这些选项是互相排斥的。

注意

对于 UNIX 类型的系统，urchinctl 应该作为已安装 Urchin 的用户/UID 运行，以确保 urchinwebd 和 urchind 进程作为此 UID 启动。
