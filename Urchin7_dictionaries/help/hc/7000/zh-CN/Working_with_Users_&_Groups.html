使用用户和组

概述

通过 Urchin 的“用户和组”功能，Urchin 管理员可轻松设置任意数量的用户，并授予他们适当的报告访问权。 如果用户数量很多，管理员还可以将他们分组，以加速并简化管理工作。 如果某个用户组获得了报告访问权，该组中的所有用户都可以登录系统阅读报告。 用户只有在经过特别许可后才能获得报告访问权。

如何添加用户

  1. 以管理员身份登录 Urchin 系统。 请注意： 无论何种用户类型，访问 Urchin 系统的网址都是相同的。
  2. 点击菜单右上角的“配置”。
  3. 点击“用户和组”。
  4. 点击右上角的“添加”按钮，进入“用户向导”。
  5. 选择用户名（用户名应该为小写且不含空格），并选择密码。
  6. 输入用户的真实姓名，该姓名将在 Urchin 系统中显示。 点击“下一步”按钮。

  7. 决定该用户应具有的控制级别。 可根据需要授予用户对 AdWords 工具的使用权（适用于 Urchin 6.601 及更高版本）。 点击“完成”。

  8. 点击“更多选项”，为该用户修改“用户信息”、“偏好设置”、“组”或“报告访问权”。
  9. 点击“报告访问权”标签。 左侧框中会显示所有可用的配置文件。
    1. 选择一个或多个配置文件。 要选择多个配置文件，请根据您的平台使用 Command 或 Ctrl 键。
    2. 点击右向箭头，将配置文件移至“准许访问”框。
    3. 点击“更新”以保存更改。

从 Urchin 6.602 开始，您可以在 Urchin 配置数据库中为用户单独定义“首页”默认视图。 要了解详情，请参阅汇总报告。 您可以通过“用户设置”->“偏好设置”，向相应用户授予或撤销 AdWords 工具访问权。

如何使用组

  1. 以管理员身份登录 Urchin 系统，然后点击菜单右上角中的“配置”。
  2. 点击“用户和组”。
  3. 点击“组”。
  4. 点击右上角的“添加”按钮。
  5. 输入组名称（可以是任何描述性的名称）。
  6. 输入“组描述”（可以是有关组的位置或构成的信息）。
  7. 点击“完成”。
  8. 点击“更多选项”。
  9. 点击“组中的用户”标签，选择要添加到组的用户。要选择多个用户，请根据您的平台使用 Command 或 Ctrl 键。
    1. 点击右向箭头，将用户移至“组中的用户”框。
    2. 点击“更新”按钮以保存更改。

建议

  • 所有“超级管理员”级别的用户对 Urchin 系统都有完整的控制权，因此建议只将此权限授予一名用户。
  • 为提高安全性，密码最好包含一个或多个大写字母和/或符号。
