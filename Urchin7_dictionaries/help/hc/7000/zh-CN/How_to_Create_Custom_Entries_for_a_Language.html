如何创建自定义的语言条目

  1. 将文件夹 <Urchin>/lib/admin/languages/ 下的语言文件 xx-XX.txt 复制到 <Urchin>/lib/custom/languages/。

  2. 在上述文件的末尾追加新条目，并为每个新条目分配一个新编号。

  3. 必须用 <Tab> 作为编号与值之间的分隔符，否则 Urchin 无法解析。

  4. 确保条目编号是连续的（增量为 1）。
  5. 重新加载 Urchin。

  6. 重新加载之后，Urchin 应在文件夹 <Urchin>/lib/custom/languages/ 下创建一个名为 xx-XX.db 的语言 DB 文件。

定义完成后，就可以在报告部分将自定义条目用作列标签、标题或帮助说明。

语言文件示例

第 738、739 和 740 条为添加的自定义条目。 

  ...
  731     Executive
  732     Marketer
  733     Webmaster
  734     Dashboards
  735     All Reports
  736     ja-jp-mac (Japanese)
  737     ja-jp-osaka (Japanese)
  738     Modules
  739     Actions
  740     Modules / Actions

重要注意事项： 升级 Urchin 时可能会替换原来的语言文件并增加默认的条目。 请务必重新创新自定义的语言文件并更新自定义报告。
