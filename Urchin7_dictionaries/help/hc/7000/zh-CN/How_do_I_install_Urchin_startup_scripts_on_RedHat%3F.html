如何在 RedHat 上安装 Urchin 启动脚本？

安装后会创建一个 Urchin 启动脚本。脚本命名为"urchin_daemons" ，保存在 Urchin util 目录中。"/path/to/urchin/util/"

请将此脚本复制到 /etc/rc.d/init.d/ directory 中。

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • 查看 /etc/inittab 文件，并判断默认的运行级别（通常为 3）。[[参见下文]]
  • 然后将目录更改为 /etc/rc.d/rcX.d（其中的 X 是运行级别）。
  • cd /etc/rc.d/rc3.d
  • 创建名为 S95urchin 和 K05urchin 的符号链接，将其指向 /etc/rc.d/init.d/urchin_daemons。
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # 默认 runlevel。RHS 使用的 runlevel 为：

  • 0 - 暂停（不要将 initdefault 设为此值）
  • 1 - 单一用户模式
  • 2 - 多用户，不采用 NFS（如果没有网络，则此设置与 3 相同）
  • 3 - 完整多用户模式
  • 4 - 未使用
  • 5 - X11
  • 6 - 重新启动（不要将 initdefault 设为此值）
