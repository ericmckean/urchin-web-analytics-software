导入配置文件 (Windows)

概述

对于系统中运行 Microsoft Internet Information Server 的用户来说，要为他们的每一个 IIS 网站设置配置文件，Urchin 的导入配置文件功能提供了一种轻松快捷的方法。Urchin 可以读取 IIS 配置、确定服务器上正在运行哪些网站，并为将 IIS 日志作为日志源的各个网站建立基本的配置文件。然后，您可以对这些配置文件进行自定义，还可以为导入后的网站添加所需的其他配置文件。

如何导入配置文件

要开始导入配置文件，请以管理员身份登录到 Urchin 管理系统，然后点击"配置"（界面的右上方）点击右上方的"导入"按钮，您将进入到"导入配置文件"屏幕。在此屏幕中，您可以选择要导入哪些网站（如果有）。选择要导入的网站之后，点击"导入"按钮。填写完所有导入选项之后，点击"完成"。

建议

建议您为服务器上的每个网站至少创建一个配置文件，这样您可以通过 Urchin 的汇总报告了解全面的点击量情况。汇总报告为您提供服务器的总体点击量信息，以及各个网站按不同点击量参数的排名。如果您是主机端并且根据带宽用量进行结算，这将非常方便。请注意，汇总报告仅基于所配置的配置文件显示数据，其中不包含未设置配置文件的网站。
