如何从 Windows 2000 删除 Apache 和 Urchin 调度程序服务？

极少数情况下，即使已经从系统中删除 Urchin 网络服务器和 Urchin 调度程序服务，若未完整删除 Urchin 软件，还是可能会在 Windows 2000 注册表中留下这些服务的条目。要删除注册表条目，请按照下列步骤操作：

警告： 不恰当地使用注册表编辑器可能会导致需要重新安装 Windows 等重大问题。使用风险自担。

  • 以管理员身份登录
  • 右键点击"我的电脑"，然后选择"管理"

  • 进入"服务和应用程序"->"服务"

  • 右键点击 Urchin 网络服务器，然后选择"属性"
  • 在"启动类型"下，选择"已禁用"，然后点击"确定"
  • 右键点击 Urchin 调度程序，然后选择"属性"
  • 在"启动类型"下，选择"已禁用"，然后点击"确定"

  • 启动"注册表编辑器"：开始->运行->regedt32.exe

  • 打开 HKEY_LOCAL_MACHINE 窗口

  • 进入 System->CurrentControlSet->Services

  • 点击 Urchin6Webserver 文件夹，然后按下 Delete 键

  • 点击 Urchin6MasterScheduler 文件夹，然后按下 Delete 键

  • 点击 Urchin6SlaveScheduler 文件夹，然后按下 Delete 键

  • 关闭注册表编辑器并保存更改
  • 重新启动您的系统
  • 运行 Urchin 安装文件
  • 选择"删除"Urchin，然后点击"下一步"

现在，系统应该不会再引用 Urchin 网络服务器和 Urchin 调度程序服务，可以重新安装 Urchin 了。
