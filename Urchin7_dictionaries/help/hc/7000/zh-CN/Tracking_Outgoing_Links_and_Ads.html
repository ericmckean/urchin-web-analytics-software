跟踪出站链接和广告

Urchin 提供了跟踪横幅广告或其他出站链接上的点击的简单方法。因为这些链接不指向您网站上包含 UTM JavaScript 的网页，所以如果您要跟踪这些点击，就需要用 urchinTracker JavaScript 标记这些链接本身。这段 JavaScript 会将综合浏览量分配给针对链接的任何点击，综合浏览量归属于您指定的文件名。

对于动画 GIF 或其他类型的静态横幅广告，请将 urchinTracker JavaScript 添加到横幅广告的 <a> 标记中： 

    <a href="http://www.example.com/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
