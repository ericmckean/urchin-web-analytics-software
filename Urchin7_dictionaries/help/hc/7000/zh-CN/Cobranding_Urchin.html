Urchin 联合品牌

概述

Urchin 在管理界面、报告界面及登录屏幕中加入了联合品牌功能。要在界面最上方加入 HTML，需要修改 3 个文件。如果进行全面的门户网站集成，应用程序服务器就可以在框架集或表格内发送 Urchin 报告。

联合品牌说明

要在界面上使用联合品牌，需要修改位于 Urchin 安装目录中的下列文件： 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • 删除".sample"，重命名文件（文件名必须类似于 cobrand_admin.tpl 形式）
  • 根据需要在这些文件中添加 HTML 内容，以包含您的品牌说明

第一和第二个文件分别控制管理界面和报告界面中的联合品牌功能。第三个文件则控制登录屏幕中的联合品牌功能。这些文件中提供的 HTML 会置于 Urchin 界面的顶端。
