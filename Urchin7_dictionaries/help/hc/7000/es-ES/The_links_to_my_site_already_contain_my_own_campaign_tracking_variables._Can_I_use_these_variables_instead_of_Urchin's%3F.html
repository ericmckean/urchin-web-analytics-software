Los enlaces al sitio web ya contienen mis propias variables de seguimiento de campañas. ¿Puedo utilizar estas en lugar de las variables de Urchin?

Si ya ha etiquetado sus enlaces online con sus propias variables de seguimiento, las puede asignar a las variables de seguimiento de campañas de Urchin. Los enlaces que etiquete deben contener al menos tres variables, que se asignan a:

  1. utm_source: la fuente de su campaña (por ejemplo, Google, Yahoo),
  2. utm_medium: el medio de su campaña (cpc, banner, correo electrónico),
  3. utm_campaign: el nombre de su campaña (rebajas_primavera)

A modo de ejemplo, supongamos que su sitio posee la siguiente dirección URL: 

            www.ejemplo.es/index.html?camp=GADS&kwd=memory&version=47&type=ppc&adver=google

Paso 1. Establezca una correspondencia entre sus variables y las de Urchin. Cree una tabla con el formato que se indica en el ejemplo a continuación: 

               Campaña      Fuente del anuncio  Tipo de anuncio     Palabra clave   contenido A/B  Códigos
de usuario maestros=    camp        adver       tipo        palabra clave       versión            
Códigos de Urchin= utm_campaign utm_source      utm_medium  utm_term  utm_content  utm_id

Paso 2. Modifique el archivo de UTM.

  • Abra el archivo urchin.js en un editor de textos. En la parte superior del archivo aparece una sección definida por el usuario que recoge las variables de UTM: 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="utm_campaign";
                var _ucmd="utm_medium";
                var _ucsr="utm_source";
                var _uctr="utm_term";
                var _ucct="utm_content";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";
  • Si utiliza el enlace etiquetado del ejemplo anterior, esta sección se modificará de la siguiente manera: 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="camp";
                var _ucmd="type";
                var _ucsr="adver";
                var _uctr="kwd";
                var _ucct="version";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";

Paso 3. Cree filtros para estas nuevas variables de seguimiento.

  • Esta es la configuración de filtro básica que debe emplear con cada filtro: 

                Tipo de filtro: Avanzado
                Campo A: utm_campaign_uri (sin procesar)
                Extracto A: new_variable = ([^&]*) 
                Campo B: (Ninguno)
                Extracto B: (vacío)
                Enviar resultados a: utm _(old_variable)
                Constructor: $A1
                El campo A es obligatorio: Sí
                El campo B es obligatorio: No
                Sobrescribir el campo de resultados: Sí
                Distinción entre mayúsculas y minúsculas: No
  • old_variable se refiere a la variable de seguimiento predeterminada de Urchin. new_variable se refiere a su propia variable de seguimiento. Recuerde que necesita un filtro para cada variable que modifique.
