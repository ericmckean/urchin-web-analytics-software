Cómo crear entradas personalizadas para un idioma

  1. Copie el archivo del idioma xx-XX.txt de la carpeta <Urchin>/lib/admin/languages/ a <Urchin>/lib/custom/languages/.

  2. Añada nuevas entradas al final del archivo y asigne un nuevo número a cada una de ellas.

  3. Separe el número del valor mediante un <Tab> para que Urchin pueda realizar el análisis.

  4. Asegúrese de que los números de entrada sean consecutivos (con incrementos de uno).
  5. Vuelva a cargar Urchin.

  6. A continuación, Urchin debería crear un archivo DB de idioma con el nombre xx-XX.db ubicado en la carpeta <Urchin>/lib/custom/languages/.

Cuando estén definidas, las entradas personalizadas pueden utilizarse en la sección de informes como columna, etiquetas, títulos o descripciones de ayuda.

Ejemplo de un archivo de idioma

Las entradas 738, 739 y 740 son entradas personalizadas añadidas. 

  ...
  731     Executive
  732     Marketer
  733     Webmaster
  734     Dashboards
  735     All Reports
  736     ja-jp-mac (Japanese)
  737     ja-jp-osaka (Japanese)
  738     Modules
  739     Actions
  740     Modules / Actions

Nota importante: la actualización de Urchin puede sustituir los archivos de idioma originales y añadir entradas predeterminadas. No olvide volver a crear los archivos de idioma personalizados y actualizar los informes personalizados.
