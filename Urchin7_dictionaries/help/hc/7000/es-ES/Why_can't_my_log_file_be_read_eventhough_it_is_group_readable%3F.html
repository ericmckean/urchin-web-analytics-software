¿Por qué no se puede leer mi archivo de registro aunque sí puedan leerlo los grupos?

El error número "7026-76-*" que produce un archivo de registro ilegible, aunque sí pueden leerlo los grupos, es un error conocido de la funcionalidad setuid/setgid del binario de urchin. Por motivos de seguridad, el motor urchin se ejecuta como el propietario de dicho binario, incluso si se inicia como raíz. Bajo esta condición, la aplicación solo visualiza el número de identificación de grupo exacto, y no la lista de grupos.

Para arreglar este problema debe cambiar los permisos para que pueda leer el archivo cualquier usuario o bien indicar el grupo exacto del usuario de urchin. Esto solo afecta a arquitecturas de tipo UNIX.
