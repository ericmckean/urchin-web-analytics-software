¿Qué debo hacer para realizar el seguimiento de los eventos de Flash?

Urchin le permite realizar el seguimiento de todos los eventos basados en el navegador, incluidos los eventos de Flash y de JavaScript. Para supervisar un evento, active la función urchinTracker de JavaScript con un argumento que especifique un nombre para el evento. Si, por ejemplo, activa 

    javascript:urchinTracker('/homepage/flashbuttons/button1');

cada vez que aparezca ese evento de Flash, se registra como si se hubiera producido una visita de página /homepage/flashbuttons/button1. El argumento debe comenzar con una barra inclinada. Puede organizar los nombres de evento en la estructura de directorios que desee y no es necesario que existan páginas reales con esos nombres de archivo.

Por ejemplo, si desea organizar los eventos de flash por página y por tipo de evento, le recomendamos que lo haga en una jerarquía similar a esta: 

    /homepage/flashbuttons/button1
    /homepage/clips/clip1

Ejemplos de código Flash 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file');");
    }
