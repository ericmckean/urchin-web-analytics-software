¿Cómo puedo realizar el seguimiento de todos los subdominios del sitio en perfiles separados?

  • Autor ivantage Limited, asesor autorizado del software Urchin

El seguimiento de los subdominios en perfiles individuales con Urchin depende de la configuración del servidor web. Es posible configurarlo para registrar archivos de registro separados para todos y cada uno de los subdominios. Urchin podrá procesar así cada archivo de registro del subdominio por separado. Como alternativa, si se crea solo un archivo de registro y la configuración del registro de acceso web permite registrar el nombre de host, incluido el subdominio, se puede construir un filtro Incluir para todos los perfiles del subdominio. El filtro en un perfil incluiría solo tráfico orientado a, por ejemplo, ayuda.ejemplo.co.uk y otro filtro Incluir incluiría, por ejemplo, solo tráfico orientado a www.ejemplo.co.uk.

De esta manera, para realizar un seguimiento de sus subdominios en perfiles separados, aplique un filtro Incluir a cada perfil para incluir solo un nombre de host particular. El primer paso es crear un perfil para cada subdominio. Una vez que lo haya hecho, podrá crear los filtros descritos a continuación.

Por ejemplo, si desea efectuar el seguimiento de sub.ejemplo.es y www.ejemplo.es en dos perfiles separados, aplique los siguientes filtros:

  • Para su perfil sub.ejemplo.es 

    Tipo de filtro: Incluir SOLO patrón
    Campo del filtro: utm_request_hostname (AUTO)
    Patrón del filtro: ^sub\.ejemplo\.es
  • Para su perfil www.ejemplo.es 

    Tipo de filtro: Incluir SOLO patrón
    Campo del filtro: utm_request_hostname (AUTO)
    Patrón del filtro: ^www\.ejemplo\.es
