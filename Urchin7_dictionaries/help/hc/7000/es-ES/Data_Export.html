Exportación de datos

Los botones de exportación de datos de Urchin facilitan la exportación de datos de cualquier informe de Urchin. Esta función es útil para hacer un análisis más detallado de los datos de un informe en una hoja de cálculo, procesador de textos, base de datos, etc.

Para exportar datos de un informe, seleccione el tipo de exportación apropiada basándose en la aplicación que desee usar para trabajar con los datos. Para la importación a bases de datos comunes, utilice el formato de texto separado por tabulaciones.

  • Tabulador: haga clic en el botón "T" para exportar datos en el formato delimitado por tabulaciones.
  • Word: haga clic en el icono de Word para exportar datos en el formato nativo de Microsoft Word.
  • Excel: haga clic en el icono de Excel para exportar datos en el formato nativo de Microsoft Excel.

Impresión: haga clic en el icono de impresora para obtener una vista imprimible de los datos y, a continuación, haga clic en el enlace Imprimir página de esa pantalla para imprimir el informe.
