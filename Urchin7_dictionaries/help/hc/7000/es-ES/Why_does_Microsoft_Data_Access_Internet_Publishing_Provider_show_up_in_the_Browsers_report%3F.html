¿Por qué aparece Microsoft Data Access Internet Publishing Provider en el Informe de navegadores?

Este tipo de navegador suele aparecer si tiene archivos en su sitio que pueden descargarse y que se abren con un producto de Microsoft Office (es decir, archivos Word.doc y archivos Excel .xls). Los complementos que utiliza Internet Explorer para ver este tipo de archivos se registran como Microsoft Data Access Internet Publishing Provider al ver esa página.
