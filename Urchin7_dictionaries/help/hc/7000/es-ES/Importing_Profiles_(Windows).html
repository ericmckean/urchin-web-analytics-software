Importación de perfiles (Windows)

Visión general

Con la función "Importación de perfiles" de Urchin los usuarios que disponen de sistemas que ejecutan Microsoft Internet Information Server pueden configurar perfiles de manera práctica y rápida para cada uno de sus sitios de IIS. Urchin puede leer la configuración de IIS, determinar los sitios web que se ejecutan en el servidor y, a continuación, crear perfiles básicos para cada sitio web que utilice los registros de IIS como fuentes de registros. Posteriormente puede personalizar los perfiles o añadir otros perfiles para los sitios importados.

Instrucciones para importar perfiles

Para empezar a importar perfiles, acceda al sistema de administración de Urchin como administrador y haga clic en "Configuración" (parte superior derecha de la interfaz). Haga clic en el botón "Importar" situado en la parte superior derecha y accederá a la pantalla "Importar perfiles". Aquí puede seleccionar los sitios web que desee importar, si procede. Una vez seleccionados, haga clic en el botón "Importar". Haga clic en "Finalizado" cuando haya terminado.

Recomendaciones

Es aconsejable crear al menos un perfil para cada sitio web en el servidor para obtener una idea general del tráfico que se genera al servidor a través de los Informes de resumen de Urchin. Estos informes proporcionan datos generales de tráfico del servidor, así como una clasificación de cada sitio según distintos parámetros de tráfico. Esto resulta muy útil si es un host y factura según el uso del ancho de banda. Recuerde que los informes de resumen solo muestran datos basados en los perfiles que se hayan configurado; no se incluyen sitios que no dispongan de perfiles activos.
