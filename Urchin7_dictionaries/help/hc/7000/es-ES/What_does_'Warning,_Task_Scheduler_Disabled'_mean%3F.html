¿Qué significa "Advertencia: el programador de tareas está inhabilitado?"

El programador de tareas de Urchin es el servicio responsable de ejecutar la función ejecutable de procesamiento de Urchin. Urchin no puede procesar tareas si el programador está inhabilitado. Motivos habituales para que aparezca el error: Advertencia: el programador de tareas inhabilitado":

  • El servicio urchind no se está ejecutando. Esto puede ocurrir cuando se reinicia el servidor y Urchin no está incluido en las secuencias de comandos de arranque del sistema.
  • (Sólo para Unix) El servicio urchind se está ejecutando, pero pertenece a un usuario diferente al propietario de la distribución de Urchin.

Para determinar si el programador se está ejecutando, realice las siguientes tareas:

  • Abra un shell de comandos y escriba CD para el directorio /ruta/a/urchin/bin/.
  • Escriba: urchinctl status (en Unix, escriba ./urchinctl status).

Si el programador no se está ejecutando, puede iniciarlo desde la línea de comandos realizando las siguientes tareas:

  • Abra un shell de comando y escriba CD para el directorio /ruta/a/urchin/bin/.
  • escriba: urchinctl -s start (en Unix, escriba ./urchinctl -s start).
  • si ninguno de los dos servicios se activa, escriba: urchinctl start

Si el servicio se activa, verifique el proceso de ejecución en el servidor para determinar quién es el propietario del proceso de Urchin de la siguiente manera: ps -ef | grep urchind

Observe el "Nombre-usuario" del titular en los datos que aparecen: 

Nombre-usuario 33597 0.0 0.2 352 204 ?? Ss 27Feb08 17:42.18 /usr/local/urchin6/bin/urchind

Si el usuario no es el mismo que el propietario de la distribución de Urchin, deberá detener el servicio y reiniciarlo con el usuario adecuado. Esto se puede lograr "cambiando el usuario" por el que actualmente posee el proceso de urchind y deteniendo la aplicación urchinctl. Utilice la opción"-s" como se describe más arriba. A continuación, vuelva a cambiar al usuario de Urchin e inicie el urchinctl, utilizando nuevamente la opción "-s".
