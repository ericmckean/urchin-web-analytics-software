Seguimiento de transacciones seguras en tiendas

Autor: ivantage Limited, asesor autorizado del software Urchin

Urchin permite realizar el seguimiento de la información del carrito de la compra del dominio o de otros dominios con tan solo añadir un fragmento de código. Si desea configurar las páginas para informar a Urchin sobre los datos de transacción, consulte ¿Cómo se realiza el seguimiento de las transacciones de e-commerce?

Además, si la tienda online se encuentra en un dominio o subdominio distinto del sitio web principal, siga las instrucciones ofrecidas a continuación para configurar las páginas.

Si el proceso de formalización de compra se inicia en un subdominio distinto, por ejemplo, dirija a los usuarios de www.mitienda.es a carritocompra.misitio.es:

  • Añada la siguiente línea (en negrita) al código de seguimiento tanto de las páginas de la tienda online como del carrito de la compra: 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

Si el proceso de formalización de compra se inicia en un sitio distinto, por ejemplo, dirija a los usuarios de www.mitienda.es a www.carritoseguro.es:

  • Añada las siguientes líneas (en negrita) al código de seguimiento tanto de las páginas de la tienda online como del carrito de la compra: 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • Cambie los enlaces que dirigen al sitio web principal por enlaces que llevan al sitio web seguro para utilizar utmLinker tal y como se indica a continuación. Si los enlaces actuales son parecidos a estos: 

     <a href="https://www.securecart.com/?store=parameters">Comprar ahora</a>
  • cámbielos por: 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">Comprar ahora</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">Comprar ahora</a>
   </noscript>

  • El código anterior ofrece enlaces tanto para usuarios que tienen habilitada la función de JavaScript como los que no. Tenga en cuenta que es necesario convertir los apóstrofos en caracteres de escape con una barra invertida allí donde aparezcan en el enlace o en el texto del mismo. Importante: si las páginas incluyen la invocación de urchinTracker(), utmLinker(), utmSetTrans() o utmLinkPost(), el código de seguimiento de Analytics debe insertarse en el código HTML encima de dichas invocaciones. En estos casos, se puede colocar el código de seguimiento en cualquier lugar entre la etiqueta <body> de apertura y la invocación de JavaScript.

Si envía información al carrito de la compra mediante formularios, utilice la función utmLinkPost() que añade información de la cookie a la URL del formulario facilitado. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • Funciona incluso en formularios con method="GET".

En estos casos, se puede colocar el código de seguimiento en cualquier lugar entre la etiqueta de <body> apertura y la invocación de JavaScript.
