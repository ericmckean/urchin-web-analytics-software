Filtros avanzados

Introducción

La opción "Filtros avanzados" le permite crear campos para ofrecer información de uno o dos campos existentes. Las expresiones regulares POSIX y las variables correspondientes se pueden utilizar para obtener campos completos o parte de ellos y combinar el resultado en el orden deseado.

Utilización de filtros avanzados

En la ilustración anterior, el Filtro avanzado utiliza un máximo de dos campos: "Campo A" y "Campo B" y crea el "Campo de resultados". La creación se produce del siguiente modo: la expresión "Extracto A" se aplica al "Campo A" y la expresión "Extracto B" se aplica al "Campo B". Estas expresiones pueden emplear concordancias de texto tanto parcial como total y también incluir comodines. La siguiente es una lista de los comodines más habituales y sus significados. Las expresiones se rigen por expresiones regulares POSIX. 

.   coincide con cualquier carácter individual
*   coincide con cero o más elementos anteriores
+   coincide con uno o más elementos anteriores
?   coincide con cero o uno de los elementos anteriores
()  recuerda el contenido del paréntesis como elemento
[]  coincide con un elemento de la lista
-   crea un intervalo en una lista
|   o
^   coincide con el inicio del campo
$   coincide con el final del campo
\   usa caracteres de escape con cualquiera de los elementos anteriores

Emplee el paréntesis () para obtener parte de los campos. Se pueden indicar en el Constructor utilizando la notación $A1, $A2, $B1, $B2. "A" o "B" hace referencia al campo, mientras que el número indica el paréntesis que hay que obtener. En el ejemplo anterior, se obtienen el "Campo A" y el "Campo B" íntegros y se unen para formar el nuevo campo. El campo "Enviar resultados" puede ser un campo distinto o el mismo que el "Campo A" o "Campo B".

Controles

El control 'Sobrescribir el campo de resultados' le permite decidir qué hacer si ya existe el campo de resultados. La opción 'El campo x es obligatorio' le permite decidir qué hacer si una de las expresiones no coincide.
