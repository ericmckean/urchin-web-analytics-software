¿Qué debo hacer para realizar el seguimiento de los clics en los enlaces salientes?

Si dispone de enlaces salientes en el sitio web, existe una forma muy sencilla de efectuar el seguimiento de los enlaces en los que los usuarios hacen clic para salir de su sitio web.

Para efectuar un seguimiento los clics salientes, añada el siguiente código dentro de la etiqueta <a>: 

    <a href="http://www.site.com" onClick="javascript:urchinTracker('/outboundlinks/sitename/pagename');">

Este código permite que cada clic que se haga en el enlace saliente se registre como si fuera una visita con el nombre /outboundlinks/sitename/pagename. Sería conveniente registrar todos los enlaces salientes en una estructura de directorios lógica, tal y como se indica en el ejemplo. De este modo, podría identificar fácilmente las páginas en las que el usuario ha hecho clic para salir del sitio web.
