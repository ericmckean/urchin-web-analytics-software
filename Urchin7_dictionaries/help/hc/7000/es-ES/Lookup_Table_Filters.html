Filtros "Tabla de consulta"

El filtro "Tabla de consulta" puede utilizarse para:

  • Implementar los códigos maestros de seguimiento para realizar un seguimiento de una campaña. Obtenga información sobre Cómo utilizar códigos maestros de seguimiento.

  • Implementar una tabla de datos externa para buscar y reemplazar valores de campos con caracteres cuando se produce una concordancia. Las Tablas de consulta pueden corresponderse con un único campo y actualizar varios campos.

Utilización del filtro:

  1. En la pantalla "Asistente para filtros:Configuración", introduzca el nombre del filtro que desee (campo "Nombre del filtro").
  2. Seleccione "Tabla de consulta" como tipo de filtro.
  3. Seleccione el campo que desee actualizar en el menú desplegable Campo de filtro.

  4. Seleccione una tabla preconfigurada (obtenga información sobre Cómo utilizar códigos maestros de seguimiento) del menú desplegable "Nombre de tabla".
