Urchin 6.602 Geo-databases

Urchin 6.602 se suministra con dos bases de datos de información geográfica:

  • geo.db: contiene información detallada sobre áreas metropolitanas (country/region/city), redes (empresas) y dominios. Contiene el mismo tipo de información (aunque actualizada) que las versiones anteriores de urchin 6 (6.400 - 6.601).

  • geo.light.db: contiene información de área metropolitana relativa al país únicamente. No incluye información sobre regiones, ciudades, redes o empresas. La memoria de esta base de datos es bastante más reducida y el tiempo de carga también es inferior. Se recomienda utilizarla cuando el rendimiento del motor de procesamiento de registros sea clave. El uso de esta base de datos con contenido geográfico y de versión reducida afecta a los informes siguientes.

    • o En los informes siguientes sobre "visitas por ubicación", se muestran datos de área metropolitana relativos al país únicamente. No se ofrece información sobre la región ni sobre la ciudad.

      • “Vista de ejecutivo” -> “Visión general de ejecutivo” (panel),

      • “Vista de ejecutivo” -> “Gráfico de visitas por ubicación”,

      • “Vista de profesional del marketing” -> “Visión general de profesional del marketing” (panel),

      • “Vista de webmaster” -> “Visión general de webmaster” (panel),

      • “Optimización de marketing” -> “Rendimiento del segmento de usuarios” -> “Gráfico de visitas por ubicación”.

      o El informe "Ubicaciones de red" (“Optimización de marketing” -> “Rendimiento del segmento de usuarios” -> “Ubicación de red”) no muestra datos.

      o El informe "Dominios" (“Optimización de marketing” -> “Rendimiento del segmento de usuarios” -> “Dominios”) no ofrece datos.

La base de datos que se utiliza de forma predeterminada es geo.db. Para cambiar a geo.light.db, elimine el comentario de la línea correspondiente del archivo urchin.conf: 


# Configuración de la base de datos con información geográfica

# Elimine el comentario de la línea siguiente para habilitar la versión reducida de la base de datos con información geográfica.
# GeodbLight: ON
