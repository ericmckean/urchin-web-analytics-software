¿Cómo me recupero del error "No se puede abrir la base de datos para escribir en ella porque se ha archivado"?

Al intentar ejecutar un perfil en Urchin 6, recibe este error: 

WARNING: (7063-54-63) Unable to open database for writing since it has been archived

Este error se produce debido a que, con Urchin, las bases de datos de seguimiento del registro se guardan mensualmente. Durante el procesamiento, Urchin lee el primer par de líneas de un archivo de registro, determina a qué mes pertenece el registro y trata de abrir las bases de datos correspondientes a dicho mes. Dado que ya se han archivado las bases de datos de dicho mes, se devuelve un error, ya que Urchin no permite la lectura/escritura en bases de datos archivadas. Existen dos posibles soluciones:

  1. Inhabilitar la función de archivo, descomprimir los archivos *-archive.zip de los perfiles afectados y eliminar dichos archivos *-archive.zip.
  2. Cambiar los comodines de forma que incluyan una cadena de concordancia de fecha (quizá pueda utilizar YY* o YYYY* para identificar archivos que correspondan solo a este año).
