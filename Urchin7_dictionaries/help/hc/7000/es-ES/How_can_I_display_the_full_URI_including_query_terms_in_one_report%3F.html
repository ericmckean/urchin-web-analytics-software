¿Qué debo hacer para mostrar la URI completa, incluidos los términos de consulta de un informe?

En Urchin 6, los patrones de consulta se muestran en un menú desplegable especial de 'Optimización de contenido -> Rendimiento del contenido -> Términos de consulta de la página'. Si prefiere mostrar la página completa y el patrón de consulta en el informe 'Contenido principal', utilice el filtro siguiente para cambiar la forma en que se visualizan estos datos.

Nota: esto aumenta el tiempo de procesamiento de cualquier perfil al que se aplique. Además, debido al incremento de nombres de página únicos, es posible que se supere el límite de la base de datos.

Cree un 'Filtro de perfiles' nuevo con los siguientes ajustes:

  • Tipo de filtro: Avanzado
  • Filtro A: request_stem (AUTO)- Extracto A (.*)
  • Filtro B: request_query (AUTO) - Extracto B (.*)
  • Enviar resultados a: request_stem (AUTO) - Constructor $A1/$B1
  • Sobrescribir el campo de resultados: 'Sí'
  • Campos obligatorios: 'Ambos campos'
  • Distinción de mayúsculas y minúsculas: 'No'
