¿Por qué tengo tantas entradas "Sin referencia" en mi informe de referencia?

La entrada "Sin Referencia" en el informe de referencia de Urchin normalmente aparece por una o varias de las siguientes razones:

  • Los archivos de registro de su servidor web no presentan un formato correcto.
  • Los usuarios acceden al sitio directamente (introduciendo la URL o incluyendo el sitio en sus marcadores).
  • Los usuarios acceden al sitio directamente haciendo un clic en un enlace incluido en un correo electrónico.
  • Las arañas del motor de búsqueda web identifican las páginas web.
  • Las referencias internas que proceden de las solicitudes de las páginas almacenadas en caché (no aplicable cuando el Supervisor de tráfico de Urchin (UTM) no está correctamente configurado).

Formato del registro: para que Urchin informe sobre las referencias, los archivos de registro deben configurarse de modo que incluyan un campo "Referencia". Consulte Registro - IIS y Apache para más obtener más detalles sobre la configuración de los archivos de registro de Apache o IIS.

Dirección directa: Si un usuario escribe la dirección del sitio web en la barra de direcciones del navegador, no se transferirá ninguna información de referencia al servidor web. Las solicitudes de archivo -o "resultado"- de un servidor web se registran en un archivo y cada entrada tiene un campo de referencia que puede o no contener información. Cuando se accede a una dirección manualmente, como se describe anteriormente, no se registra ninguna información en el campo de referencia porque el usuario no accedió a través de un enlace de otro sitio.

Enlace de correo electrónico: cuando un usuario hace clic en un enlace de un mensaje de correo electrónico (por ejemplo una campaña publicitaria por correo electrónico) puede que no se transfiera ninguna información de referencia. Se pueden crear enlaces de correo electrónico que transfieran la información de referencia al servidor web en el que reside el sitio. Pero esto lo debe configurar la persona que crea el contenido del mensaje.

Arañas web: los rastreadores de robots y las arañas que producen los motores de búsqueda suelen representar en su mayoría resultados "Sin referencia". Todos estos programas visitan sitios web y leen sus páginas y otra información para crear entradas para un índice del motor de búsqueda.

Referencias internas: Urchin no incluye ninguna referencia interna en sus informes si estos se configuran correctamente. Una referencia interna es una referencia del mismo sitio web al que pertenece la primera página solicitada. Si se descubre una referencia interna en el campo de referencia, se la incluirá en el informe como un resultado "Sin referencia". No es común que una referencia interna aparezca en la solicitud inicial de una página web. Pero existe una circunstancia concreta en la que son comunes estas referencias.

Muchos de los sitio ISP y de motores de búsqueda más importantes utilizan servidores almacenados en caché para capturar contenido del sitio web. Al hacerlo, pueden reducir el tiempo y el tráfico de red necesarios para mostrar estas páginas más rápido a los usuarios. Pero, cuando un usuario solicita su página web y un servidor almacenado en caché responde a esa solicitud, su registro de servidor web no advierte esa solicitud. Como resultado, su archivo de registro del servidor web tampoco contendrá ninguna información sobre la solicitud. El resultado final de esta acción es que no se mostrará un nuevo usuario a su servidor web hasta que se realice una segunda o tercera solicitud de página. Para ese entonces, se accederá a las páginas que solicite el usuario desde enlaces en otras páginas de su sitio web. Por lo tanto, el campo de referencia en el archivo de registro mostrará sus propias páginas como referencia. Como resultado, urchin añade los resultados de referencia interna al informe "Sin referencia". Urchin 6 puede recuperar los datos del archivo de registro que faltan si configura el Supervisor de tráfico de Urchin (UTM). Para obtener más información, consulte Cómo funciona el Supervisor de tráfico de Urchin (UTM).
