¿Qué debo hacer para localizar mi código de serie?

Existen dos modos de mostrar el código de serie de Urchin.

  1. Consola del administrador
    • Inicie la consola del administrador de Urchin desde cualquier navegador web.

      Haga clic en 'Configuración --> Ajustes --> Licencia.

  2. Ejecute la función de inspector.
    • Abra una ventana de símbolo del sistema y ejecute el comando cd hasta llegar al directory basedir/urchin/util/. Ejecute la función de inspector. En la pantalla se muestra el número de serie junto con otros datos de Urchin. Recuerde que en Unix/Linux, todos los comandos deben ir precedidos de './'
