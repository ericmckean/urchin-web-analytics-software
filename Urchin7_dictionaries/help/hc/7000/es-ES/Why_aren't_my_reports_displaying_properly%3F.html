¿Por qué mis informes no se muestran correctamente?

Si tiene problemas para ver los informes, probablemente se deba a que está utilizando una versión anterior de Flash. Para ver los informes correctamente, debe instalar Flash 7.x o una versión superior. Para actualizar el programa, visite el sitio de Macromedia en la dirección http://www.macromedia.com/software/flashplayer/.
