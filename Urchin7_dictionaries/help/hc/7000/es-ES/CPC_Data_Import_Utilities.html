Utilidades de importación de datos de CPC (Urchin 6.5 y versiones superiores)

Urchin 6.5 incluye utilidades nuevas para trabajar con datos de CPC.

urchincpc - La herramienta Importación de datos de CPC de Urchin extrae los datos de CPC mediante el API de marketing basado en búsquedas de Google AdWords y Yahoo! para procesarlos e imprimirlos.

  • Nota: los datos de CPC archivados que se descarguen de AdWords no podrán superar los 250 MB (Urchin 6.501 y versiones superiores).

urchincpcd - El Programador de fuentes de CPC de Urchin (o, simplemente, Programador de CPC) se utiliza para descargar tareas de CPC en función de la programación que defina el usuario (similar al Programador de perfiles).

El Programador de CPC (urchincpcd) se ejecuta en un host local junto con el Programador principal de Urchin (urchind). Este último debe estar en funcionamiento para poder programar las fuentes de CPC. El Programador de CPC comprueba periódicamente la cola de tareas de descarga de datos de CPC y controla su ejecución.

Si hay alguna tarea programada, la herramienta de Importación de CPC (urchincpc) se inicia automáticamente. El Programador de CPC genera procesos de trabajo de forma interna, aunque se los puede invocar directamente a través de la línea de comandos para comenzar a ejecutar las tareas que estén disponibles en la cola.

Los datos de CPC descargados se almacenan en /path/to/urchin/data/cpc/ y el motor de Urchin los recopila de forma automática mientras se procesan los perfiles enlazados.

La utilidad urchinctl administra el Programador de CPC de Urchin. Para utilizar sólo el Programador de fuentes de CPC, utilice la opción "-c". Por ejemplo, 

urchinctl -c start/stop/restart

inicia/detiene/reinicia únicamente el Programador de CPC.

Para programar una descarga inmediata de datos de CPC, vaya a "Configuración" -> "Administrador de fuentes de CPC" -> "Descargar o Editar" -> pestaña "Programación" de una fuente determinada de CPC y haga clic en el botón "Descargar ahora". Recuerde que el Programador de CPC debe estar en funcionamiento para poder procesar la solicitud.

Uso: Programador de CPC - urchincpcd

El Programador de CPC se encuentra disponible en la carpeta bin de la instalación de Urchin 6.5 y versiones superiores. Puede utilizar las opciones siguientes a través de la línea de comandos:

  • -d habilita el modo de depuración. Los resultados registran más información de depuración, que en Windows sólo es compatible con la opción -console. Por ejemplo: 

    •            urchincpcd.exe -console -d

  • -h imprime la información de ayuda y sale del programa.

  • -v imprime el número de versión y sale del programa.

En Windows, proporcione uno de los comandos siguientes como primer argumento:

  • -install para instalar el servicio del Programador de CPC de Windows.

  • -remove para suprimir el servicio del Programador de CPC de Windows.

  • -start para iniciar el servicio del Programador de CPC de Windows.

  • -stop para detener el servicio del Programador de CPC de Windows.

  • -console para iniciar el Programador de CPC en el modo consola.

Uso: herramienta Importación de CPC (urchincpc)

Puede utilizar las opciones siguientes a través de la línea de comandos:

  • -c [id] descarga la fuente de coste indicada; "id" es el identificador de la fuente de CPC.

  • -d habilita el modo de depuración.

  • -h imprime la información de ayuda y sale del programa.

  • -v imprime el número de versión y sale del programa.

Tenga en cuenta que, si el Programador de CPC se inicia sin parámetros o sólo con la opción -d, se procesarán todas las tareas que estén disponibles en la cola y se cerrará el programa cuando ya no queden tareas por procesar.

Descarga de datos de CPC a través del proxy (6.501 y versiones superiores)

Para habilitar la descarga de datos de CPC a través del servidor proxy, edite el archivo urchin.conf, elimine los comentarios de las entradas WebServiceProxyHost, WebServiceProxyPort, WebServiceProxyUsername y WebServiceProxyPassword e indique los valores que desee. 

 ######################################################
# CPC Source Manager Settings
#...
# WebServiceProxyHost, WebServiceProxyHost: Host and port of the HTTP/HTTPS
# proxy CPC Source Manager must use to access external CPC sources.
# WebServiceProxyUsername, WebServiceProxyPassword: username and password for
# proxy that enforces authorization.
######################################################
#WebServiceProxyHost: <proxy host>
#WebServiceProxyPort: <proxy port>
#WebServiceProxyUsername: <username>
#WebServiceProxyPassword: <password>

Para WebServiceProxyHost, puede introducir una dirección IP o un nombre de dominio completo.
