¿Por qué obtengo un error cuando intento volver a ejecutar un perfil después haber fallado?

Problema: No se puede procesar su perfil y se produce el error siguiente: 

------------------------------------------------------
Urchin 6.4.00 (windows) starting: 20080229 15:50:13
------------------------------------------------------
Processing profile: {profileName} (on machineName 5764)

WARNING: (7050-323-271) Starting automatic rollback


WARNING: (7054-323-308) Unable to restore databases since backup does not exist
DETAIL: C:\Program Files\Urchin6\data\reports\{accountName}\{profileName}\200508-backupv6-*.zip

Motivo del error: Este error se produce cuando falla la ejecución del procesamiento y no existe una copia de seguridad para realizar la restauración.

Solución: Elimine todos los datos de informes de los meses afectados y vuelva a procesar dichos meses.
