Registro de la licencia de Urchin

Los usuarios pueden obtener uno de los tres tipos de licencia siguientes:

  • Una licencia normal (anterior a Urchin 6.6) no caduca nunca.
  • Una licencia de USP (Urchin 6.6 o versiones superiores) posee una fecha de caducidad que se establece durante la compra.
  • Con una licencia para la versión de prueba puede disfrutar de las mismas funciones que con una licencia normal o de USP, pero no permite más de cinco perfiles y cinco fuentes del registro.

Obtención y activación de una licencia para la versión de prueba

Para obtener una licencia para la versión de prueba, acceda a la interfaz de administración de Urchin con un navegador web (normalmente se accede mediante la URL http://your.server.com:9999). Haga clic en "Obtener la licencia de la demostración" y acepte el acuerdo de licencia para completar el proceso.

Cuando haya obtenido su licencia para la versión de prueba, accederá a una pantalla que le mostrará los detalles referentes a esta. Entre ellos se incluyen: el tipo de licencia, el código de serie, el número de perfiles, las fuentes del registro paralelas, los registros por perfil y el sistema operativo. Siempre es buena idea guardar los datos de la licencia, especialmente el código de serie para futuras referencias. Cuando haya instalado Urchin, podrá ver los detalles de la licencia (el estado de la licencia, la fecha de caducidad de la licencia, el código de serie y la versión) accediendo a Página principal->Configuración->Ajustes->Licencia.

Cuando revise estos datos, active la licencia para la versión de prueba. Haga clic en el enlace 'Instalar la licencia', en la pantalla 'Información de la licencia'. Se abre un asistente. Es recomendable que sustituya la contraseña de administración predeterminada por otra nueva.

NOTA: al seleccionar las opciones correspondientes a Configuración de acceso remoto y Configuración del panel de control del administrador, asegúrese de seleccionar la opción adecuada para Configuración del panel de control del administrador, que le permite otorgar niveles de derechos de acceso como administrador a uno o a varios perfiles de un grupo. No podrá deshacer este ajuste cuando haya finalizado la instalación.

Cuando haya finalizado correctamente la activación de la licencia para la versión de prueba de Urchin, podrá acceder a la pantalla de la página principal de administración del programa. Desde esta pantalla, podrá crear perfiles, procesar registros, ver informes, establecer preferencias, etc.

Cuando caduca la licencia para la versión de prueba, aparece la pantalla Licencia de Urchin con el estado Licencia válida - versión de prueba (caducada). Para instalar una licencia preadquirida, haga clic en el enlace 'Activar licencia'.

Cuando la licencia para la versión de prueba deja de ser válida, aparece la pantalla Licencia de Urchin con el siguiente estado de licencia: Licencia no válida. Para volver a activar la versión de prueba, haga clic en el enlace 'Reactivar licencia' o en 'Activar licencia' para instalar una licencia preadquirida.

Adquisición de una licencia normal

Póngase en contacto con un asesor autorizado del software Urchin para comprar una licencia.

Activación de una licencia normal adquirida

Al recibir su nuevo código de serie para activar la licencia permanente de Urchin, deberá proceder de la siguiente manera:

  • Diríjase a Configuración -> Ajustes -> Licencia

  • Haga clic en el enlace Activar licencia
  • En la pantalla 'Activar código de serie', introduzca una dirección de correo electrónico válida y su código de serie.

Una vez realizado el proceso, podrá utilizar todas las funciones de Urchin. La licencia durará indefinidamente.

Importante: asegúrese de introducir los datos de contacto correctos para obtener o activar su licencia para la versión de prueba, ya que esta información será necesaria para procesos de facturación y servicios de asistencia. Cuando haya instalado Urchin, podrá ver los detalles de la licencia (el estado de la licencia, la fecha de caducidad de la licencia, el código de serie y la versión) accediendo a Página principal->Configuración->Ajustes->Licencia.

Si una licencia normal deja de ser válida, aparece la pantalla Licencia de Urchin con el siguiente estado de licencia: Licencia no válida. Para volver a activar una licencia normal, haga clic en el enlace 'Reactivar licencia' o haga clic en 'Activar licencia' para instalar una licencia preadquirida de USP.

Instalación de una licencia sin acceso a Internet

Es posible registrar una licencia de Urchin sin disponer de acceso a Internet, por ejemplo, tras un cortafuegos. Para ello, debe ejecutar la función "inspector" de Urchin, que encontrará en el directorio "util". (Para obtener instrucciones sobre cómo ejecutar la función "inspector", consulte inspector: Verificador de integridad de la instalación de Urchin.) Cuando tenga el resultado de "inspector", póngase en contacto con su asesor autorizado del software Urchin para que le proporcione las instrucciones para la activación manual.

Adquisición de una licencia de USP

Póngase en contacto con un asesor autorizado del software Urchin para comprar una licencia.

Activación de una licencia de USP adquirida

Al recibir su nuevo código de serie para activar la licencia permanente de Urchin, deberá proceder de la siguiente manera:

  • Diríjase a Configuración -> Ajustes -> Licencia

  • Haga clic en Activar licencia
  • En la pantalla 'Activar código de serie', introduzca el código de serie.

Cuando haya realizado estos pasos, podrá utilizar todas las funciones de Urchin hasta que caduque la licencia.

Si la licencia de USP caduca, póngase en contacto con un asesor autorizado del software Urchin para comprar una licencia.

Actualización a Urchin 6.6

No se recomienda actualizar las copias de Urchin que utilicen una licencia para la versión de prueba, ya que puede generar problemas de acceso o de licencia. No se deberían realizar cambios de licencia tras la actualización si la versión anterior de Urchin utilizaba una licencia normal en Windows. Se debería reactivar la licencia tras una actualización si la versión anterior de Urchin utilizaba una licencia normal en Linux/FreeBSD (haga clic en el enlace 'Reactivar licencia' en la pantalla Licencia de Urchin).

Si una licencia normal deja de ser válida (en cualquier plataforma), aparece la pantalla Licencia de Urchin con el siguiente estado de licencia: Licencia no válida. Para volver a activar una licencia normal, haga clic en el enlace 'Reactivar licencia' o haga clic en 'Activar licencia' para instalar una licencia preadquirida de USP.
