¿Cuál es el número máximo de objetivos que puede tener un perfil?

Cada perfil puede tener hasta cuatro objetivos. Si desea realizar el seguimiento de más de cuatro objetivos para una propiedad web, deberá configurar perfiles duplicados para ese sitio. Para ello, haga clic en el enlace "Crear perfil adicional para un sitio web existente" en el área Administración de la cuenta.

Si su sitio es muy frecuentado, debe ser consciente de que cada perfil procesa las visitas de página de forma individual y de que estas visitas de página duplicadas se incluyen en el recuento total.
