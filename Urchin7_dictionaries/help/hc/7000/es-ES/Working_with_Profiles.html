Uso de perfiles

Visión general

Un perfil es un conjunto de informes para un sitio web. Es necesario configurar al menos un perfil por sitio. En ocasiones, es posible que desee configurar más de un perfil por sitio, pero en términos generales, la proporción entre perfiles y sitios es 1:1.

En términos más técnicos, un perfil es un conjunto de reglas que determinan el procesamiento de archivos de registro y la creación de informes de un sitio web o de las partes que lo componen.

Los perfiles son herramientas organizativas fundamentales para el sistema Urchin, por eso es importante entender cómo funcionan.

Información de la licencia

La licencia básica de Urchin 6 incluye 1.000 perfiles, cada uno de los cuales puede tener una fuente del registro. Un perfil puede compartir más de una fuente del registro (útil cuando un perfil es "maestro", y otros tienen filtros aplicados).

Cada sitio web debe tener al menos un perfil asociado a él para obtener informes. Normalmente uno es suficiente, pero no hay límite en el número de informes que se pueden crear para un análisis especializado de una propiedad web única.

Modo de utilización de perfiles

Para crear un perfil, acceda a la interfaz de administración de Urchin como administrador de Urchin y haga clic en el enlace Configuración que se encuentra en el área superior derecha de la pantalla. Accede al Asistente para añadir perfiles, que le ofrece una serie de pasos sencillos encaminados a ayudarle a configurar el perfil de forma rápida y sencilla. En cada pantalla del asistente se incluye información de ayuda explícita.

Recomendaciones

  • Para obtener informes de calidad óptima con Urchin, es importante activar el Urchin Traffic Monitor (UTM), especialmente si se desea información de usuario único. Hay tres pasos básicos necesarios para utilizar el UTM:
    1. En la pestaña "Ajustes" de cualquier perfil, seleccione "Urchin Traffic Monitor (UTM)" como método de seguimiento de usuarios.
    2. Instale el sensor de UTM en todas las páginas de las que desee realizar un seguimiento con la mayor precisión. (Si no utiliza el UTM, se realizará el seguimiento de las páginas; sin embargo, no se informará de algunas acciones como recargas o el uso del botón "Atrás".)
    3. Modifique el comportamiento de registro de su servidor web para activar las cookies.
  • Si intenta configurar filtros junto con su perfil, se recomienda que disponga de más de un perfil para dicho sitio web o las partes que lo componen. Se recomienda que disponga de un perfil "maestro" -- que lo contenga todo. Si, por ejemplo, desea filtrar arañas o robots, puede ser una buena idea poner estos filtros en un segundo perfil. Esto le ayuda a comparar los resultados de los perfiles de datos filtrados con el perfil maestro.
