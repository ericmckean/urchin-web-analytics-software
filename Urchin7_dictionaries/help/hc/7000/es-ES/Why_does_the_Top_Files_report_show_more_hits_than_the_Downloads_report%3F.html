¿Por qué el informe Archivos principales muestra más resultados que el informe Descargas?

Es habitual que el informe "Archivos principales" muestre un mayor número de resultados que el informe "Descargas" del mismo archivo. A continuación, le indicamos los motivos. En el caso de archivos PDF, muchos navegadores incluyen un complemento que muestran una página a la vez. La solicitud de la página inicial tiene como resultado un código de estado 200 del servidor web. Todas las páginas subsiguientes tienen como resultado un código de estado 206 (descarga parcial) ya que el complemento del navegador solo descarga la página que está viendo y no el documento entero. Por lo tanto, en el caso de que vea cinco páginas PDF del mismo documento, el servidor web registra un resultado con un código de estado 200 y cuatro resultados con un código de estado 206, todos registrados como el mismo documento PDF.

A continuación, le indicamos cómo se muestra esta información en los informes de Urchin:

Descargas: un resultado de descarga requiere un código de estado 200, 302 o 304, de manera que el informe muestre sólo una descarga del archivo PDF, por ejemplo, el único resultado con el código de estado 200.

El informe "Archivos principales" muestra todas las solicitudes de archivo correctas en las que el código de estado es 2XX, 302 o 304 y en las que el archivo concuerda con los tipos de MIME adecuados para que pueda considerarse una visita de página. De manera predeterminada, Urchin trata a todos los tipos de MIME como visitas de página, excepto los archivos con los siguientes sufijos: gif,jpg,jpeg,png,js,css,cur,ico,ida.

De este modo, las cinco solicitudes PDF en el ejemplo anterior se contarían como resultados de este documento en el informe "Archivos principales", ya que cada resultado contiene ya sea un código de estado 200 o uno 206.

Cada resultado de descarga parcial (código de estado 206) cuenta como una visita de página discreta ya que cada resultado corresponde, en mayor o menor medida, a una única página dentro de un documento PDF de varias páginas.
