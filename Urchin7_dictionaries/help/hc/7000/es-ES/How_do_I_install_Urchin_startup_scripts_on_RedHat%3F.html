¿Qué debo hacer para instalar secuencias de comandos de inicio de Urchin en RedHat?

Existe una secuencia de comandos de inicio de Urchin que se crea tras la instalación. Su nombre es "urchin_daemons" y se encuentra en el directorio Urchin util. "/path/to/urchin/util/"

Copie esta secuencia de comandos en el directorio /etc/rc.d/init.d/.

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • Consulte el archivo /etc/inittab y determine el nivel de ejecución predeterminado (normalmente 3). [[Consulte el ejemplo siguiente]]
  • A continuación, cámbielo a /etc/rc.d/rcX.d (donde X es el nivel de ejecución).
  • cd /etc/rc.d/rc3.d
  • Cree enlaces simbólicos que apunten a /etc/rc.d/init.d/urchin_daemons con el nombre S95urchin y K05urchin.
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # Nivel de ejecución predeterminado. Los niveles de ejecución que utiliza RHS son:

  • 0 - detener (NO establezca initdefault en este valor)
  • 1 - Modo de un solo usuario
  • 2 - Multiusuario, sin NFS (igual que 3, si no tiene red)
  • 3 - Modo multiusuario completo
  • 4 - no utilizado
  • 5 - X11
  • 6 - reiniciar (NO establezca initdefault en este valor)
