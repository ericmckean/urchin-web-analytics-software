¿Por qué instala Urchin un servidor web Apache?

Urchin instala un servidor web Apache ligero para proporcionar las funciones de administración basadas en la Web y para la generación de informes dinámicos. Al integrar una versión independiente de Apache con Urchin, no es necesario modificar la configuración del servidor web existente para que sea compatible con Urchin. Además, el hecho de que el servidor web Apache de Urchin se ejecute de manera independiente con respecto al servidor web existente reporta otras ventajas:

  • Urchin no se ve afectado por los cambios que se produzcan en el servidor web estándar.
  • Los servicios de Urchin pueden administrarse sin afectar al servidor web estándar.
  • Es posible utilizar la GUI de administración de Urchin para cambiar determinados parámetros de funcionamiento del servidor web Apache independiente.
  • Las actualizaciones de Urchin pueden ser totalmente independientes y cualquier cambio necesario, como la compatibilidad con SSL, puede ser gestionado exclusivamente por el proceso de actualización de Urchin, sin necesidad de realizar cambios en el servidor web estándar. La instalación de Apache de Urchin no debe afectar a ningún otro servicio web existente en su equipo ni traducirse en un uso importante de los recursos informáticos. Urchin puede configurarse de tal modo que permita el acceso remoto a los informes a través de la Web.
