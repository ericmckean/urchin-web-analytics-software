¿Cómo restablezco los datos de un perfil al realizar una prueba?

Es posible que desee ejecutar un perfil de Urchin varias veces para probar el rendimiento, los filtros u otros elementos, en el que los datos reales generados por Urchin se rechazan tras cada prueba. El modo más sencillo de realizar esto es eliminar el directorio/carpeta en el que se guardan los datos del perfil después de cada ejecución. Estos directorios se guardan según el nombre del perfil en la distribución de Urchin en el área de data/reports. La eliminación de todo el directorio que tiene como nombre el del perfil que se está probando restablece todos los datos, incluida la información de seguimiento de registros.
