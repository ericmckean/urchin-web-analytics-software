Recomendaciones de configuración

Visión general

Una vez que Urchin se haya instalado, hay ciertos parámetros de funcionamiento que se deben configurar. El proceso se realiza mediante un Asistente de configuración que se ejecuta cuando se conecta a la interfaz de administración de Urchin por primera vez, y durante los primeros pasos, en la creación de los perfiles. Esta primera configuración incluye:

  • registro de la licencia de Urchin,
  • configuración del administrador para el acceso remoto a los informes y a los sistemas de administración, así como establecer el modo de centro de datos,
  • configuración de la contraseña de la cuenta de administrador de Urchin,
  • programación de tareas para cada uno de sus perfiles para procesar datos,
  • administración de registros.

Procedimiento

Acceda a la interfaz de administración de Urchin. En sistemas Windows, vaya a “Inicio->Programas->Urchin->Administración de Urchin”. En sistemas UNIX, utilice la URL http://hostname:9999, donde "hostname" es el nombre del host registrado del sistema Urchin. Para instalaciones nuevas, utilice la siguiente información de acceso:

Nombre de usuario: Contraseña de administración: urchin

Accederá a la pantalla de bienvenida del Asistente de configuración de Urchin. Haga clic en “Continuar” para acceder a cada una de las pantallas subsiguientes. Tenga en cuenta que las opciones que elija en esta primera configuración podrá modificarlas en otro momento si así lo desea.

Licencia de Urchin

Siga las instrucciones que se indican en Registro de la licencia de Urchin.

Configuración del administrador

  • Configuración de acceso remoto: seleccione Activar para permitir la conexión remota de cada uno de los navegadores. Si selecciona “Desactivar” para alguno de ellos, el único acceso posible será desde la consola del sistema en el que se encuentra instalado Urchin.
  • Modo de centro de datos: este ajuste determina si Urchin está configurado para permitir la creación de cuentas, lo que permite organizar de forma lógica los perfiles, grupos y usuarios en categorías de acceso restringidas. Si no está seguro, se recomienda configurar la opción en “Activar” ya que no supone ningún coste adicional y permite utilizarlo en el futuro.

Usuario administrativo

  • Restablezca la contraseña para la cuenta de administrador y anote la contraseña en un lugar seguro.

Programación de tareas

Durante la creación de perfiles se le ofrece la opción de programar la hora de ejecución de la tarea que procesa los datos para ese perfil. Le recomendamos verificar la configuración de cada perfil para asegurarse de que la hora definida para la tarea no entre en conflicto con la disponibilidad de los datos del registro, para saber el tiempo que se tardará en procesar los datos y definir cuándo desea que los informes actualizados estén disponibles para los usuarios.

Administración de registros

Urchin incluye un módulo de supervisión de registros para saber la cantidad de datos procesados en cada registro. Por lo tanto, no es necesario que la rotación del archivo de registro esté coordinada con el funcionamiento de Urchin. Sin embargo, en “Origen del registro”, “Configuraciones avanzadas”, Urchin ofrece la función de automatización para la rotación o eliminación del registro.

Consulte Administración de registros para obtener más información al respecto.
