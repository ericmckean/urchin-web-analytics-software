Seguimiento de anuncios y enlaces salientes

Urchin proporciona una forma sencilla de efectuar el seguimiento de los clics en anuncios de banner u otros enlaces salientes. Dado que estos enlaces no llevan a páginas del sitio que contienen el código UTM JavaScript, debe etiquetar el enlace con el código urchinTracker JavaScript si desea efectuar el seguimiento de estos clics. Este fragmento de JavaScript asigna una visita de página a cualquier clic que se hace en un enlace; la visita de página se atribuye al nombre de archivo que especifique.

Para un archivo GIF animado u otro tipo de anuncio de banner estático, añada el código JavaScript urchinTracker a la etiqueta <a> de su banner: 

    <a href="http://www.ejemplo.es/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
