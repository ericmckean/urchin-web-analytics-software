¿Qué variables de UTM debo utilizar para las pruebas A/B?

Las pruebas A/B son útiles para probar diferentes versiones de correos electrónicos y anuncios. Utilice utm_source, utm_medium, utm_content y utm_campaign para diferenciar enlaces de distintos anuncios y correos electrónicos, y para diferenciar los numerosos enlaces dentro de cada anuncio y de cada correo electrónico.

El siguiente ejemplo muestra el uso de las variables UTM en un boletín informativo por correo electrónico. En el ejemplo, el boletín contiene varios enlaces a una única página del sitio web. Tenga en cuenta que el utm_content es el único valor que difiere entre los dos enlaces.

El primer enlace es de texto: 

    http://www.ejemplo.es/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buytext&utm_campaign=brand_x

El segundo es una imagen con un hiperenlace: 

    http://www.ejemplo.es/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buypict&utm_campaign=brand_x

Utilice la herramienta de creación de URL http://www.google.com/support/analytics/bin/answer.py?hl=es&answer=55578 de Google Analytics para crear sus enlaces. El uso de esta herramienta garantiza que sus enlaces contienen la sintaxis correcta.

Importante: utilice los mismos nombres y la misma ortografía en todas las variables de su campaña.
