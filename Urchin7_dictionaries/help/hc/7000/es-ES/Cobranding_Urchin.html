Asociación de marcas en Urchin

Visión general

Urchin permite asociar una marca en la interfaz de administración, la interfaz de creación de informes y la pantalla de acceso. Para incluir código HTML en la parte superior de la interfaz deben editarse tres archivos. Si se realiza la integración completa de un portal, su servidor de aplicaciones puede mostrar informes de Urchin en varios marcos o en una tabla.

Instrucciones para la asociación de marca

Para incluir su marca en la interfaz, debe editar los archivos siguientes que se encuentran en la instalación de Urchin: 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • Cambie el nombre de los archivos eliminando '.sample' (el nombre de archivo debe ser, por ejemplo, cobrand_admin.tpl).
  • Añada el contenido HTML necesario de la marca

El primer archivo controla el uso de marca compartida de su interfaz de administración, el segundo, el de la de creación de informes. El tercer archivo controla el uso de marca compartida en la pantalla de acceso. El código HTML incluido en estos archivos se coloca en la parte superior de la interfaz de Urchin.
