읽기 가능 그룹에 있는데도 불구하고 내 로그 파일을 읽을 수 없는 이유는 무엇입니까?

읽기 가능 그룹에 있는데도 불구하고 읽을 수 없는 로그 파일에 대해 발생하는 오류 번호 '7026-76-*'은 urchin 바이너리의 setuid/setgid 기능과 관련하여 이미 알려진 문제입니다.보안상의 이유로 urchin 엔진은 루트로 시작된 경우에도 바이너리의 소유자로 실행됩니다.그렇기 때문에 응용 프로그램은 정확한 group-id만 볼 수 있고 그룹 목록은 볼 수 없습니다.

이 문제를 해결하려면 로그 파일에서 사용 권한을 전체 읽기 가능으로 변경하거나 그룹을 urchin 사용자의 정확한 그룹으로 변경해야 합니다.이는 UNIX 계열 아키텍처에만 영향을 줍니다.
