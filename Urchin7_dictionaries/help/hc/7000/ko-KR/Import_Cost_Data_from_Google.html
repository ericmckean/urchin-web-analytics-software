Google에서 비용 데이터 가져오기

Urchin 6.5+

CPC 소스 일정 예약과 전체 CPC 소스 업데이트를 사용하면 자동으로 CPC 데이터를 다운로드할 수 있습니다. 자세한 내용은 CPC 데이터 가져오기 유틸리티 및 CPC 소스를 참조하시기 바랍니다.

Urchin 6.5 이전 버전

프로필을 정기적으로 실행하거나 수동으로 실행하기 전에 다음 단계에 따라 매일 또는 매주 비용 데이터를 다운로드하십시오.

  1. adwords.google.co.kr 사이트에서 Google 애드워즈 계정에 로그인합니다.
  2. '보고서' 탭에서 '보고서 생성'을 클릭합니다.

  3. 보고서 유형 섹션에서 'URL 실적'을 선택한 다음 설정섹션에서 다음과 같이 선택합니다.

    • 보기(시간 단위) - 풀다운 메뉴에서 '일별'을 선택합니다.
    • 기간
      • 캠페인 추적을 위해 데이터를 처음으로 다운로드하는 경우에는 키워드 태깅을 시작한 날짜부터 시작하여 어제 날짜로 끝나는 기간을 입력합니다.
      • 이전 데이터를 이미 다운로드한 적이 있으면 이전 다운로드 종료 날짜의 다음 날부터 시작하여 어제 날짜로 끝나는 기간을 입력합니다.
      • 일별 다운로드(권장)의 경우에는 어제 날짜부터 시작하여 어제 날짜로 끝나는 기간을 입력합니다.
    • 캠페인 및 광고그룹 - 모든 캠페인 및 모든 광고그룹 라디오 버튼을 선택합니다.

    • 고급 설정(선택사항) 섹션에서 ‘열 추가/삭제’를 클릭하고 다음을 선택합니다.

      • '세부정보 수준':
        • 키워드
        • 캠페인
        • 광고그룹
        • 도착 URL
      • '실적 통계':
        • 노출수
        • 클릭수
        • 클릭률
        • 평균 CPC
        • 비용
        • 평균 게재순위
      • '전환 열':
        • (선택사항입니다. Google 애드워즈 계정에서 전환 추적을 사용하는 경우에는 '전환', '전환율', '비용/전환'을 선택합니다. 전환 추적을 사용 설정하지 않은 경우에는 이 단계를 건너뜁니다.)

    • 템플릿, 예약 및 이메일 섹션에서 다음과 같이 지정합니다.

      • 보고서 이름 지정 - 보고서 이름을 입력합니다.
      • 예약 - 이 상자를 선택하면 이 보고서를 다시 정의할 필요가 없습니다. 보고서 형식이 저장되고 선택한 내용에 따라 매일, 매주 또는 매달 자동으로 실행됩니다.
      • 이메일 - 이 상자를 선택하면 보고서가 실행될 때 Google로부터 이메일을 받게 됩니다.
  4. '보고서 생성' 버튼을 클릭합니다.
  5. 보고서가 수동으로 실행되거나 Google에서 자동으로 예약되어 실행된 후에는 '다운로드 센터'에 나타납니다 (페이지 상단의 '다운로드 센터' 링크를 클릭하십시오). 원하는 날짜 또는 주에 대한 보고서를 선택하고 .tsv 파일로 다운로드합니다.
