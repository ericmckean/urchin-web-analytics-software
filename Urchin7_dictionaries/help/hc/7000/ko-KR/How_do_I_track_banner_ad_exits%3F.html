Flash를 사용하고 있어 내 사이트에 하나의 URL만 있습니다.목표 유입경로는 어떻게 설정합니까?

웹사이트에서 Flash 이벤트를 비롯한 브라우저 기반 이벤트를 추적할 수 있습니다.urchinTracker 자바스크립트 함수를 사용하여 Flash 액션에 페이지 파일 이름을 지정하고 적합한 목표 또는 유입경로 단계에서 해당 파일 이름을 입력할 수 있습니다.

이벤트를 추적하려면 이벤트에 대한 이름을 지정하는 인수를 사용하여 urchinTracker() 자바스크립트 함수를 호출하십시오.예를 들어 다음을 호출할 수 있습니다. 

    javascript:urchinTracker('/purchase_funnel/page1.html');

그러면 Flash 이벤트 호출이 발생할 때마다 /purchase_funnel/page1.html 이름 아래 페이지뷰처럼 기록됩니다.인수는 슬래쉬로 시작해야 합니다.

원하는 디렉토리 스타일 구조로 이벤트 이름을 구성할 수 있습니다.urchinTracker에 대한 경로/파일 이름 인수는 웹사이트의 실제 URL을 나타내지 않아도 됩니다.

Flash 코드 예 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }

    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }

    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
