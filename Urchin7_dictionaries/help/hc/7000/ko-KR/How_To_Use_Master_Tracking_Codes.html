마스터 추적 코드의 사용 방법

Urchin에서는 마스터 추적 코드(utm_id)를 사용하여 개별 변수 대신에 링크에 태그를 추가할 수 있습니다.링크의 utm_id를 사용하고 테이블에 각 utm_id의 의미를 정의해 두십시오.예를 들어 다음과 같은 URL 대신, 

http://www.hostsite.com/?utm_source=overture&utm_medium=cpc&utm_campaign=springpromo

다음과 같은 UTM 변수 utm_id를 사용할 수 있습니다. 

http://www.hostsite.com/?utm_id=2

utm_id를 사용하면 수정 및 변경이 쉬운 테이블에 캠페인 추적 변수 값이 지정되므로 웹 사용자들에게 캠페인 추적 변수가 표시되지 않고 태그 추가 시 오류가 줄어듭니다.

마스터 추적 코드를 사용하려면 다음과 같이 하십시오.

  • 테이블에 코드를 정의합니다.
  • 테이블을 프로필에 필터로 적용합니다.
  • 링크에 마스터 추적 코드를 사용합니다..

코드 정의

코드를 정의하려면 다음과 같이 하십시오.

  1. Excel에서 코드를 일련의 캠페인 변수에 매핑하는 테이블을 만듭니다.
    • 파일의 첫 번째 행은 '#Fields:'로 시작하고 이어서 UTM 변수 이름이 임의의 순서로 나와야 합니다. 각 행은 #Fields 항목에 배치한 마스터 추적 코드에 대한 캠페인 변수 설정을 정의합니다.값이 없음을 나타내려면 하이픈(-)을 사용하십시오. 코드에 대한 값이 없는 테이블의 입력란은 생략할 수 있습니다.파일 끝에 비어 있는 줄을 추가합니다.파일의 마지막 줄에는 텍스트가 있어서는 안 됩니다.
  2. Excel 테이블을 Urchin 폴더의 lib/custom/lookuptables 디렉토리에 탭 구분 일반 텍스트 파일로 저장합니다.파일은 확장자를 '.lt'로 하여 저장해야 합니다.

테이블을 프로필에 필터로 적용

테이블을 프로필에 적용하려면 다음과 같이 하십시오.

  1. Urchin에 관리자로 로그인하고 인터페이스 오른쪽 상단에 있는 '구성'을 클릭합니다.
  2. 마스터 추적 코드를 적용할 프로필을 수정합니다.
  3. '필터' 탭에서 '추가' 아이콘을 클릭합니다.
  4. 필터 마법사에서 '필터 유형'의 '조회 테이블'을 선택합니다.
    • 마법사에 '테이블 이름' 입력란이 나타납니다.
  5. '테이블 이름' 드롭다운 목록에서 생성할 테이블 이름을 선택합니다.드롭다운 목록에 테이블이 나타나지 않는 경우 테이블 파일 이름 확장자가 .lt이고 Urchin 폴더의 lib/custom/lookuptables 디렉토리에 저장되었는지 확인하십시오.
  6. '필터 입력란' 드롭다운 목록에서 'utm_id(자동)'을 선택합니다.
  7. '마침'을 클릭합니다.

링크의 코드 사용

조회 테이블의 '입력란' 항목 값을 utm_id 값으로 사용하십시오.예를 들어 위의 이 문서에서 만든 조회 테이블을 사용하여 다음과 같이 링크에 태그를 넣을수 있습니다. 

http://www.hostsite.com/?utm_id=1

추가 설명

utm_id를 링크의 기타 UTM 변수(예: utm_source)와 함께 사용하는 경우 기타 변수의 값은 조회 테이블에 정의된 값으로 덮어써집니다.
