이미 Urchin이 구성되었는데 데이터 센터 모드로 변경하려면 어떻게 해야 합니까?

  • 이 문서는 Urchin 소프트웨어 공인 컨설턴트인 ROI Revolution이(가) 작성했습니다.

데이터 센터 모드를 이용하면 그룹 내 하나 이상의 프로필에 다양한 수준의 관리 권한을 추가할 수 있습니다. 이미 최초 설정 과정을 완료했으며 데이터 센터 모드를 사용하려는 경우 MySQL이나 PostgreSQL에서 쉽게 설정할 수 있습니다.

MySQL: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

PostgreSQL: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
