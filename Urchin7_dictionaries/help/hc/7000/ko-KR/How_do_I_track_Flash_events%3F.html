Flash 이벤트는 어떻게 추적합니까?

Urchin을 사용하면 Flash 및 자바스크립트 이벤트를 비롯한 모든 브라우저 기반 이벤트를 추적할 수 있습니다.이벤트를 추적하려면 이벤트에 대한 이름을 지정하는 인수를 사용하여 urchinTracker 자바스크립트 함수를 호출하십시오.예를 들어 다음을 호출할 수 있습니다. 

    javascript:urchinTracker('/homepage/flashbuttons/button1');

위와 같이 호출하면 해당 Flash 이벤트가 발생할 때마다 마치 /homepage/flashbuttons/button1의 페이지뷰인 것처럼 기록됩니다.인수는 슬래시로 시작해야 합니다.원하는 디렉토리 스타일 구조로 이벤트 이름을 구성할 수 있으며, 해당 파일 이름을 가진 실제 페이지가 존재하지 않아도 됩니다.

예를 들어 flash 이벤트를 페이지, 유형별로 구성하려는 경우 다음 행과 함께 계층을 구성할 수 있습니다. 

    /homepage/flashbuttons/button1
    /homepage/clips/clip1

Flash 코드 예 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file');");
    }

    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }

    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file');");
    }
