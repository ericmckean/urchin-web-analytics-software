Urchin 6.602 Geo-databases

Urchin 6.602에는 두 개의 geo databases가 있습니다.

  • geo.db - 자세한 대도시(country/region/city), 네트워크(조직) 및 도메인 정보를 포함하고 있습니다. 이전 버전의 urchin 6(6.400 - 6.601)에서와 동일한 수준이지만 업데이트된 정보를 포함하고 있습니다.

  • geo.light.db - 국가 수준까지만 대도시 정보를 포함하고 있습니다. 지역, 도시, 네트워크 및 조직 정보는 포함되어 있지 않습니다. 이 데이터베이스는 필요한 메모리 공간과 로드 시간이 훨씬 적기 때문에 로그 처리 엔진의 성능이 중요할 때마다 사용하십시오. 이 경량급 geo database는 다음 보고서에 영향을 줍니다.

    • o 다음 "Geo Map" 보고서는 국가 수준까지만 대도시 데이터를 표시합니다(지역 및 도시 정보 없음).

      • 경영진 보기 -> 경영진 개요(대시보드)

      • 경영진 보기 -> Geo Map 오버레이

      • 마케팅 담당자 보기 -> 마케팅 담당자 개요(대시보드)

      • 웹마스터 보기 -> 웹마스터 개요(대시보드)

      • 마케팅 최적화 -> 방문자 세그먼트 실적 -> Geo Map 오버레이

      o "네트워크 위치" 보고서에 데이터가 표시되지 않습니다(마케팅 최적화 -> 방문자 세그먼트 실적 -> 네트워크 위치).

      o "도메인" 보고서에 데이터가 표시되지 않습니다(마케팅 최적화 -> 방문자 세그먼트 실적 -> 도메인).

기본적으로 geo.db가 사용됩니다. geo.light.db database로 전환하려면 urchin.conf 파일에서 해당 행의 주석을 해제하십시오. 


# Geo Database 설정

# 아래 행의 주석을 해제하여 경량급 geo-database를 사용합니다.
# GeodbLight: ON
