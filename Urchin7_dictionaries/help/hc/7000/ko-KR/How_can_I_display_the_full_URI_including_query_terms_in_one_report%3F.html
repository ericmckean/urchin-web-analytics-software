검색어를 포함한 전체 URI를 한 보고서에 표시하려면 어떻게 해야 합니까?

Urchin 6에서는 특수 드릴다운 '콘텐츠 최적화' -> '콘텐츠 실적' -> '페이지 검색어'에 검색 패턴이 표시됩니다.'상위 콘텐츠' 보고서에 전체 페이지 및 검색 패턴을 표시하고자 할 경우 아래 필터를 사용하여 이 데이터의 표시 방식을 변경할 수 있습니다.

참고: 이렇게 할 경우에는 해당 설정이 적용된 프로필의 처리 시간이 길어집니다.또한 고유한 페이지 이름이 많아지기 때문에 데이터베이스 한도를 초과할 수도 있습니다.

다음 설정으로 새 '프로필 필터'를 만드십시오.

  • 필터 유형:고급
  • 필터 A:request_stem(AUTO) - 추출 A(.*)
  • 필터 B:request_query(AUTO) - 추출 B(.*)
  • 출력 대상:request_stem(AUTO) - 생성자 $A1/$B1
  • 출력 입력란 무시 :'예'
  • 필수 입력란:'두 입력란 모두'
  • 대소문자 구분:'아니요'
