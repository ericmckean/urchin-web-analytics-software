'세션 오류 - 응용 프로그램을 호출할 수 없습니다.(4002)...'는 어떤 의미입니까?

Urchin 보고서를 보는 경우 브라우저는 사용자가 요청한 데이터 범위 및 보고서 보기와 일치하는 데이터를 Urchin 데이터베이스 파일에서 쿼리합니다.데이터 쿼리에 대한 응답이 기본값인 45초 이내에 수신되지 않으면 시간초과가 발생하고 '세션 오류 - 응용 프로그램을 호출할 수 없습니다. (4002) 다시 로그인하십시오.' 오류가표시됩니다.이 오류가 발생하면 시간초과 값을 알맞게 늘려야 합니다.값을 늘리려면 명령행에서 Urchin 폴더 파일에 액세스해야 합니다.그렇지 않은 경우 보고서를 제공하는 담당자에게 이 문서를 확인하도록 요청하십시오.

응용 프로그램 시간초과 값을 늘리려면 다음을 수행하십시오.

  • 명령 쉘을 열고 Urchin 서버로 이동합니다.
  • 다음 파일을 엽니다. /path/to/urchin/etc/session.conf

  • 다음 지시문을 찾습니다.APPLICATION_TIMEOUT:45(초)

  • 값을 알맞게 늘리고 파일을 저장합니다.변경 사항은 즉시 적용됩니다.
