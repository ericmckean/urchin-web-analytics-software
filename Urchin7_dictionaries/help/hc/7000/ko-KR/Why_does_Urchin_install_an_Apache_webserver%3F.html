Urchin에서 Apache 웹서버를 설치하는 이유는 무엇입니까?

Urchin에서는 웹 기반 관리 및 동적 보고서 전송용 기능을 제공하기 위해 경량급 Apache 웹서버를 설치합니다.독립형 버전의 Apache를 Urchin과 함께 번들로 제공하기 때문에 Urchin을 지원하기 위해 웹서버의 기존 웹서버 구성을 수정하지 않아도 됩니다.Urchin의 Apache 웹서버가 기존 웹서버와 별도로 실행될 때의 다른 이점도 있습니다.

  • Urchin이 독립 웹서버의 영향을 받지 않습니다.
  • 독립 웹서버에 영향을 주지 않고 Urchin 서비스를 관리할 수 있습니다.
  • Urchin의 관리 GUI를 사용하여 독립형 Apache 웹서버의 특정 운영 매개변수를 변경할 수 있습니다.
  • Urchin 업그레이드를 완벽하게 내부적으로 진행할 수 있으며, 독립 웹서버를 변경하지 않고도 SSL 지원과 같은 필수 변경사항을 Urchin 업그레이드 프로세스에서 단독으로 처리할 수 있습니다.Urchin의 Apache 설치는 컴퓨터에 있는 기존의 다른 웹 서비스에 영향을 주지 않으며 컴퓨터 리소스를 많이 소비하지도 않습니다. 웹을 통해 보고서에 원격 액세스할 수 있도록 Urchin을 구성할 수도 있습니다.
