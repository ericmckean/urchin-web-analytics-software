사용자 및 그룹 작업

개요

Urchin의 사용자 및 그룹 기능을 통해 Urchin 관리자는 사용자 수를 손쉽게 설정하고 해당 보고서에 대해 적절한 권한을 부여할 수 있습니다. 그런 다음 사용자를 그룹화하여 사용자 수가 많은 경우에도 신속하고 간단하게 관리할 수 있습니다. 사용자 그룹에 보고서 액세스 권한이 부여된 경우 해당 그룹 내 모든 사용자는 시스템에 로그인하여 액세스할 수 있습니다. 보고서 액세스 권한이 없는 사용자는 달리 허용되지 않는 한 액세스가 불가능합니다.

사용자 추가 방법

  1. Urchin 시스템에 관리자로 로그인합니다. 참고: Urchin 시스템에 액세스하기 위한 URL은 사용자 유형에 관계없이 동일합니다.
  2. 오른쪽 상단 메뉴의 구성 버튼을 클릭합니다.
  3. 사용자 및 그룹을 클릭합니다.
  4. 사용자 Wizard를 시작하려면 오른쪽 상단의 추가 버튼을 클릭합니다.
  5. 사용자 이름을 선택하고(소문자를 사용하고 공백이 없어야 함) 비밀번호를 정합니다.
  6. 사용자의 실명 입력 – 실명이 Urchin 시스템에 나타납니다. 다음 버튼을 클릭합니다.

  7. 사용자의 제어 수준을 결정합니다. 필요한 경우 AdWords 도구에 대한 액세스 권한을 부여합니다(Urchin 6.601+). 마침을 클릭합니다.

  8. 이 사용자의 사용자 정보, 환경설정 또는 보고서 액세스를 수정하려면 추가 옵션을 클릭합니다.
  9. 보고서 액세스 탭을 클릭합니다. 사용 가능한 프로필이 왼쪽의 상자에 나타납니다.
    1. 하나 이상의 프로필을 선택합니다. 여러 개의 프로필을 선택하려면 플랫폼에 따라 Ctrl 또는 Command 키를 사용하십시오.
    2. 오른쪽 방향 화살표를 클릭하여 프로필을 액세스 권한 부여 상자로 옮깁니다.
    3. 업데이트를 클릭하여 변경사항을 저장합니다.

Urchin 6.602부터 사용자에 대한 홈 페이지 기본 보기를 Urchin 구성 데이터베이스에서 정의할 수 있습니다. 자세한 내용은 전체 보고서를 참조하십시오. AdWords 도구에 대한 액세스 권한은 사용자 설정 -> 환경설정을 통해 해당하는 사용자에게 부여되고 취소될 수 있습니다.

그룹 사용 방법

  1. Urchin에 관리자로 로그인하고 오른쪽 상단에 있는 구성을 클릭합니다.
  2. 사용자 및 그룹을 클릭합니다.
  3. 그룹을 클릭합니다.
  4. 오른쪽 상단의 추가 버튼을 클릭합니다.
  5. 그룹 이름을 입력합니다(그룹을 설명하는 이름이면 어떤 이름이나 상관 없음).
  6. 그룹 설명을 입력합니다. 그룹 설명은 그룹의 위치 또는 구성과 관련이 있을 수도 있습니다.
  7. 마침을 클릭합니다.
  8. 추가 옵션을 클릭합니다.
  9. 그룹에 속한 사용자 탭을 클릭하여 그룹에 추가할 사용자를 선택합니다. 여러 명의 사용자를 선택하려면 플래폼 종류에 따라 Command 키나 Ctrl 키를 사용하십시오.
    1. 오른쪽 방향 화살표를 클릭하여 사용자를 그룹에 속한 사용자 상자로 옮깁니다.
    2. 업데이트를 클릭하여 변경사항을 저장합니다.

권장사항

  • "수퍼 관리자" 수준 사용자가 Urchin 시스템 전체를 통제하는 경우 한 사람에게만 이 권한을 부여하는 것이 좋습니다.
  • 비밀번호는 하나 이상의 대문자 및 부호를 포함시켜 추측하기 어렵게 만들어야 합니다.
