Urchin 공동 브랜딩

개요

Urchin은 관리 인터페이스, 보고 인터페이스 및 로그인 화면에 공동 브랜딩을 사용합니다.인터페이스 상단에 HTML을 포함하도록 수정할 수 있는 3개 파일이 있습니다.전체 포털 통합이 완료된 경우 응용 프로그램 서버에 의해 프레임셋 또는 테이블 안으로 Urchin 보고가 전달될 수 있습니다.

공동 브랜딩 도움말

인터페이스를 공동 브랜딩하려면 Urchin 설치 폴더에서 다음 파일을 수정해야 합니다. 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • '.sample'(파일 이름은 cobrand_admin.tpl 형태여야 함)을 삭제하여 파일 이름 변경
  • 필요한 경우 이러한 파일에 브랜딩을 포함하는 HTML 콘텐츠 추가

첫 번째 파일은 관리 인터페이스에서 공동 브랜딩을 제어하고 두 번째 파일은 보고 인터페이스에서 공동 브랜딩을 제어합니다.세 번째 파일은 로그인 화면에서 공동 브랜딩을 제어합니다.이러한 파일에 제공되는 HTML은 Urchin 인터페이스 상단에 배치됩니다.
