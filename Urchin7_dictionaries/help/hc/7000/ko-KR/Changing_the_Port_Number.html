포트 번호 변경

Urchin 웹서버가 수신하는 기본 포트 번호는 9999입니다.이 번호를 변경하려면 두 가지 기본 단계가 필요합니다.

  • '서버 설정' 화면에서 포트 번호 변경
  • Urchin 서비스를 중지한 후 시작(이 프로세스는 Windows 시스템과 Unix 시스템에서 서로 약간 다름)

세부적인 프로세스는 다음과 같습니다.

  • Urchin 관리 인터페이스에 로그인합니다.

  • '구성'->'글로벌 설정'->'액세스 설정'으로 이동합니다

  • '서버 포트 번호' 상자에서 새 포트 번호를 설정합니다.
  • '업데이트' 버튼을 클릭합니다.

이제 Urchin 서비스를 다시 시작해야 합니다.

  • Unix 계열의 시스템에서는 Urchin 폴더의 bin 디렉토리로 이동하고 다음을 실행합니다. 

          ./urchinctl restart

  • Windows 시스템에서는 콘솔에서 '시작'->'프로그램'->'Urchin 6'으로 이동하고 'Urchin 6 서비스 사용 안 함'을 선택한 후에 'Urchin 6 서비스 사용'을 선택합니다.

그러면 웹서버가 새 포트 번호에서 연결 요청을 수신합니다.즉, 보고서를 보고 Urchin 소프트웨어를 구성하는 데 사용되는 URL이 변경되었기 때문에 사용자에게 새 URL을 알려줘야 합니다.

참고 많은 시스템에서는 1024보다 낮은 숫자의 포트 번호를 사용하려는 경우 루트 권한이 필요할 수도 있습니다.또한 지정된 포트에서 다른 서비스가 이미 실행 중인 경우에는 Urchin이 시작되지 않습니다.
