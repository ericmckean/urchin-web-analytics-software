브라우저 보고서에 Microsoft Data Access Internet Publishing Provider가 표시되는 이유는 무엇입니까?

일반적으로 이 브라우저 유형은 사이트에 Word .doc 파일 및 Excel .xls 파일 같은 Microsoft Office 제품에서 열리는 다운로드 가능한 파일이 있는 경우 표시됩니다.Internet Explorer에서 이러한 파일 유형의 보고서를 보기 위해 사용하는 플러그인은 해당 페이지를 볼 때 Microsoft Data Access Internet Publishing Provider로 보고됩니다.
