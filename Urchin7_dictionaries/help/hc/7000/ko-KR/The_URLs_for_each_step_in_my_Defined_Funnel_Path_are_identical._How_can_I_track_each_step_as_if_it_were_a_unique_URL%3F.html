정의된 유입경로에서 각 단계의 URL이 똑같을 경우각 단계를 하나의 고유한 URL처럼 추적하려면 어떻게 해야 합니까?

모든 단계에 동일한 URL이 사용되는 유입경로에서 방문자의 경로를 추적하는 경우가 있습니다.예를 들어 가입 경로는 다음과 같습니다.

  • 1단계(가입)
    • www.mysite.com/sign_up.cgi
  • 2단계(동의)
    • www.mysite.com/sign_up.cgi
  • 3단계(마침)
    • www.mysite.com/sign_up.cgi

이 문제를 해결하려면 다음과 같이 각 단계(onload 이벤트 내)에서 urchinTracker Javascript 함수를 호출하십시오.

  • urchinTracker("/funnel_G1/step1.html")
  • urchinTracker("/funnel_G1/step2.html")
  • urchinTracker("/funnel_G1/step3.html")

그런 다음 다음과 같이 유입경로의 각 단계를 설정합니다.

  • http://www.mysite.com/funnel_G1/step1.html

  • http://www.mysite.com/funnel_G1/step2.html

  • http://www.mysite.com/funnel_G1/step3.html

urchinTracker()에 대한 경로/파일 이름 인수가 기존 경로 또는 파일 이름을 나타내지 않아도 됩니다.urchinTracker에 대한 인수는 Urchin에서 페이지뷰를 첨부할 수 있는 가공된 페이지 이름만 제공합니다.
