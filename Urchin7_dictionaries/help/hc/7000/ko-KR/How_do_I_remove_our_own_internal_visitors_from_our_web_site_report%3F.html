웹사이트 보고서에서 내부 방문자는 어떻게 삭제합니까?

개요

특정 IP 주소의 IP 주소만 포함되고 내부 모니터링 시스템 등의 IP 주소는 제외되도록 IP 주소 필터를 프로필과 함께 사용하는 경우는 매우 흔합니다.Urchin에서는 이러한 작업을 쉽게 수행할 수 있습니다.그러나 IP 주소 필터를 설정할 때 올바른 '필터 입력란'을 사용하는 것이 중요합니다.

처리 절차

적절한 내부 트래픽 필터를 구성하는 가장 간단한 방법은 다음과 같습니다.

  • Urchin 관리 인터페이스에서 '구성' -> 'Urchin 프로필' -> '필터 관리자'로 이동합니다.

  • '추가' 버튼을 클릭하여 새 필터를 만듭니다.

  • '=> 예제 필터 선택' 드롭다운 메뉴에서 '내부 트래픽 필터링'을 선택합니다.

  • '필터 패턴' 상자에 있는 예제 IP 주소를 필터링할 IP 주소로 대체합니다.IP 주소의 각 점 앞에 역슬래시('\') 문자를 반드시 입력해야 합니다.
  • '마침'을 클릭하고 다음 창에서 '추가 옵션'을 클릭합니다.
  • '사용 가능한 프로필' 상자에서 필터를 적용할 모든 프로필을 선택하고 오른쪽 화살표를 클릭하여 '적용 대상' 상자로 이동합니다.
  • '업데이트'를 클릭하여 필터를 선택한 프로필에 적용합니다.

기타 고려사항

Urchin 필터 패턴은 POSIX 정규식입니다.따라서 IP 주소 속성에서 '.' 문자를 이스케이프하여 Urchin이 IP 주소를 정규식의 메타 문자로 해석하지 않도록 해야 합니다.예를 들어 IP 주소 '172.16.1.1'을 필터링하려면 필터 패턴을 '172\.16\.1\.1'으로 지정합니다.그리고 IP 주소 범위 '192.168.xxx.xxx를 필터링하려면 '필터' 입력란을 '192\.168\.'로 지정해야 합니다. 다음은 몇 가지 추가적인 예입니다.

IP 주소가 여러 개일 경우에는 파이프 기호로 구분됩니다(192\.168\.12\.32$|10\.10\.1\.23$|63\.212\.171\.5$). 여기서 $는 마지막 숫자 이후로 다른 숫자가 있어서는 안 된다는 것을 의미합니다.따라서 63.212.171.5와 63.212.171.53을 모두 포함하려는 경우에는 필터를 63\.212\.171\.5로 작성하면 됩니다.하지만 63\.212\.171\.5만 지정하려면 필터 패턴을 63\.212\.171\.5$로 작성해야 합니다.

IP 주소 범위는 단일 문자열로 표현할 수 있습니다. 192.168.10.10에서 192.168.10.50까지는 192.168.10.[1-5][0-9]$로 작성하면 됩니다.

행 앞에 있을 때는 필터 패턴 63\.212\.171\.5가 163\.212\.171\.5와도 일치하게 됩니다.특정 문자로 시작하는 IP만 포함하려면 '문자열의 시작에 맞추는 식 ''을 사용하십시오(예: 63\.212\.171\.5$).

전체 정규식 목록 

 *'''와일드카드'''   '''의미'''
 *'''.'''       단일 문자를 찾습니다.
 *'''*'''       없거나 하나 이상 있는 이전 항목을 찾습니다.
 *'''+'''       하나 이상의 이전 항목을 찾습니다.
 *'''?'''       없거나 하나 있는 이전 항목을 찾습니다.
 *'''()'''      괄호 안에 있는 콘텐츠를 항목으로 인식합니다.
 *'''[]'''      이 목록의 한 항목에 일치됩니다.
 *'''-'''       목록에서 범위를 생성합니다.
 *'''|'''       또는
 *'''^'''       입력란의 시작에 맞춥니다.
 *'''$'''       입력란의 끝에 맞춥니다.
 *'''\'''       위의 모든 옵션을 이스케이프합니다.
