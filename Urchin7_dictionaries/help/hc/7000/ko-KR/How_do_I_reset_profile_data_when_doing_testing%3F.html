테스트 시 프로필 데이터를 재설정하려면 어떻게 해야 합니까?

성능, 필터 등을 테스트하기 위해서 Urchin 프로필을 여러 번 실행하는 것이 바람직할 수 있습니다. 이 경우 Urchin에서 생성한 실제 데이터는 테스트를 수행한 후 매번 삭제됩니다.가장 간단한 방법은 실행한 후 매번 프로필 데이터가 저장된 디렉토리/폴더를 삭제하는 것입니다.이러한 디렉토리는 Urchin 폴더의 data/reports 영역에 프로필 이름으로 저장되어 있습니다.프로필을 테스트한 후 전체 디렉토리를 삭제하면 로그 추적 정보를 포함한 모든 데이터가 재설정됩니다.
