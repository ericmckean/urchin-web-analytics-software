안전한 상점 거래 추적

이 문서는 Urchin 소프트웨어 공인 컨설턴트인 ivantage Limited에서 작성했습니다.

Urchin의 경우 코드를 몇 개 추가하면 귀하의 도메인뿐만 아니라 다른 도메인에서 쇼핑 바구니의 데이터를 추적할 수 있습니다.페이지에서 Urchin으로 거래 데이터를 보고하도록 설정하려면 '전자상거래는 어떻게 추적합니까?'를 참조하십시오.

또한 상점 사이트가 귀하의 기본 사이트와 다른 하위 도메인이나 도메인에 있는 경우 아래 안내에 따라 페이지를 구성할 수 있습니다.

귀하의 웹사이트가 다른 하위 도메인에서 결제처리 절차를 시작하는 경우(예: www.mystore.com에서 shoppingcart.mystore.com으로 고객을 보내는 경우) 다음 단계를 따르시기 바랍니다.

  • 상점 사이트 및 쇼핑 바구니 페이지의 추적 코드에 진하게 표시된 다음 행을 추가합니다. 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

귀하의 웹사이트가 개별 상점 사이트에서 결제처리 절차를 시작하는 경우(예: www.mystore.com에서 www.securecart.com으로 고객을 보내는 경우) 다음 단계를 따르시기 바랍니다.

  • 상점 사이트 및 쇼핑 바구니 페이지의 추적 코드에 진하게 표시된 다음 행을 추가합니다. 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • 다음과 같이 utmLinker를 사용하려면 기본 사이트에서 보안 사이트로 링크를 변경합니다.현재 링크가 다음과 같이 표시되는 경우, 

     <a href="https://www.securecart.com/?store=parameters">지금 구매</a>
  • 아래와 같이 변경하십시오. 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">지금 구매</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">지금 구매</a>
   </noscript>
  • 위의 코드를 사용하면 자바스크립트 사용 여부에 상관없이 사용자에게 링크를 제공할 수 있습니다.링크나 링크 텍스트에 아포스트로피가 있는 경우 역슬래시를 사용하여 이스케이프 처리해야 합니다.

    중요사항:페이지에 urchinTracker(), utmLinker(), utmSetTrans() 또는 utmLinkPost()에 대한 호출이 포함된 경우 Analytics 추적 코드는 HTML 코드에서 이러한 호출 위에 삽입되어야 합니다.이 경우 추적 코드는 여는 <body> 태그와 자바스크립트 호출 사이에 삽입될 수 있습니다.

일정 양식을 사용하여 정보를 쇼핑 바구니로 보내시려면 utmLinkPost() 기능을 사용하십시오.그러면 쿠키 데이터가 해당 양식의 URL에 추가됩니다. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • 이 사항은 method="GET"인 양식에도 적용됩니다.

    주의사항:페이지에 urchinTracker(), utmLinker(), utmSetTrans() 또는 utmLinkPost()에 대한 호출이 포함된 경우 Analytics 추적 코드는 HTML 코드에서 이러한 호출 위에 삽입되어야 합니다.이 경우 추적 코드는 여는 <body> 태그와 자바스크립트 호출 사이에 삽입될 수 있습니다.
