자바스크립트 이벤트는 어떻게 추적합니까?

이벤트를 추적하려면 이벤트에 대해 이름을 지정하는 urchinTracker 자바스크립트 함수를 사용하십시오.예를 들어 다음을 호출하실 수 있습니다. 

    javascript:urchinTracker('/homepage/link1');

이 경우 호출 이벤트가 발생할 때마다 URL /homepage/link1에 대해 페이지뷰가 기록됩니다.인수는 슬래쉬로 시작해야 합니다.원하는 디렉토리 스타일 구조로 이벤트 이름을 구성할 수 있습니다.

예를 들어 페이지 > 이벤트 유형별로 이벤트를 구성하려는 경우 다음 행과 함께 계층을 구성할 수 있습니다. 

    * '/homepage/links/link1'
    * '/homepage/radiobuttons/button1'
    * '/contactform/rollovers/image1'

HTML 코드 예

다음은 onClick 이벤트의 기록 방법을 보여줍니다. 

    <a href="javascript:void(0);" onClick="javascript:urchinTracker('/folder/file');">

또는 다음과 같이 롤오버(rollover) 이벤트를 기록할 수 있습니다. 

    <a href="javascript:void(0);" onMouseOver="javascript:urchinTracker ('/folder/file');">
