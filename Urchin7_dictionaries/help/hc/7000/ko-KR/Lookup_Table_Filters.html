조회 테이블 필터

조회 테이블 필터는 다음과 같은 용도로 사용할 수 있습니다.

  • 캠페인 추적을 위한 마스터 추적 코드를 구현합니다.마스터 추적 코드의 사용 방법을 참조하시기 바랍니다.

  • 조회할 외부 데이터 테이블을 구현하고 일치하는 항목이 있을 경우 문자 입력란 값을 바꿉니다.조회 테이블은 한 입력란과 일치할 수 있으며 여러 입력란을 업데이트할 수 있습니다.

필터 사용 방법:

  1. '필터 마법사: 설정' 화면에서 원하는 필터 이름을 입력합니다('필터 이름' 입력란).
  2. 필터 유형으로 '조회 테이블'을 선택합니다.
  3. '필터 입력란' 드롭다운 메뉴에서 업데이트할 입력란을 선택합니다.

  4. '테이블 이름' 드롭다운 메뉴에서 미리 구성된 테이블(마스터 추적 코드의 사용 방법 참조)을 선택합니다.
