개별 방문자와 연관된 캠페인

Urchin 6에는 사이트의 캠페인에 대한 상세정보를 보여주는 보고서를 제공합니다.이러한 보고서는 '보고서' -> '마케팅 최적화' -> '마케팅 캠페인 결과'에서 찾아볼 수 있습니다.특정 캠페인을 통해 사이트를 방문한 방문자수를 보려면 다음과 같이 하십시오.

  • '모든 보고서' -> '마케팅 최적화' -> '마케팅 캠페인 결과' -> '캠페인 전환'으로 이동합니다(또는 '소스 전환', '매체 전환', '추천 전환').

  • 특정 캠페인 옆에 있는 '+' 기호를 클릭합니다(예: 'organic').

  • 검색엔진(예: 'google[organic]') 옆에 있는 '분석 옵션' 버튼(http://www.google.com/urchin/images/wiki/Analysis_Options.jpg)을 클릭하고 '방문자 기록 드릴다운' 옵션을 선택합니다.

  • 방문자 ID 중 하나를 클릭합니다.

해당 방문자수 목록과 함께 참조/캠페인 정보가 표시됩니다.
