평균 점수 이해

일부 Urchin 보고서에 평균 점수 통계가 포함됩니다.평균 점수는 사이트에서 가장 가치 있는 페이지를 파악하는 한 방법입니다.평균 점수는 그 자체로는 유용하지 않지만 사이트의 페이지와 링크를 비교하는 데 있어 유용합니다.점수가 가장 높은 페이지는 방문 도중 높은 목표값으로 전환한 사용자들이 가장 많이 방문한 페이지입니다.

페이지의 평균 점수는 다음과 같이 계산됩니다.

페이지 방문으로 이어진 총 전환값 / 총 페이지뷰 수

예

사용자가 사이트에 10번 방문하여 총 100달러의 가치가 있는 한 번의 전환이 발생했다고 가정해 보겠습니다.이때 5회의 방문이 다음 경로를 거칩니다.

홈페이지 --> 페이지 A --> 값이 $100인 목표 페이지

나머지 5회의 방문은 다음 경로를 거칩니다.

홈페이지 --> 페이지 B --> 종료 사이트

각 페이지의 평균 점수는 다음과 같습니다.

홈페이지 평균 점수 = 50(= 5개의 후속 전환* 100달러 / 10개의 총 페이지뷰) 페이지 A의 평균 점수 = 100(= 5개의 후속 전환* 100달러 / 5개의 총 페이지뷰) 페이지 B의 평균 점수 = 0(목표에 도달하지 않았으므로)

점수는 목표 또는 전자상거래로부터 계산할 수 있습니다.

Urchin은 전자상거래와 목표 전환값에서 모두 평균 점수를 계산합니다.전자상거래의 경우 평균 점수를 계산하는 데 사용되는 전환값은 거래 수익에서 직접 가져옵니다.목표의 경우 Urchin에서는 귀하의 목표에 할당된 값을 사용하여 평균 점수를 계산합니다.
