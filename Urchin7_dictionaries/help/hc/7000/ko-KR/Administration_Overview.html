관리 개요

소개

Urchin 관리 인터페이스는 Urchin 실행과 관련된 모든 사항을 실제로 제어할 수 있는 브라우저 기반 명령 센터로, 프로필 설정, 로그 처리 이벤트 예약, 사용자/그룹 관리, 필터 구성 등의 작업이 가능합니다.

시작하려면 브라우저를 사용하여 Urchin 시스템에 로그인합니다.설치하는 동안 기본 포트를 사용한 경우 URL은 http://your.server.com:9999/여야 하며 여기서 'your.server.com'을 Urchin이 실행 중인 실제 서버 이름으로 바꿉니다.또는 해당 시스템에서 직접 실행하는 경우 http://localhost:9999/를 사용할 수 있습니다.Windows 플랫폼의 경우 시작 메뉴에 'Urchin 관리' 바로 가기가 제공됩니다.'admin' 계정의 기본 비밀번호는 'urchin'입니다.이 비밀번호를 보다 안전한 비밀번호로 변경하십시오.

관리 인터페이스 사용

  • 화면 오른쪽 상단의 '구성'을 클릭하여 관리 인터페이스를 엽니다.
  • 이 인터페이스에서 구성 컨트롤에 액세스할 수 있습니다.왼쪽 탐색 화면에서 화살표를 클릭하여 특정 섹션을 펼칩니다(예: Urchin 프로필, 사용자 및 그룹, 스케줄러, 설정 등).어두운 색상은 현재 표시된 화면을 나타냅니다.
  • 특정 항목 옆의 '수정' 버튼을 클릭하여 해당 항목에 대한 구성을 수정할 수 있습니다.새 항목을 추가하려면 테이블 오른쪽 상단의 '추가' 버튼('+' 기호 아이콘)을 클릭하십시오.특정 항목에서 '수정' 버튼을 클릭하면 해당 항목에 대해 사용 가능한 구성 집합 화면이 상단에 탭 페이지로 표시됩니다.
  • 아무 탭이나 클릭하여 해당 탭의 구성 설정에 액세스합니다.설정을 변경한 후에는 반드시 각 화면 하단의 '업데이트' 버튼을 클릭해야 합니다.
  • 특정 영역에 항목에 대한 긴 목록이 있는 경우 항목 목록 테이블 위/아래에 있는 '다음/이전' 버튼을 사용하여 스크롤할 수 있습니다.테이블 목록 위에 표시되는 '#표시' 값(기본값은 10)을 변경하여 테이블에 표시되는 항목 수를 변경할 수 있습니다.
  • + - 필터(포함/제외) 옵션을 통해 특정 항목을 신속하게 찾을 수 있습니다.항목 이름을 모두 또는 일부만 입력하고 '+(포함)' 또는 '-(제외)' 버튼을 누릅니다.
  • 각 화면에 대한 도움말 정보는 테이블 아래에 설명되어 있습니다.
