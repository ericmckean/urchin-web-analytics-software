데이터 내보내기

Urchin의 데이터 내보내기 버튼을 사용하면 Urchin 보고서에서 간편하게 데이터를 추출할 수 있습니다.이 기능은 추가 분석을 위해 보고서 데이터를 스프레드시트, 워드 프로세서 또는 데이터베이스 등으로 가져올 때 유용합니다.

보고서에서 데이터를 내보내려면 데이터 조작에 사용할 응용 프로그램을 기준으로 해당하는 내보내기 유형을 선택하십시오.일반적인 데이터베이스 내보내기에서는 탭으로 구분된 형식을 사용합니다.

  • 탭:탭으로 구분된 형식으로 데이터를 내보내려면 'T' 버튼을 클릭합니다.
  • Word: Microsoft Word 기본 형식으로 데이터를 내보내려면 Word 아이콘을 클릭합니다.
  • Excel: Microsoft Excel 기본 형식으로 데이터를 내보내려면 Excel 아이콘을 클릭합니다.

인쇄:인쇄가 가능한 형식의 데이터를 얻으려면 프린터 아이콘을 클릭하고 화면에서 '페이지 인쇄' 링크를 클릭하여 보고서를 실제로 인쇄합니다.
