RedHat에 Urchin 시작 스크립트는 어떻게 설치합니까?

설치할 때 생성되는 Urchin 시작 스크립트가 있습니다.해당 스크립트의 이름은"urchin_daemons"이고 Urchin util 디렉토리("/path/to/urchin/util/")에 있습니다.

이 스크립트를 /etc/rc.d/init.d/ directory에 복사합니다.

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • /etc/inittab 파일을 보고 기본 실행 수준(일반적으로 3)을 확인합니다.[[아래 참조]]
  • 그런 다음 /etc/rc.d/rcX.d(여기에서 X는 실행 수준)으로 변경합니다.
  • cd /etc/rc.d/rc3.d
  • /etc/rc.d/init.d/urchin_daemons를 가리키는 S95urchin 및 K05urchin 기호 링크를 만듭니다.
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # 기본 실행 수준.RHS에서 사용하는 실행 수준은 다음과 같습니다.

  • 0 - 중지(initdefault는 이 값으로 설정하면 안 됨)
  • 1 - 단일 사용자 모드
  • 2 - 다중 사용자, NFS 없음(네트워킹을 사용하지 않는 경우 3과 동일)
  • 3 - 전체 다중 사용자 모드
  • 4 - 사용 안 함
  • 5 - X11
  • 6 - 재부팅(initdefault는 이 값으로 설정하면 안 됨)
