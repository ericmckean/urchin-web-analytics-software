특정 키워드나 URL을 키워드가 아닌 직접 추천으로 처리하도록 Urchin을 구성하려면 어떻게 해야 합니까?

특정 검색 키워드 또는 URL을 키워드 추천이 아닌 직접 추천으로 처리할 수 있습니다.이를 위해 Urchin이 보고서에 직접 추천으로 보고하도록 urchin.js 파일을 구성할 수 있습니다.

  1. 텍스트 편집기를 사용하여 urchin.js 파일을 엽니다.
  2. 아래 제목으로 스크롤합니다. 

          //-- Auto/Organic Keywords to Ignore
  1. 이 양식 목록 끝에 무시할 새 유기적 키워드를 추가합니다. 

          _uOno[#]="keyword";
  • ..다음 예 참조: 

        //-- Auto/Organic Keywords to Ignore
        var _uOno=new Array();
        _uOno[0]="urchin";
        _uOno[1]="urchin.com";
        _uOno[2]="www.urchin.com";
