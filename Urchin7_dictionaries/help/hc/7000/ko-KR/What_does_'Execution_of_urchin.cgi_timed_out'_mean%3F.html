'urchin.cgi 실행 시간 종료'는 어떤 의미입니까?

이 오류는 Urchin의 보고서 직접 링크 기능이 사용되었거나 Urchin 세션 컨트롤러가 우회된 경우에만 나타납니다.이러한 경우 Urchin 보고서를 보는 도중 사용되는 URL을 확인하여 식별할 수 있습니다.보고서 직접 링크가 사용된 경우 URL에 'report.cgi'가 포함되며 표준 Urchin 세션 컨트롤러가 사용된 경우 URL에 'session.cgi'가 포함됩니다.

보고서 직접 링크를 통해 Urchin 보고서를 보는 경우 브라우저는 사용자가 요청한 데이터 범위 및 보고서 보기와 일치하는 데이터를 report.cgi 응용 프로그램을 통해 Urchin 데이터베이스 파일에서 쿼리합니다.데이터 쿼리에 대한 응답이 기본값인 60초 이내에 수신되지 않으면 CGI 응용 프로그램이 시간 종료되고 브라우저에 'urchin.cgi 실행 시간 종료' 오류가 표시됩니다.이 오류가 발생하면 CGI 응용 프로그램 시간초과 값을 알맞게 늘려야 합니다.이 시간초과 값을 변경하려면 명령행에서 Urchin 폴더에 액세스해야 합니다.

보고서 직접 링크에 대한 응용 프로그램 시간초과 값을 늘리려면 다음을 수행하십시오.

  • 명령 쉘을 열고 Urchin 서버로 이동합니다.
  • 그런 다음/path/to/urchin/htdocs/.report.conf 파일을 수정합니다. 이 파일이 없으면 .report.conf.sample 파일을 .report.conf로 복사하여 하나 만듭니다.
  • "#Timeout = 60" 행을 찾습니다.
  • 맨 앞의 '#' 문자를 삭제하여 행의 주석을 해제합니다.
  • 기본값 60초를 더 적절한 값으로 늘립니다.
  • 파일을 저장합니다.변경사항은 즉시 적용됩니다.
