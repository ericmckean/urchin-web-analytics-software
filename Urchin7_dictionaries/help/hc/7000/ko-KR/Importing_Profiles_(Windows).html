프로필 가져오기(Windows)

개요

Urchin의 프로필 가져오기 기능은 Microsoft Internet Information Server를 실행하는 시스템의 사용자가 각각의 IIS 사이트에서 프로필을 빨리 설정할 수 있는 편리한 방법입니다. Urchin은 IIS 구성을 읽고 서버에서 실행 중인 웹사이트를 확인한 다음 IIS 로그를 로그 소스로 사용하는 각 웹사이트에 대한 기본 프로필을 생성할 수 있습니다.그런 다음 프로필을 맞춤설정하거나 가져온 사이트에 대해 프로필을 원하는 대로 추가할 수 있습니다.

프로필을 가져오는 방법

프로필 가져오기를 시작하려면 Urchin 관리 시스템에 관리자로 로그인하고 인터페이스 오른쪽 상단에 있는 '구성'을 클릭합니다.오른쪽 상단의 '가져오기' 버튼을 클릭하면 '프로필 가져오기' 화면이 나타납니다.이 화면에서 가져올 웹사이트를 선택할 수 있습니다.가져올 사이트를 선택한 후에 '가져오기' 버튼을 클릭합니다.모든 가져오기 선택을 마친 후 '완료'를 클릭합니다.

권장사항

Urchin의 요약 보고서를 통해 서버에 대한 트래픽을 완벽하게 확인할 수 있도록 서버에 있는 각 웹사이트에 대해 적어도 하나의 프로필을 만드는 것이 좋습니다.요약 보고서는 서버에 대한 전체적인 트래픽 정보뿐만 아니라 다양한 트래픽 매개변수를 기준으로 매긴 각 사이트의 순위도 제공합니다.이러한 정보는 대역폭 사용을 기준으로 청구하는 호스트에게 특히 유용합니다.요약 보고서는 구성된 프로필만을 기준으로 데이터를 표시합니다. 프로필이 작동하지 않는 사이트는 포함되지 않습니다.
