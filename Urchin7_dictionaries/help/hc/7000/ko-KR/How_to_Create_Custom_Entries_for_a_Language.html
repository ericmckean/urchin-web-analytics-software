언어별 맞춤 항목 만들기

  1. <Urchin>/lib/admin/languages/ 폴더의 xx-XX.txt 언어 파일을 <Urchin>/lib/custom/languages/ 폴더로 복사합니다.

  2. 파일 끝에 새 항목을 추가하고 각 항목에 새 번호를 할당합니다.

  3. 번호와 값 사이에는 <Tab> 구분기호가 있어야 Urchin에서 구문을 분석할 수 있습니다.

  4. 각 항목 번호는 연속적으로 1씩 증가해야 합니다.
  5. Urchin을 다시 로드합니다.

  6. 로드 후 Urchin은 <Urchin>/lib/custom/languages/ 폴더에 xx-XX.db라는 이름의 언어 DB 파일을 만듭니다.

정의된 맞춤 항목은 보고서 섹션에서, 열 라벨, 제목 또는 도움말 설명 등에 사용될 수 있습니다.

언어 파일의 예

아래에서 738, 739 및 740 항목이 새로 추가된 맞춤 항목입니다. 

  ...
  731     Executive
  732     Marketer
  733     Webmaster
  734     Dashboards
  735     All Reports
  736     ja-jp-mac (Japanese)
  737     ja-jp-osaka (Japanese)
  738     Modules
  739     Actions
  740     Modules / Actions

중요사항: Urchin 업그레이드 시 기존 언어 파일의 항목이 기본 항목으로 바뀔 수 있으므로, 업그레이드 후에는 맞춤 언어 파일을 다시 만들고 맞춤 보고서를 업데이트해야 합니다.
