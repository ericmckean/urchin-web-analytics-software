날짜 디렉토리를 다른 위치로 옮길 수 있습니까?

개요

기본적으로 Urchin 데이터 디렉토리는 루트 Urchin 프로그램 디렉토리의 하위 디렉토리입니다.이 위치는 ~urchin/etc/urchin.conf 파일을 수정하고 기존 ~urchin/data 디렉토리를 원하는 위치로 직접 이동하여 쉽게 변경할 수 있습니다.그러나 이 경우 기존 라이센스를 손상시킬 수 있으며 Urchin 소프트웨어 공인 컨설턴트에 문의하여 일련번호를 다시 설정해야 합니다.

데이터 디렉토리 이동 절차:

Urchin이 보고서 데이터를 저장하는 위치는urchin.conf 파일에서 변경할 수 있으며 이 파일은 주 Urchin 설치 디렉토리의 etc 디렉토리에 있습니다.

UNIX 계열 시스템:

  1. Urchin을 설치한 사용자로 명령 쉘을 엽니다.
  2. cd 명령을 통해 Urchin이 설치된 디렉토리로 이동합니다.
  3. 다음 명령을 통해 Urchin 서비스를 중지합니다.bin/urchinctl stop
  4. 텍스트 편집기를 사용하여 Urchin 폴더의 etc 디렉토리에서 urchin.conf 파일을 엽니다.
  5. 맨 앞의 '#' 문자를 삭제하여

    • #dataDirectory:./data/ 행의 주석을 해제하고 기본 디렉토리 대신 사용할 전체 디렉토리 경로로 대체합니다(예: dataDirectory: /bigdisk/urchin/data/ )

  6. 필요한 경우 새 데이터 디렉토리를 생성합니다.중요사항: 이 디렉토리는 Urchin을 실행하는 사용자가 읽고 쓸 수 있습니다.

  7. 기존 데이터를 새 위치로 복사합니다.모든 권한 및 데이터를 올바르게 유지하려면 다음 명령을 사용하는 것이 좋습니다.
    • cd data; tar cf - .| (cd /bigdisk/urchin/data; tar xpf -)
  8. 데이터 디렉토리 이름을 data.old로 변경합니다.원하는 경우 완전히 삭제할 수 있으나 삭제하기 전 모든 사항이 올바르게 작동하는지 확인해야 합니다.
  9. 관리 편의를 위해 기본 Urchin 디렉토리에 심볼릭 링크를 생성하여 새 위치를 가리키도록 할 수 있습니다
    • (예: -s /bigdisk/urchin/data ./data).
  10. 다음 명령으로 Urchin 서비스를 다시 시작합니다.bin/urchinctl start
  11. Urchin에 관리 사용자로 로그인합니다.Urchin 라이센스 화면이 표시됩니다.'라이센스 재활성화' 링크를 클릭하면 완료됩니다.

Windows 시스템:

  1. Urchin 서비스를중지합니다('시작'->'프로그램'->'Urchin'->'Urchin 서비스 사용 안 함').

  2. Windows 탐색기를 열고 Urchin 폴더의 etc 폴더로 이동합니다.기본적으로 이 위치는 C:\Program Files\Urchin\etc입니다.
  3. 텍스트 편집기를 사용하여 urchin.conf 파일을 엽니다.
  4. 맨 앞의 '#' 문자를 삭제하여

    • #dataDirectory:./data/ 행의 주석을 해제하고 드라이브 문자와 기본 폴더 대신 사용할 전체 디렉토리 경로로 대체합니다(예: dataDirectory: E:\Urchin\data).

  5. 필요한 경우 새 데이터 폴더를 생성합니다.중요사항:이 폴더의 권한은 Urchin 서비스에 대한 읽기 및 쓰기 권한을 허용해야 합니다.
  6. 데이터 폴더의 내용을 새 폴더로 복사합니다.
  7. 데이터 폴더 이름을 data.old로 변경합니다.원하는 경우 완전히 삭제할 수 있으나 삭제하기 전 모든 사항이 올바르게 작동하는지 확인해야 합니다.

  8. Urchin 서비스를 다시시작합니다('시작'->'프로그램'->'Urchin'->'Urchin 서비스 사용').

주의사항

UNIX 계열 시스템의 변경 방지를 위한 라이센스 구현 방식으로 인해 Urchin의 데이터 디렉토리를 이동하는 경우 Urchin 라이센스를 재활성화해야 할 수 있습니다.그러나 라이센스 재발급 작업은 매우 간단합니다. Urchin 관리 사용자로 로그인하여 '라이센스 재활성화'를 클릭하면 됩니다.
