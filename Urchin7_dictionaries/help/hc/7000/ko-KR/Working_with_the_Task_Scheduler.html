작업 스케줄러 사용하기

개요

작업 스케줄러는 Urchin의 핵심 기능으로 모든 프로필에 대한 Urchin 로그 처리 이벤트의 실제 예약 및 실행을 수행합니다. 스케줄러를 통해 작업을 즉시 실행하거나 작업을 Urchin 이벤트 목록에 추가하여 원하는 간격으로 반복 실행할 수 있습니다.

스케줄러 사용 방법

  1. Urchin 6에 로그인하고 인터페이스 오른쪽 상단에 있는 '구성'을 클릭합니다.
  2. 예약할 프로필을 찾고 '수정'을 클릭합니다.
  3. '실행/예약' 탭을 선택합니다.
  4. '작업 설정'에서 원하는 간격을 선택합니다. '일별'이 권장됩니다.
  5. 작업 시간을 설정합니다.
  6. '업데이트'를 클릭하여 변경사항을 저장합니다.
  7. 작업을 즉시 실행하려면 '지금 실행'을 클릭합니다. 이후 예약된 작업이 설정한 일정에 따라 실행됩니다.

권장 사항

  • 많은 웹서버에서 일 단위로 로그가 로테이션되므로 대부분의 작업은 일 단위로 실행되도록 예약해야 합니다. 그러나 Urchin의 로그 추적 기능은 같은 로그를 데이터 중복 없이 여러 번 읽을 수 있으므로 반드시 일 단위일 필요는 없습니다.

스케줄러 작업에 대한 참고사항

  • Urchin에서 모든 작업은 순차적으로 처리되므로 여러 작업을 같은 시간에 실행하도록 지정해도 한 번에 하나씩 작업이 처리됩니다.
  • 작업이 모두 실행되었는지 보려면 왼쪽 탐색 창에서 '스케줄러'와 '작업 기록'을 차례로 선택하십시오.

스케줄러 섹션:

  • 프로필 작업 스케줄러 - 이 화면에는 구성에 있는 각 프로필에 대해 실행이 예약된 작업이 표시됩니다. 프로필 이름, 실행 빈도, 실행 상태(예약되지 않음, 보류 중, 대기, 실행 중) 및 실행 중인 작업의 완료율과 시작 시간을 포함하여 각 프로필에 대한 여러 세부 정보가 표시됩니다.
  • 프로필 작업 기록 - 이 화면에는 프로필 작업 스케줄러 또는 지금 실행 버튼을 통해 실행된 모든 작업 기록이 표시됩니다. 작업 요약 정보가 표시되기 때문에 각 작업에 대한 중요 정보를 쉽게 파악할 수 있습니다. 상태 입력란을 클릭하면 해당 작업 기록의 실행 시간 내역을 볼 수 있습니다.

  • 지리 DB 다운로드 기록(Urchin 6.500+) – 프로필 작업 스케줄러의 일부였던 geo-update가 별도로 분리된 섹션입니다. 이 섹션에는 프로필 작업 스케줄러를 통해 수행된 모든 지리 DB 다운로드 내역이 표시됩니다. 다운로드 요약 정보가 표시되어 각 다운로드에 대한 중요 정보를 쉽게 파악할 수 있습니다. 상태 입력란을 클릭하면 해당 작업 기록의 실행 시간 내역을 볼 수 있습니다. 설정 -> 글로벌 설정 -> 지리 데이터 업데이트 섹션에서 지리 DB 다운로드를 예약할 수 있습니다. 또한 설정 -> 글로벌 설정 -> CPC 소스 업데이트 섹션에서 CPC 스케줄러를 수정할 수 있습니다.

  • CPC 다운로드 기록(Urchin 6.500+) – 이 화면에는 CPC 작업 스케줄러를 통해 실행된 모든 CPC 다운로드 작업 기록이 표시됩니다. CPC 다운로드 요약 정보가 표시되어 각 다운로드에 대한 중요 정보를 쉽게 파악할 수 있습니다. 상태 입력란을 클릭하면 해당 작업 기록의 실행 시간 내역을 볼 수 있습니다. CPC 소스 관리자 보기에서 CPC 소스 다운로드를 예약할 수 있습니다. CPC 소스 관리자 -> 다운로드 또는 수정 -> 일정 탭을 선택한 후 일정 다운로드를 클릭하거나 지금 다운로드 버튼을 클릭하여 바로 다운로드합니다.
