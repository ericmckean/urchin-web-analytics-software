특정 키워드나 URL을 키워드가 아닌 직접 추천으로 처리하도록 Urchin을 구성하려면 어떻게 해야 합니까?

특정 검색 키워드 또는 URL을 키워드 추천이 아닌 직접 추천으로 처리할 수 있습니다.이를 위해 Urchin에서 보고서에 직접 추천으로 보고하도록 urchin.js 파일을 구성할 수 있습니다.

  1. 텍스트 편집기를 사용하여 urchin.js 파일을 엽니다.
  2. 아래 제목으로 스크롤합니다. 

          //-- Auto/Organic Keywords to Ignore
  1. 이 양식 목록 끝에 무시할 새 유기적 키워드를 추가합니다. 

          _uOno[#]="keyword";
  • ..다음 예 참조: 

        //-- Auto/Organic Keywords to Ignore
        var _uOno=new Array();
        _uOno[0]="urchin";
        _uOno[1]="urchin.com";
        _uOno[2]="www.urchin.com";

보고서를 볼 때 브라우저 시간초과가 발생하는 이유는 무엇입니까?

응용 프로그램이 Urchin 데이터베이스를 검색하고 각 보고서를 생성할 때 45초 이내에 데이터가 반환되지 않으면 응용 프로그램 시간초과가 발생합니다.데이터베이스 크기가 큰 경우 필요한 보고서 데이터를 모두 검색하는 데 45초 이상이 소요될 수 있습니다.이 경우 시간초과가 발생하고 대개 오류가 표시됩니다.

Urchin을 실행하는 서버에 대해 충분한 액세스 권한이 있는 경우 시간초과 값을 연장할 수 있습니다.

응용 프로그램 시간초과 값 연장

  • 명령 쉘을 사용하여 Urchin 서버에 연결합니다.
  • .../[urchin-install-dir]/etc/ 디렉토리로 이동합니다.
  • 텍스트 편집기를 사용하여 'session.conf' 파일을 엽니다.
  • 'APPLICATION_TIMEOUT:'(초) 값을 늘립니다.
  • 파일을 저장하면 변경사항이 즉시 업데이트됩니다.
