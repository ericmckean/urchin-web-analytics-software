カスタム レポートの作成方法

カスタム レポートを作成する方法は下記のとおりです。

  1. データ マップの表を選択するか、新しい表を定義します (カスタムのデータ マップ をご覧ください);

  2. <Urchin>/lib/custom/profiletypes/Default/default.rs2.sample をコピーして新しい (".rs2") レポートファイルを作成します。

  3. レポート ファイルに新しい行を追加して、新しいレポートを定義します。

レポート ファイルの形式につきましては下記をご覧ください。

  • ID - 固有のレポート ID:
    • C= カテゴリ
    • D= データ
    • M= メニュー
    • R= レポート
    • S= サブレポート
    • P= ページ レポート
    • B= マイレポート
  • RS - マイレポート/レポート セクション:
    • 1= 管理
    • 2= マーケティング
    • 4= ウェブマスター
  • RR - レポート要件:
    • 1= ソフトウェア
    • 2= 上級
    • 4= e コマース
    • 8=UTM
  • NTITLE - ナビゲーション タイトル (辞書エントリ #)
  • ONCOLOR - ナビゲーション オン カラー
  • OFFCOLOR - ナビゲーション オフ カラー
  • HELP - レポート ヘルプ (辞書エントリ #)
  • NDATA - データ要素の数 (直後にエントリが来る)
    • このフィールドの直後に追加のタイプ D （データ) のサブレポートが来ても使用できます
  • CTITLES - 列タイトル (辞書エントリ #)
  • CTYPES - フォーマットに使用される列形式
    • 10= ストリング
    • 11= 日/年月
    • 12= 日付
    • 20= 数字
    • 21= 収益
    • 22= バイト数
    • 23= 時間比率
    • 24= 比率
    • 25= 割合
    • 26= 時間
    • 30= 合計割合
    • 40= 収益
    • 41= 収益/数
    • 42= 数/収益
  • TABLE - データ表の数字:
    • T= 合計
    • TR= トランザクション
    • H= 棒グラフ
    • V= ユーザー
    • 1-n 表
  • DXFORMAT - データ抽出形式
  • RF - 必須フィールド
  • RT - レポート形式:
    • 1= グラフの合計
    • 2= 棒グラフ
    • 3= スタック
    • 4= トップ
    • 5= ツリー
    • 6= ディレクトリ
    • 7= グラフ専用
    • 8= 点
    • 9= クリック
    • 10= 目標到達プロセスの定義
    • 11= ホームページ上のデータ表示
    • 12= 地図上のデータ表示
    • 13= 目標の確認
    • 14= 目標パス
    • 20= KPI 日付レポート
    • 21= KPI 上位 10 位
    • 22= トレンドの概要
    • 23= コンバージョンの概要
    • 24= Map Mini
  • GTYPES - グラフ形式:
    • 1= 棒
    • 2= 線
    • 3= 円
    • 4= 水平棒
    • 5=fun
    • 6= 多重線
  • FSEP - フィールドの区切り
  • SP - 特別オプション (ビット マップ番号を追加):
    • 1= 合計
    • 2= 平均
    • 4= 日付の比較
    • 8= 小文字
    • 16= 一定期間のデータ
    • 32= データ表示
    • 64= 言語ルックアップ
    • 128= セグメント
    • 256= 有効値
    • 512= トランザクションの詳細
    • 1024= ユーザーの詳細
    • 2048= 用語のルックアップ
  • SR - 特別レポート フラグ:
    • 1= キーワード アドバイス
    • 2= クリック
    • 3= 時間別の詳細
    • 4= ユーザー リスト
    • 5= 地図上のデータ表示
    • 6= カスタム使用のカスタム辞書
    • 7= 日付の比較
  • FNT - フィルタ数とフィルタ形式:
    • 1: =

    • 2: >=

    • 3: <=

    • 4: >

    • 5: <

  • FNF - フィルタ数のフィールド
  • FNV - フィルタの値
  • FST - フィルタ ストリングの形式:
    • 1= 除外
    • 2= 含む
  • FSF - フィルタ ストリングのフィールド
  • FSX - 正規表現のストリング
