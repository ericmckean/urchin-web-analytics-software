Urchin を Apache ウェブサーバーにインストールする理由

Urchin は、軽量の Apache ウェブサーバーにインストールすることで、ウェブベースの管理と動的なレポート配信を実現します。 Urchin に単体の Apache をバンドルすることにより、Urchin に対応するために、ウェブサーバーの既存のウェブサーバー設定を変更する必要がなくなります。 他にも Urchin の Apache ウェブサーバーを既存のウェブサーバーとは別に実行することで次のようなメリットがあります。

  • 標準のウェブサーバーを変更しても Urchin への影響はありません。
  • 標準のウェブサーバーに影響を与えることなく Urchin サービスを管理できます。
  • Urchin の管理 GUI を使用して単体の Apache ウェブサーバーの特定の運用パラメータを変更できます。
  • Urchin のアップグレードを自己完結で行うことができ、SSL サポートなどの必要な変更もすべて Urchin のアップグレード プロセスのみで処理できます。標準のウェブサーバーを変更する必要はありません。 また、Urchin の Apache インストールによって、コンピュータ上の他の既存のウェブ サービスが影響を受けたり、コンピュータのリソースを著しく消費することもありません。Urchin では、ウェブを介したレポートへのリモート アクセスを許可するよう設定することもできます。
