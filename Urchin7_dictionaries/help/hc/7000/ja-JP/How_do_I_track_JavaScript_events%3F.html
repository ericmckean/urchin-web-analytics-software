JavaScript イベントをトラッキングする

イベントをトラッキングするには、urchinTracker の JavaScript 関数を使用してイベント名を指定します。 たとえば、次のように呼び出します。 

    javascript:urchinTracker('/homepage/link1');

これにより、呼び出しイベントが発生するたびに、/homepage/link1 の URL のページビューが 1 回記録されます。 引数 (任意のURL) はスラッシュ (/) で始める必要があります。 任意のファイル名を割り当てることができるので、イベント名を分かりやすい任意のディレクトリ構造に分類できます。

たとえば、イベントをページとイベントの種類で分類する場合は、次のようなディレクトリ名で分類します。 

    * '/homepage/links/link1'
    * '/homepage/radiobuttons/button1'
    * '/contactform/rollovers/image1'

HTML コードの例

onClick イベントを記録する場合は、次のように指定します。 

    <a href="javascript:void(0);" onClick="javascript:urchinTracker('/folder/file');">

rollover イベントを記録する場合は、次のように指定します。 

    <a href="javascript:void(0);" onMouseOver="javascript:urchinTracker ('/folder/file');">
