Urchin を設定した後でデータセンター モードに切り替える

  • この記事は、Urchin ソフトウェア認定コンサルタントの ROI Revolution によって作成されています。

データセンター モードでは、グループ内の 1 つまたは複数のプロファイルに管理者権限を追加することができます。 初期設定が既に完了している場合は、MySQL または PostgreSQL を使用してデータセンター モードを簡単に有効にできます。

MySQL の場合: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

PostgreSQL の場合: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
