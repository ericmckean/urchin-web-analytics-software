プロファイルのインポート (Windows)

概要

Urchin のプロファイル インポート機能は、Microsoft Internet Information Server (IIS) を実行しているシステムで、各 IIS サイトのプロファイルを設定する際に便利です。Urchin は、IIS の設定を読み込んでサーバー上で実行しているウェブサイトを特定し、IIS ログをログ ソースとして使用する各ウェブサイトの基本的なプロファイルを作成します。 その後、プロファイルをカスタマイズしたり、必要に応じてインポートしたサイトのプロファイルを追加できます。

プロファイルをインポートする方法

プロファイルをインポートするには、まず Urchin 管理システムに管理者としてログインし、画面右上の [構成] をクリックします。 右上の [インポート] ボタンをクリックすると、[プロファイルのインポート] 画面が表示され、 インポートするウェブサイトを選択できます (対象のウェブサイトがある場合)。 インポートするサイトを選択して [インポート] ボタンをクリックします。 選択したサイトのインポートがすべて完了したら、[終了] をクリックします。

推奨

サーバー上のウェブサイトごとに、1 つ以上のプロファイルを作成することをおすすめします。これにより、Urchin's のサマリー レポートでサーバーのトラフィックに関して詳細な情報を取得できます。 サマリー レポートでは、サーバーの全体的なトラフィック情報や、さまざまなトラフィック パラメータごとの各サイトのランキングを確認できます。 これらの情報は、ホスト プロバイダが帯域幅に応じた請求を行う場合に非常に便利です。 サマリー レポートには、設定済みのプロファイルに基づくデータのみが表示されます。有効なプロファイルが設定されていないサイトのデータは表示されません。
