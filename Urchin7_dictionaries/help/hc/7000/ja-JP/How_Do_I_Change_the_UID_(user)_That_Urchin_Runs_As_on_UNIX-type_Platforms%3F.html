UNIX プラットフォームで Urchin を実行する UID (ユーザー) を変更する

Urchin を実行する UID/ユーザーや GID/グループは簡単に変更できます。 次の操作は「root」から行う必要があるため、多くの場合はシステム管理者が行う必要があります。

  1. システムで新しいユーザーやグループをまだ作成していない場合は、パスワード ファイルとグループ ファイルに適切なエントリを作成します。
  2. 「urchinctl stop」を実行して urchin のサービスを停止します。 「urchinctl」バイナリは Urchin の「bin」ディレクトリにあります。
  3. Urchin の root ディレクトリで「chown -R newuser:newgroup」を実行します。 「urchin.conf」ファイルで、「var」または「data」ディレクトリの代わりの場所を指定した場合は、これらのディレクトリで「chown」コマンドを実行する必要があります。
  4. Urchin の「var」ディレクトリにある「urchinwebd.conf.template」ファイルを編集して、次の行を変更します。
    • User newuser Group newgroup
  5. 「urchinctl start」を使用して Urchin のサービスを再起動します。
