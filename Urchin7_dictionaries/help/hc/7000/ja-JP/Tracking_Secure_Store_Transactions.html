セキュリティで保護されたショップ トランザクションのトラッキング

この記事は、Urchin ソフトウェア認定コンサルタントの ivantage Limited によって作成されています。

Urchin ではいくつかのコードを追加することで、所有するドメインだけでなく、他のドメインにあるショッピング カートのデータもトラッキングできます。 Urchin でトランザクション データを集計するようページを設定するには、e コマース トランザクションをトラッキングするをご覧ください。

ショップ サイトがメイン サイトとは異なるサブドメインやドメインにある場合は、次の手順でページを設定します。

精算手続きの開始ページが異なるサブドメインにある場合 (たとえば、顧客が www.mystore.com から shoppingcart.mystore.com に移動する場合など):

  • ショップ サイトとショッピング カートのページのトラッキング コードに次の太字の行を追加します。 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

精算手続きの開始ページが異なるドメインにある場合 (たとえば、顧客が www.mystore.com から www.securecart.com に移動する場合):

  • ショップ サイトとショッピング カートのページのトラッキング コードに次の太字の行を追加します。 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • utmLinker を使用して、メイン サイトからセキュリティで保護されたサイトにリンクを変更する方法は次のとおりです。 現在のリンクが次のようになっているとします。 

     <a href="https://www.securecart.com/?store=parameters">今すぐ購入</a>
  • このリンクを次のように変更します。 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">今すぐ購入</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">今すぐ購入</a>
   </noscript>

  • JavaScript が有効かどうかに関係なく、上記のコードでユーザーへのリンクが表示されます。 リンクまたはリンクのテキストにアポストロフィがある場合、バックスラッシュ (\) でエスケープ処理する必要があります。 重要: ページに urchinTracker()、utmLinker()、utmSetTrans() または utmLinkPost() へのコールが含まれている場合、トラッキング コードをこれらの関数より上部の HTML コードに配置する必要があります。 この場合、トラッキング コードは、開始 <body> タグと JavaScript 関数の間の任意の場所に配置してください。

フォームを使用してショッピング カートに情報を送信する場合は、utmLinkPost() 関数を使用します。 この関数を使用すると、該当フォームの URL に Cookie データが追加されます。 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • これは、method="GET" のフォームにも使用できます。

    重要: ページに urchinTracker()、utmLinker()、utmSetTrans() または utmLinkPost() へのコールが含まれている場合、トラッキング コードをこれらの関数より上部の HTML コードに配置する必要があります。 この場合、トラッキング コードは、開始 <body> タグと JavaScript 関数の間の任意の場所に配置してください。
