データ ディレクトリを別の場所に移動する

概要

Urchin のデータ ディレクトリは、デフォルトで Urchin プログラムのルート ディレクトリ内に作成されます。 この場所を変更するには、~urchin/etc/urchin.conf ファイルを編集して、既存の ~urchin/data ディレクトリを希望の場所に手動で移動します。 この操作は既存のライセンスに違反する場合があるため、担当の Urchin ソフトウェア認定コンサルタントに連絡し、シリアル番号をリセットしてもらう必要があります。

データ ディレクトリの移動手順:

Urchin がレポート データを保存する場所は、urchin.conf ファイルで変更できます。このファイルは、Urchin のメイン インストール ディレクトリの etc ディレクトリにあります。

UNIX システムの場合:

  1. Urchin をインストールしたユーザーでコマンド シェルを開きます。
  2. cd コマンドを使用して Urchin がインストールされているディレクトリに移動します。
  3. bin/urchinctl stop コマンドを実行して Urchin サービスを停止します。
  4. テキスト エディタを使用して Urchin の etc ディレクトリにある urchin.conf ファイルを開きます。
  5. 「#dataDirectory: ./data/」のコメントを外し (先頭の「#」文字を削除)、デフォルトのディレクトリの代わりに使用するディレクトリ パス全体 (例: dataDirectory: /bigdisk/urchin/data/) に置き換えます。

  6. 必要に応じて、新しいデータ ディレクトリを作成します。 注: このディレクトリは、Urchin を実行するユーザーによって読み書き可能である必要があります。

  7. 既存のデータを新しい場所にコピーします。 すべての権限設定とデータが適切に保存されていることを確認するには、コマンド「cd data; tar cf - . | (cd /bigdisk/urchin/data; tar xpf -)」を使用することをおすすめします。
  8. データ ディレクトリの名前を「data.old」に変更します。 このディレクトリは完全に削除することができますが、すべてが適切に機能していることを確認してから削除することをおすすめします。
  9. 簡単に管理できるよう、Urchin のメイン ディレクトリにこの新しい場所を指す symlink を作成することをおすすめします (例: ln -s /bigdisk/urchin/data ./data)。
  10. bin/urchinctl start コマンドを実行して Urchin サービスを再起動します。
  11. 管理者で Urchin にログインします。 Urchin のライセンス画面が表示されます。 [ライセンスの再登録] をクリックして完了します。

Windows システムの場合:

  1. [スタート]、[プログラム]、[Urchin]、[Urchin サービスの無効化] の順にクリックし、Urchin サービスを停止します。
  2. Windows エクスプローラを開き、Urchin の etc フォルダに移動します。 デフォルトでは C:\Program Files\Urchin\etc になります。
  3. テキスト エディタを使用して urchin.conf ファイルを開きます。
  4. 「#dataDirectory: ./data/」のコメントを外し (先頭の「#」文字を削除)、デフォルトのフォルダの代わりに使用するドライブ名とパス名全体 (例: dataDirectory: E:\Urchin\data) に置き換えます。
  5. 必要に応じて、新しいデータ フォルダを作成します。 注: このフォルダの権限設定では、Urchin サービスへの読み書きアクセスを許可する必要があります。
  6. データ フォルダの内容を新しいフォルダにコピーします。
  7. データ フォルダの名前を「data.old」に変更します。 このディレクトリは完全に削除することができますが、すべてが適切に機能していることを確認してから削除することをおすすめします。
  8. [スタート]、[プログラム]、[Urchin]、[Urchin サービスの有効化] の順にクリックし、Urchin サービスを再起動します。

アドバイス

不正使用を防ぐため、UNIX システムでは特別な方法でライセンスが実装されているため、Urchin のデータ ディレクトリを移動する場合は Urchin ライセンスを再登録する必要があります。 ライセンスを再登録するには、Urchin の管理者でログインし、[ライセンスの再登録] をクリックします。
