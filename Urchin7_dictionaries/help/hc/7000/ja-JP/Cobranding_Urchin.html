Urchin のブランド設定

概要

Urchin では、管理インターフェース、レポート インターフェース、ログイン画面のブランド設定に対応しています。 インターフェースの上部に HTML を含めることができる編集可能なファイルが 3 つあります。 ポータルの統合が完了すると、Urchin レポートはアプリケーション サーバーによってフレームセット内または表内に表示されます。

ブランド設定の手順

インターフェースをブランド設定するには、Urchin パッケージの次のファイルを編集する必要があります。 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • 「.sample」の部分を削除してファイル名を変更します (ファイル名は cobrand_admin.tpl などのようにする必要があります)。
  • 必要に応じて HTML コンテンツを追加し、ブランド要素を含めます。

1 つ目のファイルは管理インターフェース、2 つ目のファイルはレポート インターフェース、 3 つ目のファイルはログイン画面のブランド設定を管理します。 これらのファイルで提供される HTML は、Urchin インターフェスの上部に配置されます。
