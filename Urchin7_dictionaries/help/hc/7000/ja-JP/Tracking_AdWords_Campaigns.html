外部へのリンクや広告のトラッキング

Urchin では、バナー広告やその他の外部へのリンクのクリックを簡単にトラッキングできます。 これらのリンクはサイト内の UTM JavaScript を設定したページを経由しないため、トラッキングするには urchinTracker JavaScript を使用してリンク自体にタグを設定する必要があります。 この JavaScript コードにより、リンクがクリックされるたびに 1 回のページビューが割り当てられます。ページビューは指定したファイル名でカウントできます。

アニメーション GIF や他の静的なバナー広告などの場合は、urchinTracker JavaScript をバナー広告の <a> タグ内に次のように追加します。 

    <a href="http://www.example.com/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
