RedHat に Urchin の起動スクリプトをインストールする

Urchin の起動スクリプトは、インストール時に作成され、 Urchin の util ディレクトリである /path/to/urchin/util/ に urchin_daemons という名前で保存されます。

このスクリプトを /etc/rc.d/init.d/ ディレクトリにコピーします。

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • /etc/inittab ファイルを参照してデフォルトのランレベルを確認します (通常は「3」に設定されています)。 [[下記を参照]]
  • /etc/rc.d/rcX.d に移動します (X にはランレベルが入ります)。
  • cd /etc/rc.d/rc3.d
  • S95urchin と K05urchin という名前で、/etc/rc.d/init.d/urchin_daemons へのシンボリック リンクを作成します。
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # デフォルトのランレベルです。RHS で使用するランレベルは次のとおりです。

  • 0 - 休止 (初期値にこのランレベルを設定しないでください)
  • 1 - シングル ユーザー モード
  • 2 - NFS なしのマルチユーザー (ネットワークを使用しない場合は 3 と同じになります)
  • 3 - フル マルチユーザー モード
  • 4 - 未使用
  • 5 - X11
  • 6 - 再起動 (初期値にこのランレベルを設定しないでください)
