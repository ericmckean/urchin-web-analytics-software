Urchin 6.602 Geo-databases

Urchin 6.602 には下記の地域データベースが含まれています。

  • geo.db - 大都市圏の詳細情報（country/region/city）と、ネットワーク（組織）およびドメイン情報が含まれています。 urchin 6 （6.400 - 6.601）以前のバージョンと同じレベルの情報が含まれています。

  • geo.light.db - 最大で国レベルの大都市圏情報のみが含まれ、地域、都市、ネットワーク、組織情報は含まれていません。 このデータベースのメモリ フットプリントは極めて小さく、ロード時間も短いため、ログ処理エンジンのパフォーマンスが重要な場合はこのデータベースを使用します。 この軽量な地域データベースを使用した場合、レポートの表示は次のようになります。

    • o 次の "地域地図" レポートには、最大で国レベルの大都市圏情報のみが表示され、地域や都市情報は表示されません。
      • [経営層向けの表示] の [経営層向けの概要（ダッシュボード）]
      • [経営層向けの表示] の [地図上のデータ表示]
      • [マーケティング担当者向けの表示] の [マーケティング担当者向けの概要（ダッシュボード）]
      • [ウェブマスター向けの表示] の [ウェブマスター向けの概要（ダッシュボード）]
      • [マーケティングの最適化] の [ユーザー セグメントのパフォーマンス] の [地図上のデータ表示]
      o "利用ネットワーク" レポート（[マーケティングの最適化] の [ユーザー セグメントのパフォーマンス] の [利用ネットワーク]）には、データが表示されません o "ドメイン" レポート（[マーケティングの最適化] の [ユーザー セグメントのパフォーマンス] の [ドメイン]）には、データが表示されません

デフォルトでは geo.db が使用されます。 geo.light.db データベースに切り替えるには、urchin.conf ファイルで対応する行をコメントアウトします。 


# 地域データベースの設定

# 軽量な地域データベースを有効にするには下記の行をコメントアウトします
# GeodbLight: ON
