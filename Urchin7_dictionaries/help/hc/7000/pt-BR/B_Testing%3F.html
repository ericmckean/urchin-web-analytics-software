Que variáveis do UTM devo usar para Testes A/B?

Testes A/B são úteis para testar diferentes versões de e-mails e anúncios.Use utm_source, utm_medium, utm_content e utm_campaign para diferenciar links de diferentes anúncios e e-mails e para diferenciar os diversos links dentro de cada anúncio e e-mail.

O exemplo a seguir ilustra o uso de variáveis do UTM e um boletim informativo por e-mail.Nesse exemplo, o boletim informativo contém diversos links com uma única página do seu site.Observe que utm_content é o único valor que difere entre os dois links.

O primeiro link é um link de texto: 

    http://www.example.com.br/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buytext&utm_campaign=brand_x

O segundo link é uma imagem realçada: 

    http://www.example.com.br/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buypict&utm_campaign=brand_x

Use a ferramenta de criação de URLs do Google Analytics para criar os seus links.Essa ferramenta garante a sintaxe correta dos seus links.

Importante:use nomes e grafias consistentes em todos os valores de variáveis da sua campanha.
