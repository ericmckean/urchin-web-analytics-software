Posso receber relatórios do Urchin por e-mail?

  • Atualmente, o Urchin não fornece relatórios por e-mail.Para visualizar os seus relatórios, faça login na sua conta do Urchin. Entretanto, se você deseja manipular os seus dados fora da sua conta do Urchin, é possível usar a função de exportação de dados do Urchin para extrair dados manualmente de qualquer relatório do Urchin.Isso é muito útil para transferir dados de relatório a uma planilha, processador de texto ou banco de dados para análise mais detalhada.
