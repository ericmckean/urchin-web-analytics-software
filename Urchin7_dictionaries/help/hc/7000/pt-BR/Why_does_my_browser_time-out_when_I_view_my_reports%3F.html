Como faço para configurar o Urchin de modo que ele trate certas palavras-chave ou URLs como referências diretas em vez de palavras-chave?

Você pode tratar certas palavras-chave ou URLs como referências diretas em vez de palavras-chave.Para fazer isso, configure o arquivo urchin.js de forma que o Urchin registre esses elementos como referências diretas nos seus relatórios.

  1. Use qualquer editor de texto para abrir o seu arquivo urchin.js.
  2. Role para baixo até o título a seguir: 

          //-- Auto/Organic Keywords to Ignore
  1. Adicione ao final da lista uma nova palavra-chave estrutural a ser ignorada, no formato: 

          _uOno[#]="keyword";
  • ..como mostra o exemplo a seguir: 

        //-- Auto/Organic Keywords to Ignore
        var _uOno=new Array();
        _uOno[0]="urchin";
        _uOno[1]="urchin.com";
        _uOno[2]="www.urchin.com";

Por que o meu navegador atinge o tempo limite quando visualizo meus relatórios?

O aplicativo que consulta o banco de dados do Urchin e produz cada relatório irá atingir o tempo limite se os dados não forem retornados dentro de 45 segundos.Se o banco de dados for grande o bastante, 45 segundos talvez não sejam suficientes para recuperar todos os dados de relatório solicitados.Quando isso acontece, ele simplesmente atinge o tempo limite, exibindo um erro na maioria das vezes.

Você poderá estender o valor de tempo limite se tiver acesso suficiente ao servidor no qual o Urchin é executado.

Como estender o tempo limite do aplicativo

  • Conecte-se ao servidor do Urchin usando um shell de comando.
  • Navegue até o diretório .../[urchin-install-dir]/etc/.
  • Usando um editor de texto, abra "session.conf".
  • Aumente o valor de 'APPLICATION_TIMEOUT:' (em segundos).
  • Salve o arquivo. As alterações serão atualizadas imediatamente.
