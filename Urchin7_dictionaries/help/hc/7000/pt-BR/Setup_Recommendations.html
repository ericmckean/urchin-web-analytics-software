Recomendações de configuração

Visão geral

Depois que o Urchin é instalado, existem alguns parâmetros operacionais iniciais que devem ser configurados. Isso é feito por meio de um assistente de configuração que é executado quando você se conecta à interface de administração do Urchin pela primeira vez e durante os primeiros estágios de definição de perfis. Essas ações de configuração inicial incluem:

  • Licenciamento do Urchin
  • Definição de configurações de administrador para acesso remoto a relatórios e administração, bem como a definição do “Modo Datacenter”
  • Configuração da senha da conta de administrador do Urchin
  • Agendamento de tarefas para cada um de seus perfis para o processamento de dados
  • Gerenciamento de registros

Procedimento

Conecte-se à interface de administração do Urchin. Nos sistemas Windows, vá até “Iniciar” -> “Programas” -> “Urchin” -> “Administração do Urchin”. Em sistema tipo UNIX, você pode usar o URL http://hostname:9999, onde o nome do host é o nome do host registrado do seu sistema Urchin. Para uma nova instalação, use o seguinte para fazer login:

Nome de usuário: admin Senha: urchin

Será exibida uma tela inicial do Assistente de configuração do Urchin. Clique em “Continuar” para percorrer cada uma das seguintes telas do assistente. As opções feitas nessa configuração inicial podem ser alteradas posteriormente.

Licenciar Urchin

Siga as instruções de Licenciamento do Urchin.

Configurações administrativas

  • Configurações de acesso remoto - selecione “Ativar” em cada caso se desejar permitir a conexão de navegadores remotos. Se você selecionar “Desativar” em alguma opção, o único acesso permitido será no console do sistema onde o Urchin está instalado.
  • “Modo Datacenter” - essa configuração determina se o Urchin está configurado para permitir a criação de contas, que permitem a organização lógica de perfis, grupos e usuários em categorias de acesso restrito. Se não tiver certeza, será melhor selecionar “Ativar”, que não adiciona nenhuma sobrecarga e fornece a flexibilidade para usá-lo no futuro.

Usuário administrativo

  • Redefina a senha da conta de administrador e grave essa senha por segurança.

Agendamento de tarefas

Ao criar perfis, você pode agendar o momento de execução da tarefa que processa os dados do perfil em questão. Você deve verificar as configurações de cada perfil para assegurar que o tempo da tarefa seja adequado com relação ao momento em que os dados do registro estarão disponíveis, quanto tempo levará o processamento desses dados e quando deseja disponibilizar os relatórios atualizados aos usuários.

Gerenciamento de registros

O Urchin inclui um módulo de controle do registro que acompanha quanto cada registro foi processado até agora. Desse modo, a rotação do arquivo de registro não precisa estar necessariamente coordenada com a operação do Urchin. No entanto, o Urchin automatiza a rotação ou a remoção de registros nas configurações avançadas de cada origem do registro.

Leia Gerenciamento de registros para obter mais informações.
