Como posso alterar para o modo Datacenter agora que o Urchin já está configurado?

  • Este artigo foi escrito por ROI Revolution consultor autorizado do software Urchin

O modo Datacenter permite que você adicione níveis de direitos administrativos a um ou a vários perfis de um grupo. Se você já tiver concluído o processo de configuração inicial e quiser ativar o modo Datacenter, poderá fazê-lo facilmente em MySQL ou em PostgreSQL.

Para MySQL: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

Para PostgreSQL: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
