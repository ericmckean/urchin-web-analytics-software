Reprocessamento de registros

Visão geral

Certas circunstâncias podem garantir o reprocessamento de dados de registro, como a inatividade de um servidor DNS na ocasião do processamento, filtros incorretamente aplicados e assim por diante.O documento a seguir descreve o procedimento apropriado que é necessário para o retrocesso e o reprocessamento dos dados de registro de um servidor da web.

Observe que o reprocessamento de registros requer o uso de utilitários do Urchin que apenas estão disponíveis em um ambiente shell de linha de comandos.Não é possível realizar o procedimento completo exclusivamente a partir da GUI administrativa com base na web do Urchin.

Como reprocessar um único dia

  • Na GUI administrativa do Urchin, edite o Perfil e desative "Controle do registro", na guia "Armazenamento/banco de dados".Verifique se clicou em "Atualizar" para salvar as suas alterações.
  • Na guia "Origens do registro", verifique se os arquivos de registro apropriados a serem reprocessados estão especificados.Os dados de registro devem conter apenas os hits referentes às datas cujas estatísticas você está zerando.
  • Chame um shell de comando no sistema Urchin e 

cd /<URCHIN_DIR>/util
  • Execute o utilitário udb-sanitizer no diretório/pasta 'util' da distribuição do Urchin , com o comando 

udb-sanitizer -p profile-name -d YYYYMM
  • em que YYYYMM corresponde ao ano e ao mês contendo o dia que você deseja reprocessar

  • Selecione a opção 5, "Zerar um ou mais dias".O utilitário solicitará que você especifique o dia correto e irá zerar as estatísticas referentes a esse dia particular.Para visualizar mais exemplos de como usar o udb-sanitizer, consulte udb-sanitizer:Manutenção do banco de dados.

  • Clique no botão "Executar agora", na guia "Executar/agendar" referente ao Perfil, para reprocessar os dados de registro.
  • Redefina a Origem do registro alterando o Caminho do arquivo de registro de volta à sua configuração original
  • Na guia "Armazenamento/banco de dados", na área de edição do perfil, ative novamente o Controle do registro

Como reprocessar um mês inteiro:

O procedimento para reprocessar um mês inteiro de dados é semelhante ao procedimento anterior para um único dia: basta usar opções diferentes do udb-sanitizer.Consulte udb-sanitizer:Manutenção do banco de dados para obter exemplos.
