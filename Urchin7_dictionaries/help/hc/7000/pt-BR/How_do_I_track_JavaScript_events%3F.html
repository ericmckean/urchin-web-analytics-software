Como faço para monitorar eventos em JavaScript?

Para monitorar um evento, use a função JavaScript urchinTracker para especificar um nome para o evento.Por exemplo, chamar: 

    javascript:urchinTracker('/homepage/link1');

fará com que cada ocorrência do evento de chamada seja registrada como se fosse uma exibição de página para o URL /homepage/link1.O argumento deve começar com uma barra.Os nomes de eventos podem ser organizados em qualquer estrutura de diretórios desejada.

Por exemplo, se quiser organizar os eventos por página > tipo de evento, você poderá organizar uma hierarquia como esta: 

    * '/homepage/links/link1'
    * '/homepage/radiobuttons/button1'
    * '/contactform/rollovers/image1'

Exemplos de código HTML

Veja a seguir como registrar um evento onClick: 

    <a href="javascript:void(0);" onClick="javascript:urchinTracker('/folder/file');">

Ou, para registrar um evento de rolagem: 

    <a href="javascript:void(0);" onMouseOver="javascript:urchinTracker ('/folder/file');">
