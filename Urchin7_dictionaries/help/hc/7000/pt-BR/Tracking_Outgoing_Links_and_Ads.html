Como monitorar links e anúncios externos

O Urchin permite monitorar facilmente os cliques feitos em links de anúncios de banner ou outros links externos.Como esses links não levam a uma página no seu site que contenha o JavaScript do UTM, será necessário codificar o link em questão com o JavaScript urchinTracker, se você quiser monitorar esses cliques.Esse JavaScript atribui uma exibição de página a qualquer clique em um link - a exibição de página é atribuída ao nome de arquivo especificado.

Para um GIF animado ou outro tipo de anúncio de banner estático, adicione o JavaScript urchinTracker à tag JavaScript à tag <a> do seu banner: 

    <a href="http://www.example.com.br/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
