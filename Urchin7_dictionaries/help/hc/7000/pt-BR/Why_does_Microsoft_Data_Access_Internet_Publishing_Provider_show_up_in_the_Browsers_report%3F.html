Por que o Microsoft Data Access Internet Publishing Provider está aparecendo no relatório Navegadores?

Em geral, esse tipo de navegador irá aparecer se você tiver no site arquivos transferíveis por download que costumam ser abertos por um produto Microsoft Office (por exemplo, arquivos .doc do Word e arquivos .xls do Excel).Os plug-ins usados pelo Internet Explorer para visualizar esses tipos de arquivos se auto-registram como Microsoft Data Access Internet Publishing Provider quando a página está sendo visualizada.
