Como trabalhar com usuários e grupos

Visão geral

A funcionalidade Grupos e usuário do Urchin's permite que os administradores do Urchin configurem facilmente qualquer número usuários e conceda acesso a quaisquer relatórios considerados apropriados. Esses usuários poderão, então, ser colocados em grupos para expedir e simplificar o gerenciamento de muitos usuários. Se houver um grupo de usuários tiver acesso concedido, todos os usuários desse grupo terão acesso para fazer login no sistema. Usuários não têm acesso a relatórios a menos que especificamente permitido.

Como adicionar usuários

  1. Faça login em seu sistema do Urchin como um administrador. Observação: O URL de acesso ao sistema do Urchin é idêntico independentemente do tipo de usuário.
  2. Clique em “Configuração” (menu superior direito).
  3. Clique em “Usuários e grupos”.
  4. Clique no botão “Adicionar” no canto superior direito para inserir o “Wizard de usuário”.
  5. Selecione um nome de usuário ― ele deverá estar em letras minúsculas e não ter espaços ― e escolha uma senha.
  6. Insira o nome real do usuário ― ele será exibido no sistema do Urchin. Clique no botão "Avançar".

  7. Determine que nível de controle o usuário deverá ter. Conceda acesso às ferramentas do Google AdWords se necessário (Urchin 6.601+). Clique em “Concluir”.

  8. Clique em “Mais opções” para editar as Informações de usuário, Preferências, Grupos ou Acesso a relatórios do Usuário.
  9. Clique na guia "Acesso a relatórios". Os Perfis disponíveis serão mostrados na caixa à esquerda.
    1. Selecione um ou mais perfis. Para selecionar vários Perfis, use a tecla de comando ou a tecla de controle, dependendo de sua plataforma.
    2. Clique na seta para a direita a fim de mover o(s) Perfil(is) para a caixa Acesso concedido.
    3. Clique em “Atualizar” para salvar as alterações.

A partir do Urchin 6.602, a visualização padrão da Página inicial padrão para um usuário pode ser definida manualmente no banco de dados de configuração do Urchin. Leia Relatórios de totalização para obter mais informações. O acesso às Ferramentas do Google AdWords pode ser concedido/cancelado ao usuário correspondente via Configurações do usuário -> Preferências.

Como usar grupos

  1. Faça login em seu sistema do Urchin como administrador e clique em “Configuração” (menu no canto superior direito).
  2. Clique em “Usuários e grupos”.
  3. Clique em “Grupos”.
  4. Clique no botão “Adicionar”, na parte superior direita.
  5. Insira o Nome do grupo ― ele pode ser descritivo.
  6. Insira a Descrição do grupo ― ele deve ser alguma coisa a ser feita com o local ou a composição do grupo.
  7. Clique em “Concluir”.
  8. Clique em “Mais opções”.
  9. Clique na guia “Usuários no grupo” para selecionar usuários a serem adicionados ao grupo ― para selecionar vários usuários, use a tecla de comando ou a tecla de controle, dependendo de sua plataforma.
    1. Clique na seta para a direita a fim de mover os usuários para a caixa “Usuários no grupo”.
    2. Clique no botão “Atualizar” para salvar as alterações.

Recomendações

  • Qualquer usuário de nível "Superadministrador" tem total controle sobre o sistema Urchin, de modo que é aconselhável conceder esse privilégio a somente uma pessoa.
  • Senhas devem conter uma ou mais letras maiúsculas e/ou símbolos para que sejam difíceis de serem adivinhadas.
