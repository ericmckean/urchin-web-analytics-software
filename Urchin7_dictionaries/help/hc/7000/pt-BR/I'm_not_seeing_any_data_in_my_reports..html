Não estou vendo dados nos relatórios.

Se os relatórios não estiverem exibindo dados:

  • Verifique se o UTM está corretamente referenciado nas suas páginas.Se o arquivo urchin.js não estiver instalado no diretório raiz do seu site, você precisará atualizar a referência em cada uma das suas páginas da web de forma a apontar para o local real.
  • Se o UTM estiver instalado corretamente, talvez você tenha verificado os relatórios antes de os dados serem exibidos.Geralmente, leva várias horas para os dados de relatórios aparecerem na sua conta após a instalação do UTM.
  • Relatórios do Urchin não podem ser visualizados em navegadores Safari no Mac OS X.
