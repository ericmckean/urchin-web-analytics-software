Como faço para instalar scripts de inicialização do Urchin no RedHat?

Existe um script de inicialização do Urchin que é criado na ocasião da instalação.Seu nome é"urchin_daemons" e ele está localizado no diretório util do Urchin."/path/to/urchin/util/"

Copie este script no diretório /etc/rc.d/init.d/.

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • Examine o arquivo /etc/inittab e determine o nível de execução padrão (3 em geral).[[Veja abaixo]]
  • Em seguida, altere para /etc/rc.d/rcX.d (em que X é o nível de execução).
  • cd /etc/rc.d/rc3.d
  • Crie links simbólicos que apontem para /etc/rc.d/init.d/urchin_daemons e com os nomes S95urchin e K05urchin.
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # Nível de execução padrão.Os níveis de execução usados pelo RHS são:

  • 0 - interromper (NÃO defina initdefault para isso)
  • 1 - Modo de usuário único
  • 2 - Vários usuários, sem NFS (O mesmo que 3, se você não tiver conexão de rede)
  • 3 - Modo completo de vários usuários
  • 4 - não utilizado
  • 5 - X11
  • 6 - reinicializar (NÃO defina initdefault para isso)
