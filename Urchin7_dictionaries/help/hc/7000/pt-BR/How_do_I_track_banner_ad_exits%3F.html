Meu site só tem um URL porque estou usando Flash.Como faço para configurar um funil de meta?

É possível monitorar qualquer evento com base no navegador, inclusive eventos em Flash, no seu site.Usando a função JavaScript urchinTracker, é possível atribuir um nome de arquivo de página a qualquer ação do Flash e inserir esse nome na meta ou na etapa de funil adequada.

Para monitorar um evento, chame a função JavaScript urchinTracker() com um argumento que especifique um nome para o evento.Por exemplo, chamar: 

    javascript:urchinTracker('/purchase_funnel/page1.html');

fará com que cada ocorrência do evento em Flash de chamada seja registrada como se fosse uma exibição de página com o nome /purchase_funnel/page1.html.O argumento deve começar com uma barra.

Os nomes de eventos podem ser organizados em qualquer estrutura de diretórios desejada.O argumento caminho/nome do arquivo para urchinTracker não precisa representar um URL real no seu site.

Exemplos de código em Flash 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
