Por que estou vendo "Outros" nos meus relatórios?

Cada registro em um relatório do Urchin reside em um banco de dados de tamanho fixo.Os arquivos que compreendem esse banco de dados são alternados mensalmente.Por isso, existe um certo limite sobre quantos registros exclusivos podem ser relacionados no banco de dados em cada mês.

Se esse limite for excedido, o Urchin criará um último registro denominado "outros".O registro "outros" pode revelar vários relatórios.Assim que o limite for atingido, todos os registros exclusivos adicionais que forem adicionados aos dados de relatório serão movidos para o registro "Outros".Desse modo, se cinco novas páginas forem visualizadas quando o banco de dados estiver lotado, o registro "outros" contará cinco exibições de página adicionais.

O limite para os registros do banco de dados pode ser aumentado.Por padrão, esse limite está definido como 100.000 registros.Isso significa que o Urchin irá registrar um total de 100.000 itens de registro exclusivo todos os meses, para cada perfil.Você poderá aumentar esse número se tiver acesso de "Super administrador".Se não tiver o acesso apropriado, entre em contato com o Administrador do Urchin e solicite que ele efetue as alterações necessárias para você.

Como aumentar o limite por tabela de banco de dados globalmente:

  • Clique em "Configuração" (canto superior direito da interface).
  • Clique em "Configurações" (menu de navegação à esquerda).
  • Clique na guia "Configurações do banco de dados".
  • Use a seta da lista suspensa para aumentar o "Limite da tabela de banco de dados" até o valor desejado.
  • Clique em "Atualizar para salvar as alterações".

Como aumentar o Limite por tabela de banco de dados para um Perfil existente:

  • Clique em "Configuração" (canto superior direito da interface).
  • Clique no ícone de chave inglesa "Editar" referente ao Perfil em questão.
  • Clique na guia "Armazenamento/banco de dados".
  • Altere o valor de Limite por tabela de banco de dados para o novo limite de registro desejado.A utilização de "Padrão global" irá corresponder ao limite do Perfil ao limite da Configuração global atual, conforme observado anteriormente.
  • Clique em "Atualizar".

Observação:lembre-se de que isso irá aumentar a utilização do espaço em disco, bem como o tempo necessário para abrir várias visualizações de relatórios, já que a interface de relatório do Urchin terá que recuperar os dados de um banco de dados maior.
