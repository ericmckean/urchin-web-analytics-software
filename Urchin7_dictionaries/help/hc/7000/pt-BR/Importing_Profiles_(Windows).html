Como importar perfis (Windows)

Visão geral

A função "Importar perfis" do Urchin é uma maneira conveniente de os usuários com sistemas que executam o Microsoft Internet Information Server configurarem Perfis rapidamente para cada um dos seus sites do IIS. O Urchin pode ler a configuração do IIS, determinar que sites da web estão em execução no servidor e, então, criar Perfis básicos para cada site que utiliza os registros do IIS como suas Origens do registro.Dessa forma, você pode personalizar os Perfis ou incluir Perfis adicionais conforme desejar para os sites importados.

Como importar perfis

Para começar a importar Perfis, faça login no sistema de administração do Urchin como admin e clique em "Configuração" (canto superior direito da interface).Clique no botão Importar, no canto superior direito, para acessar a tela "Importar perfis".Essa tela permite selecionar os sites da web que serão importados (se houver).Depois de selecionar os sites a serem importados, clique no botão "Importar".Clique em "Concluído" quando terminar de escolher suas preferências de importação.

Recomendações

Convém criar pelo menos um Perfil para cada site no servidor, para que você possa ter uma perspectiva completa do tráfego até o servidor, por meio dos Relatórios de resumo do Urchin.Os Relatórios de resumo fornecem informações gerais sobre o tráfego do servidor, além de uma classificação para cada site de acordo com vários parâmetros de tráfego.Essas informações serão muito práticas se você for um host e gerar faturas de acordo com o uso da largura de banda.Observe que os Relatórios de resumo somente mostram dados com base nos Perfis que foram configurados -- sites sem Perfis operacionais não são incluídos.
