#format wiki #language en Filtros da tabela de consulta

O filtro "Tabela de consulta" pode ser usado para:

  • implementar códigos de monitoramento master para monitoramento de campanhas.Leia Como usar códigos de monitoramento master.

  • implementar uma tabela de dados externa para consultar e substituir valores de campos de caracteres quando uma correspondência ocorrer.Tabelas de consulta podem ser correspondidas com base em um único campo e podem atualizar vários campos.

Como usar o filtro:

  1. Na tela "Assistente de Filtro:Configurações", insira o nome do filtro desejado (nome "Nome do filtro").

  2. Selecione "Tabela de consulta" como o tipo de filtro.
  3. Selecione o campo que você deseja atualizar no menu "Campo de filtro".

  4. Selecione a tabela pré-configurada (consulte Como usar códigos de monitoramento master) no menu suspenso Nome da tabela.
