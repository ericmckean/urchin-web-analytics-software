Os URL de cada etapa são idênticos para cada etapa do meu Caminho de funil definido.Como posso monitorar cada etapa como se fosse um URL exclusivo?

Você pode monitorar o progresso dos visitantes por meio de um funil que tem o mesmo URL para cada etapa.Por exemplo, o funil de inscrição pode ser:

  • Etapa 1. (Inscreva-se)
    • www.meusite.com.br/sign_up.cgi
  • Etapa 2 (Aceite o contrato)
    • www.meusite.com.br/sign_up.cgi
  • Etapa 3 (Conclua)
    • www.meusite.com.br/sign_up.cgi

Para evitar isso, é possível chamar a função urchinTracker Javascript em cada etapa (provavelmente em um evento onload), desta maneira:

  • urchinTracker("/funnel_G1/step1.html")
  • urchinTracker("/funnel_G1/step2.html")
  • urchinTracker("/funnel_G1/step3.html")

Em seguida, configure cada etapa do funil como:

  • http://www.meusite.com.br/funnel_G1/step1.html

  • http://www.meusite.com.br/funnel_G1/step2.html

  • http://www.meusite.com.br/funnel_G1/step3.html

Observe que o argumento caminho/nome do arquivo para o urchinTracker() não precisa representar um caminho ou nome de arquivo existente.O argumento para urchinTracker fornece de maneira simples um nome de página inventado ao qual o Urchin pode anexar exibições de página.
