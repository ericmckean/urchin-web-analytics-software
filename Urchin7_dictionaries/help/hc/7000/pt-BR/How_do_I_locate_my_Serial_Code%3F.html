Como faço para localizar meu Código de série?

Existem duas maneiras de exibir o seu código de série atual do Urchin.

  1. Console administrativo
    • Inicie o Console administrativo do Urchin a partir de qualquer navegador da web.

      Clique em 'Configuração--> Configurações --> Licença".

  2. Executar o utilitário "inspector"
    • Abra uma janela de comando e acesse o diretório basedir/urchin/util/ directory. Execute o utilitário "inspector".O número de série será impresso na tela, junto com outras informações do Urchin. Observe que no Unix/Linux, todos os comandos devem ser precedidos por './'
