Por que o meu arquivo de registro não pode ser lido, mesmo sendo legível para o grupo?

O número de erro "7026-76-*" que resulta em um arquivo de registro que não pode ser lido, mesmo sendo legível para o grupo, é um problema conhecido na funcionalidade setuid/setgid do binário do urchin.Por questões de segurança, o mecanismo do urchin será executado como proprietário do binário, mesmo sendo iniciado como raiz.Nessas condições, apenas o id de grupo exato, e não a lista de grupos, será visto pelo aplicativo.

Para corrigir o problema, talvez seja necessário alterar as permissões no arquivo de registro para universalmente legível ou alterar o grupo para o grupo exato do usuário do urchin.Isso afeta apenas arquiteturas do tipo UNIX.
