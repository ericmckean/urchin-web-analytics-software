Como faço para remover os meus próprios visitantes internos do relatório de sites da web?

Visão geral

É muito comum usar filtros de endereços IP com Perfis para incluir apenas os endereços IP de um endereço IP específico, excluir os endereços IP de sistemas de monitoramento interno e assim por diante.Isso é facilmente feito com o Urchin. Entretanto, é importante usar a especificação correta de 'Campo de filtro' ao configurar um filtro de endereço IP.

Procedimento

A maneira mais simples de configurar o filtro de tráfego interno apropriado é seguir estas etapas:

  • Navegue até "Configuração -> Perfis do Urchin -> Gerenciador de filtro", na interface de administração do Urchin.

  • Clique no botão "Adicionar" para criar um novo filtro.

  • Escolha "Filtrar tráfego interno" no menu suspenso "=> Selecionar filtros de exemplo".

  • Substitua o exemplo de endereço IP na caixa "Termo de filtragem" pelo endereço IP que você deseja filtrar.Não se esqueça de colocar um caractere de barra invertida ("\") antes de cada ponto no endereço IP.
  • Clique em "Concluir" e em "Mais opções" na janela seguinte.
  • Selecione todos os Perfis aos quais você deseja aplicar o filtro na caixa "Perfis disponíveis" e clique na seta à direita para transferi-los à caixa "Aplicado a".
  • Clique em "Atualizar" para aplicar o filtro aos Perfis selecionados.

Outras considerações

Lembre-se de que os termos de filtragem do Urchin são expressões regulares POSIX.Desse modo, verifique se fez o escape dos caracteres '.' na sua especificação de endereço IP para garantir que o Urchin não os interprete como metacaracteres da expressão regular.Por exemplo, para filtrar o endereço IP '172.16.1.1', especifique o termo de filtragem como '172\.16\.1\.1'.Como outro exemplo, para filtrar o intervalo de endereços IP '192.168.xxx.xxx, o campo de filtro seria especificado como '192\.168\.'.Veja alguns outros exemplos:

Vários endereços IP são separados por símbolos de barra vertical 192\.168\.12\.32$|10\.10\.1\.23$|63\.212\.171\.5$ Observe o uso de $, que indica que nenhum outro número deve ser o último número impresso.Desse modo, para incluir 63.212.171.5 e 63.212.171.53, o seu filtro pode ser escrito como 63\.212\.171\.5.Porém, para corresponder somente 63\.212\.171\.5, seu termo de filtragem deve ser escrito como 63\.212\.171\.5$

Um intervalo de endereços IP pode ser expresso em uma única string. O intervalo de 192.168.10.10 a 192.168.10.50 pode ser escrito como 192.168.10.[1-5][0-9]$.

Corresponder o termo de filtragem 63\.212\.171\.5 ao início da linha também irá corresponder 163\.212\.171\.5.Para incluir somente o IP que começa com um certo número, use a 'expressão de correspondência ao início da string '', como 63\.212\.171\.5$.

Lista completa de expressões regulares 

 *'''Curinga''' '''Significado'''
 *'''.'''       corresponder um caractere qualquer
 *'''*'''       corresponder zero ou mais dos itens anteriores
 *'''+'''       corresponder um ou mais dos itens anteriores
 *'''?'''       corresponder zero ou um item anterior
 *'''()'''      lembrar o conteúdo dos parênteses como um item
 *'''[]'''      corresponder um item nesta lista
 *'''-'''       criar um intervalo em uma lista
 *'''|'''       ou
 *'''^'''       corresponder ao início do campo
 *'''$'''       corresponder ao final do campo
 *'''\'''       fazer escape de todos os curingas acima
