Como trabalhar com contas

Visão geral

Uma Conta é uma associação de alto nível que é usada para agrupar Perfis relacionados, Origens de registro, Usuários e Grupos em um único rótulo de identificação, que, em geral, é um nome de empresa ou cliente.Para instalações do Urchin nas quais é necessário oferecer suporte a várias organizações complexas de clientes, criar uma Conta permite que o administrador do Urchin controle facilmente todos os componentes de geração de relatórios do Urchin para uma pessoa física ou jurídica.Direitos de acesso a relatórios do Urchin podem ser controlados por meio de uma associação de Conta e, dentro de uma Conta, direitos de acesso ainda mais granulares a certos relatórios podem ser atribuídos a Grupos ou Usuários, protegendo, assim, os seus dados em vários níveis desejados.Além disso, direitos de administração em nível de Conta podem ser atribuídos a um usuário que, então, pode atuar como um administrador do Urchin para a Conta.Isso possibilita a distribuição da responsabilidade pelo gerenciamento, pela configuração e pela manutenção dos relatórios do Urchin em uma Conta.

É importante observar que, uma vez que Contas são o "elemento" organizacional de maior nível na interface de administração do Urchin, você deve criar as Contas antes de criar Perfis, Origens de registro, Usuários ou Grupos que serão associados à Conta.A opção de Conta deve ser feita quando um elemento do Urchin é criado e não pode ser alterada depois disso.Na ocasião da criação, se você não escolher uma Conta específica, a Conta padrão (NENHUMA) será definida.Isso informa o Urchin que o elemento em questão não tem Conta.

Como criar contas

Para criar uma Conta, acesse a tela "Configuração->Usuários e grupos -> Contas" e clique no botão "Adicionar".Apenas o "Nome da conta" é necessário. "Contato" e "E-mail de contato" não são usados pelo Urchin, sendo a rigor campos exclusivamente informativos para beneficiar o administrador."Localização dos dados de relatório" (opcional) especifica onde armazenar os dados de relatórios para todos os Perfis que pertencem à Conta.A localização padrão é o diretório "data/reports" na distribuição do Urchin.Alterar a Localização dos dados de relatório permite separar fisicamente no seu sistema de arquivos os dados de relatórios para diferentes organizações.O campo "Localização de navegação do diretório" (opcional) é fornecido como medida de segurança ao conceder ao administrador de uma Conta acesso para criar Perfis.Um diretório inserido nesse campo irá limitar a capacidade do administrador da Conta de navegar pelos arquivos de registro somente nesse diretório.Por padrão, não há restrições quanto ao lugar em que um administrador de Conta pode procurar arquivos de registro.

Como usar contas

Para atribuir um Perfil, uma Origem de registro, um Usuário ou um Grupo a uma Conta, use o menu suspenso denominado Conta opcional, na tela inicial do assistente de configuração, ao criar o elemento em questão.Quando uma Conta estiver atribuída a um Perfil, uma Origem de registro ou a um Usuário, entre outros, a interface de administração do Urchin irá restringir as opções de modificação aos elementos que estiverem associados à Conta.Dessa maneira, a Conta atua no sentido de controlar os direitos de acesso em alto nível, de forma que você possa isolar as organizações uma das outras sem precisar definir permissões de acesso específicas em cada relatório.

Contas também auxiliam na distribuição de responsabilidades de gerenciamento para a configuração do Urchin.Em uma Conta, o administrador principal do Urchin pode atribuir privilégios administrativos locais a usuários dessa Conta.Existem quatro níveis administrativos de controle que podem ser atribuídos a um Usuário.Consulte Como trabalhar com usuários e grupos.

Ao visualizar as telas de administração para os parâmetros de configuração do Urchin, você pode filtrar as entradas de forma a mostrar seletivamente apenas aquelas com um nome de Conta particular, usando o menu suspenso Conta na barra superior da tabela.Embora (NENHUMA) signifique nenhuma associação de conta, para fins de filtragem, (NENHUMA) aparece como uma opção no menu suspenso Conta, para que você possa visualizar apenas as entradas que não estão associadas a uma conta.
