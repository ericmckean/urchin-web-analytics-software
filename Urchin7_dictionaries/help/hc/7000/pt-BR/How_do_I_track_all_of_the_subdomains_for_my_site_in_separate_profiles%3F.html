Como faço para monitorar todos os subdomínios do meu site em perfis separados?

  • Este artigo foi escrito por ivantage Limited, consultor autorizado do software Urchin

O monitoramento de subdomínios em perfis individuais com o Urchin é uma tarefa que depende da configuração do seu servidor da web.É possível configurar o seu servidor da web para registrar arquivos de registro separados para cada subdomínio. Dessa forma, o Urchin poderá processar cada arquivo de registro de subdomínio separadamente.Como alternativa, se apenas um arquivo de registro for gerado e o registro de acesso à web estiver definido para registrar o nome do host, incluindo o subdomínio, um filtro Incluir poderrá ser construído para o perfil de cada subdomínio.Por exemplo, o filtro em um perfil incluiria exclusivamente o tráfego para ajuda.example.com.br, enquanto outro filtro Incluir incluiria apenas o tráfego para www.example.com.br.

Desse modo, para monitorar seus subdomínios em perfis separados, você pode aplicar um filtro Incluir a cada perfil, para incluir somente um nome de host específico.A primeira etapa é criar um perfil para cada subdomínio.Feito isso, você pode criar os filtros descritos a seguir.

Por exemplo, se deseja monitorar sub.example.com.br e www.example.com.br em dois perfis separados, aplique os filtros a seguir:

  • Para o perfil do sub.example.com.br 

    Tipo de filtro:SOMENTE padrão Incluir
    Campo de filtragem:utm_nomehost_solicitação (AUTO)
    Termo de filtragem:^sub\.example\.com.br
  • Para o perfil do www.example.com.br 

    Tipo de filtro:SOMENTE padrão Incluir 
    Campo de filtragem:utm_solicitação_nomedehost (AUTO)
    Termo de filtragem:^www\.example\.com.br
