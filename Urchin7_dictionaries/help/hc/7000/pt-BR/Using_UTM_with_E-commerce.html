Como usar o UTM com E-commerce

Visão geral

Como o principal aspecto do UTM é a capacidade de identificar e correlacionar a atividade dos visitantes, é possível monitorar nos seus sites e registrar coletivamente em relatórios a parcela dessa atividade que é capaz de gerar receita.As transações no servidor que hospeda o seu software de carrinho de compras podem estar correlacionadas com sessões no seu outro servidor da web, permitindo que variáveis de sessão, como referências e palavras-chave, sejam registradas em comparação à receita que elas geram.O UTM oferece monitoramento entre várias sessões, que monitora o visitante desde a origem até a compra ou a meta.Proporções de conversão e relatórios de ROI oferecem resultados detalhados dos esforços de marketing on-line, incluindo compras por palavra-chave, campanhas de e-mail e trocas de links.

Configuração no mesmo domínio

Se o site da web front-end e o site seguro de e-commerce usarem o mesmo domínio, a instalação do UTM no seu site de e-commerce não será diferente da instalação de outros tipos de sites da web.As informações nas outras áreas desta seção sobre a instalação do UTM fornecerão os detalhes específicos da instalação.Atenção especial deve ser dada às áreas que explicam como definir o domínio do UTM apropriadamente para o site de e-commerce e outros sites.Informações adicionais sobre formatos de registro de transações de e-commerce são fornecidas na seção E-commerce da documentação.

Configuração entre domínios

É cada vez mais comum para sites da web com carrinhos de compra de e-commerce terceirizar o componente de e-commerce para outra organização, como o Amazon, o PayPal ou o Yahoo Stores.Isso pode criar um problema para o monitoramento do UTM, pois o domínio do UTM muda conforme um visitante se dirige do site principal para a loja segura.Para que a loja segura utilize o mesmo ID de visitante do UTM que o site principal, esse ID de visitante deve ser transferido no link para a loja segura. O UTM contém uma função utmLinker, que irá transportar o link com o id apropriado antes de enviar esse link para a loja.Em vez de se vincular diretamente com a loja, basta transferir o link para a função utmLinker.Veja as instruções específicas para o uso da função utmLinker:

  1. Edite o arquivo urchin.js na raiz de documentos de ambos os sites da web, defina a variável _ulink como 1 e a variável _udn como "none".
  2. Defina o Domínio UTM do perfil como nada (em branco).
  3. Altere os links do site principal para o site seguro, conforme indicado a seguir: 

   <a
   href='javascript:__utmLinker("https://previous_link?with_parameters");'>
   link-para-carrinho-de-compras</a>
