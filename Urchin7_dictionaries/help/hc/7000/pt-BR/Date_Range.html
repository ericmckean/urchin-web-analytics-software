Período

Visão geral

A função Período do Urchin permite visualizar dados de relatório para qualquer cronograma, desde um único dia até o espaço de tempo completo no qual você possui dados, ou para qualquer parte desse espaço. Você pode clicar em um ano, um mês, uma semana ou um dia para selecioná-lo ou pode inserir um cronograma personalizado (como de 3 a 14 de março de 2002).Também, pode comparar dados referentes a dois períodos.O período (e o período de comparação) selecionados permanecerão ativos até que você os altere ou saia da interface de geração de relatórios.

Usando a função Período

  • Período padrão:para visualizar dados de acordo com um cronograma padrão, como um dia, uma semana, um mês ou um ano, clique no período desejado no calendário do Período, e os dados do relatório serão modificados de forma correspondente.
  • Período personalizado:clique no ícone de calendário simples.É exibido um painel mostrando dois calendários.Selecione a data de início no primeiro calendário e a data de término no segundo calendário e clique em Aplicar intervalo.Os dados no relatório atual serão alterados para refletir o Período que você selecionou.
  • Período de comparação:clique no ícone de calendário duplo.Aparecerá um segundo calendário abaixo do primeiro.Defina o período para o seu período de comparação no segundo calendário.

Recomendações

  • Se estiver examinando um site com pouco tráfego, tente usar um cronograma maior para obter dados mais significativos.
  • Se estiver interessado em tendências de tráfego no decorrer da vida útil do seu site (e se houver dados para isso), tente analisar um ano ou mais de dados equivalentes. O Urchin irá ajustar o tamanho dos elementos do gráfico de barras para acomodar a seleção.
