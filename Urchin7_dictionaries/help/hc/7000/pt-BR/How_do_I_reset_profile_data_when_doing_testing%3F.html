Como faço para redefinir dados de perfil ao efetuar testes?

Talvez seja preferível executar um perfil do Urchin várias vezes com o objetivo de efetuar testes de desempenho, filtros e assim por diante, nos quais os dados reais gerados pelo Urchin são descartados depois de cada teste.A maneira mais simples de fazer isso é remover o diretório ou a pasta em que os dados de perfil são armazenados depois de cada execução.Esses diretórios são armazenados de acordo com o nome do perfil na distribuição do Urchin, na área "data/reports".A remoção do diretório inteiro, nomeado de acordo com o perfil que está sendo testado, irá redefinir todos os dados, incluindo as informações de monitoramento de registros.
