Visão geral dos utilitários de administração

Visão geral

O Urchin vem acompanhado de vários programas utilitários que são usados para fins de diagnóstico e configuração. Esses utilitários estão localizados no diretório util da distribuição do Urchin. Este documento tem como objetivo proporcionar uma visão geral introdutória desses utilitários. Não é um guia abrangente para a sua utilização.

Todos os utilitários são compatíveis com os argumentos de linha de comando “-h” e “-v”. Invocar um utilitário com o argumento "-h" fornecerá um resumo da utilização e das opções disponíveis dessa ferramenta. Invocar com um argumento "-v" imprime a versão do Urchin do utilitário.

geo-update

Esse utilitário é usado para verificar as atualizações do banco de dados DNS interno do Urchin e para fazer download das atualizações, se estiverem disponíveis. Ele permite fazer download das atualizações dos servidores de download do Google, bem como de alguns diretórios locais (novidade no urchin 6).

inspector

Esse utilitário executa verificações básicas em sua distribuição instalada do Urchin, garantindo que a estrutura geral dessa distribuição permaneça intacta, que todos os binários fornecidos com o produto tenham a versão adequada e que as permissões subjacentes estejam corretas (nas plataformas UNIX). O utilitário também informa o status operacional dos Agendadores de perfil do Urchin (urchind), dos Agendadores de CPC (urchincpcd, urchincpc) e do servidor da web Apache fornecido (urchinwebd). Ele também verifica as informações de licenciamento.

uconf-manager (Urchin 6.402+)

O Utilitário gerenciador de configuração do Urchin (uconf-manager) permite adicionar, editar e excluir registros de bancos de dados.

  • Urchin 6.402: esse utilitário foi lançado pela primeira vez no Urchin 6.402 e só funcionava na tabela 'uusers' do banco de dados de configuração. Ele pode ser usado para redefinir a senha de administrador do Urchin, automatizar a criação de usuários e mover um usuário para uma conta diferente.
  • Urchin 6.500+: no urchin 6.5 e nas versões posteriores, esse utilitário é compatível com operações de banco de dados em mais tabelas (uaccounts, ucosts, ufilters, ugoals, ugroups, ugroups_uusers, ulogs, ulogs_ufilters, uprofiles, uprofiles_ucosts, uprofiles_ufilters, uprofiles_ugroups, uprofiles_ulogs, uprofiles_uusers, e uusers). Esse utilitário pode ser usado para executar ações que não podem ser realizadas pela interface de administração do Urchin.

uconf2sql

O Urchin 5 e as versões mais antigas armazenam informações de configuração em arquivos proprietários. No entanto, no Urchin 6, essas informações são armazenadas em um banco de dados relacional (MySql ou PostgreSql). Para suavizar a transição das informações de configuração, o Urchin 5 e o Urchin 6 fornecem utilitários para exportar e importar informações de configuração. O Urchin 5 tem um utilitário, chamado uconf-export, para exportar o banco de dados de configuração em um arquivo (em formato XML) e o Urchin 6 tem um utilitário, chamado uconf2sql, para importar os dados de configuração desse arquivo XML para o banco de dados do Urchin 6.

convert-u5data

Como no banco de dados de configuração, a estrutura do banco de dados de relatórios (que armazena as informações de relatório) também foi alterada no Urchin 6. Esse utilitário, convert-u5data, oferece uma opção para importar o banco de dados de relatórios diretamente da instalação do Urchin 5 para a instalação do urchin 6.

udb-sanitizer

O utilitário udb-sanitizer é usado para executar operações relacionadas ao gerenciamento no banco de dados de relatórios. Isso inclui a reversão para um banco de dados arquivado, a exclusão de informações sobre uma data específica ou um intervalo de datas (em caso de necessidade de um novo processamento).

urchinctl

Observação: esse utilitário fica no diretório bin, e não no diretório util. Ele oferece meios de iniciar e interromper os serviços Agendador do Urchin e Servidor da web do Urchin. Em sistemas UNIX, urchinctl geralmente é chamado de um dos scripts em tempo de inicialização do sistema para iniciar ou encerrar os serviços do Urchin automaticamente. Na plataforma windows, ele também oferece opções para instalar e desinstalar os serviços do urchin.

urchincpc (Urchin 6.5+)

Observação: esse utilitário fica no diretório bin, e não no diretório util. A ferramenta de importação de dados de CPC do Urchin extrai dados de CPC através da API do Google AdWords para fins de processamento e geração de relatórios. Para obter mais informações, consulte Utilitários de importação de dados de CPC.

urchincpcd (Urchin 6.5+)

Observação: esse utilitário fica no diretório bin, e não no diretório util. O Agendador de origem de CPC do Urchin (ou simplesmente Agendador de CPC) é usado para agendar tarefas de download de CPC com base em uma programação definida pelo usuário (similar ao Agendador de perfil). Para obter mais informações, consulte Utilitários de importação de dados de CPC.

urchin_daemons

Em máquinas UNIX, o Urchin executa um daemon agendador (urchind) e um servidor da web Apache (urchinwebd). Esses programas devem ser iniciados no momento da inicialização. O script urchin_daemons pode ser adicionado aos scripts de inicialização do sistema no local apropriado para o seu sistema operacional UNIX e ele fará com que os daemons de serviços do Urchin sejam iniciados adequadamente. Em plataformas windows, esses binários são executados como serviços do sistema e podem ser iniciados e interrompidos com o uso do gerenciador de serviços do windows.

ugetlicense

O Utilitário de operação de licenças do Urchin, ugetlicense, permite executar operações em um número de série. Essas operações serão necessárias se uma instalação atual do Urchin se tornar indisponível e precisar ser reinstalada ou migrada para um servidor diferente, entre outras situações.
