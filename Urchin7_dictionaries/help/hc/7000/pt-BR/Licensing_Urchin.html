Licenciamento do Urchin

O Urchin pode ser licenciado de três maneiras:

  • As licenças normais (antes do Urchin 6.6) nunca expiram.
  • As licenças USP (Urchin 6.6+) têm uma data de validade que é determinada durante a compra.
  • As licenças de demonstração têm a mesma funcionalidade das licenças normais e USP, mas é possível ter no máximo cinco perfis e cinco origens de registro.

Como obter e ativar uma licença de demonstração

Para obter uma licença de demonstração, faça login na interface de administração do Urchin com um navegador da web (geralmente, http://your.server.com:9999). Clique em "Obter licença de demonstração" e aceite o Contrato de licença para concluir o processo.

Após obter sua licença de demonstração, será exibida uma tela mostrando os novos detalhes de sua licença. Esses detalhes incluem: tipo de licença, código de série, número de perfis, origens de registro paralelas, registros por perfil e sistema operacional. É sempre recomendável salvar as informações de licença, especialmente o código de série, para referência futura. Após instalar o Urchin com êxito, os detalhes da licença (status e validade da licença, código de série e versão) poderão ser visualizados navegando em Página inicial -> Configuração -> Configurações -> Licença.

Após examinar os detalhes da licença, ative a licença de demonstração. Clique no link "Instalar a licença" exibido na tela "Informações da licença". Isso iniciará o assistente. É recomendável substituir a senha administrativa padrão por uma nova.

OBSERVAÇÃO: ao selecionar as opções para "Configurações de acesso remoto" e "Configurações do painel de controle administrativo", não se esqueça de selecionar a opção em "Configurações do painel de controle administrativo" que permite adicionar níveis de direitos administrativos a um ou vários perfis em um grupo. Você não poderá reverter essa configuração após a conclusão da instalação.

Após a conclusão bem-sucedida da ativação na nova licença de demonstração, você verá a tela inicial "Administração" do Urchin. Nessa tela, é possível criar perfis, processar registros, visualizar relatórios, configurar preferências etc.

Quando a licença de demonstração expirar, a tela "Licença do Urchin" será exibida com o seguinte status de licença em detalhes: "Licença válida - Demonstração (vencida)". Para instalar uma licença adquirida previamente, clique no link "Ativar licença".

Quando a licença de demonstração ficar inválida, a tela "Licença do Urchin" será exibida com o seguinte status de licença em detalhes: "Licença inválida". Para reativar a demonstração existente, clique no link "Reativar licença" ou clique em "Ativar licença" para instalar a licença adquirida previamente.

Como adquirir uma licença normal

Entre em contato com um Consultor autorizado do software Urchin para adquirir uma licença.

Como ativar uma licença normal adquirida

Depois de receber seu novo código de série para uma licença permanente do Urchin, você precisará ativá-la da seguinte maneira:

  • Navegue até Configuração -> Configurações -> Licença.

  • Clique no link "Ativar licença".
  • Na tela “Ativar código de série”, insira seu endereço de e-mail válido e o código de série.

Após concluir esse processo, o Urchin funcionará completamente e sua licença nunca irá expirar.

Importante: não se esqueça de inserir informações de contato precisas ao obter ou ativar sua licença de demonstração, pois essas informações serão necessárias para o faturamento e o suporte. Após instalar o Urchin com êxito, os detalhes da licença (status e validade da licença, código de série e versão) poderão ser visualizados navegando em Página inicial -> Configuração -> Configurações -> Licença.

Se uma licença normal ficar inválida, a tela "Licença do Urchin" será exibida com o seguinte status de licença em detalhes: "Licença inválida". Para reativar uma licença normal existente, clique no link "Reativar licença" ou clique em "Ativar licença" para instalar a licença USP adquirida previamente.

Como instalar uma licença sem acesso à internet

É possível licenciar o Urchin sem acesso à Internet, por exemplo, por trás de um firewall. Para fazer isso, você precisará executar o utilitário "inspector", fornecido junto com o Urchin e disponível no diretório "util". Para obter instruções sobre como executar o utilitário "inspector", leia inspector: verificador da integridade da instalação do Urchin. Depois de obter a saída do "inspector", entre em contato com seu Consultor autorizado do software Urchin para obter instruções manuais de ativação.

Como adquirir uma licença USP

Entre em contato com um Consultor autorizado do software Urchin para adquirir uma licença.

Como ativar uma licença USP adquirida

Depois de receber seu novo código de série para uma licença permanente do Urchin, você precisará ativá-la da seguinte maneira:

  • Navegue até Configuração -> Configurações -> Licença

  • Clique em "Ativar licença"
  • Na tela “Ativar código de série”, insira seu código de série.

Depois que você realizar essas etapas, o Urchin funcionará completamente até a licença expirar.

Se a sua licença USP expirar, entre em contato com um Consultor autorizado do software Urchin para adquirir uma licença.

Atualização para o Urchin 6.6

A atualização de uma instância do Urchin que está usando uma licença de demonstração não é recomendada e pode resultar em problemas de login/licenciamento. Nenhuma alteração de licenciamento deverá ser realizada depois da atualização se a versão anterior do Urchin tiver sido usada com uma licença normal no Windows. A licença deverá ser reativada depois de uma atualização se a versão anterior do Urchin tiver sido instalada com uma licença normal no Linux/FreeBSD (clique no link "Reativar licença" na tela "Licenciar Urchin").

Se uma licença normal ficar inválida (em qualquer plataforma), a tela "Licença do Urchin" será exibida com o seguinte status de licença: "Licença inválida". Para reativar uma licença normal existente, clique no link "Reativar licença" ou, para instalar uma licença USP adquirida previamente, clique em "Ativar licença".
