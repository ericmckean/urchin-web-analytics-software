Como faço para recuperar do erro "Não é possível abrir o banco de dados para gravação porque ele foi arquivado"?

Ao tentar executar um perfil no Urchin 6, você recebe este erro: 

WARNING: (7063-54-63) Unable to open database for writing since it has been archived

Esse erro ocorre porque os bancos de dados de controle de registro são armazenados mensalmente com o Urchin.Durante o processamento, o Urchin faz a leitura das primeiras linhas no arquivo de registro, determina a que mês esse registro pertence e tenta abrir os bancos de dados referentes a esse mês.Como os bancos de dados do mês foram arquivados, um erro é retornado, já que o Urchin não permitirá a leitura/gravação em bancos de dados que foram arquivados. Existem duas soluções:

  1. Desative o recurso de arquivamento e descompacte os arquivos *-archive.zip para os perfis afetados e remova os arquivos *-archive.zip.
  2. Altere a atribuição de curingas do registro de forma a incluir uma string de correspondência de data (é possível usar YY* ou YYYY* para identificar arquivos referentes ao ano atual).
