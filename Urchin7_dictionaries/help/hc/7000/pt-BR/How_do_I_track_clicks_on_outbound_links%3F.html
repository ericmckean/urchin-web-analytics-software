Como faço para monitorar cliques em links externos?

Se você possui links externos no seu site, é fácil monitorar em quais links os visitantes clicam para sair do seu site.

Para monitorar cliques externos, adicione o seguinte código à tag <a>: 

    <a href="http://www.site.com" onClick="javascript:urchinTracker('/outboundlinks/sitename/pagename');">

Esse código faz com que cada clique no link externo seja registrado como se fosse uma exibição de página com o nome /outboundlinks/sitename/pagename.É conveniente registrar todos os seus links externos em uma estrutura lógica de diretório, como mostra o exemplo.Assim, você irá identificar facilmente as páginas em que os visitantes clicaram para sair do site.
