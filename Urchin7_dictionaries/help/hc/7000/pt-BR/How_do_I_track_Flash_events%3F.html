Como faço para monitorar eventos em Flash?

O Urchin permite monitorar qualquer evento com base no navegador, inclusive eventos em Flash e Javascript.Para monitorar um evento, chame a função JavaScript urchinTracker com um argumento que especifique um nome para o evento.Por exemplo, chamar: 

    javascript:urchinTracker('/homepage/flashbuttons/button1');

fará com que cada ocorrência desse evento em Flash seja registrada como se fosse uma exibição de página de /homepage/flashbuttons/button1.O argumento deve começar com uma barra.Os nomes do evento podem ser organizados em qualquer estrutura de diretórios desejada, e as páginas reais com esses nomes de arquivo não precisam existir.

Por exemplo, se quiser organizar os eventos em flash por página, por tipo de evento, você pode organizar uma hierarquia como esta: 

    /homepage/flashbuttons/button1
    /homepage/clips/clip1

Exemplos de código em Flash 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file');");
    }
