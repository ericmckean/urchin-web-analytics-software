Como criar entradas personalizadas para um idioma

  1. Copie o arquivo de idioma xx-XX.txt da pasta <Urchin>/lib/admin/languages/ para <Urchin>/lib/custom/languages/.

  2. Anexe novas entradas ao final do arquivo e atribua um novo número a cada entrada.

  3. O separador entre o número e o valor deve ser um <Tab>. Caso contrário, o Urchin não o analisará.

  4. Verifique se os números de entrada são consecutivos (acréscimo de um número)
  5. Recarregue o Urchin.

  6. Após o recarregamento, o Urchin deverá criar um arquivo DB de idioma chamado xx-XX.db, localizado na pasta <Urchin>/lib/custom/languages/.

Depois de definidas, as entradas personalizadas podem ser usadas na seção do relatório como rótulos ou títulos de colunas ou como descrições de ajuda.

Exemplo de arquivo de idioma

As entradas 738, 739 e 740 são entradas personalizadas adicionadas. 

  ...
  731     Executive
  732     Marketer
  733     Webmaster
  734     Dashboards
  735     All Reports
  736     ja-jp-mac (Japanese)
  737     ja-jp-osaka (Japanese)
  738     Modules
  739     Actions
  740     Modules / Actions

Observação importante: A atualização do Urchin pode substituir os arquivos de idioma originais e adicionar entradas padrão. Lembre-se de recriar os seus arquivos de idioma personalizados e atualizar os relatórios personalizados.
