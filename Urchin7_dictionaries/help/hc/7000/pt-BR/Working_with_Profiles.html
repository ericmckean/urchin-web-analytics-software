Como trabalhar com perfis

Visão geral

Um perfil é um conjunto de relatórios para um siteÉ necessário configurar pelo menos um perfil por sitee em diversas circunstâncias, convém configurar mais de um perfil por site. Porém, em linhas gerais, existe uma proporção de 1:1 entre perfis e sites.

Sob um ponto de vista mais técnico, um perfil consiste em um conjunto de regras que controlam o processamento de arquivos de registro e a criação de relatórios para um site ou partes dele.

Os perfis são as principais ferramentas organizacionais do sistema Urchin e, por isso, é importante compreender como eles funcionam.

Informações de licenciamento

A licença base do Urchin 6 inclui 1000 Perfis.Cada um deles pode ter uma Origem de registro, ou mais de um Perfil pode compartilhar uma Origem de registro (o que é útil quando um Perfil é o "master" e um ou mais dos restantes possuem Filtros aplicados).

Cada site deve ter pelo menos um Perfil associado a ele para obter relatórios.Em geral, um é suficiente, mas não há limite quanto ao número de perfis que podem ser criados para a análise especializada de uma única propriedade da web.

Como usar perfis

Para criar um Perfil, faça login na interface de administração do Urchin como administrador do Urchin e clique no link "Configuration", no canto superior direito da tela.Você será levado ao "Add Profile Wizard ".Esse assistente contém uma série de etapas simples projetadas para ajudar você a configurar o Perfil de maneira rápida e fácil em seu formato básico.Cada tela do assistente inclui informações de ajuda explícitas.

Recomendações

  • Para obter a qualidade ideal de geração de relatórios do Urchin, é importante ativar o Urchin Traffic Monitor (UTM), especialmente se forem desejadas informações sobre visitantes únicos.Há três etapas básicas necessárias para usar o UTM:
    1. Na guia "Configurações" de qualquer Perfil, selecione "Urchin Traffic Monitor (UTM)" como o Método de monitoramento de visitantes.
    2. Instale o sensor UTM em cada página que você deseja monitorar com a maior precisão possível.Se você não usar o UTM, as páginas ainda serão monitoradas, mas algumas ações, como recargas e a utilização do botão Voltar, não serão reportadas.
    3. Modifique o comportamento de registro do servidor da web para ativar os cookies.
  • Se você pretende configurar um ou mais Filtros juntamente com o seu perfil, convém possuir mais de um perfil para o site ou para parte desse site.Recomendamos que você possua um perfil "master" -- que contenha tudo.Se você quiser, por exemplo, filtrar indexadores ou robôs, é uma boa idéia colocar esses filtros em um segundo perfil.Com isso, será possível comparar os resultados do perfil de dados filtrado com os do perfil master:
