Como monitorar transações de loja seguras

Este artigo foi escrito por ivantage Limited, consultor autorizado do software Urchin

O Urchin pode monitorar dados de um carrinho de compras no seu próprio domínio ou em outros domínios com a adição de algumas linhas de código.Para configurar suas páginas de forma a comunicar dados de transações ao Urchin, leia Como posso monitorar transações de e-commerce?.

Além disso, se o seu site de loja estiver em um subdomínio ou domínio diferente do que o seu site principal, siga as instruções abaixo para configurar as suas páginas.

Se seu site iniciar um processo de pagamento de compras em um subdomínio diferente (por exemplo, se você enviar clientes de www.minhaloja.com.br para carrinho.minhaloja.com.br):

  • Adicione a linha a seguir (em negrito) ao seu código de monitoramento, tanto no site da loja quanto nas páginas do carrinho de compras: 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

Se o seu site iniciar um processo de pagamento de compras em um site de loja separado (por exemplo, se você enviar os clientes de www.minhaloja.com.br para www.carrinhoseguro.com.br):

  • Adicione as linhas a seguir (em negrito) ao seu código de monitoramento, tanto no site da loja quanto nas páginas do carrinho de compras: 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • Altere os links do site principal para o site seguro, de forma a usar utmLinker da seguinte maneira.Se os seus links atuais forem semelhantes a: 

     <a href="https://www.securecart.com/?store=parameters">Comprar agora</a>
  • mude-os para: 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">Comprar agora</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">Comprar agora</a>
   </noscript>

  • O código acima fornece links para usuários com ou sem JavaScript ativado.Lembre-se de que os apóstrofos precisam ser seguidos de uma barra invertida no lugar em que aparecem no link ou no texto do link. Importante:se as suas páginas incluírem uma chamada para urchinTracker(), utmLinker(), utmSetTrans() ou utmLinkPost(), o código de monitoramento do Analytics deverá ser inserido no código HTML, acima de todas essas chamadas.Nesses casos, o código de monitoramento pode ser inserido em qualquer lugar entre a tag <body> de abertura e a chamada JavaScript.

Se você enviar informações ao carrinho de compras usando formulários, use a função utmLinkPost().Essa função acrescenta dados de cookies ao URL do formulário fornecido. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • Isso funcionará mesmo para formulários em que method="GET"
