Visão geral da administração

Introdução

A Interface de administração do Urchin é uma central de comandos com base em navegador a partir da qual você pode controlar praticamente tudo aquilo que se relaciona à execução do Urchin, incluindo a configuração de Perfis, o agendamento de eventos de processamento de registros, o gerenciamento de Usuários e Grupos, a configuração de Filtros e muito mais.

Para começar, faça login no sistema Urchin usando um navegador.Se a porta padrão tiver sido usada durante a instalação, o URL deverá ser http://seu.servidor.com.br:9999/, substituindo "seu.servidor.com.br" pelo nome real do sistema no qual o Urchin está sendo executado.Como alternativa, será possível usar http://localhost:9999/ se você estiver diretamente no sistema.Em plataformas Windows, existe um atalho para "Administração do Urchin" no menu Iniciar.A senha padrão da conta "admin" é "urchin".Lembre-se de alterá-la para uma senha mais segura.

Como usar a interface de administração

  • Clique em "Configuração", no canto superior direito da tela, para acessar a interface de administração.
  • Essa interface fornece acesso a controles de configuração.Na navegação esquerda, clique nas setas para expandir uma seção específica (ou seja, "Perfis do Urchin", "Usuários e grupos", "Agendador" e "Configurações").A cor mais escura indica qual tela está sendo exibida no momento.
  • Clicar no botão "Editar" ao lado de uma entrada particular permitirá que você modifique a configuração referente a essa entrada.Para adicionar novas entradas, clique no botão "Adicionar" (o ícone de sinal de adição "+"), no canto superior direito da tabela.Depois que você clicar em "Editar" em uma entrada específica, o conjunto de telas de configuração disponíveis para essa entrada será apresentado com o uso de guias distribuídas pela parte superior da visualização.
  • Clique em qualquer guia para acessar as definições de configuração na guia em questão.Depois de alterar qualquer configuração, não deixe de clicar no botão "Atualizar", na parte inferior de cada tela.
  • Quando você tiver uma longa lista de entradas em uma área particular, poderá rolar usando os botões "Próxima/Anterior", localizados logo acima e abaixo da lista de entradas da tabela.Também é possível alterar o número de entradas exibidas na tabela, modificando para isso o valor "N° mostrado" (o padrão é 10) que é exibido acima da lista.
  • A opção "Filtro + - (incluir/excluir) pode ajudar você a localizar rapidamente uma entrada particular.Basta inserir o nome completo ou parcial da entrada e pressionar o botão de adição (incluir) ou subtração (excluir).
  • Informações de ajuda para cada tela são fornecidas abaixo da tabela.
