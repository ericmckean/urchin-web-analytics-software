Como posso fazer com que o Urchin identifique mecanismos de pesquisa adicionais nos relatórios de referências?

Por padrão, o Urchin monitora referências de 20 mecanismos de pesquisa relacionados no seu arquivo do UTM.Também, é possível configurar o Urchin para identificar referências a partir de mecanismos adicionais de pesquisa, editando para isso o arquivo UTM, da seguinte maneira:

  1. Abra o arquivo urchin.js em um editor de texto.
  2. Role para baixo até o seguinte: 

          //-- Auto/Organic Sources and Keywords
          var _uOsr=new Array();
          var _uOkw=new Array();
  1. Você verá entradas para mecanismos de pesquisa numeradas de 0 a 19.Esses são os mecanismos de pesquisa que o Urchin monitora por padrão.Para adicionar outro mecanismo de pesquisa à lista, basta adicionar uma nova entrada que siga o formato das outras entradas.Por exemplo, veja como você pode adicionar uma entrada para um mecanismo de pesquisa "new": 

          _uOsr[17]="voila"; _uOkw[17]="kw";
          _uOsr[18]="virgilio"; _uOkw[18]="qs";
          _uOsr[19]="teoma"; _uOkw[19]="q";
          _uOsr[20]="new"; _uOkw[20]="s";
  • O valor para _uOsr deve ser o nome do mecanismo de pesquisa, enquanto o valor para _uOkw deve ser a variável de consulta que armazena a palavra-chave.Por exemplo, se alguém pesquisar "motocicleta" e o URL do resultado da pesquisa for: 

          http://www.google.com/search?q=motorcycle
  • o valor de _uOkw será "q". o valor de _uOsr será "google".
