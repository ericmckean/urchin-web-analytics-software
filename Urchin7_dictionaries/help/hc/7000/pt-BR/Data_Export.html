Exportação de dados

Os botões de exportação de dados do Urchin facilitam a extração de dados de qualquer relatório do Urchin.Isso é muito útil para transferir dados de relatório a uma planilha, processador de texto, banco de dados, etc. para análise mais detalhada.

Para exportar dados de qualquer relatório, selecione o tipo de exportação apropriado com base no aplicativo que você planeja usar para manipular os dados.Para importação geral de banco de dados, use o formato separado por tabulação.

  • Tabulação:clique no botão "T" para exportar dados no formato delimitado por tabulação.
  • Word: clique no ícone do Word para exportar dados no formato nativo do Microsoft Word.
  • Excel: clique no ícone do Excel para exportar dados no formato nativo do Microsoft Excel.

Impressão:clique no ícone de impressora para obter uma visualização dos dados para impressão; clique no link Imprimir página nessa tela para imprimir o relatório.
