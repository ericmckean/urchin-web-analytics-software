Posso mover o diretório "data" para outro local?

Visão geral

O diretório "data" do Urchin é um subdiretório diretório raiz do programa Urchin por padrão.Este local pode ser facilmente alterado por meio da edição do arquivo ~urchin/etc/urchin.conf e movendo manualmente o diretório ~urchin/data existente para o local desejado.É importante observar que esse procedimento pode violar a licença existente e, nesse caso, você precisará entrar em contato com o seu Consultor autorizado do software Urchin e solicitar a redefinição do seu número de série.

Procedimento para mover o diretório de dados:

O local em que o Urchin armazena seus dados de relatório pode ser alterado no arquivo urchin.conf, que está localizado no diretório "etc" do diretório de instalação principal do Urchin.

Para sistemas UNIX:

  1. Abra um shell de comando como o usuário usado pelo Urchin para instalação. 2.Especifique "cd" para o diretório no qual o Urchin está instalado.
  2. Pare os serviços do Urchin com o comando:bin/urchinctl stop
  3. Usando um editor de texto, abra o arquivo urchin.conf no diretório "etc" da distribuição do Urchin.
  4. Exclua as barras de comentário na linha a seguir, removendo o caractere "#" inicial:

    • #dataDirectory:./data/ e substitua o caminho de diretório completo que você deseja usar no lugar do diretório padrão; por exemplo dataDirectory: /bigdisk/urchin/data/

  5. Se necessário, crie o novo diretório de dados.Observação importante: esse diretório deve ser legível e gravável pelo usuário com o qual o Urchin é executado.

  6. Copie os dados existentes para o novo local.Para garantir que todas as permissões e dados sejam devidamente preservados, convém usar o seguinte comando:
    • cd data; tar cf - .| (cd /bigdisk/urchin/data; tar xpf -)
  7. Renomeie o diretório de dados como "data.old".Você poderá removê-lo completamente se desejar, mas convém garantir que tudo esteja funcionando corretamente antes de fazer isso.
  8. Para facilitar a administração, convém criar um symlink no diretório principal do Urchin que aponte para esse novo local; por exemplo,
    • ln -s /bigdisk/urchin/data ./data
  9. Reinicie os serviços do Urchin com o comando:bin/urchinctl start
  10. Faça login no Urchin como o usuário admin.A tela "Licença" do Urchin será apresentada.Para concluir o processo, basta clicar no link "Reativar licença".

Para sistemas Windows:

  1. Pare os Serviços do Urchin:"Iniciar -> Programas -> Urchin -> Desativar serviços do Urchin".

  2. Abra o Windows Explorer e navegue até a pasta "etc" da distribuição do Urchin.Por padrão, essa pasta está em C:\Program Files\Urchin\etc.
  3. Usando um editor de texto, abra o arquivo urchin.conf.
  4. Exclua as barras de comentário na linha a seguir, removendo o caractere "#" inicial:

    • #dataDirectory:./data/ e substitua a letra da unidade e o nome do caminho completo que você deseja usar no lugar da pasta padrão; por exemplo,

      • dataDirectory: E:\Urchin\data
  1. Se necessário, crie a nova pasta de dados.Observação importante:as permissões nessa pasta devem permitir acesso de leitura e gravação aos serviços do Urchin.
    1. Copie o conteúdo da pasta de dados para a nova pasta.
    2. Renomeie a pasta de dados como "data.old".Você poderá removê-lo completamente se desejar, mas convém garantir que tudo esteja funcionando corretamente antes de fazer isso.

    3. Reinicie os Serviços do Urchin:"Iniciar -> Programas -> Urchin -> Ativar serviços do Urchin".

Considerações

Devido à maneira como as licenças são implementadas em sistemas UNIX para impedir a adulteração, mover o diretório "data" do Urchin exigirá a reativação da licença do Urchin.Entretanto, a operação de relicenciamento é extremamente simples - basta fazer login como usuário admin do Urchin e clicar em "Reativar licença".
