Installazione rapida di UTM (Apache)

Articolo di Panalysis, partner autorizzato del software Urchin

Nel seguente paragrafo viene rapidamente illustrato come installare UTM per i siti web gestiti su un server Apache con tutte le piattaforme.

Passaggio 1. Copia i file UTM nella root dei documenti del sito web. I file, urchin.js e __utm.gif sono ubicati nella directory "util/utm" della distribuzione Urchin. Copia questi due file nella directory principale dei contenuti del sito web.

Passaggio 2. Riferimento UTM nel codice HTML. In tutte le pagine HTML inserisci la riga indicata di seguito. Puoi posizionarli in qualsiasi punto delle pagine, tuttavia, è consigliabile inserirla nella sezione <head>. Se utilizzi una normale inclusione o un modello, puoi inserire il codice in quel punto. IMPORTANTE: il nome del file GIF comincia con due trattini di sottolineatura. 

<script src="/urchin.js" type="text/javascript"></script>
<script type="text/javascript">
        urchinTracker();
</script>

Se utilizzi un pacchetto come "HTML Tidy", puoi inserire la riga Javascript nell'area HEAD della pagina, ad esempio: 

<html>
<head>
  <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1">
  <script src="/urchin.js" type="text/javascript"></script>
  <script type="text/javascript">
        urchinTracker();
  </script>
  ...
</head>

Passaggio 3. Attivazione dei cookie nella registrazione Apache. Se non l'hai già fatto, per abilitare la registrazione dei cookie puoi utilizzare il seguente esempio di httpd.conf: 

LogFormat "%h %v %u %t \"%r\" %>s %b \"%{Referer}i\"\"%{User-Agent}i\"\"%{Cookie}i\"" special
...
<VirtualHost 63.212.171.6>
ServerName www.mysite.com
DocumentRoot /www/www.mysite.com
CustomLog /var/logs/mysite.log special
</VirtualHost>

Passaggio 4. Impostazione dell'uso di UTM nel profilo Urchin. Nell'interfaccia di amministrazione di Urchin, modifica il profilo in questione e fai clic sulla scheda dei rapporti. In Metodo monitoraggio visitatore, imposta UTM. In Dominio UTM indica l'indirizzo del tuo sito web, senza www. Al termine, fai clic sul pulsante Aggiorna. Fai clic sulla scheda Impostazioni profilo e nel set di rapporti predefinito scegli UTM-Abilita tutti e fai nuovamente clic sul pulsante Aggiorna. È tutto. Il sito web comincia a registrare i dati UTM nel normale file di log che verrà identificato al successivo avvio di Urchin.

Funziona? Per verificare se il sistema UTM esegue correttamente le registrazioni nel file di log, esamina il log dopo aver installato l'UTM e selezionato alcune pagine del sito. In fondo al file di log dovrebbe essere riportata una voce simile alla seguente: 

... "GET /__utm.gif?..." 200 ..."__utma=..."

Se le voci __utma non sono presenti, verifica che i cookie siano stati attivati correttamente nella registrazione. Se il codice di stato non è 200, verifica che i file siano stati copiati correttamente nella root dei documenti.
