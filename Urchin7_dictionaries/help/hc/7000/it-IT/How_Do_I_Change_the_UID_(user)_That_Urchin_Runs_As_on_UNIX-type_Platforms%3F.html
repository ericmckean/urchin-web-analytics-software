Come faccio a modificare lo UID (utente) che Urchin esegue sulle piattaforme di tipo UNIX?

Modificare lo UID (utente) e/o il GID (gruppo) eseguito da Urchin è piuttosto semplice. È probabile che sia necessario disporre di privilegi di utente "root" per eseguire le operazioni descritte di seguito, che nella maggior parte dei casi dovranno quindi essere effettuate dall'amministratore di sistema.

  1. Se il nuovo utente e gruppo non sono già esistenti nel sistema, crea le relative voci nei file di password e dei gruppi.
  2. Interrompi i servizi urchin con "urchinctl stop". Il file binario "urchinctl" si trova nella directory "bin" della distribuzione Urchin.
  3. Esegui un comando "chown -R newuser:newgroup" sulla directory root della distribuzione Urchin. Se hai specificato ubicazioni alternative per le directory "var" o "data" nel file "urchin.conf", devi eseguire questo comando "chown" anche su tali directory.
  4. Modifica il file "urchinwebd.conf.template" nella directory "var" della distribuzione Urchin e modifica le righe seguenti:
    • User newuser Group newgroup
  5. Riavvia i servizi Urchin con "urchinctl start".
