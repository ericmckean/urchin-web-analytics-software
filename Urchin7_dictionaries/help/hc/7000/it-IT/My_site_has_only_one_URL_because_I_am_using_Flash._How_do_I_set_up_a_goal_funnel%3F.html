Il mio sito ha un solo URL perché utilizzo Flash. Come faccio a impostare un percorso obiettivo?

Puoi monitorare qualsiasi evento basato su browser sul tuo sito web, inclusi gli eventi Flash. Utilizzando la funzione JavaScript urchinTracker, puoi assegnare un nome file di pagina a qualsiasi azione Flash e inserire quindi quel nome di file nella fase dell'obiettivo o del percorso obiettivo corrispondente.

Per monitorare un evento, richiama la funzione JavaScript urchinTracker() con un argomento specificando un nome per l'evento. Ad esempio, richiamando: 

    javascript:urchinTracker('/purchase_funnel/page1.html');

ciascuna occorrenza dell'evento che richiama Flash verrà registrata come se fosse una pagina visualizzata sotto il nome /purchase_funnel/page1.html. L'argomento deve iniziare con una barra (/).

I nomi di evento possono essere organizzati secondo il tipo di struttura di directory desiderato. L'argomento del percorso/nome di file che richiama urchinTracker non deve rappresentare necessariamente un URL reale sul tuo sito web.

Esempi di codice Flash 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
