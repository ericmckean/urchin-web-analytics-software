Come utilizzare i codici di monitoraggio principali

Urchin ti consente di codificare i link utilizzando codici di monitoraggio principali (un utm_id) anziché variabili individuali. È sufficiente utilizzare utm_id nei tuoi link e definire il significato di ciascun utm_id in una tabella. Ad esempio, anziché l'espressione: 

http://www.sitohost.it/?utm_source=overture&utm_medium=cpc&utm_campaign=springpromo

puoi utilizzare la variabile UTM utm_id nel seguente modo: 

http://www.sitohost.it/?utm_id=2

L'utilizzo di utm_id nasconde le tue variabili di monitoraggio delle campagne ai visitatori e rende meno soggetta a errori la procedura di inserimento dei codici, poiché i valori delle variabili di monitoraggio delle campagne vengono specificati in una tabella, dove è possibile apportare modifiche e correzioni con facilità.

Per utilizzare i codici di monitoraggio principali, è necessario:

  • Definire i codici in una tabella.
  • Applicare la tabella al tuo profilo come un filtro.
  • Utilizzare i codici di monitoraggio principali nei tuoi link.

Definizione dei codici

Per definire i codici, procedi nel seguente modo:

  1. Crea una tabella in Excel per mappare i codici con un gruppo di variabili di campagna.
    • La prima riga del file deve iniziare con "#Fields:" seguito dai nomi delle variabili UTM in qualsiasi ordine. Ciascuna riga definisce le impostazioni delle variabili di campagna per il codice di monitoraggio principale che hai inserito nella colonna "#Fields". Utilizza il trattino (-) per non indicare alcun valore. Puoi omettere i campi nella tabella per i quali non sono disponibili valori per alcun codice. Aggiungi una riga vuota in fondo al file (l'ultima riga del file non deve contenere testo).
  2. Salva la tabella Excel come file di testo delimitato da tabulazioni nella directory lib/custom/lookuptables della distribuzione Urchin. È necessario salvare il file con l'estensione ".lt".

Applicazione della tabella al tuo profilo

Per applicare la tabella a un profilo, procedi nel seguente modo:

  1. Accedi a Urchin come amministratore e fai clic su "Configurazione" (nella parte in alto a destra dell'interfaccia).
  2. Modifica il profilo al quale desideri applicare i codici di monitoraggio principali.
  3. Nella scheda Filtri, fai clic sull'icona "Aggiungi".
  4. Nella procedura guidata filtro, seleziona "Tabella di ricerca" sotto "Tipo filtro".
    • Viene visualizzato il campo "Nome tabella" nella procedura guidata.
  5. Dall'elenco a discesa "Nome tabella" seleziona il nome della tabella creata. Se la tabella non viene visualizzata nell'elenco a discesa, assicurati che il nome del file della tabella abbia l'estensione .lt e che sia stato salvato nella directory lib/custom/lookuptables della distribuzione Urchin.
  6. Dall'elenco a discesa "Campo da filtrare" seleziona utm_id (AUTO).
  7. Fai clic su "Fine".

Utilizzo dei codici nei link

Utilizza i valori nella colonna "Filtri" della tua tabella di ricerca come valori utm_id. Ad esempio, se utilizzi la tabella di ricerca creata seguendo la procedura sopra descritta, puoi inserire i codici in un link nel seguente modo: 

http://www.sitohost.it/?utm_id=1

Ulteriori indicazioni

Se in un link utilizzi utm_id insieme ad altre variabili UTM (ad esempio, utm_source), i valori delle altre variabili verranno sovrascritti con i valori definiti nella tabella di ricerca.
