Posso spostare la directory dei dati in un altro percorso?

Panoramica

Per impostazione predefinita, la directory dei dati di Urchin è una sottodirectory della directory principale di programma di Urchin. Questa posizione può essere cambiata facilmente modificando il file ~urchin/etc/urchin.conf e spostando manualmente la directory ~urchin/data esistente nella posizione desiderata. Tieni presente che questa operazione potrebbe alterare la licenza esistente, nel qual caso dovrai contattare un Consulente autorizzato per il software Urchin e richiedere la reimpostazione del numero di serie.

Procedura di spostamento della directory dei dati:

La posizione in cui Urchin memorizza i dati dei rapporti può essere modificata nel file urchin.conf, che si trova nella directory etc della directory di installazione principale di Urchin.

Per sistemi di tipo UNIX:

  1. Apri un'interfaccia di riga di comando come utente che ha installato Urchin.
  2. Specifica il comando cd per la directory in cui è installato Urchin.
  3. Interrompi i servizi Urchin con il comando: bin/urchinctl stop
  4. Utilizzando un editor di testo, apri il file urchin.conf nella directory etc della distribuzione Urchin.
  5. Rimuovi il commento dalla seguente riga eliminando il carattere iniziale "#":
    • #dataDirectory: ./data/ e sostituisci la directory predefinita con il percorso completo che desideri utilizzare, ad esempio dataDirectory: /bigdisk/urchin/data/

  6. Se necessario, crea la nuova directory dei dati. Nota importante. Questa directory deve essere leggibile e scrivibile dall'utente che ha eseguito l'accesso a Urchin.

  7. Copia i dati esistenti nella nuova posizione. Per accertarsi che tutti i dati e le autorizzazioni vengano mantenuti correttamente, è consigliabile utilizzare il seguente comando:
    • cd data; tar cf - . | (cd /bigdisk/urchin/data; tar xpf -)
  8. Rinomina la directory dei dati in data.old. Puoi rimuoverla completamente, se lo desideri, ma prima di farlo ti consigliamo di accertarti che tutto funzioni correttamente.
  9. Per facilità di amministrazione, puoi creare un link simbolico nella directory principale di Urchin che punta al nuovo percorso, ad esempio
    • ln -s /bigdisk/urchin/data ./data
  10. Riavvia i servizi Urchin con il comando: bin/urchinctl start
  11. Esegui l'accesso a Urchin come utente con privilegi di amministratore. Viene visualizzata la schermata Licenza Urchin. Fai semplicemente clic sul link Riattiva licenza per completare l'operazione.

Per sistemi Windows:

  1. Interrompi i servizi Urchin: Start->Programmi->Urchin->Disabilita servizi Urchin.

  2. Apri Windows Esplora risorse e naviga fino alla cartella etc della distribuzione Urchin. Per impostazione predefinita, C:\Program Files\Urchin\etc.
  3. Utilizzando un editor di testo, apri il file urchin.conf.
  4. Rimuovi il commento dalla seguente riga eliminando il carattere iniziale "#":
    • #dataDirectory: ./data/ e sostituisci la cartella predefinita con la lettera di unità e il percorso completo che desideri utilizzare, ad esempio dataDirectory: E:\Urchin\data
  5. Se necessario, crea la nuova cartella dei dati. Nota importante. Le autorizzazioni su questa cartella devono consentire l'accesso in lettura e scrittura al servizio Urchin.
  6. Copia il contenuto della cartella dei dati nella nuova cartella.
  7. Rinomina la cartella dei dati in data.old. Puoi rimuoverla completamente, se lo desideri, ma prima di farlo ti consigliamo di accertarti che tutto funzioni correttamente.

  8. Riavvia i servizi Urchin: Start->Programmi->Urchin->Abilita servizi Urchin.

Considerazioni

A causa del modo in cui è implementata la licenza sui sistemi di tipo UNIX per impedire sabotaggi, lo spostamento della directory dei dati di Urchin richiede la riattivazione della licenza Urchin. Tuttavia, l'operazione di riattivazione è estremamente semplice: devi solo eseguire l'accesso come utente con privilegi di amministratore di Urchin e fare clic su Riattiva licenza.
