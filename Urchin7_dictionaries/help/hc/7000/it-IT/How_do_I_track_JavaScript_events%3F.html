Come faccio a eseguire il monitoraggio degli eventi JavaScript?

Per monitorare un evento, utilizza la funzione JavaScript urchinTracker per specificare un nome per l'evento. Ad esempio, richiamando: 

    javascript:urchinTracker('/homepage/link1');

ciascuna occorrenza dell'evento che richiama la funzione verrà registrata come se fosse una pagina visualizzata dell'URL /homepage/link1. L'argomento deve iniziare con una barra (/). I nomi di evento possono essere organizzati secondo il tipo di struttura di directory desiderato.

Ad esempio, per organizzare eventi per pagina > tipo di evento, puoi organizzare una struttura gerarchica in queste righe: 

    * '/homepage/links/link1'
    * '/homepage/radiobuttons/button1'
    * '/contactform/rollovers/image1'

Esempi di codice HTML

L'esempio seguente illustra il modo in cui è possibile registrare un evento onClick: 

    <a href="javascript:void(0);" onClick="javascript:urchinTracker('/folder/file');">

In alternativa, per registrare un evento rollover: 

    <a href="javascript:void(0);" onMouseOver="javascript:urchinTracker ('/folder/file');">
