I link diretti al mio sito contengono già le mie variabili personali di monitoraggio delle campagne. Posso utilizzare queste variabili al posto di quelle di Urchin?

Se hai già codificato i tuoi link online con le tue variabili personali di monitoraggio, puoi indirizzare le variabili esistenti verso le variabili di monitoraggio delle campagne di Urchin. I link codificati devono contenere almeno 3 variabili, le quali saranno indirizzate verso:

  1. utm_source: La sorgente della tua campagna (ad esempio, Google, Yahoo).
  2. utm_medium: Il mezzo della campagna (costo per clic, banner, email).
  3. utm_campaign: Il nome della campagna (ad esempio, saldi_primaverili).

Nell'esempio usato per queste istruzioni ipotizzeremo che il sito abbia il seguente URL: 

            www.esempio.it/index.html?camp=GADS&kwd=memory&version=47&type=ppc&adver=google

Passaggio 1. Esegui la corrispondenza delle variabili di Urchin con le variabili personalizzate. Crea una tabella applicando il formato indicato nel seguente esempio: 

               Campagna     Sorgente annuncio   Tipo di annuncio     Parola chiave   Contenuto A/B  Principale
Codici utente=    camp      adver       type        kwd       version      
Codici Urchin=  utm_campaign utm_source         utm_medium  utm_term  utm_content  utm_id

Passaggio 2. Modifica il file UTM.

  • Apri il file urchin.js in un editor di testo. Nella parte superiore del file è presente una sezione definita dall'utente che include le variabili utm: 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="utm_campaign";
                var _ucmd="utm_medium";
                var _ucsr="utm_source";
                var _uctr="utm_term";
                var _ucct="utm_content";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";
  • Utilizzando l'esempio di link codificato sopra riportato, la sezione viene modificata come indicato di seguito: 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="camp";
                var _ucmd="type";
                var _ucsr="adver";
                var _uctr="kwd";
                var _ucct="version";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";

Passaggio 3. Crea i filtri per queste nuove variabili di monitoraggio.

  • Questa è la configurazione di base che dovrai usare per ciascun filtro: 

                Tipo filtro: avanzato
                Campo A: utm_campaign_uri (non elaborato)
                Estrai A: new_variable =([^&]*)
                Campo B: (nessuno)
                Estrai B: (vuoto)
                Output in: utm _(old_variable)
                Constructor: $A1
                Campo A obbligatorio: sì
                Campo B obbligatorio: no
                Sostituisci campo di output: sì
                Maiuscole/minuscole: no
  • "old_variable" si riferisce alla variabile di monitoraggio Urchin predefinita. "new_variable" si riferisce alla tua variabile di monitoraggio personale. Tieni presente che è necessario applicare un filtro per ciascuna variabile modificata.
