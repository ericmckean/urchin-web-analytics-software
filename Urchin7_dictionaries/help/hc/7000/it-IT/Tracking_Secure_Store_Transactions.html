Monitoraggio delle transazioni di negozio protetto

Articolo di ivantage Limited, consulente autorizzato del software Urchin

Urchin è in grado di monitorare i dati di un carrello degli acquisti presente sul tuo dominio o altri domini mediante l'aggiunta di un frammento di codice. Per fare in modo che le tue pagine producano rapporti sui dati delle transazioni in Urchin, consulta la sezione Come faccio a monitorare le transazioni di e-commerce?

Inoltre, se il sito riservato alla vendita si trova in un sottodominio o in un dominio diverso dal sito principale, segui le istruzioni riportate più avanti per configurare le tue pagine.

Se il tuo sito web avvia una procedura di checkout degli acquisti in un sottodominio diverso (ad esempio, se i clienti vengono reindirizzati da www.mionegozio.it a carrelloacquisti.mionegozio.it):

  • Aggiungi la seguente riga (in grassetto) al codice di monitoraggio sia nel sito di vendita che nelle pagine del carrello degli acquisti: 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

Se il tuo sito web avvia una procedura di checkout degli acquisti in un sito di vendita distinto (ad esempio, se i clienti vengono reindirizzati da www.mionegozio.it a www.carrelloprotetto.it):

  • Aggiungi le seguenti righe (in grassetto) al codice di monitoraggio sia nel sito di vendita che nelle pagine del carrello degli acquisti: 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • Modifica i link che collegano il sito principale al sito protetto utilizzando utmLinker nel seguente modo. Se i tuoi link hanno il seguente formato: 

     <a href="https://www.securecart.com/?store=parameters">Acquista adesso</a>
  • modificali in: 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">Acquista adesso</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">Acquista adesso</a>
   </noscript>

  • Il codice indicato qui sopra offre link sia agli utenti che hanno JavaScript attivato sia a quelli che lo hanno disattivato. Tieni presente che gli apostrofi devono essere convertiti in caratteri escape utilizzando una barra rovesciata (\) quando sono inclusi all'interno del link o del testo del link. Importante. Se le tue pagine includono una chiamata a urchinTracker(), utmLinker(), utmSetTrans() o utmLinkPost(), il tuo codice di monitoraggio Analytics deve essere collocato all'interno del tuo codice HTML sopra una di queste chiamate. In questi casi, il codice di monitoraggio può essere inserito in qualsiasi punto tra il tag iniziale <body> e la chiamata JavaScript.

Se invii informazioni al tuo carrello degli acquisti utilizzando dei moduli, utilizza la funzione utmLinkPost(). Aggiunge dati sui cookie all'URL del modulo fornito. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • Questo funziona anche per i moduli con method="GET"
