Monitoraggio di link in uscita e annunci

Urchin offre un modo semplice per monitorare i clic sugli annunci banner o su altri link in uscita. Dal momento che questi link non rimandano a una pagina sul tuo sito che contiene l'UTM JavaScript, dovrai codificare il link stesso con la funzione JavaScript urchinTracker se desideri monitorare questi clic. Questo spezzone di codice JavaScript assegna una pagina visualizzata a ciascun clic effettuato su un link (la pagina visualizzata viene attribuita al nome di file da te specificato).

Per un GIF animato o un altro tipo di banner statico, aggiungi il codice JavaScript urchinTracker al tag <a> del tuo banner: 

    <a href="http://www.esempio.it/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
