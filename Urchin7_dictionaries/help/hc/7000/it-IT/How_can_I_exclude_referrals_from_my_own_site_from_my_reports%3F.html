Come faccio a escludere i referral del mio sito dai rapporti?

Per escludere i referral del tuo sito dai rapporti Referral, è necessario apportare una modifica al file urchin.js. Apri il file in un editor di testo e inserisci la seguente sezione: 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";

In questo caso, qualsiasi hit che contiene urchin.com verrà escluso dai tuoi referral. Se desideri aggiungere ulteriori domini, modifica il file includendo un altro valore _uRno[x] e aumenta il valore numerico di x. Ad esempio: 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";
    _uRno[1]=".urchin.net";
    _uRno[2]=".urchinsoftware.com"

Tieni presente che ogniqualvolta apporti delle modifiche a urchin.js, è necessario caricare il file aggiornato nel server web per applicarle.
