Cobranding di Urchin

Panoramica

Urchin consente il cobranding nell'interfaccia di amministrazione, nell'interfaccia rapporti e nella schermata di accesso. Per includere codice HTML nella parte superiore dell'interfaccia, è necessario modificare tre file. Se si sta effettuando un'integrazione completa del portale, i rapporti di Urchin possono essere pubblicati all'interno di un frameset o di una tabella dal tuo server di applicazioni.

Istruzioni di cobranding

Per eseguire il cobranding dell'interfaccia, devi modificare i seguenti file che si trovano nell'installazione di Urchin: 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • Rinomina i file rimuovendo ".sample" (il nome di file deve essere, ad esempio, cobrand_admin.tpl).
  • Aggiungi a questi file il contenuto HTML richiesto per includere il tuo branding.

Il primo file controlla il cobranding nell'interfaccia di amministrazione e il secondo controlla il cobranding nell'interfaccia rapporti. Il terzo file controlla il cobranding nella schermata di accesso. Il codice HTML fornito in questi file verrà inserito nella parte superiore dell'interfaccia di Urchin.
