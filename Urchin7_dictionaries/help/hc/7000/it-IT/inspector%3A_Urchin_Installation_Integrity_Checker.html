inspector: Utilità di controllo integrità dell'installazione di Urchin

Panoramica

inspector, l'utilità di controllo integrità dell'installazione di Urchin, fornisce un metodo per controllare distribuzione e rapporti di Urchin:

  • lo stato dei server Urchin 6 (server web e programma di pianificazione).
  • problematiche relative a licenze o autorizzazioni.
  • file mancanti.
  • altri problemi di configurazione.

Per impostazione predefinita, fornisce anche un riepilogo delle informazioni sull'installazione di Urchin e sul tipo di piattaforma su cui è installato.

L'utilità inspector consente di effettuare le seguenti operazioni:

  • Controllo dell'integrità della distribuzione Urchin, incluse esistenza dei file e autorizzazioni appropriate.
  • Controllo della disponibilità di una porta del server web (per il server web Urchin).
  • Reimpostazione delle autorizzazioni appropriate per la distribuzione Urchin.

Utilizzo

L'utilità inspector è ubicata nella directory util della distribuzione Urchin 6.

L'utilità può essere utilizzata nei seguenti modi: 

  inspector [-h] (consente di stampare il messaggio sull'utilizzo e di uscire).

  inspector [-v] consente di stampare la versione di inspector.

  inspector [-p] controlla una porta specificata per determinarne la disponibilità.

  inspector [-r] stabilisce le autorizzazioni sui file della distribuzione.

Se richiamata senza argomenti della riga di comando, l'utilità inspector esegue una serie di controlli di integrità sull'installazione di Urchin al fine di assicurarne l'integrità. Specificamente, l'utilità esegue le operazioni seguenti:

  1. Stampa la versione di Urchin e informazioni sul server quali versione del sistema operativo e nome host.
  2. Verifica che siano correttamente presenti tutti i file binari e le utilità di Urchin e che la loro versione sia corretta. Sulle piattaforme UNIX, sono inoltre rilevate le discrepanze nelle autorizzazioni di file binari e file nella distribuzione Urchin.
  3. Controlla lo stato del server web Apache di Urchin (urchinwebd) e genera i relativi rapporti.
  4. Controlla lo stato di Urchin Scheduler (urchind) e genera i relativi rapporti.
  5. Controlla la configurazione di Urchin e genera rapporti sul numero totale di record e su informazioni riepilogative sulla licenza.
  6. Informazioni di configurazione di Urchin 6 (numero di serie, stato della licenza, versione della licenza).

Quando viene attivata con l'argomento "-p", l'utilità inspector controlla la disponibilità della porta specificata per l'utilizzo da parte del server web Urchin6.

L'opzione "-r" (riparazione) fa in modo che l'utilità tenti di riparare le autorizzazioni relative a qualsiasi file della distribuzione Urchin non conformi all'installazione originale. Questa operazione è in genere utile solo per la piattaforme UNIX, e nella maggior parte dei casi richiederà che l'utilità venga eseguita come utente "root".
