Suggerimenti per la configurazione

Panoramica

Una volta installato Urchin è necessario configurare alcuni parametri operativi iniziali. Questa operazione viene eseguita tramite una procedura guidata di installazione quando ti colleghi per la prima volta all'interfaccia di amministrazione di Urchin, e durante le prime fasi di creazione dei profili. Queste operazioni di configurazione iniziali includono:

  • Rilascio della licenza Urchin
  • Configurazione delle impostazioni amministratore per i rapporti remoti e l'accesso di amministrazione, oltre a stabilire la modalità Datacenter.
  • Impostazione della password dell'account amministratore Urchin.
  • Pianificazione di attività per ciascun profilo per elaborare i dati
  • Gestione log.

Procedura

Collegati all'interfaccia amministratore di Urchin. Nei sistemi Windows, seleziona Start->Programmi->Urchin->Amministrazione Urchin. Nei sistemi UNIX, puoi utilizzare l'URL http://hostname:9999, in cui il nome host è il nome host registrato del tuo sistema Urchin. Per una nuova installazione devi usare i seguenti dati di accesso:

Nome utente: password amministratore: urchin

Viene visualizzata la schermata iniziale della procedura guidata di installazione di Urchin. Fai clic su Continua per passare alle procedura guidata di procedura guidata di installazione. Tieni presente che è sempre possibile modificare in un secondo momento le selezioni eseguite in questa configurazione iniziale.

Licenza Urchin

Segui le istruzioni contenute in Rilascio della licenza Urchin.

Impostazioni amministratore

  • Impostazioni di accesso remoto: seleziona Attivato per ogni caso in cui desideri consentire le connessioni remote per i browser. Se selezioni Disattivato per una di queste impostazioni, l'unico accesso consentito è sulla console del sistema in cui è installato Urchin.
  • Modalità Datacenter: questa impostazione determina se Urchin è configurato per consentire la creazione di account, che ti consente di organizzare logicamente profili, gruppi e utenti in categorie di accesso limitato. Se sei indeciso, è preferibile impostare questa modalità su Attivato, in quanto non causa sovraccarico e ti fornisce la flessibilità di poterla utilizzare in futuro.

Utente amministrativo

  • Reimposta la password per l'account amministratore e conservala al sicuro.

Attività di pianificazione

Quando crei i profili ti viene fornita un'opzione per pianificare l'orario di esecuzione dell'attività di elaborazione dei dati relativi al profilo specifico. Verifica le impostazioni di ogni profilo per assicurarti che l'orario della tua attività tenga presente della disponibilità dei dati di log, del tempo necessario a elaborare questi dati e di quando si desidera rendere disponibili questi dati agli utenti.

Gestione log

Urchin include un modulo di monitoraggio log che tiene traccia del livello di elaborazione di ogni singolo log. In questo modo, la rotazione log non deve essere necessariamente coordinata con il funzionamento di Urchin. Tuttavia, Urchin fornisce l'automazione della rotazione o della rimozione dei log nelle impostazioni avanzate di ogni sorgente log.

Per ulteriori informazioni, consulta Gestione log.
