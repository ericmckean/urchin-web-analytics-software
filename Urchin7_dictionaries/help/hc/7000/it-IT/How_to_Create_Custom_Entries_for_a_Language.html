Come creare voci personalizzate per una lingua

  1. Copia il file della lingua xx-XX.txt dalla cartella <Urchin>/lib/admin/languages/ in <Urchin>/lib/custom/languages/.

  2. Aggiungi nuove voci alla fine del file e assegna un nuovo numero a ciascuna voce.

  3. Il separatore tra il numero e il valore deve essere un carattere <Tab> altrimenti Urchin non lo analizzerà.

  4. Assicurati che i numeri delle voci siano progressivi (aumentino di uno).
  5. Ricarica Urchin.

  6. Una volta ricaricato, Urchin crea un file database della lingua denominato xx-XX.db nella cartella <Urchin>/lib/custom/languages/.

Una volta definite, le voci personalizzate possono essere utilizzate nella sezione dei rapporti come etichette di colonne, titoli o descrizioni della guida.

Esempio di file della lingua

Le voci 738, 739 e 740 vengono aggiunte come voci personalizzate. 

  ...
  731     Executive
  732     Marketer
  733     Webmaster
  734     Dashboards
  735     All Reports
  736     ja-jp-mac (Japanese)
  737     ja-jp-osaka (Japanese)
  738     Modules
  739     Actions
  740     Modules / Actions

Nota importante. L'upgrade di Urchin può sostituire il file della lingua originale e aggiungere voci predefinite. Accertati di ricreare i file della lingua personalizzati e di aggiornare i rapporti personalizzati.
