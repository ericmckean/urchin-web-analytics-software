Installazione rapida di UTM (IIS)

  • Articolo di EpikOne, consulente autorizzato del software Urchin

Nel seguente paragrafo viene rapidamente illustrato come installare UTM per i siti web gestiti su un server Microsoft IIS su qualsiasi piattaforma Windows. Per informazioni dettagliate su UTM, consulta l'articolo intitolato "Urchin Traffic Monitor (UTM)" in questa sezione.

Passaggio 1. Copia dei file UTM nella root dei documenti del sito web. I file, urchin.js e __utm.gif sono ubicati nella cartella "utils\utm" della distribuzione di Urchin. Copia questi due file nella cartella principale dei contenuti del sito web. IMPORTANTE: Il nome del file GIF comincia con due trattini di sottolineatura.

Passaggio 2. Inserimento di un riferimento UTM nel codice HTML del sito web. Inserisci la seguente riga in tutte le pagine HTML. Puoi posizionarla in qualsiasi punto delle pagine, tuttavia, è consigliabile inserirla direttamente prima del tag </body> di chiusura. Se utilizzi una normale inclusione o un modello, puoi inserire il codice in quel punto. 

<script src="/urchin.js" type="text/javascript"></script>

<script type="text/javascript">

        urchinTracker();

</script>

Se utilizzi un pacchetto come "HTML Tidy", puoi inserire la riga Javascript nell'area HEAD della pagina, ad esempio: 

<html>

<head>

  <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1">

  <script src="/urchin.js" type="text/javascript"></script>

  <script type="text/javascript">

        urchinTracker();

  </script>

  ...

</head>

Passaggio 3. Attivazione dei cookie nella registrazione log IIS. Apri l'IIS Manager e visualizza la finestra delle proprietà del tuo sito web. Assicurati che la registrazione log sia attivata e impostata sul formato W3C esteso. Fai clic sul pulsante Proprietà accanto al formato e nella scheda Proprietà estese, seleziona la casella accanto a Cookie.

Passaggio 4. Impostazione dell'uso di UTM (Urchin Traffic Monitor) nel profilo Urchin. Nell'interfaccia di amministrazione di Urchin, modifica il profilo in questione e fai clic sulla scheda Impostazioni. Imposta il metodo monitoraggio dei visitatori su UTM. In Dominio UTM indica l'indirizzo del tuo sito web, senza www. Scegli quindi l'impostazione predefinita per il set di rapporti e fai clic sul pulsante Aggiorna.

È tutto. Il sito web comincia a registrare i dati UTM nel normale file di log che verrà identificato al successivo avvio di Urchin.

Funziona? Per verificare se il sistema UTM esegue correttamente le registrazioni nel file di log, esamina il log dopo avere installato l'UTM e selezionato alcune pagine del sito. In fondo al file di log dovrebbe essere riportata una voce simile alla seguente: 

... "GET /__utm.gif?..." 200 ..."__utma=..."

Se le voci utma non sono presenti, verifica che i cookie siano stati attivati correttamente nella registrazione. Se il codice di stato non è 200, verifica che i file siano stati copiati correttamente nella root dei documenti.
