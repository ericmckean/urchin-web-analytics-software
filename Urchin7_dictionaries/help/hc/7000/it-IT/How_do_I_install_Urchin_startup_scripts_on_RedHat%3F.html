Come faccio a installare gli script di avvio di Urchin su RedHat?

Durante l'installazione viene creato uno script di avvio di Urchin che si chiama "urchin_daemons" ed è posizionato nella directory Urchin util. "/path/to/urchin/util/"

Copia questo script nella directory /etc/rc.d/init.d/ directory.

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • Individua il runlevel predefinito (normalmente 3) nel file /etc/inittab. [[Vedi sotto]]
  • Quindi passa alla directory /etc/rc.d/rcX.d (dove X rappresenta il runlevel).
  • cd /etc/rc.d/rc3.d
  • Crea link simbolici che puntano a /etc/rc.d/init.d/urchin_daemons, denominati S95urchin e K05urchin.
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # Runlevel predefinito. I runlevel utilizzati da RHS sono:

  • 0 - Arresta il sistema (NON impostare initdefault su questo livello)
  • 1 - Modalità a utente singolo
  • 2 - Multiutente, senza NFS (analogo al livello 3, se non si dispone di rete)
  • 3 - Modalità multiutente completa
  • 4 - Non utilizzato
  • 5 - X11
  • 6 - Riavvia il sistema (NON impostare initdefault su questo livello)
