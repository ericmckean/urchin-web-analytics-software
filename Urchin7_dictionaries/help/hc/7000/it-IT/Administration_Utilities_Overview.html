Panoramica delle utilità di amministrazione

Panoramica

Urchin offre una serie di utilità che possono essere utilizzate per le operazioni di configurazione e diagnostica. Tali utilità sono ubicate nella directory util della distribuzione Urchin. Questo documento fornisce una panoramica introduttiva, non una guida completa all'uso di tali utilità.

Tutte le utilità supportano gli argomenti "-h" e "-v" della riga di comando. Se si attiva un'utilità con l'argomento "-h", è possibile visualizzare un riepilogo delle funzioni e le opzioni disponibili per lo strumento. Se si attiva un'utilità con l'argomento "-v", viene stampata la versione Urchin dell'utilità.

geo-update

Questa utilità consente di verificare gli aggiornamenti del database DNS interno di Urchin e di scaricare gli aggiornamenti disponibili, sia dai server di download di Google, sia da una directory locale (novità di urchin 6).

inspector

Questa utilità consente di eseguire i controlli di base sulla distribuzione Urchin installata, verificando che la struttura complessiva della distribuzione sia intatta, che tutti i file binari inclusi nel prodotto siano nella versione prevista e che le autorizzazioni sottostanti siano corrette (nelle piattaforme UNIX). L'utilità segnala inoltre lo stato operativo del Programma di pianificazione dei profili di Urchin (urchind), dei Programmi di pianificazione CPC (urchincpcd, urchincpc) e del server web integrato Apache (urchinwebd). Inoltre, verifica le informazioni sulla licenza.

uconf-manager (Urchin 6.402+)

L'Utilità di gestione della configurazione di Urchin (uconf-manager) consente di aggiungere, modificare ed eliminare record db.

  • Urchin 6.402: Questa utilità è stata introdotta per la prima volta in Urchin 6.402 e funzionava esclusivamente con la tabella 'uusers' del database di configurazione. Può essere utilizzata per reimpostare la password amministratore di Urchin, automatizzare la creazione di utenti e trasferire un utente in un account diverso.
  • Urchin 6.500+: In urchin 6.5 e versioni successive, questa utilità supporta operazioni di database su più tabelle (uaccounts, ucosts, ufilters, ugoals, ugroups, ugroups_uusers, ulogs, ulogs_ufilters, uprofiles, uprofiles_ucosts, uprofiles_ufilters, uprofiles_ugroups, uprofiles_ulogs, uprofiles_uusers e uusers). Questa utilità può essere utilizzata per eseguire azioni che non sono disponibili nell'interfaccia di amministrazione di Urchin.

uconf2sql

Urchin 5 e le versioni precedenti memorizzano i dati di configurazione nei file delle proprietà, mentre Urchin 6 li memorizza in un database relazionale (MySql o PostgreSql). Per semplificare la transizione dei dati di configurazione, sia Urchin 5 sia Urchin 6 includono utilità per l'esportazione e l'importazione di questi dati. Urchin 5 include un'utilità chiamata uconf-export che consente di esportare il database di configurazione in un file (in formato XML), mentre Urchin 6 include un'utilità chiamata uconf2sql che consente di importare i dati di configurazione da questo file XML in un database Urchin 6.

convert-u5data

Analogamente al database di configurazione, in Urchin 6 è stata modificata inoltre la struttura del database dei rapporti (che memorizza i dati dei rapporti). Tale utilità, chiamata convert-u5data, include un'opzione per l'importazione del database dei rapporti direttamente dall'installazione di Urchin 5 all'installazione di urchin 6.

udb-sanitizer

L'utilità udb-sanitizer consente di eseguire operazioni correlate alla gestione nel database dei rapporti, tra le quali il ripristino di un database archiviato e la cancellazione delle informazioni di una data o intervallo di date specifico (nel caso in cui sia necessaria la rielaborazione dei dati).

urchinctl

(Nota. Questa utilità è ubicata nella directory bin, non nella directory util.) Questa utilità fornisce uno strumento per attivare e disattivare il Programma di pianificazione di Urchin e i servizi del server web di Urchin. Di solito, nei sistemi UNIX, l'utilità urchinctl viene richiamata da uno degli script di avvio del sistema per attivare o disattivare automaticamente i servizi Urchin. Nella piattaforma Windows l'utilità fornisce anche opzioni per l'installazione e la disinstallazione dei servizi Urchin.

urchincpc (Urchin 6.5+)

(Nota. Questa utilità è ubicata nella directory bin, non nella directory util.) Lo strumento di importazione dei dati CPC di Urchin estrae dati CPC tramite l'API di Google AdWords per consentire un'ulteriore elaborazione dei dati e generazione di rapporti. Per ulteriori informazioni consulta Utilità di importazione dei dati CPC.

urchincpcd (Urchin 6.5+)

(Nota. Questa utilità è ubicata nella directory bin, non nella directory util.) Il Programma di pianificazione del sorgente CPC di Urchin (o semplicemente Programma di pianificazione CPC) viene utilizzato per programmare attività di download del CPC sulla base di una pianificazione definita dall'utente (simile al Programma di pianificazione dei profili). Per ulteriori informazioni consulta Utilità di importazione dei dati CPC.

urchin_daemons

Nei computer UNIX, Urchin esegue un daemon del programma di pianificazione (urchind) e un server web Apache (urchinwebd). Tali programmi devono essere attivati all'avvio del sistema. Lo script urchin_daemons può essere aggiunto agli script di inizializzazione del sistema nell'ubicazione idonea al sistema operativo UNIX per OS fare in modo che i daemon dei servizi Urchin vengano attivati correttamente. Nelle piattaforme windows, tali file binari vengono eseguiti come servizi di sistema e possono essere attivati e disattivati tramite il gestore dei servizi windows.

ugetlicense

L'Utilità per il funzionamento della licenza Urchin, ugetlicense, consente di eseguire operazioni su un numero di serie. Queste operazioni sono necessarie se un'installazione corrente di Urchin diventa non disponibile e deve essere reinstallata o spostata su un server diverso e così via.
