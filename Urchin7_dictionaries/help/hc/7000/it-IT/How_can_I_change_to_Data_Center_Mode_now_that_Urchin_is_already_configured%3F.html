Come faccio a passare alla modalità Data Center una volta che Urchin è stato configurato?

  • Articolo di ROI Revolution, consulente autorizzato del software Urchin

La modalità Data Center consente di aggiungere livelli di privilegi amministrativi a uno o più profili di un gruppo. Se la procedura di configurazione iniziale è già stata completata, è possibile attivare facilmente la modalità Data Center da MySQL o PostgreSQL.

Da MySQL: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

Da PostgreSQL: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
