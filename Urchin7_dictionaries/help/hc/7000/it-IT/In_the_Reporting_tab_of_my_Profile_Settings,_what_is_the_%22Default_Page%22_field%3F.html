Che cosa indica il campo "Pagina predefinita" nella scheda Rapporti della pagina Impostazioni profilo?

L'opzione "Pagina predefinita " deve essere impostata sulla pagina predefinita (o sull'indice) nella directory del sito, ovvero il file caricato quando l'utente inserisce solo il nome di dominio senza estensione di file (www.esempio.it). Nella maggior parte dei casi si tratta della pagina "index.html", ma sono comunemente utilizzate anche le pagine "index.php", "Default.htm" o "default.asp".

Urchin utilizza questa informazione per aggregare URL quali http://www.example.com/ e http://www.example.com/index.html, considerandoli la stessa pagina. Se le informazioni sulla Pagina predefinita non sono immesse correttamente, queste due pagine vengono considerate come due pagine differenti.
