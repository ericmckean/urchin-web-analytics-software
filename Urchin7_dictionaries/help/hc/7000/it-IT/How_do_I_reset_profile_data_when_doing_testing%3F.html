Come faccio a reimpostare i dati del profilo quando eseguo i test?

È opportuno eseguire un profilo Urchin più volte allo scopo di eseguire i test su rendimento, filtri e altro, considerando che i dati effettivi generati da Urchin vengono eliminati dopo ogni test. Il modo più semplice per effettuare questa operazione è di rimuovere la directory/cartella in cui sono memorizzati i dati del profilo dopo ogni esecuzione. Queste directory sono memorizzate per nome profilo nell'area dati/rapporti della distribuzione Urchin. La rimozione della directory completa denominata in base al profilo in fase di test reimposterà tutti i dati, incluso le informazioni sul monitoraggio log.
