Modifica del numero della porta

Il numero predefinito della porta di ascolto del server web Urchin è 9999. La modifica di questo numero comporta due passaggi fondamentali:

  • Modifica del numero della porta nella schermata Impostazioni server
  • Interruzione e avvio dei servizi Urchin, con una procedura lievemente diversa per i sistemi Windows e Unix

La procedura dettagliata è descritta di seguito.

  • Accedi all'interfaccia di amministrazione di Urchin.

  • Vai alla pagina Configurazione -> Impostazioni globali -> Impostazioni di accesso.

  • Imposta il nuovo dato nella casella Numero porta server.
  • Fare clic sul pulsante Aggiorna.

A questo punto è necessario riavviare i servizi Urchin.

  • Nei sistemi di tipo Unix accedi alla directory bin della distribuzione Urchin ed esegui: 

          ./urchinctl restart

  • Nei sistemi Windows, dalla console vai a Start -> Programmi -> Urchin 6 e scegli Disattiva servizi Urchin 6, quindi Attiva servizi Urchin 6.

Il server web ora si porrà in ascolto delle richieste di connessione sul nuovo numero di porta. Ciò significa che l'URL utilizzato per visualizzare i rapporti e configurare il software Urchin è cambiato e che occorre comunicare il nuovo URL ai tuoi utenti.

Note Su molti sistemi, possono essere necessari privilegi di utente root per utilizzare numeri di porta inferiori a 1024. Se sulla porta specificata è già in esecuzione un altro servizio, inoltre, Urchin non si avvia.
