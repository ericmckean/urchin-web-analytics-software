Importazione di profili (Windows)

Introduzione

La funzione Importa profili di Urchin rappresenta un modo pratico che gli utenti di sistemi su cui è installato Microsoft Internet Information Server possono utilizzare per impostare velocemente i profili per ognuno dei loro siti IIS. Urchin è in grado di leggere la configurazione IIS, determinare quali siti web sono in esecuzione sul server e quindi creare profili di base per ogni sito web che utilizza i log IIS quale sorgente log. Puoi quindi personalizzare i profili o aggiungerne di altri secondo le tue preferenze per i siti importati.

Come importare i profili

Per iniziare l'importazione di profili, accedi al sistema di amministrazione Urchin come amministratore e fai clic su Configurazione (in alto a destra nell'interfaccia). Facendo clic sul pulsante Importa in alto a destra viene visualizzata la schermata Importa profili, che ti consente di selezionare quali siti web eventualmente importare. Dopo aver effettuato la selezione, fai clic sul pulsante Importa. Fai clic su Completato al termine di tutte le selezioni di importazione.

Suggerimenti

È consigliabile creare almeno un profilo per ogni sito web sul server in modo da avere un quadro completo del traffico in ingresso sul server mediante i rapporti di riepilogo di Urchin. Tali rapporti forniscono informazioni sul traffico complessivo del server, oltre al posizionamento di ogni sito in base a vari parametri di traffico. Questi dati sono molto utili se sei un host e la tua fatturazione si basa sull'utilizzo della larghezza di banda. Tieni presente che i rapporti di riepilogo presentano solo i dati sulla base dei profili configurati, i siti privi di profili in funzione non vi sono inclusi.
