Perché Urchin installa un server web Apache?

Urchin installa un server web Apache leggero per fornire le funzionalità necessarie per l'amministrazione basata su Web e la generazione dinamica dei rapporti. Se si include in bundle con Urchin una versione standalone di Apache, non è necessario modificare la configurazione del server web esistente per supportare Urchin. Inoltre, l'esecuzione del server web Apache di Urchin indipendente dal server web esistente, offre altri vantaggi:

  • Urchin non viene influenzato dalle modifiche apportate al server web standard
  • È possibile gestire i servizi Urchin senza influenzare il server web standard
  • È possibile utilizzare la GUI di amministrazione di Urchin per modificare determinati parametri operativi del server web Apache standalone
  • È possibile eseguire gli upgrade di Urchin in modo completamente indipendente e le eventuali modifiche necessarie, quali il supporto SSL, possono essere gestite esclusivamente tramite la procedura di upgrade di Urchin, senza dover apportare modifiche al server web standard. L'installazione Apache di Urchin non deve influenzare altri servizi web in uso, né impegnare troppe risorse del computer. Urchin è configurabile per consentire l'accesso remoto ai rapporti tramite Web.
