Come si creano i rapporti personalizzati

Per creare un rapporto personalizzato:

  1. Scegli una tabella mappa di dati o definiscine una nuova (consulta Mappe di dati personalizzate);

  2. Crea un nuovo file rapporto (".rs2") copiando <Urchin>/lib/custom/profiletypes/Default/default.rs2.sample.

  3. Definisci il nuovo rapporto con l'aggiunta di nuova riga nel file del rapporto.

Il formato del file del rapporto viene descritto di seguito:

  • ID - ID rapporto univoco:
    • C=Categoria
    • D=Dati
    • M=Menu
    • R=Rapporto
    • S=Sottorapporto
    • P=Rapporto pagina
    • B=Pannello di visualizzazione
  • RS - Sezioni Pannello di visualizzazione/Rapporti:
    • 1=exec
    • 2=operatore di marketing
    • 4=webmaster
  • RR - Requisiti rapporto:
    • 1=Software
    • 2=Avanzato
    • 4=E-commerce
    • 8=UTM
  • NTITLE - titolo navigazione (voce dizionario n.)
  • ONCOLOR - colore quando attivo
  • OFFCOLOR - colore quando non attivo
  • HELP - guida del rapporto (voce dizionario n.)
  • NDATA - numero di elementi dati (le voci devono essere inserite subito dopo)
    • Questo campo può essere utilizzato per ulteriori sottorapporti con tipo D (dati)
  • CTITLES - titoli di colonna (voce dizionario n.)
  • CTYPES - tipi di colonna utilizzati per la formattazione
    • 10=stringa
    • 11=data/annomese
    • 12=data
    • 20=numero
    • 21=entrate
    • 22=byte
    • 23=rapporto temporale
    • 24=rapporto
    • 25=pct
    • 26=tempo
    • 30=pct totale
    • 40=entrate
    • 41=entrate/num
    • 42=num/entrate
  • TABLE - numero tabella dati:
    • T=totale
    • TR-transazione
    • H=istogramma
    • V=visitatore
    • 1-n tabella
  • DXFORMAT - formato estrazione dati
  • RF - campo obbligatorio
  • RT - tipo di rapporto:
    • 1=totali grafico
    • 2=istogramma
    • 3=stack
    • 4=principali
    • 5=albero
    • 6=directory
    • 7=limitato dal grafico
    • 8=punto
    • 9=click through
    • 10=canalizzazione definita
    • 11=overlay home page
    • 12=copertura geografica
    • 13=verifica obiettivo
    • 14=percorso obiettivo
    • 20=rapporto data kpi
    • 21=dieci principali kpi
    • 22=riepilogo tendenza
    • 23=riepilogo conversioni
    • 24=Map Mini
  • GTYPES - tipi di grafici:
    • 1=barra
    • 2=riga
    • 3=torta
    • 4=barra orizzontale
    • 5=creativo
    • 6=multiriga
  • FSEP - separatore di campo
  • SP - opzioni speciali (aggiungi numeri bit map):
    • 1=totali
    • 2=medie
    • 4=confronto date
    • 8=inminuscolo
    • 16=dati nel tempo
    • 32=Overlay
    • 64=ricerca lingua
    • 128=segmento
    • 256=life time value
    • 512=dettaglio transazione
    • 1024=Dettaglio visitatore
    • 2048=ricerca termine
  • SR - flag rapporto speciale:
    • 1=suggerimenti di parole chiave
    • 2=click through
    • 3=analisi stratificata per fasce orarie
    • 4=elenco visitatori
    • 5=Copertura geografica
    • 6=Dizionario personalizzato
    • 7=confronto date
  • FNT - tipo filtro numero filtro:
    • 1: =

    • 2: >=

    • 3: <=

    • 4: >

    • 5: <

  • FNF - campo numero filtro
  • FNV - valore filtro
  • FST - tipo stringa filtro:
    • 1=escludi
    • 2=includi
  • FSF - campo stringa filtro
  • FSX - stringa espressione regolare
