Combien d'objectifs puis-je associer à chaque profil ?

Vous pouvez configurer jusqu'à 4 objectifs par profil. Si vous souhaitez effectuer le suivi de plus de quatre objectifs pour une même propriété Web, vous devez configurer d'autres profils identiques pour ce site. Il vous suffit pour cela de cliquer sur le lien "Créez un profil supplémentaire pour un site Web existant" dans la section Gestion de compte de votre compte.

Si votre site enregistre un trafic important, sachez que vos profils traitent les consultations de page séparément et que les consultations de page en double sont comptabilisées dans le nombre total des consultations.
