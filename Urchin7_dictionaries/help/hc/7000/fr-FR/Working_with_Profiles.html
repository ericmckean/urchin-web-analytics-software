Gestion des profils

Présentation

Chaque profil réunit un ensemble de rapports relatifs à un site Web. Vous devez configurer au moins un profil par site. De nombreuses situations peuvent nécessiter la configuration de plusieurs profils pour un site. Toutefois, la grande majorité des sites sont associés à un seul profil.

Techniquement, le profil constitue un ensemble de règles organisant le traitement des fichiers journaux et la création des rapports pour un site Web donné, ou une partie de celui-ci.

Les profils sont les principaux outils de gestion du système Urchin, il est donc essentiel de comprendre leur fonctionnement.

Informations relatives à la licence

La licence de base d'Urchin 6 inclut 1 000 profils. Chacun d'eux peut être associé à une source de journal. Vous pouvez associer une même source de journal à plusieurs profils, afin d'exploiter un profil "maître" et un ou plusieurs autres profils filtrés.

Chaque profil doit être associé à au moins un profil pour générer des rapports. Un profil suffit généralement, mais vous pouvez en créer tant que nécessaire, pour réaliser des analyses spécialisées d'une propriété Web par exemple.

Utilisation des profils

Pour créer un profil, connectez-vous à l'interface d'administration Urchin en tant qu'administrateur et cliquez sur le lien Configuration dans la partie supérieure droite de l'écran. L'assistant d'ajout de profil s'affiche. Il s'agit d'une procédure en quelques étapes qui vous permettra de configurer l'essentiel de votre profil en toute simplicité. Chaque étape de l'assistant est accompagnée d'explications claires et précises.

Recommandations

  • Pour obtenir d'Urchin une qualité de rapports optimale, il est important que vous activiez Urchin Traffic Monitor (UTM), en particulier si vous souhaitez disposer d'informations sur les visiteurs uniques. La procédure à suivre pour utiliser l'UTM se déroule en trois étapes :
    1. Sous l'onglet Paramètres d'un profil, sélectionnez "Urchin Traffic Monitor (UTM)" comme Méthode de suivi des visiteurs.
    2. Installez le détecteur UTM dont vous souhaitez effectuer le suivi avec la plus grande précision. Si vous ne faites pas appel à l'UTM, le suivi des pages est tout de même effectué, toutefois certaines actions telles que les rechargements et l'utilisation du bouton Page précédente ne sont pas comptabilisées.
    3. Modifiez le fonctionnement de la journalisation sur votre serveur Web afin d'activer les cookies.
  • Si vous envisagez d'appliquer un ou plusieurs filtres à votre profil, il est préférable de le dupliquer afin de traiter séparément le site Web ou les différentes sections qui le composent. Nous vous recommandons de créer un profil "maître" -- réunissant toutes les informations. Si vous souhaitez, par exemple, exclure les explorateurs et les robots, il est judicieux d'appliquer ces filtres dans un second profil. Cela vous permettra de comparer les résultats du profil filtré avec ceux du profil maître.
