Comment puis-je masquer les variables de suivi de campagne UTM dans mes liens ?

Urchin vous permet de créer vos liens avec des codes de suivi principaux (nommés utm_id) plutôt que d'inclure une liste de variables individuelles. Il vous suffit d'utiliser la valeur utm_id dans vos liens et de définir chaque utm_id dans un tableau. Par exemple, au lieu de : 

    http://www.example.com/?utm_source=overture&utm_medium=cpc&utm_campaign=springpromo

vous pouvez utiliser un utm_id comme suit : 

    http://www.example.com/?utm_id=2

L'utilisation d'un utm_id masque vos variables de suivi de campagne aux yeux des internautes et rend votre processus de balisage moins sujet aux erreurs puisque les valeurs des variables de suivi de campagne sont spécifiées dans un tableau ce qui facilite les corrections et les modifications.

Pour utiliser les codes de suivi principaux, vous devez :

  1. Définir vos codes dans un tableau
  2. Utiliser les codes de suivi principaux dans vos liens

Lisez la rubrique Utilisation des codes de suivi principaux pour plus d'informations.
