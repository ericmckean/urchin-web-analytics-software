Puis-je déplacer le répertoire data vers un autre emplacement ?

Présentation

Par défaut, le répertoire data d'Urchin est situé dans le répertoire racine du programme Urchin. Vous pouvez aisément changer son emplacement, en modifiant le fichier ~urchin/etc/urchin.conf et en déplaçant le répertoire ~urchin/data existant vers l'emplacement voulu. Notez toutefois que cette opération peut annuler la licence en cours. Vous devez alors contacter votre consultant Urchin agréé pour demander la réinitialisation de votre numéro de série.

Procédure de déplacement du répertoire data :

Vous pouvez modifier l'emplacement dans lequel Urchin stocke les données de rapports via le fichier urchin.conf qui se trouve dans le répertoire etc, lui-même situé dans le répertoire principal de l'installation Urchin.

Avec un système de type UNIX :

  1. Ouvrez un interpréteur de commande avec l'utilisateur ayant servi à installer Urchin.
  2. Utilisez la commande cd pour ouvrir le répertoire dans lequel Urchin est installé.
  3. Arrêtez les services Urchin avec la commande : bin/urchinctl stop.
  4. Avec un éditeur de texte, ouvrez le fichier "urchin.conf" situé dans le répertoire etc de la distribution Urchin.
  5. Supprimez de la ligne suivante le caractère "#" correspondant à un commentaire :
    • #dataDirectory: ./data/ et remplacez le répertoire par défaut par le chemin complet du répertoire que vous souhaitez utiliser, par exemple dataDirectory: /bigdisk/urchin/data/
    • dataDirectory: /bigdisk/urchin/data/

  6. Le cas échéant, créez ce nouveau répertoire de données. Important : Ce répertoire doit être accessible en lecture et en écriture à l'utilisateur correspondant à Urchin.

  7. Copiez les données existantes dans le nouvel emplacement. Pour garantir que toutes les autorisations et les données restent intactes, nous vous conseillons d'utiliser la commande suivante :
    • cd data; tar cf - . | (cd /bigdisk/urchin/data; tar xpf -)
  8. Renommez le répertoire data en data.old. Vous pouvez le supprimer entièrement si vous le souhaitez, mais vérifiez au préalable que tout fonctionne correctement.
  9. Pour simplifier l'administration, vous pouvez créer un lien symbolique (symlink) pointant vers ce nouvel emplacement dans le répertoire principal d'Urchin, par exemple
    • ln -s /bigdisk/urchin/data ./data.
  10. Redémarrez les services Urchin via la commande bin/urchinctl start.
  11. Connectez-vous à Urchin en tant qu'admin. L'écran Urchin relatif à la licence devrait s'afficher. Cliquez simplement sur le lien Réactiver la licence pour terminer la procédure.

Pour les systèmes Windows

  1. Arrêtez les services Urchin : menu Démarrer->Programmes->Urchin->Désactiver les services Urchin.

  2. Lancez l'Explorateur Windows et ouvrez le dossier etc de la distribution Urchin. Le chemin d'accès par défaut est C:\Program Files\Urchin\etc.
  3. Ouvrez le fichier "urchin.conf" dans un éditeur de texte.
  4. Supprimez le caractère "#" correspondant à un commentaire de la ligne suivante :
    • #dataDirectory: ./data/ et remplacez le dossier par défaut par la lettre du lecteur suivie du chemin d'accès complet au dossier que vous souhaitez utiliser, par exemple dataDirectory: E:\Urchin\data
  5. Le cas échéant, créez ce nouveau dossier de données. Important : Le service Urchin doit disposer d'une autorisation d'accès à ce dossier en lecture et en écriture.
  6. Copiez le contenu du dossier data dans le nouveau dossier.
  7. Renommez le dossier data en data.old. Vous pouvez le supprimer entièrement si vous le souhaitez, mais vérifiez au préalable que tout fonctionne correctement.

  8. Redémarrez les services Urchin : menu Démarrer->Programmes->Urchin->Activer les services Urchin.

Considérations

En raison du mode de mise en œuvre des licences sur les systèmes de type UNIX et pour prévenir toute tentative de falsification, vous devez réactiver votre licence Urchin lorsque vous déplacez le répertoire data d'Urchin. Cependant, vous obtiendrez une nouvelle licence très facilement, il suffit en effet de se connecter à Urchin en tant qu'utilisateur admin et de cliquer sur Réactiver la licence.
