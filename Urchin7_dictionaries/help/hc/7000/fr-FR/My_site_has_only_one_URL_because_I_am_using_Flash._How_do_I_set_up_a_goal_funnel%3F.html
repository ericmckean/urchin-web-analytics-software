Mon site ne comporte qu'une seule URL, car j'utilise Flash. Comment puis-je définir un entonnoir de conversion ?

Vous pouvez effectuer le suivi de tous les événements de navigateur qui se produisent sur votre site Web, notamment des événements Flash. La fonction JavaScript urchinTracker vous permet d'attribuer des noms de page à vos actions Flash. Vous pouvez alors associer ces noms de page à l'objectif approprié ou aux différentes étapes de l'entonnoir.

Pour effectuer le suivi d'un événement, appelez la fonction JavaScript urchinTracker() avec un argument spécifiant le nom de l'événement. Par exemple, si vous appelez : 

    javascript:urchinTracker('/purchase_funnel/page1.html');

chaque occurrence de l'événement Flash correspondant sera consignée comme s'il s'agissait d'une consultation de la page nommée /purchase_funnel/page1.html. L'argument doit débuter par une barre oblique.

Vous pouvez organiser les noms des événements selon la structure de répertoires de votre choix. L'argument chemin/nom de fichier de la fonction urchinTracker ne doit pas nécessairement représenter une URL existante de votre site Web.

Exemples de code Flash 

    on (release) {
    // Track with no action
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
    on (release) {
    //Track with action
    getURL("javascript:urchinTracker('/folder/file.html');");
    _root.gotoAndPlay(3);
    myVar = "Flash Track Test"
    }
    onClipEvent (enterFrame) {
    getURL("javascript:urchinTracker('/folder/file.html');");
    }
