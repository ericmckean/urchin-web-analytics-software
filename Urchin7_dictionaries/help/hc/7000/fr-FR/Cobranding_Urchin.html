Votre marque dans Urchin

Présentation

Urchin offre la possibilité d'apposer une marque dans l'interface d'administration, des rapports et sur l'écran de connexion. Trois fichiers sont à votre disposition pour inclure du contenu HTML dans la partie supérieure de l'interface. Si vous réalisez une intégration complète dans un portail, la création des rapports Urchin peut être fournie via votre serveur d'applications au sein d'un jeu de cadres ou dans un tableau.

Instructions pour l'apposition de votre marque

Pour personnaliser votre interface, vous devez modifier les fichiers suivants situés dans le répertoire d'installation Urchin : 

     [urchin install dir]/lib/custom/cobrands/cobrand_admin.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_report.tpl.sample
     [urchin install dir]/lib/custom/cobrands/cobrand_session.tpl.sample
  • Renommez les fichiers en supprimant la portion ".sample" (cobrand_admin.tpl par exemple).
  • Ajoutez à ces fichiers le contenu HTML nécessaire pour représenter votre marque.

Le premier fichier contrôle l'affichage de votre marque dans l'interface admin et le second dans l'interface des rapports. Le troisième fichier sert à présenter votre marque sur l'écran de connexion. Le code HTML que vous incluez dans ces fichiers sera placé en haut de l'interface Urchin.
