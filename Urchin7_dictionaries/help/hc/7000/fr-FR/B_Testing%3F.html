Quelles variables UTM faut-il utiliser pour les tests A/B ?

Les tests A/B vous permettent d'essayer différentes versions d'e-mails et d'annonces. Utilisez utm_source, utm_medium, utm_content et utm_campaign pour distinguer les liens des différents e-mails et annonces et les liens contenus dans chaque e-mail et annonce.

L'exemple suivant illustre l'utilisation des variables UTM dans une lettre d'information envoyée par courrier électronique. Dans cet exemple, la lettre d'information contient plusieurs liens vers une même page de votre site Web. Notez que utm_content est la seule valeur qui varie d'un lien à l'autre.

Le premier lien est un lien textuel : 

    http://www.example.com/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buytext&utm_campaign=brand_x

Le second lien est une image associée à un lien hypertexte : 

    http://www.example.com/buy_page?utm_source=newsletter&utm_medium=email&utm_content=buypict&utm_campaign=brand_x

Utilisezl'outil de création d'URL de Google Analytics pour créer vos liens. L'utilisation de cet outil garantit que vos liens auront une syntaxe correcte.

Important : Utilisez une orthographe et des noms identiques pour toutes les variables de la campagne.
