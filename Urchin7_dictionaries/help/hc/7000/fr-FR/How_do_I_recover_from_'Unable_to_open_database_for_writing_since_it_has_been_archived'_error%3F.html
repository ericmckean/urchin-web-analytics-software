Je rencontre l'erreur "Unable to open database for writing since it has been archived". Comment puis-je résoudre ce problème ?

Vous recevez l'erreur suivante lorsque vous tentez d'exécuter un profil dans Urchin 6 : 

WARNING: (7063-54-63) Unable to open database for writing since it has been archived

Cette erreur se produit lorsque les bases de données de suivi des journaux sont stockées de façon mensuelle avec Urchin. Lors d'un traitement, Urchin analyse les premières lignes du fichier journal, détermine le mois correspondant à ce journal, puis tente d'ouvrir les bases de données associées à ce mois. Les bases de données du mois concerné ayant été archivées, une erreur apparaît, car Urchin n'autorise pas la lecture ni l'écriture dans des bases de données archivées. Vous pouvez contourner ce problème de deux manières :

  1. Désactivez la fonction d'archivage, décompressez les fichiers *-archive.zip des profils concernés, puis supprimez les fichiers *-archive.zip.
  2. Modifiez les caractères génériques désignant les journaux en incluant une chaîne correspondant à une date (YY* ou YYYY* peuvent par exemple servir à identifier uniquement les fichiers de cette année).
