Importation de profils (Windows)

Présentation

La fonction Urchin d'importation de profils fournit aux utilisateurs de systèmes exécutant Microsoft Internet Information Server un outil efficace permettant de configurer rapidement les profils de leurs sites IIS. En effet, Urchin est capable de lire la configuration IIS, de détecter les sites Web présents sur le serveur et de constituer des profils de base pour les sites Web utilisant les journaux IIS comme source de journal. Vous pouvez ensuite personnaliser les profils ainsi créés et ajouter des profils relatifs aux sites importés.

Importation des profils

Pour commencer, connectez-vous à l'interface de gestion d'Urchin en tant qu'administrateur et cliquez sur Configuration (en haut à droite de l'interface). Cliquez sur le bouton Importer dans l'angle supérieur droit pour accéder à l'écran d'importation. Ce dernier vous permet de sélectionner les sites Web à importer, le cas échéant. Sélectionnez les sites à importer et cliquez sur le bouton Importer. Cliquez sur Terminé lorsque vous avez effectué toutes les importations voulues.

Recommandations

Il est conseillé de créer au moins un profil par site Web présent sur le serveur, afin d'obtenir une vue précise du trafic enregistré grâce aux rapports récapitulatifs d'Urchin. Ces rapports fournissent des informations globales sur le trafic associé au serveur, ainsi que le classement de chaque site en fonction de divers paramètres du trafic. Cela est particulièrement utile si vous êtes un hôte qui facture en fonction de la bande passante utilisée. Notez que les rapports récapitulatifs affichent uniquement les données des profils configurés. Les sites dont le profil ne fonctionne pas ne sont pas inclus.
