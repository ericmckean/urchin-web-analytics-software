Comment puis-je effectuer le suivi des clics sur les liens sortants ?

Si votre site comporte des liens sortants, vous pouvez facilement savoir sur quels liens les visiteurs ont cliqué pour sortir de votre site.

Pour effectuer le suivi des clics sortants, ajoutez le code suivant à la balise <a> correspondante : 

    <a href="http://www.site.com" onClick="javascript:urchinTracker('/outboundlinks/sitename/pagename');">

Ce code permet d'enregistrer chaque clic sur le lien sortant concerné comme s'il s'agissait d'une consultation de la page nommée /outboundlinks/sitename/pagename. Il est conseillé de répertorier l'ensemble de vos liens sortants dans une structure de répertoires logique, comme illustré dans l'exemple. Vous pourrez ainsi identifier facilement les pages sur lesquelles les visiteurs ont cliqué avant de quitter votre site.
