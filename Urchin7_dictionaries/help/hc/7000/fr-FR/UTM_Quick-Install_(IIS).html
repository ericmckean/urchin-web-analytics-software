Installation rapide de l'UTM (IIS)

  • Cet article a été rédigé par EpikOne, un consultant agréé du logiciel Urchin.

La procédure présentée ci-après permet une installation rapide de l'UTM pour des sites Web fonctionnant sur un serveur Microsoft IIS, quelle que soit la plate-forme Windows associée. Pour des informations plus détaillées sur l'UTM, consultez l'article intitulé "Urchin Traffic Monitor (UTM)" que vous trouverez dans la présente section.

Étape 1 : Copie des fichiers UTM à la racine du site Web. Les fichiers urchin.js et __utm.gif sont situés dans le répertoire "util\utm" de la distribution Urchin. Copiez ces deux fichiers dans le dossier principal du contenu de votre site Web. IMPORTANT : Le nom du fichier GIF commence par deux traits de soulignement.

Étape 2 : Référencement de l'UTM dans le code HTML de votre site Web. Saisissez la ligne ci-après dans toutes vos pages HTML. Cette ligne peut être placée n'importe où dans la page, mais nous vous recommandons de l'insérer juste avant la balise de fermeture </body>. Si vous utilisez une inclusion ou un modèle, vous pouvez l'insérer à cet endroit. 

<script src="/urchin.js" type="text/javascript"></script>

<script type="text/javascript">

        urchinTracker();

</script>

Si vous utilisez un logiciel du type "HTML Tidy", vous devez inclure la ligne JavaScript dans la section HEAD de votre page, par exemple : 

<html>

<head>

  <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1">

  <script src="/urchin.js" type="text/javascript"></script>

  <script type="text/javascript">

        urchinTracker();

  </script>

  ...

</head>

Étape 3 : Activation des cookies dans votre consignation IIS. Ouvrez le gestionnaire IIS et affichez la boîte de dialogue Propriétés de votre site Web. Assurez-vous que la consignation est activée et définie sur le format étendu W3C. Cliquez sur le bouton Propriétés en regard du format. Sous l'onglet Propriétés étendues, cochez la case en regard de Cookie.

4e étape : Définition du profil Urchin de sorte à utiliser Urchin Traffic Monitor (UTM). Dans l'interface d'administration Urchin, modifiez le profil en question et cliquez sur l'onglet Paramètres. Définissez la méthode de suivi des visiteurs sur Urchin Traffic Monitor (UTM). Attribuez au Domaine UTM l'adresse de votre site Web sans préfixe www. Définissez le paramètre par défaut pour l'option Groupe de rapports, puis cliquez sur le bouton Mettre à jour.

L'opération est terminée. Votre site Web consignera désormais les données UTM dans votre fichier journal habituel, qui sera identifié la prochaine fois que vous lancerez Urchin.

Cela fonctionne-t-il ? Pour vérifier que l'UTM génère bien des entrées dans votre fichier journal, examinez votre journal après l'installation de l'UTM et cliquez sur quelques pages du site. Une entrée semblable à la suivante devrait apparaître à la fin du fichier journal : 

... "GET /__utm.gif?..." 200 ..."__utma=..."

Si les entrées utma n'apparaissent pas, vérifiez que les cookies sont correctement activés dans la consignation. Si un autre code d'état que 200 s'affiche, vérifiez que les fichiers ont bien été copiés à la racine du site.
