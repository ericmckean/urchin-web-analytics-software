Les liens menant à mon site contiennent déjà mes propres variables de suivi de campagne. Puis-je utiliser ces variables à la place de celles d'Urchin ?

Si vous avez déjà créé des liens avec vos propres variables de suivi, vous pouvez les associer aux variables de suivi de campagne d'Urchin. Vos liens devront alors contenir au moins trois variables. Elles seront associées aux éléments suivants :

  1. utm_source: la source de votre campagne (par exemple Google, Yahoo) ;
  2. utm_medium: le support de votre campagne (cpc, bannière, e-mail) ;
  3. utm_campaign: le nom de votre campagne (soldes_printemps).

Dans le cadre de ces instructions, nous supposerons que votre site contient l'URL suivante : 

            www.example.com/index.html?camp=GADS&kwd=memory&version=47&type=ppc&adver=google

Étape 1 : Associez vos variables à celles d'Urchin. Créez un tableau au format indiqué dans l'exemple ci-dessous : 

               Campagne      Source de l'annonce        Type d'annonce      Mot clé   Contenu A/B  Principal
Codes utilisateur =   camp          adver       type        kwd       version      
Codes Urchin =  utm_campaign utm_source         utm_medium  utm_term  utm_content  utm_id

Étape 2 : Modifiez le fichier UTM.

  • Ouvrez le fichier urchin.js dans un éditeur de texte. Au début du fichier se trouve une section définie par l'utilisateur qui comporte les variables utm : 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="utm_campaign";
                var _ucmd="utm_medium";
                var _ucsr="utm_source";
                var _uctr="utm_term";
                var _ucct="utm_content";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";
  • Avec l'exemple du lien balisé ci-dessus, vous devez modifier cette section comme suit : 

                //-- UTM Campaign Tracking Settings
                var _uctm=1;
                var _ucto="15768000";
                var _uccn="camp";
                var _ucmd="type";
                var _ucsr="adver";
                var _uctr="kwd";
                var _ucct="version";
                var _ucid="utm_id";
                var _ucno="utm_nooverride";

Étape 3 : Créez des filtres pour ces nouvelles variables de suivi.

  • Voici la configuration de base que vous devez utiliser pour chaque filtre : 

                Type de filtre : Avancé
                Champ A : uri_campagne_utm (brut)
                Extraire A : nouvelle_variable =([^&]*)
                Champ B : (Aucun)
                Extraire B : (vide)
                Sortie vers : utm _( ancienne_variable )
                Constructeur : $A1
                Champ A obligatoire : Oui
                Champ B obligatoire : Non
                Remplacer le champ de sortie : Oui
                Sensible à la casse : Non
  • ancienne_variable fait référence à la variable de suivi d'Urchin par défaut. nouvelle_variable fait référence à votre propre variable de suivi. Rappelez-vous que vous aurez besoin d'un filtre pour chaque variable modifiée.
