Comment puis-je exclure de mes rapports des sites référents de mon site ?

Pour exclure de vos rapports les sites référents de votre propre site, vous devez apporter une légère modification au fichier urchin.js. Ouvrez ce fichier dans votre éditeur de texte courant et insérez la section suivante : 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";

Dans le cas ci-dessus, tout appel de fichier contenant .urchin.com sera exclu de vos sites référents. Si vous avez des domaines supplémentaires à ajouter, modifiez le fichier pour inclure une autre valeur _uRno[x] et augmentez la valeur numérique de x dans le même temps. Par exemple : 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";
    _uRno[1]=".urchin.net";
    _uRno[2]=".urchinsoftware.com"

Rappelez-vous que chaque fois que vous modifiez le fichier urchin.js, vous devez télécharger le fichier modifié sur votre serveur Web pour que les modifications deviennent effectives.
