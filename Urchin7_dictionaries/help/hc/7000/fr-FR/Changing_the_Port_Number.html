Modification du numéro de port

Par défaut, le serveur Web Urchin écoute le numéro de port 9999. Deux étapes sont nécessaires pour modifier ce numéro de port :

  • Modification du numéro de port sur l'écran Paramètres du serveur
  • Arrêt et redémarrage des services Urchin (les procédures diffèrent légèrement entre les systèmes Windows et Unix)

Voici la procédure à suivre :

  • Connectez-vous à l'interface d'administration Urchin.

  • Suivez Configuration->Paramètres globaux->Paramètres d'accès.

  • Saisissez le nouveau numéro de port dans le champ Numéro de port du serveur.
  • Cliquez sur le bouton Mettre à jour.

Vous devez à présent redémarrer les services Urchin.

  • Avec un système de type Unix, accédez au répertoire bin de votre distribution Urchin et exécutez la commande : 

          ./urchinctl restart

  • Avec un système Windows, ouvrez le menu Démarrer->Programmes->Urchin 6 à partir du Bureau et sélectionnez Désactiver les services Urchin 6, puis Activer les services Urchin 6.

Le serveur Web écoute alors les demandes de connexion sur le nouveau numéro de port. Cela signifie également que l'URL utilisée pour consulter les rapports et configurer le logiciel Urchin a changé et que vous devez communiquer cette nouvelle URL à vos utilisateurs.

Remarques Dans de nombreux systèmes, vous devez disposer des droits d'administrateur (root) pour utiliser les numéros de port inférieurs à 1024. Sachez également que le démarrage d'Urchin échoue lorsqu'un autre service s'exécute sur le numéro de port spécifié.
