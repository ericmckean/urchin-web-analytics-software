Les cinq dimensions du suivi de campagne

Urchin 6 effectue le suivi des campagnes en ligne en associant les cinq dimensions marketing suivantes :

  • Source
  • Support
  • Terme
  • Contenu
  • Campagne

Source Chaque accès à un site Web a une origine ou source. En voici quelques exemples : moteur de recherche Google ou AOL, nom d'une lettre d'information ou d'un site Web.

Support Le support permet de qualifier la source ; ensemble, la source et le support fournissent des informations particulières sur la provenance. Par exemple, si la source est "moteur de recherche Google", le support peut être "coût par clic", ce qui indique un lien commercial payé par l'annonceur, ou "naturel", ce qui indique un lien référencé gratuitement dans les résultats du moteur de recherche. Si la source est une lettre d'information, le support peut être "e-mail" ou "papier".

Terme Le terme ou mot clé représente le mot ou l'expression tapé par un utilisateur dans un moteur de recherche.

Contenu La dimension contenu décrit la version d'une annonce sur laquelle un visiteur a cliqué. Elle est utilisée pour la publicité ciblée sur le contenu et pour le test comparatif (A/B) afin de déterminer la version d'une publicité la plus efficace pour attirer des prospects intéressants.

Campagne La dimension campagne vise à différencier les promotions de produits, promotions de printemps sur les skis, par exemple, ou campagnes avec slogans du type "Préparez-vous pour l'été".
