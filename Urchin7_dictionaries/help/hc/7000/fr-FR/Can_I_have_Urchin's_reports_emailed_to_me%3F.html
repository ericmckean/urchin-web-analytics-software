Est-il possible de recevoir des rapports d'Urchin par e-mail ?

  • Actuellement, Urchin ne fournit pas de rapports par e-mail. Pour consulter vos rapports, connectez-vous à votre compte Urchin. Toutefois, si vous souhaitez manipuler vos données en dehors de votre compte Urchin, il est possible d'utiliser la fonction d'exportation des données d'Urchin pour extraire manuellement les données des rapports Urchin. Ceci est utile pour importer les données dans un tableur, un traitement de texte ou une base de données pour analyse ultérieure.
