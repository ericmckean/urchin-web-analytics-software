Filtres de tableau de conversion

Le filtre Tableau de conversion vous permet de :

  • Mettre en œuvre des codes de suivi principaux pour le suivi de campagne. Reportez-vous à la rubrique Utilisation des codes de suivi principaux.

  • Mettre en œuvre des données de tableaux externes pour convertir et remplacer des caractères de champs lorsqu'une correspondance est détectée. Les tableaux de conversion peuvent établir des correspondances avec un seul champ et mettre à jour plusieurs champs.

Pour utiliser ce filtre :

  1. Dans la fenêtre Assistant de filtres : Paramètres, saisissez le nom de filtre voulu (champ Nom du filtre).
  2. Sélectionnez le type de filtre "Tableau de conversion".
  3. Sélectionnez le champ à mettre à jour dans le menu déroulant Champ de filtrage.

  4. Sélectionnez un tableau prédéfini dans le menu déroulant Nom du tableau (voir la rubrique Utilisation des codes de suivi principaux).
