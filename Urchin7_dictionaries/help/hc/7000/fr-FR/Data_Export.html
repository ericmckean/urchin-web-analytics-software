Exportation des données

Les boutons d'exportation des données d'Urchin permettent d'extraire facilement les données de rapports Urchin. Ceci est utile pour importer les données dans un tableur, un traitement de texte, une base de données, etc. pour analyse ultérieure.

Pour exporter des données d'un rapport, sélectionnez le type d'exportation approprié en fonction de l'application que vous utiliserez pour manipuler les données. Pour les importations générales vers une base de données, utilisez le format séparé par des tabulations.

  • Tabulation : cliquez sur le bouton "T" pour exporter les données dans un format délimité par les tabulations.
  • Word: cliquez sur l'icône Word pour exporter les données dans le format natif Microsoft Word.
  • Excel: cliquez sur l'icône Excel pour exporter les données dans le format natif Microsoft Excel.

Impression : cliquez sur l'icône de l'imprimante pour obtenir une vue prête à imprimer. Pour imprimer ce rapport, cliquez sur le lien Imprimer la page de cet écran.
