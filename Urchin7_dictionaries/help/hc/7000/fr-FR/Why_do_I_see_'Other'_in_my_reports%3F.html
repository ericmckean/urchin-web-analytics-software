Pourquoi la mention "Autre" apparaît-elle dans mes rapports ?

Chaque enregistrement d'un rapport Urchin réside dans une base de données à taille fixe. Les fichiers qui composent la base de données sont soumis à une rotation mensuelle. Il y a donc une limite au nombre d'enregistrements uniques qui peuvent être répertoriés chaque mois dans la base de données.

Si cette limite est atteinte, Urchin crée un dernier enregistrement nommé "autre". L'enregistrement "autre" peut être en fait constitué de plusieurs enregistrements. Une fois cette limite atteinte, tous les enregistrements uniques supplémentaires qui sont ajoutés aux données de rapport sont transférés dans l'enregistrement "Autre". Ainsi, si cinq nouvelles pages sont consultées alors que la base de données est pleine, l'enregistrement appelé "autre" comptera cinq consultations de page supplémentaires.

La limite pour les enregistrements de la base de données peut être augmentée. Par défaut, cette limite est définie à 100 000 enregistrements. Cela signifie qu'Urchin enregistrera chaque mois un total de 100 000 enregistrements uniques pour chaque profil. Vous pouvez augmenter ce nombre si vous disposez de l'accès "Super administrateur". Si vous ne disposez pas de l'accès approprié, vous devrez contacter votre administrateur Urchin et lui demander d'effectuer pour vous les modifications nécessaires.

Augmentation de la limite globale de la base de données :

  • Cliquez sur le bouton Configuration (en haut à droite de l'interface).
  • Cliquez sur Paramètres (menu de navigation de gauche).
  • Cliquez sur l'onglet Paramètres de la base de données.
  • Utilisez la flèche de la liste déroulante pour augmenter la limite de la base de données conformément à la valeur souhaitée.
  • Cliquez sur Mettre à jour pour appliquer les modifications.

Augmentation de la limite de la base de données pour un profil existant :

  • Cliquez sur le bouton Configuration (en haut à droite de l'interface).
  • Cliquez sur la clé de modification dans le profil à modifier.
  • Cliquez sur l'onglet Stockage/Base de données.
  • Modifiez la limite de la base de données conformément à la valeur souhaitée. L'utilisation de l'option "Valeur globale par défaut" fait concorder la limite du profil avec celle du paramètre Global courant comme indiqué plus haut.
  • Cliquez sur Mettre à jour.

Remarque : Gardez à l'esprit que cette fonction augmente la taille de l'espace disque utilisé ainsi que la durée nécessaire à l'ouverture des différents écrans du rapport. En effet, l'interface de rapports Urchin doit récupérer les données dans une base de données plus grande.
