Suivi des transactions dans une boutique sécurisée

Cet article a été rédigé par ivantage Limited, un consultant agréé du logiciel Urchin.

Pour qu'Urchin puisse effectuer le suivi des données d'un panier sur votre domaine et même sur d'autres domaines, il vous suffit d'ajouter le code approprié. Si vous souhaitez configurer vos pages de manière à transmettre à Urchin des rapports relatifs aux transactions, consultez la rubrique Comment puis-je effectuer le suivi des transactions en ligne?.

En outre, si votre boutique en ligne n'est pas sur le même sous-domaine ou domaine que votre site principal, appliquez la procédure décrite ci-dessous pour configurer vos pages.

Si votre site Web lance un processus de commande sur un sous-domaine différent (par exemple si les clients sont envoyés de www.maboutique.fr à panier.maboutique.fr) :

  • Ajoutez la ligne suivante (en gras) au code de suivi des pages de votre boutique en ligne et des pages où se trouve le panier : 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

Si votre site Web lance un processus de commande sur une autre boutique en ligne (par exemple si les clients sont envoyés de www.maboutique.fr à www.paniersecurise.fr) :

  • Ajoutez les lignes suivantes (en gras) au code de suivi des pages de votre boutique en ligne et des pages où se trouve le panier : 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • Pour utiliser utmLinker, remplacez les liens du site principal par ceux du site sécurisé en procédant de la manière suivante. Si vos liens actuels sont de type : 

     <a href="https://www.securecart.com/?store=parameters">Acheter</a>
  • remplacez-les par des liens de type : 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">Acheter</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">Acheter</a>
   </noscript>

  • Le code ci-dessus vous permet d'afficher les liens avec ou sans JavaScript. Notez bien que les apostrophes doivent être précédées du caractère d'échappement (barre oblique inverse) dans le lien ou dans le texte du lien. Important : Si vos pages contiennent un appel à la fonction urchinTracker(), utmLinker(), utmSetTrans() ou utmLinkPost(), votre code de suivi Analytics doit être inséré dans votre code HTML au-dessus de cet appel. Dans ce cas, le code de suivi peut être inséré n'importe où entre la balise d'ouverture <body> et l'appel JavaScript.

Si vous envoyez des informations au panier à l'aide de formulaires, utilisez la fonction utmLinkPost(). Elle ajoute des cookies à l'URL fournie pour le formulaire. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • Cela fonctionne également pour les formulaires avec method="GET".
