Suivi des liens sortants et des annonces

Urchin permet d'assurer facilement le suivi des clics sur les bannières publicitaires ou sur d'autres liens sortants. Comme ces liens ne mènent pas à une page de votre site contenant le code JavaScript de l'UTM, vous devrez les baliser avec le code JavaScript urchinTracker si vous souhaitez effectuer le suivi des clics associés. Ce fragment de code JavaScript affecte une consultation de page à tout clic sur le lien, la consultation de page étant attribuée au nom de fichier que vous spécifiez.

Pour un GIF animé ou tout autre type de bannière statique, il vous suffit d'ajouter le code JavaScript urchinTracker à l'intérieur de la balise de votre bannière : 

    <a href="http://www.example.com/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
