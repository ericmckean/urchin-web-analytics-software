Traitement de journaux archivés

Présentation

Vous avez la possibilité de traiter vos journaux archivés après avoir installé Urchin. Cette opération est très simple à réaliser. Il suffit d'indiquer un répertoire ainsi qu'un nom de fichier partiel et/ou un caractère générique (ou une expression rationnelle) dans les écrans Paramètres du journal du Gestionnaire de journaux. REMARQUE : N'utilisez aucun caractère générique pour les sources de journal HTTP et HTTPS distantes.

Procédure de traitement de journaux archivés

Ajoutez tout d'abord une source de journal au système. Cliquez sur Configuration dans le menu situé en haut, puis sur Profils Urchin -> Gestionnaire de journaux. Dans l'écran principal, cliquez sur le bouton Ajouter en haut à droite. Sur le premier écran, sélectionnez Ajouter une source de journal locale, puis continuez. Sur l'écran suivant, cliquez sur Parcourir pour afficher un explorateur de fichiers. Recherchez le répertoire voulu dans le volet de gauche. Le volet de droite présente les fichiers du répertoire, tandis que le volet de gauche affiche le répertoire ouvert et les sous-répertoires qui lui sont associés. Une fois dans le répertoire souhaité, saisissez un nom de fichier partiel et un astérisque (ou une expression rationnelle), puis cliquez sur le bouton Vérifier. Une fenêtre apparaît contenant tous les fichiers correspondant à votre modèle. Cliquez sur un nom de fichier pour obtenir des informations le concernant : emplacement, taille, date de dernière modification et autorisation d'accès. Si la correspondance établie est satisfaisante, cliquez sur OK, puis de nouveau sur OK dans la fenêtre de l'explorateur de fichiers. Indiquez ensuite le Nom de la source de journal et cliquez sur Suivant. Sélectionnez un format de journal dans le menu déroulant et cliquez sur Terminer. Si le Nom de la source de journal est exact (ce doit être un nom unique), cliquez sur le bouton Terminer et votre liste de source de journal apparaît dans le Gestionnaire de journaux.

Si ce n'est pas encore fait, vous pouvez alors associer ce fichier journal à un Profil en cliquant sur le bouton Configuration dans le menu supérieur, puis en suivant Profils Urchin -> Profils. Une fois l'association établie, cliquez sur Exécuter/Planifier en regard du profil dans la liste principale. Vous pouvez planifier l'exécution du profil ou cliquer sur le bouton Exécuter, afin de lancer l'opération immédiatement.

Urchin empêche la duplication des données sans nécessiter de rotation des journaux, car il est doté d'une fonction de suivi garantissant que les données déjà lues ne sont pas retraitées. Toutefois, comme indiqué précédemment, les fichiers journaux peuvent rapidement occuper de grands espaces sur le disque et il est recommandé de compresser et d'archiver régulièrement les fichiers journaux. Urchin ne réanalyse jamais les fichiers journaux déjà traités et il est tout à fait envisageable de supprimer le ou les fichiers journaux après chaque traitement. Cependant, de nombreuses personnes préfèrent conserver leurs journaux sur une période donnée par mesure de précaution. En effet, les journaux sont nécessaires lorsqu'un nouveau Profil est créé ou pour réaliser des analyses de l'historique.

Recommandations

La gestion des fichiers journaux peut paraître superflue au départ, mais elle devient essentielle lorsque les journaux atteignent une taille importante. Il est donc préférable de définir une stratégie de gestion des journaux dès le déploiement du logiciel Urchin.
