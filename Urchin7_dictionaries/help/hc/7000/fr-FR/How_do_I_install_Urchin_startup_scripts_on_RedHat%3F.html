Comment puis-je installer les scripts de démarrage d'Urchin sur RedHat ?

Le script de démarrage d'Urchin est créé au cours de l'installation. Il est nommé "urchin_daemons" et est situé dans le répertoire util d'Urchin. "/path/to/urchin/util/"

Copiez ce script dans le répertoire /etc/rc.d/init.d/ directory.

  • cd /path/to/urchin/util
  • cp urchin_daemons /etc/rc.d/init.d/
  • chmod +x /etc/rc.d/init.d/urchin_daemons
  • Consultez le fichier /etc/inittab afin de déterminer le niveau d'exécution par défaut (3 en général). [[Voir ci-dessous]]
  • Accédez ensuite au script /etc/rc.d/rcX.d (où X représente le niveau d'exécution).
  • cd /etc/rc.d/rc3.d
  • Créez des liens symboliques pointant vers /etc/rc.d/init.d/urchin_daemons nommés S95urchin et K05urchin.
  • ln -s ../init.d/urchin_daemons S95urchin
  • ln -s ../init.d/urchin_daemons K05urchin

[[ # Niveau d'exécution par défaut. Les niveaux d'exécution utilisés par RHS sont les suivants :

  • 0 - Arrêt (Ne JAMAIS définir cette valeur pour initdefault)
  • 1 - Mode mono-utilisateur
  • 2 - Mode multi-utilisateur, sans NFS (Identique au niveau 3, si vous n'avez pas de mise en réseau)
  • 3 - Mode multi-utilisateur complet
  • 4 - Non utilisé
  • 5 - X11
  • 6 - Redémarrage (Ne JAMAIS définir cette valeur pour initdefault)
