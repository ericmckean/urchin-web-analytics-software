Pourquoi la mention Microsoft Data Access Internet Publishing Provider apparaît-elle dans le rapport Navigateurs ?

Ce type de navigateur apparaît généralement si votre site contient des fichiers téléchargeables qui sont généralement ouverts à l'aide d'un produit Microsoft Office (par exemple, des fichiers Word .doc et des fichiers Excel .xls). Les plug-ins utilisés par Internet Explorer pour afficher ces types de fichier se signalent eux-mêmes comme Microsoft Data Access Internet Publishing Provider lors de l'affichage de cette page.
