Comment puis-je exclure le trafic interne de mes rapports ?

  1. Cliquez sur le bouton Configuration (en haut à droite de l'interface).
  2. Cliquez sur l'icône de clé Modifier située en regard du profil dont vous souhaitez exclure le trafic interne.
  3. Cliquez sur l'onglet Filtres.
  4. Cliquez sur l'icône "+" (Ajouter).
  5. Dans le menu déroulant Exemples de filtres, sélectionnez "Filtrer le trafic interne".
  6. Cliquez sur Terminer.
