Liens directs vers les rapports Urchin

Présentation

Dans une installation Urchin standard, la distribution des rapports est gérée par un contrôleur de session intégré et par le serveur Web Apache inclus avec le logiciel Urchin. Pour afficher leurs rapports Urchin, les utilisateurs doivent s'authentifier via une procédure de connexion gérée par Urchin qui leur présente ensuite les rapports qu'ils sont autorisés à consulter.

Cependant, il est possible de contourner le contrôleur de session et l'authentification Urchin et de fournir aux utilisateurs un lien direct vers les rapports Urchin à partir d'un portail ou d'un site Web externe. Cette fonctionnalité est incluse dans Urchin 6 en tant qu'élément d'intégration standard. Urchin 6 vous permet de gérer les utilisateurs et de leur donner accès à de nombreux rapports via l'utilisation des Comptes (voir la rubrique Gestion des comptes). Avec Urchin 6, vous bénéficiez des liens directs aux rapports, que vos données soient situées ou non dans les répertoires de données/de rapports de votre distribution Urchin.

Procédure

Étape 1 : Activer l'option "Liens directs aux rapports"

  • Connectez-vous à l'interface de gestion d'Urchin en tant qu'utilisateur "admin".

  • Accédez à l'écran Paramètres->Paramètres globaux->Paramètres d'accès.

  • Définissez le champ "Liens directs aux rapports" sur "activé".

Étape 2 : Configurer les liens vers les rapports Urchin

  • Pour chaque profil (rapport) pour lequel vous souhaitez fournir un lien, créez un lien selon le format suivant : 

(baseurl)/report.cgi?profile=(profilename)&user=(username)
  • où
  • (baseurl) correspond à tous les éléments situés avant session.cgi dans l'URL actuelle,
  • (profilename) représente le nom du profil extrait de l'URL (URI-escaped),
  • et (username) correspond à l'utilisateur ayant accès au rapport.
  • Le paramètre user= définit l'autorisation de consulter le rapport, ainsi que les préférences linguistiques et de localisation.

    • http://www.hollywoodweb.com/report.cgi?profile=www.hollywoodweb.com&user=johng
