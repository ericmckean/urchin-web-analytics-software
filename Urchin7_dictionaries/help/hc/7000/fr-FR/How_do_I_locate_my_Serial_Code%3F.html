Comment puis-je trouver mon Code de série ?

Il existe deux méthodes pour connaître votre Code de série Urchin actuel.

  1. La console d'administration
    • Ouvrez votre console d'administration Urchin dans un navigateur Web.

      Suivez le menu Configuration--> Paramètres--> Licence.

  2. L'utilitaire inspector
    • Ouvrez une fenêtre de commande et accédez au répertoire basedir/urchin/util/ directory à l'aide d'une commande cd. Lancez l'utilitaire inspector. Votre numéro de série s'affiche alors à l'écran, accompagné d'autres informations relatives à Urchin. Notez que sous Unix/Linux, toutes les commandes doivent être précédées de './ '.
