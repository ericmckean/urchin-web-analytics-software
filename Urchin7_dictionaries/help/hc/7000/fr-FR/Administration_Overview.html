Vue d'ensemble de l'administration

Introduction

L'interface d'administration Urchin est un centre de gestion via navigateur à partir duquel vous pouvez contrôler pratiquement tous les aspects du fonctionnement du logiciel Urchin, notamment la configuration des profils, la planification des tâches de traitement des journaux, la gestion des utilisateurs et des groupes, la configuration des filtres et bien plus encore.

Pour commencer, connectez-vous à votre système Urchin à l'aide d'un navigateur. Si vous avez utilisé le port par défaut au cours de l'installation, l'URL à saisir est http://votre.serveur.fr:9999/. Vous devez remplacer "votre.serveur.fr" par le nom du système sur lequel Urchin est exécuté. Vous pouvez également entrer l'adresse http://localhost:9999/ si vous utilisez directement le système en question. Sur les plates-formes Windows, un raccourci "Administration Urchin" est placé dans le menu Démarrer. Le mot de passe par défaut du compte "admin" est "urchin". Veillez à choisir un mot de passe plus sûr.

Utilisation de l'interface d'administration

  • Cliquez sur Configuration en haut à droite de l'écran pour accéder à l'interface d'administration.
  • Cette interface vous donne accès aux outils de configuration. Cliquez sur les flèches dans le volet de navigation à gauche pour développer une section (Profils Urchin, Utilisateurs et groupes, Planificateur et Paramètres). L'élément de couleur sombre est celui qui est actuellement affiché à l'écran.
  • Cliquez sur le bouton "Modifier" en regard d'une entrée lorsque vous souhaitez modifier sa configuration. Pour ajouter une entrée, cliquez sur le bouton "Ajouter" (icône portant le signe "+") dans l'angle supérieur droit du tableau. Si vous cliquez sur le bouton "Modifier" d'une entrée, les écrans de configuration associés apparaissent dans les onglets situés en haut de l'écran.
  • Cliquez sur un onglet pour accéder aux paramètres de configuration correspondants. Veillez à cliquer sur le bouton "Mettre à jour" au bas de l'écran si vous modifiez des paramètres.
  • Si vos listes d'entrées sont trop longues pour s'afficher dans une zone donnée, utilisez les boutons Suivants/Précédents situés au-dessus et en dessous du tableau pour les faire défiler. Vous pouvez également choisir le nombre d'entrées affichées dans le tableau en modifiant la valeur du champ "Nombre d'éléments à afficher" (10 par défaut) situé au-dessus du tableau.
  • L'option Filtre + - (inclure/exclure) vous permet de retrouver une entrée rapidement. Il suffit de saisir tout ou partie d'une entrée et de cliquer sur le bouton plus (inclure) ou moins (exclure).
  • Des informations utiles relatives à chaque écran sont fournies en dessous du tableau.
