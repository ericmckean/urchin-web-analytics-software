Comment puis-je modifier l'UID (utilisateur) avec lequel Urchin s'exécute sur les plates-formes de type UNIX ?

La modification de l'utilisateur (UID) et/ou du groupe (GID) avec lequel Urchin s'exécute est relativement simple. Vous devrez probablement disposer de droits "root" pour effectuer les opérations ci-dessous. Votre administrateur système devra donc se charger de la plupart d'entre elles.

  1. Si l'utilisateur et le groupe concernés ne sont pas encore enregistrés sur votre système, créez les entrées appropriées dans les fichiers de mots de passe et de groupes.
  2. Arrêtez les services urchin à l'aide de la commande "urchinctl stop". Le fichier binaire "urchinctl" se trouve dans le répertoire "bin" de la distribution Urchin.
  3. Exécutez la commande "chown -R newuser:newgroup" dans le répertoire racine de la distribution Urchin. Si vous avez modifié l'emplacement des répertoires "var" et "data" dans le fichier "urchin.conf", vous devez également exécuter cette commande "chown" à partir de ces répertoires.
  4. Modifiez les lignes suivantes du fichier "urchinwebd.conf.template" situé dans le répertoire "var" de la distribution Urchin :
    • User newuser Group newgroup
  5. Redémarrez les services Urchin via la commande "urchinctl restart".
