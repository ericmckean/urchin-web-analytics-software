Obtention d'une licence Urchin

Il existe trois types de licences Urchin :

  • Une licence standard (antérieure à Urchin  6.6), à validité permanente.
  • Une licence USP (Urchin 6.6+), avec une date d'expiration déterminée à l'achat.
  • Une licence de démo, avec des fonctionnalités identiques aux licences standard et USP, mais dont le nombre de profils et de sources de journal est limité à cinq.

Obtention et activation d'une licence de démo

Pour obtenir une licence de démo, connectez-vous à l'interface d'administration Urchin avec un navigateur Web (généralement sur http://your.server.com:9999). Cliquez sur "Obtenir une licence de démo" et acceptez le contrat de licence pour mettre fin à la procédure.

Une fois votre licence de démo obtenue, un écran vous présente des informations détaillées concernant votre nouvelle licence. Il s'agit des éléments suivants : type de licence, code de série, nombre de profils, sources du journal parallèles, journaux par profil, système d'exploitation. Nous vous recommandons de toujours conserver les informations relatives à votre licence et en particulier le code de série, pour référence ultérieure. Une fois Urchin correctement installé, les détails concernant la licence (état de la licence, date d'expiration, code de série et version) peuvent être consultés en sélectionnant "Accueil" -> "Configuration" -> "Paramètres" -> "Licence".

Après avoir vérifié les détails de la licence, activez la licence de démo. Cliquez sur le lien "Installer la licence" affiché dans l'écran "Informations de licence". Un assistant apparaît. Il est recommandé de remplacer le mot de passe d'administration par défaut par votre propre mot de passe.

REMARQUE : Lorsque vous configurez les Paramètres d'accès à distance et les Paramètres du panneau de configuration admin, veillez à sélectionner l'option pour les Paramètres du panneau de configuration admin afin de pouvoir attribuer différents niveaux de droits d'administrateur à un ou plusieurs profils dans un groupe. Vous ne pourrez plus modifier ce paramètre une fois l'installation terminée.

Si vous parvenez à activer votre nouvelle licence de démo, l'accueil admin Urchin apparaît à l'écran. Cette page vous permet de créer des profils, des journaux de traitement, de consulter des rapports, de définir vos préférences, etc.

Une fois la licence de démo arrivée à expiration, l'écran de licence Urchin s'affiche et indique l'état détaillé de la licence comme suit : Licence valide - Démo (Arrivé à expiration). Pour installer une licence préachetée, cliquez sur le lien "Activer la licence".

Une fois la licence de démo activée, l'écran de licence Urchin s'affiche et indique l'état détaillé de la licence comme suit : Licence non valide. Pour réactiver la licence de démo existante, cliquez sur le lien "Réactiver la licence" ou sur "Activer la licence" pour installer une licence préachetée.

Achat d'une licence standard

Veuillez contacter un consultant Urchin agréé pour acheter votre licence.

Activation d'une licence standard achetée

Une fois en possession du code de série associé à votre licence Urchin permanente, vous devez activer celle-ci de la manière suivante :

  • Sélectionnez "Configuration" -> "Paramètres" -> "Licence".

  • Cliquez sur le lien "Activer la licence".
  • Sur la page "Activer le code de série", saisissez votre adresse e-mail et votre code de série.

À la fin de la procédure, Urchin est entièrement fonctionnel et vous disposez d'une licence permanente.

Important : Veillez à saisir des coordonnées exactes lors de l'obtention ou de l'activation de votre licence de démo. Ces informations seront nécessaires à nos services de facturation et d'assistance. Une fois Urchin correctement installé, les détails concernant la licence (état de la licence, date d'expiration, code de série et version) peuvent être consultés en sélectionnant "Accueil" -> "Configuration" -> "Paramètres" -> "Licence".

Si une licence standard devient non valide, l'écran de licence Urchin s'affiche et indique l'état détaillé de la licence comme suit : Licence non valide. Pour réactiver une licence standard existante, cliquez sur le lien "Réactiver la licence" ou sur "Activer la licence" pour installer une licence USP préachetée.

Installation d'une licence sans accès à Internet

Il est possible d'installer une licence Urchin sans accéder à Internet, avec un ordinateur situé derrière un pare-feu par exemple. Pour ce faire, vous devez lancer l'utilitaire "inspector", qui est inclus dans Urchin et situé dans le répertoire "util". Des instructions sur le fonctionnement de l'utilitaire "inspector" sont disponibles dans la rubrique inspector: vérificateur d'intégrité de l'installation.) Une fois en possession des résultats de l'utilitaire "inspector", contactez votre consultant Urchin agréé afin de procéder à une activation manuelle.

Achat d'une licence USP

Veuillez contacter un consultant Urchin agréé pour acheter votre licence.

Activation d'une licence USP achetée

Une fois en possession du code de série associé à votre licence Urchin permanente, vous devez activer celle-ci de la manière suivante :

  • Sélectionnez "Configuration" -> "Paramètres" -> "Licence".

  • Cliquez sur "Activer la licence".
  • Sur la page "Activer le code de série", saisissez votre code de série.

À la fin de la procédure, Urchin est entièrement fonctionnel jusqu'à ce que la licence arrive à expiration.

Si votre licence USP arrive à expiration, contactez un consultant Urchin agréé pour acheter une nouvelle licence.

Mise à niveau vers Urchin 6.6

Il est déconseillé de mettre à niveau une instance d'Urchin qui utilise une licence de démo, car cela peut entraîner des problèmes de connexion/licence. Après avoir effectué une mise à niveau, il est déconseillé de modifier la licence si une version précédente d'Urchin était utilisée avec une licence standard sur Windows. La licence doit être réactivée après une mise à niveau, si la version précédente d'Urchin était installée avec une licence standard sur Linux/FreeBSD (cliquez sur le lien "Réactiver la licence" dans l'écran "Licence Urchin").

Si une licence standard devient non valide (et ce, quelle que soit la plate-forme), l'écran "Licence Urchin" s'affiche et indique l'état détaillé de la licence comme suit : Licence non valide. Pour réactiver une licence standard existante, cliquez sur le lien "Réactiver la licence" ou sur "Activer la licence" pour installer une licence USP préachetée.
