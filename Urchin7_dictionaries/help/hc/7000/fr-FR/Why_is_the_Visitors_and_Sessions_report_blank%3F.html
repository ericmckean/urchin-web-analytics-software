Pourquoi le rapport sur les visiteurs et les sessions est-il vide ?

L'onglet du rapport sur les visiteurs et les sessions n'est normalement visible que si vous utilisez le détecteur UTM (Urchin Traffic Monitor) pour effectuer le suivi des visiteurs.

Le rapport sur les visiteurs et les sessions affiche principalement des données sur les visiteurs uniques. Vous devez utiliser le détecteur UTM si vous souhaitez qu'Urchin effectue le suivi du trafic des visiteurs uniques et renseigne ce rapport à l'aide de ces données. Lorsque le détecteur UTM est utilisé, Urchin peut déterminer la part des sessions visiteurs constituée par de nouveaux visiteurs de votre site Web et celle constituée par les visiteurs connus. Si vous n'utilisez pas le détecteur UTM, Urchin peut uniquement calculer le total des "sessions" visiteurs. Le total des sessions visiteurs est affiché sous le rapport "Trafic". Vous pouvez également afficher le total des "sessions" visiteurs sous le rapport sur les visiteurs et les sessions, "Sessions par jour". Pour toutes les autres vues du rapport sur les visiteurs et les sessions, le détecteur UTM doit être installé car elles affichent toutes des données sur les visiteurs individuels ou "uniques" et non sur le total des sessions visiteurs.
