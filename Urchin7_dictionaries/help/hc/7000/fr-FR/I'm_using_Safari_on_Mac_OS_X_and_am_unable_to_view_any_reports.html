J'utilise le navigateur Safari sous Mac OS X et je ne parviens pas à consulter mes rapports.

  • Les anciennes versions de Safari (v. 1.3.1 pour Mac OS X Panther et v. 2.0.1 pour Mac OS X Tiger) ne prennent pas en charge Urchin. Seuls les navigateurs du modèle Mozilla (Firefox, Mozilla et Netscape Navigator) sont pris en charge sur les ordinateurs Macintosh pour le moment. Nous vous conseillons donc d'utiliser l'un des navigateurs pris en charge pour consulter vos rapports Urchin.
