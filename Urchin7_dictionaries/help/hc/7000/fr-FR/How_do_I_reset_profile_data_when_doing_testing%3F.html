Comment puis-je réinitialiser les données d'un profil après un test ?

Vous souhaiterez probablement exécuter un même profil Urchin à plusieurs reprises, en vue de réaliser des tests de performances, de filtres et d'autres éléments, tout en supprimant les données générées par Urchin après chaque test. Il suffit pour cela de supprimer le dossier/répertoire dans lequel les données du profil sont stockées à la suite de votre test. Ces répertoires sont créés dans la distribution Urchin selon le nom de chaque profil dans le répertoire data/reports. Supprimez entièrement le répertoire portant le nom du profil testé pour réinitialiser toutes ses données, notamment les informations relatives au suivi du journal.
