Pourquoi Urchin installe-t-il un serveur Web Apache ?

Urchin installe un serveur Web Apache léger pour l'administration en ligne et la diffusion dynamique des rapports. Cette version indépendante d'Apache incluse dans Urchin évite de modifier la configuration du serveur Web existant pour prendre en charge Urchin. Le fait d'exécuter le serveur Web Apache associé à Urchin en parallèle du serveur Web existant présente également d'autres avantages :

  • Urchin n'est pas touché par les modifications apportées au serveur Web standard.
  • La gestion des services Urchin n'a aucune incidence sur le serveur Web standard.
  • L'interface d'administration Urchin permet de modifier certains paramètres de fonctionnement du serveur Web Apache indépendant.
  • Les mises à niveau d'Urchin restent parfaitement délimitées et toute modification apportée, à la prise en charge SSL par exemple, est gérée exclusivement via le processus de mise à niveau Urchin, sans requérir d'ajustement sur le serveur Web standard. L'installation du serveur Apache inclus dans Urchin ne devrait pas perturber les services Web utilisés sur votre machine, ni absorber une part significative des ressources système. Vous pouvez en outre configurer Urchin de manière à autoriser l'accès aux rapports via Internet.
