Help Center

Mes rapports ne contiennent aucune donnée.

Si vos rapports ne présentent pas de données :

  • Vérifiez que l'UTM est correctement référencé sur vos pages. Si le fichier urchin.js n'est pas installé dans le répertoire racine de votre site, vous devez modifier sa référence dans chacune de vos pages Web de sorte qu'elle pointe vers l'emplacement approprié.
  • Si l'UTM a été inséré correctement, vous n'avez peut-être pas attendu assez longtemps avant de consulter vos rapports. Après la saisie de l'UTM, les données mettent généralement plusieurs heures avant d'apparaître sur votre compte.
  • Les rapports Urchin ne s'affichent pas dans les navigateurs Safari sous Mac OS X.
