Recommandations relatives à la configuration

Présentation

Une fois Urchin installé, certains paramètres opérationnels initiaux doivent être configurés. Vous utiliserez pour cela un assistant de configuration qui s'exécute lorsque vous vous connectez à votre interface d'administration Urchin pour la première fois et lors des phases initiales d'établissement des profils. La procédure initiale de configuration inclut les opérations suivantes :

  • Obtention d'une licence Urchin
  • Configuration des paramètres Admin pour l'accès à distance aux rapports et aux fonctions d'administration, ainsi qu'établissement du fonctionnement en mode Datacenter
  • Définition du mot de passe du compte administrateur Urchin
  • Planification de tâches pour le traitement des données de chacun de vos profils
  • Gestion des fichiers journaux

Procédure

Connectez-vous à l'interface d'administration Urchin. Sous Windows, cliquez sur Démarrer->Programmes->Urchin->Administration Urchin. Sous un système d'exploitation de type UNIX, vous pouvez utiliser l'URL http://hostname:9999, dans laquelle hostname correspond au nom d'hôte enregistré de votre système Urchin. Pour une nouvelle installation, connectez-vous à l'aide des paramètres suivants :

Nom d'utilisateur : admin Mot de passe : urchin

L'écran de bienvenue de l'assistant de configuration Urchin s'affiche. Cliquez sur Continuer pour accéder à chacun des écrans suivants de l'assistant. Notez que vous pouvez toujours modifier ultérieurement les choix que vous effectuez au cours de cette configuration initiale.

Licence Urchin

Suivez les instructions de la section Obtention d'une licence Urchin.

Paramètres Admin

  • Paramètres d'accès à distance : activez tous les paramètres si vous souhaitez autoriser les connexions à distance au navigateur. Si vous désactivez l'un d'entre eux, le seul accès autorisé se fait via la console du système sur lequel Urchin est installé.
  • Mode Datacenter : ce paramètre détermine si Urchin est configuré pour autoriser la création de comptes, ce qui vous permet d'organiser de façon logique les profils, les groupes et les utilisateurs en catégories dont l'accès est restreint. Si vous n'êtes pas sûr de la meilleure marche à suivre, nous vous conseillons d'activer ce paramètre, car il n'a pas d'impact négatif sur le temps système et vous vous donnez ainsi la possibilité de l'utiliser ultérieurement.

Utilisateur administrateur

  • Réinitialisez le mot de passe du compte administrateur et conservez-le en lieu sûr.

Planification des tâches

Lorsque vous créez des profils, vous avez la possibilité de planifier l'heure d'exécution de la tâche traitant les données de chaque profil. Vérifiez les paramètres de chaque profil pour vous assurer que l'heure assignée à la tâche est logique par rapport aux facteurs suivants : l'heure à laquelle les données des fichiers journaux sont disponibles, le temps de traitement de ces données et l'heure à laquelle vous souhaitez que les rapports mis à jour soient disponibles pour vos utilisateurs.

Gestion des fichiers journaux

Urchin comporte un module de suivi des fichiers journaux qui vous indique la progression du traitement de chaque journal en temps réel. Par conséquent, la rotation des fichiers journaux ne doit pas obligatoirement être synchronisée avec le fonctionnement d'Urchin. Toutefois, Urchin propose une automatisation de la rotation ou de la suppression des fichiers journaux sous la section Paramètres avancés de chaque source de journal.

Pour plus d'information, consultez la section Gestion des fichiers journaux.
