Importation des données de coût à partir de Google

Urchin 6.5+

Utilisez la planification de sources de CPC et la mise à jour de sources de CPC globale pour télécharger automatiquement vos données de CPC. Pour plus d'informations, consultez les sections Sources de CPC et Utilitaires d'importation des données de CPC.

Versions antérieures à Urchin 6.5

Téléchargez chaque jour ou chaque semaine les données relatives à vos dépenses, avant l'exécution planifiée de votre profil (ou avant l'exécution manuelle du profil).

  1. Sur la page adwords.google.fr, connectez-vous à votre compte Google AdWords.

  2. Ouvrez l'onglet "Rapports", puis cliquez sur "Créer un rapport".

  3. Sélectionnez l'option "Performances des URL" dans la section Type de rapport. Dans la section Paramètres, sélectionnez les éléments suivants :

    • Afficher (unité de temps) : sélectionnez "Quotidien" dans le menu déroulant.
    • Période
      • S'il s'agit de votre premier téléchargement de données relatives au suivi des campagnes, indiquez une période allant de la date à laquelle vous avez débuté le marquage des mots clés jusqu'à la date d'hier.
      • Si vous avez déjà téléchargé des données d'historique, saisissez une période débutant le jour suivant la date de fin de votre dernier téléchargement et se terminant à la date d'hier.
      • Pour des téléchargements quotidiens (recommandés), saisissez la date d'hier dans les deux champs.
    • Campagnes et groupes d'annonces : activez l'option "Toutes les campagnes et tous les groupes d'annonces associés".

    • Cliquez sur le lien "Ajouter ou supprimer des colonnes" dans la section Paramètres avancés (facultatif) :

      • Dans "Niveau de détail", cochez les cases suivantes :
        • Mot clé
        • Campagne
        • Groupe d'annonces
        • URL de destination
      • Dans "Statistiques de performance", cochez les cases suivantes :
        • Impressions
        • Clics
        • CTR
        • CPC moy.
        • Coût
        • Position moy.
      • Dans les colonnes relatives aux conversions, cochez les cases suivantes :

        • (facultatif) Conversions, Taux de conversion, Coût/Conversion si le suivi des conversions est activé dans votre compte Google AdWords. Dans le cas contraire, passez à l'étape suivante.

    • Dans la section Modèles, planification et e-mail :

      • Nommer votre rapport : entrez le nom du rapport.
      • Planification : si vous cochez cette case, vous n'aurez plus à recommencer la définition de ce rapport. Le format du rapport est enregistré et exécuté automatiquement chaque jour, semaine ou mois (selon l'option choisie).
      • E-mail : si vous cochez cette case, Google vous informe par e-mail de la création du rapport.
  4. Cliquez sur le bouton "Créer un rapport".
  5. Une fois le rapport généré manuellement par vos soins, ou automatiquement par Google à la suite d'une planification, il apparaît dans le centre de téléchargement (cliquez sur le lien "Centre de téléchargement", en haut de la page). Sélectionnez le rapport quotidien ou hebdomadaire voulu et téléchargez-le sous forme de fichier .tsv.
