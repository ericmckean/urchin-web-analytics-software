Comment puis-je interpréter les rapports générés pour un site Web comportant des cadres ?

Une page Web contenant des cadres génère plusieurs consultations de page : une pour la page de cadrage (dont le code HTML contient soit une balise FRAMESET soit une balise IFRAME) et une pour chaque page affichée dans un cadre.

Dans vos rapports de navigation, chacune de ces consultations de page apparaîtra dans l'ordre suivant :

[page de cadre] -> [page de cadre 1] -> [page de cadre 2]

Par conséquent, votre rapport Optimisation du contenu > Performances du contenu > Nombre de pages visitées peut être légèrement augmenté artificiellement.

Même si une page de votre site n'apparaît que comme cadre pour une autre page, nous vous conseillons d'y insérer le script UTM. En effet, si un visiteur arrive sur cette page via un moteur de recherche ou par l'intermédiaire d'un lien direct à partir d'un autre site et que la page ne contienne pas le script UTM, les informations sur les sites référents, les mots clés et/ou les campagnes de la source seront perdues.
