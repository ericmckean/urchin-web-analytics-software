Known Constraints (REST) --Beta Release--

  • API Table of Contents

  • getData (REST)

The following limitations apply to the getData method:

  • The method returns profile data for a single specified profile.
  • If the set of requested dimensions/metrics corresponds to multiple datamaps, the method returns an error message with a list of ambiguous datamaps.
  • A filter can only be applied to field which is present in request query.
  • A single regex and numeric filter is supported per call.
