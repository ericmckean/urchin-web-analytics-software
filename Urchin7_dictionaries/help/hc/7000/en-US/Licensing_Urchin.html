Licensing Urchin

Urchin may be licensed in one of three ways:

  • A regular license (pre-Urchin 6.6) never expires.
  • A USP license (Urchin 6.6+) has an expiration date that is determined during purchasing.
  • A demo license is identical in functionality to the Regular and USP licenses, but allows to have no more than five profiles and five log sources.

Obtaining and Activating a Demo License

To obtain a demo license, log-in to your Urchin Administration Interface with a web browser (usually http://your.server.com:9999). Click "Obtain Demo License" and accept the License Agreement to complete the process.

Once you've obtained your demo license, you'll be presented with a screen showing your new license details. These include: License type, Serial Code, Number of profiles, Parallel log sources, Logs per profile, Operating system. It's always a good idea to save license information, especially the Serial Code for future reference. Once you have successfully installed Urchin, license details (License status, License Expiration, Serial Code and Version) can be viewed by browsing to Home->Configuration->Settings->License.

After reviewing License details, activate the demo license. Click the 'Install license' link that is displayed on the 'License Information' screen. This will launch a wizard. It's recommended that you replace the default administrative password with a new password.

NOTE: When selecting options for Remote Access Settings and Admin Control Panel Settings, make sure that you select the option for Admin Control Panel Settings that allows you to add levels of administrative rights to one or several profiles in a group. You will not be able to reverse this setting once installation is complete.

Upon successful completion of new Demo License activation, you will see the Urchin Admin home screen. From this screen you can create profiles, processing logs, view reports, set preferences, etc.

Once the Demo License has expired, the Urchin License screen appears with the following license status in details: Valid License - Demo (Expired). To install a pre-purchased license, click the 'Activate License' link.

Once the Demo License becomes invalid, the Urchin License screen appears with the following license status in details: Invalid License. To reactivate the existing Demo, click 'Reactivate License' link or click'Activate License' to install pre-purchased license.

Purchasing a Regular License

Contact an Urchin Software Authorized Consultant to purchase a license.

Activating a Purchased Regular License

Upon receiving your new serial code for a Permanent Urchin License, you'll need to activate it as follows:

  • Navigate to Configuration->Settings->License

  • Click on Activate License link
  • On the 'Activate Serial Code' screen, enter your valid email address and your serial code.

Upon completion of this process, Urchin will be fully functional and your license will never expire.

Important: Be sure to enter accurate contact information when obtaining/activating your demo license as this information will be necessary for billing and support. Once you have successfully installed Urchin, license details (License status, License Expiration, Serial Code and Version) can be viewed by browsing to Home->Configuration->Settings->License.

If a Regular License becomes invalid, the Urchin License screen appears with the following license status in details: Invalid License. To reactivate an existing Regular license, click 'Reactivate License' link or click'Activate License' to install pre-purchased USP license.

Installing a License Without Internet Access

It is possible to license Urchin without internet access, for example, from behind a firewall. To accomplish this, you will need to run the "inspector" utility, bundled with Urchin and found in the "util" directory. (For instructions on how to run the "inspector" utility, read inspector: Urchin Installation Integrity Checker.) Once you have the "inspector" output, contact your Urchin Software Authorized Consultant for manual activation instructions.

Purchasing a USP License

Contact an Urchin Software Authorized Consultant to purchase a license.

Activating a Purchased USP License

Upon receiving your new serial code for a Permanent Urchin License, you'll need to activate it as follows:

  • Navigate to Configuration->Settings->License

  • Click Activate License
  • On the 'Activate Serial Code' screen, enter your serial code.

Once you have completed these steps, Urchin will be fully functional until the license expires.

If your USP License expires, contact an Urchin Software Authorized Consultant to purchase a license.

Upgrading to Urchin 6.6

Upgrading an instance of Urchin that is using a Demo license is not recommended and may result in login/licensing issues. No licensing changes should be performed after upgrade if the previous Urchin version was used with a Regular license on Windows. The license should be reactivated after an upgrade if previous the Urchin version was installed with a Regular license on Linux/FreeBSD (click 'Reactivate License' link on License Urchin screen).

If a Regular License becomes invalid (on any platform), the Urchin License screen appears with the following license status: Invalid License. To reactivate an existing Regular license, click 'Reactivate License' link or, to install a pre-purchased USP license, click 'Activate License'.
