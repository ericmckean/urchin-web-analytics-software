Campaign Tracking: Why do I have 0 clicks but still show conversions?

If any of your campaigns show 0 clicks but yet show a non-zero number of conversions, your db_limit is probably not set high enough. To change your db_limit:

  1. Click Configuration (top right of interface).
  2. Edit the profile (click the Edit wrench icon).
  3. Click the Storage/DB tab.
  4. In the DB Table Limit dropdown menu, increase the number of records. (The default is 100,000.)
  5. Click Update.
  6. Click Delete All Data.
  7. Click the Run/Schedule tab.
  8. Click Run Now.

If the problem persists, repeat the above steps, but specify a larger number of records.
