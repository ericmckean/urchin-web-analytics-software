I'm using Safari on Mac OS X and am unable to view any reports.

  • Older versions of Safari (v. 1.3.1 for Mac OS X Panther and v. 2.0.1 for Mac OS X Tiger) do not support Urchin. Only Mozilla-based browsers (Firefox, Mozilla, and Netscape Navigator) are supported for Macs at this time. We recommend using one of these supported browsers to view Urchin reports.
