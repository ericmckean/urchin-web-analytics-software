Why does Urchin install an Apache webserver?

Urchin installs a lightweight Apache webserver in order to provide the functionality for web-based administration and dynamic report delivery. By bundling a standalone version of Apache with Urchin, there is no need to modify the webserver's existing webserver configuration in order to support Urchin. Also, there are other benefits to having Urchin's Apache webserver run separately from the existing webserver:

  • Urchin is unaffected by changes to the standard webserver
  • Urchin services can be managed without affecting the standard webserver
  • Urchin's administrative GUI can be used to change certain operating parameters of the standalone Apache webserver
  • Urchin upgrades can be completely self contained and any necessary changes such as SSL support can be handled exclusively by the Urchin upgrade process, without requirement for changes to the standard webserver. Urchin's Apache installation should not affect any other existing web services on your machine nor result in significant use of computer resources. Urchin is configurable to allow remote access to reports via the web.
