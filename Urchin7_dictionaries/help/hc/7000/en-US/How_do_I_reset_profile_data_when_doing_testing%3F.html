How do I reset profile data when doing testing?

It may be desirable to run an Urchin profile multiple times to for the purposes of testing performance, filters and so forth, where the actual data generated by Urchin is discarded after each test. The simplest way to do this is to remove the directory/folder where the profile data is stored after each run. These directories are stored by profile name in the Urchin distribution in the data/reports area. Removing the entire directory named after the profile being tested will reset all the data, including the log tracking information.
