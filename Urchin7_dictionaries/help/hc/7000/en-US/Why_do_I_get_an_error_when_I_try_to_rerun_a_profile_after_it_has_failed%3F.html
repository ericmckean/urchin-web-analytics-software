Why do I get an error when I try to rerun a profile after it has failed?

Problem: Your profile processing fails with the following error: 

------------------------------------------------------
Urchin 6.4.00 (windows) starting: 20080229 15:50:13
------------------------------------------------------
Processing profile: {profileName} (on machineName 5764)

WARNING: (7050-323-271) Starting automatic rollback


WARNING: (7054-323-308) Unable to restore databases since backup does not exist
DETAIL: C:\Program Files\Urchin6\data\reports\{accountName}\{profileName}\200508-backupv6-*.zip

Reason for Error: This error occurs when the last processing run failed and there is no backup to rollback to.

Solution: Remove all the reporting data for the affected month(s) and reprocess that entire month.
