Data API: Admin service (REST) --Beta Release--

API Table of Contents

The Admin Service provides operations for retrieving account and profile information.

Service URL: 

http://server_name[:port]/services/vN/adminservice

This service provides the following methods:

  • getAccountList getAccountList (REST) v1

    • The getAccountList method lets you extract account information for an authenticated user.

  • getProfileList getProfileList (REST) v1

    • The getProfileList method lets you extract profile information for an authenticated user.

The following structures are used:

  • Account

    • accountId - account identifier

    • accountName - account name

  • User

    • userId - user identifier

    • userName - user name

  • Profile

    • accountId - account identifier

    • accountName - account name

    • profileId - profile identifier

    • profileName - profile name
