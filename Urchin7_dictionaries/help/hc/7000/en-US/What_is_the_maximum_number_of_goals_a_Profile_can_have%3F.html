What is the maximum number of goals a Profile can have?

Each Profile can have up to 4 goals. If you would like to track more than 4 Goals for a web property, you will need to set up duplicate Profiles for that site. To do so, please click the 'Create additional profile for existing website' link under the Account Management area of your account.

If your site receives a large volume of traffic, you should be aware that each Profile will process each Pageview separately, and that these duplicate pageviews will count towards your total pageviews.
