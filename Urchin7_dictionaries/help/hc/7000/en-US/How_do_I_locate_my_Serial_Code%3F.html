How do I locate my Serial Code?

There are two ways to display your current Urchin serial code.

  1. Administrative Console
    • Launch your Urchin Adminstrative Console from any web browser.

      Click on 'Configuration--> Settings --> License.

  2. Run the inspector utility
    • Open a command window and cd to the basedir/urchin/util/ directory. Run the inspector utility. The serial number will be printed to the screen along with other Urchin information. Note that on Unix/Linux, all commands must be preceded by './'
