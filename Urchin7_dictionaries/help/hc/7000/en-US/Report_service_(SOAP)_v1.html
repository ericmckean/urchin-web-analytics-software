Data API: Report service (SOAP) --Beta Release--

API Table of Contents

The Report Service lets you extract data and generate reports from Urchin.

WSDL location: 

http://server_name[:port]/services/vN/reportservice?wsdl

This service provides the following methods:

  • getTableList getTableList (SOAP) v1

    • The getTableList method lets you get the list of available tables for a specified profile.

  • getData getData (SOAP) v1

    • The getData method lets you extract data from Urchin tables.

The following structures are used:

  • User

    • userId - user identifier

    • userName - user name

  • Profile

    • accountId - account identifier

    • accountName - account name

    • profileId - profile identifier

    • profileName - profile name

  • Table

    • tableId - datamap identifier

    • dimensions[] - list of available cfields from datamap

      • cfield mappings are taken from lib/reporting/logformats/fieldlist.txt with prefix "u:" - ("u" stands for urchin)
      • cfield mappings are taken from lib/custom/logformats/fieldlist.txt with prefix "uc:" ("c" stands for urchin custom)

    • metrics[] - list of available ifields from datamap

  • Record

    • recordId - record identifier

    • dimensions[] - list of extracted nameless dimension values

    • metrics[] - list of extracted named metric values
