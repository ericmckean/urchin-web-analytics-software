Es werden keine Daten in meinen Berichten angezeigt.

Wenn in Ihren Berichten keine Daten angezeigt werden:

  • Überprüfen Sie, ob auf allen Seiten Ihrer Website ordnungsgemäß auf UTM verwiesen wird. Ist die Datei "urchin.js" nicht im Stammverzeichnis Ihrer Website installiert, müssen Sie auf jeder Webseite den Verweis auf die Datei auf den tatsächlichen Speicherort aktualisieren.
  • Wenn UTM korrekt installiert ist, überprüfen Sie Ihre Berichte möglicherweise, bevor Daten angezeigt werden. Es dauert normalerweise mehrere Stunden, bis die Berichtdaten in Ihrem Konto angezeigt werden, nachdem Sie UTM installiert haben.
  • Urchin-Berichte können nicht in Safari-Browsern unter Mac OS X angezeigt werden.
