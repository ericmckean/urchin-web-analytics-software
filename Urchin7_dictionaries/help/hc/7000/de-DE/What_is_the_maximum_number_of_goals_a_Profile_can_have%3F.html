Wie viele Ziele kann ein Profil maximal erfassen?

Für jedes Profil sind bis zu vier Ziele möglich. Wenn Sie mehr als vier Ziele für eine Webeigenschaft erfassen möchten, müssen Sie für diese Website Duplikate der Profile einrichten. Klicken Sie dazu bitte im Bereich "Kontoverwaltung" Ihres Kontos auf den Link "Zusätzliches Profil für eine bestehende Website erstellen".

Beachten Sie bei einem hohen Zugriffsvolumen Ihrer Website, dass jedes Profil jeden Seitenzugriff gesondert verarbeitet und sich diese doppelt verarbeiteten Seitenzugriffe auf die Zählung der Seitenzugriffe insgesamt niederschlagen.
