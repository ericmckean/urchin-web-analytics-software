Arbeiten mit dem Aufgabenplaner

Übersicht

Der Aufgabenplaner ist der Dreh- und Angelpunkt von Urchin - er verwaltet die eigentliche Planung und Ausführung der Protokollverarbeitungsereignisse von Urchin für alle Profile. Mit dem Planer können Sie Aufgaben sofort ausführen oder zur Liste der Urchin-Ereignisse hinzufügen, um sie in jedem gewünschten Intervall wiederholt ausführen zu lassen.

Verwenden des Planers

  1. Melden Sie sich in Urchin 6 an und klicken Sie auf "Konfiguration" (auf der Benutzeroberfläche oben rechts).
  2. Klicken Sie für das Profil, das Sie planen möchten, auf "Bearbeiten".
  3. Klicken Sie auf die Registerkarte "Ausführen/Planen".
  4. Wählen Sie unter "Aufgabeneinstellungen" das gewünschte Intervall aus. Die Einstellung "Täglich" wird empfohlen.
  5. Legen Sie die Uhrzeit für die Aufgabe fest.
  6. Klicken Sie auf "Aktualisieren", um die Änderungen zu speichern.
  7. Klicken Sie auf "Jetzt ausführen", wenn die Aufgabe sofort ausgeführt werden soll. Nachfolgend geplante Aufgaben werden nach dem von Ihnen festgelegten Zeitplan ausgeführt.

Empfehlungen

  • Die meisten Aufgaben sollten täglich ausgeführt werden, da dies dem Protokollwechselplan vieler Webserver entspricht. Die Protokoll-Tracking-Funktion von Urchin lässt jedoch das mehrmalige Lesen eines Protokolls ohne Duplizierung der Daten zu, sodass dies nicht erforderlich ist.

Anmerkungen zum Betrieb des Planers

  • Alle Aufgaben werden von Urchin nacheinander ausgeführt. Mehrere Aufgaben mit derselben Ausführungszeit werden folglich jeweils der Reihe nach ausgeführt.
  • Klicken Sie im linken Navigationsmenü auf "Aufgabenprotokoll", um die Ergebnisse aller vom Planer ausgeführten Aufgaben anzuzeigen.

Bereiche des Planers:

  • Profilaufgabenplaner: Dieses Fenster dient zur Anzeige der Aufgaben, deren Ausführung für jedes Profil in der Konfiguration geplant ist. Für jedes Profil werden verschiedene Informationen angezeigt, einschließlich Profilname, Ausführungshäufigkeit und Ausführungsstatus ("Nicht geplant", "Ausstehend", "In Warteschlange", "Ausführen"). Für ausgeführte Aufgaben werden der Abschlussstatus in Prozent und die Startzeit der Aufgabe angezeigt.
  • Profilaufgabenprotokoll: In diesem Fenster wird der Verlauf jeder Aufgabe aufgeführt, die über den Profilaufgabenplaner oder die Schaltfläche "Jetzt ausführen" ausgeführt wurde. Außerdem wird eine Zusammenfassung der Aufgaben angezeigt, die wertvolle Informationen zu jeder Aufgabe bietet. Durch Klicken auf das Feld "Status" können die Ausführungsdetails für die entsprechende bereits abgeschlossene Aufgabe angezeigt werden.

  • Geo-DB-Downloadprotokoll (Urchin 6.500+): Dieser Bereich wurde vorher als "Aktualisierung der geografischen Daten" bezeichnet und war Teil des Profilaufgabenplaners. In diesem Bereich des Planers werden alle Geo-DB-Downloads angezeigt, die über den Profilaufgabenplaner ausgeführt wurden. Außerdem wird eine Zusammenfassung der Downloads angezeigt, die wertvolle Informationen zu jedem Download bietet. Durch Klicken auf das Feld "Status" können die Ausführungsdetails für die entsprechende bereits abgeschlossene Aufgabe angezeigt werden. Der Geo-DB-Download kann über den Bereich "Einstellungen -> Globale Einstellungen -> Aktualisierung der Geodaten" geplant werden. Der CPC-Planer kann über den Bereich "Einstellungen -> Globale Einstellungen -> CPC-Quellenaktualisierung" geändert werden.

  • CPC-Downloadprotokoll (Urchin 6.500+): In diesem Fenster wird das Protokoll zu jeder CPC-Downloadaufgabe angezeigt, die mittels CPC-Aufgabenplaner ausgeführt wurde. Außerdem wird eine Zusammenfassung der CPC-Downloads angezeigt, die wertvolle Informationen zu jedem Download bietet. Durch Klicken auf das Feld "Status" können die Ausführungsdetails für die entsprechende bereits abgeschlossene Aufgabe angezeigt werden. Der CPC-Quellendownload kann wie folgt geplant werden: über die Ansicht "CPC-Quellenmanager" (CPC-Quellenmanager -> Herunterladen oder Bearbeiten -> Registerkarte "Zeitplan" -> Wählen Sie "Zeitplan herunterladen" oder erzwingen Sie den Download durch Klicken auf die Schaltfläche "Jetzt herunterladen").
