Wie kann ich Nutzern Zugriff auf bestimmte Profile gewähren?

So bearbeiten Sie den Zugriff der Nutzer auf Profile:

  1. Melden Sie sich in Urchin 6 an.
  2. Klicken Sie auf "Konfiguration" und anschließend auf "Nutzer".
  3. Klicken Sie neben dem gewünschten Nutzer auf "Bearbeiten".
  4. Klicken Sie auf die Registerkarte "Berichtzugriff". Die verfügbaren Profile werden im Feld links angezeigt.
  5. Wählen Sie ein oder mehrere Profile aus. Halten Sie die Steuerungs- oder Befehlstaste gedrückt, um mehrere Profile auszuwählen.
  6. Klicken Sie auf den Pfeil nach rechts, um das Profil in das Feld "Zugriff gewährt" zu verschieben.
  7. Klicken Sie auf "Aktualisieren", um die Änderungen zu speichern.
