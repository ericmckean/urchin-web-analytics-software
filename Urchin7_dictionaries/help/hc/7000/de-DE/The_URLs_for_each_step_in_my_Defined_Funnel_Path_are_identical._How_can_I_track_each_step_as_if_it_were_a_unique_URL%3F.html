Die URLs für jeden Schritt in meinem definierten Trichterpfad sind identisch. Wie kann ich jeden Schritt so erfassen, als ob es eine eindeutige URL wäre?

Es kann sinnvoll sein, den Weg der Nutzer durch einen Trichter zu erfassen, der für jeden Schritt dieselbe URL aufweist. Der Anmeldetrichter könnte beispielsweise folgendermaßen aussehen:

  • Schritt 1 (Anmeldung)
    • www.meinewebsite.de/sign_up.cgi
  • Schritt 2 (Vereinbarung akzeptieren)
    • www.meinewebsite.de/sign_up.cgi
  • Schritt 3 (Fertig stellen)
    • www.meinewebsite.de/sign_up.cgi

Wenn Sie dies umgehen möchten, können Sie bei jedem Schritt (wahrscheinlich bei einem Onload-Ereignis) die Javascript-Funktion urchinTracker folgendermaßen aufrufen:

  • urchinTracker("/funnel_G1/step1.html")
  • urchinTracker("/funnel_G1/step2.html")
  • urchinTracker("/funnel_G1/step3.html")

Legen Sie dann die einzelnen Schritte des Trichters wie folgt fest:

  • http://www.meinewebsite.de/funnel_G1/step1.html

  • http://www.meinewebsite.de/funnel_G1/step2.html

  • http://www.meinewebsite.de/funnel_G1/step3.html

Das Pfad/Dateiname-Argument für urchinTracker() muss nicht für einen bestehenden Pfad bzw. Dateinamen stehen. Das Argument für urchinTracker enthält einfach einen erfundenen Seitennamen, an den Urchin Seitenzugriffe anfügen kann.
