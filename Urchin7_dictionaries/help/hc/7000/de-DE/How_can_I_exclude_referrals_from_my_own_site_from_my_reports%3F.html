Wie kann ich Verweise meiner eigenen Website aus meinen Berichten ausschließen?

Sie müssen eine kleine Änderung an der Datei "urchin.js" vornehmen, um Verweise Ihrer eigenen Website aus den Berichten "Verweis" auszuschließen. Öffnen Sie die Datei in einem Texteditor und fügen Sie den folgenden Abschnitt ein: 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";

Im obigen Abschnitt wird jeder Treffer aus den Verweisen ausgeschlossen, der ".urchin.com" enthält. Sie können weitere Domains hinzufügen, indem Sie die Datei bearbeiten und einen weiteren _uRno[x]-Wert einschließen. Erhöhen Sie dabei den Zahlenwert von "x". Beispiel: 

    //-- Referral domains to Ignore
    var _uRno=new Array();
    _uRno[0]=".urchin.com";
    _uRno[1]=".urchin.net";
    _uRno[2]=".urchinsoftware.com"

Hinweis: Nach der Bearbeitung der Datei "urchin.js" müssen Sie die aktualisierte Datei in Ihre Website hochladen, damit die Änderungen wirksam werden.
