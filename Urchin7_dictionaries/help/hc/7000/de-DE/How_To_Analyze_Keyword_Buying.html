Wie erfolgt die Analyse zum Keyword-Kauf?

Inwiefern ist eine Analyse zum Keyword-Kauf hilfreich?

In welche Keywords sollte ich investieren? Wie viel sollte ich für ein Keyword bieten? Wie viel erziele ich mit Keywords? Zu welchen Tageszeiten sollte ich meine Präsenz in Suchmaschinen maximieren?

Sie können diese und andere Fragen beantworten, indem Sie Ihren Keyword-Kauf mithilfe von Urchin analysieren. In diesem Artikel finden Sie eine Anleitung zu den einzelnen Schritten, vom Erfassen der Daten bis zur Analyse der Berichte.

Mit welchen Schritten wird der Keyword-Kauf analysiert?

  • Definieren eines Conversion-Ziels
  • Kauf Ihrer Keywords
    • Für jedes erworbene Keyword aus einer PPC (Pay-per-Click)-Suchmaschine (z. B. Google oder Overture) müssen Sie einen Verweislink auf Ihre Website einrichten und UTM-Variablen einbetten. Dieser Artikel beschreibt die optimale Methode zur Verwendung von UTM-Variablen für die Analyse zum Keyword-Kauf.
  • Tracking von Kampagnendaten
    • Sie müssen UTM installieren, das Protokollieren von Cookies aktivieren und UTM-Variablen in Ihre Verweislinks einbetten.

  • Importieren von Keyword-Ausgaben (nur für ROI-Berichte erforderlich). Weitere Informationen erhalten Sie unter Importieren der Kostendaten von Google.

  • Importieren von E-Commerce-Daten (nur für Analyseberichte zum E-Commerce-ROI erforderlich). Weitere Informationen erhalten Sie im Website-Bereich E-Commerce.

  • Optimieren Ihrer Keyword-Käufe
    • Informationen dazu, wie Sie mithilfe der Keyword-Berichte Ihre Keyword-Käufe optimieren können, erhalten Sie weiter unten in diesem Artikel.

Welche UTM-Variablen sollte ich für Keyword-Analysen verwenden?

  • Für bezahlte Suchmaschinenlinks, z. B. Google AdWords, verwenden Sie utm_source, utm_medium und utm_campaign. Bei der Verwendung weitgehend passender Keywords und wenn Sie Messwerte für das weitgehend passende Keyword anzeigen möchten (statt das spezifische Keyword mithilfe von UTM zu ermitteln), können Sie utm_term nutzen. Der folgende Link stellt ein Beispiel dafür dar, wie Sie UTM-Variablen in einem Google AdWords-Link verwenden können. Dabei wird angegeben, dass der Verweis über einen bezahlten Google-Suchbegriff erfolgte und dass es sich um das Medium "Cost-per-Click" handelt. Es wird ebenfalls angegeben, dass der Besucher auf einen Adidas-Werbelink geklickt hat. UTM ermittelt automatisch die Keywords, mit denen Ihre Website gefunden wurde. 

      http://www.meinunternehmen.de/?utm_source=google&utm_medium=cpc&utm_campaign=adidas

  • Verwenden Sie das Tool zur URL-Erstellung von Google Analytics zum Erstellen Ihrer Links. Durch Verwendung dieses Tools wird sichergestellt, dass Ihre Links die korrekte Syntax enthalten.

Wie verhält es sich mit Links, bei denen es sich nicht um Anzeigen handelt (d. h. unbezahlte, freie oder Indexeinträge in Suchmaschinen)?

  • Es werden nur Variablen in Anzeigen eingebettet - Links, für die Sie in einer Suchmaschine bezahlen oder Links, die Sie auf andere Weise kontrollieren, z. B. Links in einer E-Mail, die Sie an Kunden versenden. Sie müssen sich nicht um Links kümmern, bei denen es sich nicht um Anzeigen handelt, da durch das Kampagnen-Tracking in Urchin automatisch ermittelt wird, von welcher Suchmaschine der Verweislink erfolgte und welches Keyword der Besucher verwendet hat.

Achten Sie auf die Konsistenz der UTM-Variablen.

  • Es ist wichtig, dass Sie konsistente Namen und Schreibweisen für alle Ihre Kampagnen-Variablenwerte verwenden. Wählen Sie z. B. einen Code oder Namen, der "Cost-Per-Click" (Preis-pro-Klick) angibt, und verwenden Sie diesen durchgehend. Beim Kampagnen-Tracking in Urchin sind utm_medium=cpc und utm_medium=cost_per_click verschiedene Medien.

    Es steht eine Master-Tracking-Code-Funktion zur Verfügung, die die Möglichkeit von Konsistenzfehlern wesentlich reduziert. Lesen Sie bitte Verwenden von Master-Tracking-Codes.

Optimieren Sie Ihre Keyword-Käufe.

  • Welche Keywords sollte ich erwerben? Sie sollten Keywords erwerben, die die größtmögliche Anzahl von Transaktionen bzw. Zielen erreichen bzw. den größten Ertrag erzielen. Lesen Sie die Berichte "Marketingoptimierung: Suchmaschinenmarketing". Wie viel erziele ich mit einem Keyword? Sehen Sie sich im Bericht "Marketingoptimierung: Suchmaschinenmarketing: CPC-Programmanalyse" den Umsatz pro Klick und den Return on Investment (ROI) an. Sehen Sie sich in den Berichten "Marketingoptimierung: Suchmaschinenmarketing: Keyword-Conversions" und "CPC und indexbasierte Conversions" den Messwert "$/Besuche" (Wert pro Besuch) an. Zu welchen Tageszeiten sollte ich die Präsenz eines Keywords maximieren? Lesen Sie den Bericht "Marketingoptimierung: Suchmaschinenmarketing: Aufschlüsselung nach Tagesabschnitten". Zeigen Sie das zu analysierende Keyword an. In diesem Bericht wird die Anzahl der Ziele oder Transaktionen und eine Conversion-Rate nach der Stunde eines bestimmten Tages angezeigt. Die Zeitzone wird von Ihrem Administrator über das Feld "Zeitversatz" auf der Registerkarte "Berichte" unter "Konfiguration" --> "Urchin-Profile" --> "Profile" --> "Bearbeiten" festgelegt. Standardmäßig ist diese Einstellung auf "Ortszeit" festgelegt.
