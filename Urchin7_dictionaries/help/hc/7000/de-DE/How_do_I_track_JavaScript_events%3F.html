Wie kann ich JavaScript-Ereignisse erfassen?

Verwenden Sie zum Erfassen eines Ereignisses die JavaScript-Funktion urchinTracker, um einen Namen für das Ereignis anzugeben. Beispiel: Durch das Aufrufen von 

    javascript:urchinTracker('/homepage/link1');

wird jedes Auftreten des Aufrufereignisses protokolliert, als ob es ein Seitenzugriff für die URL /homepage/link1 wäre. Das Argument muss mit einem Schrägstrich beginnen. Die Ereignisnamen können in jeder gewünschten Art von Verzeichnisstruktur angeordnet werden.

Wenn Sie z. B. Ereignisse nach Seite und Ereignistyp anordnen möchten, können Sie eine Hierarchie nach diesem Schema erstellen: 

    * '/homepage/links/link1'
    * '/homepage/radiobuttons/button1'
    * '/contactform/rollovers/image1'

Beispiele für HTML-Code

Im folgenden Beispiel wird dargestellt, wie ein onClick-Ereignis protokolliert wird: 

    <a href="javascript:void(0);" onClick="javascript:urchinTracker('/folder/file');">

Beispiel für das Protokollieren eines Rollover-Ereignisses: 

    <a href="javascript:void(0);" onMouseOver="javascript:urchinTracker ('/folder/file');">
