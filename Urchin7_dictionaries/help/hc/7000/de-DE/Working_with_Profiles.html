Arbeiten mit Profilen

Übersicht

Ein Profil besteht aus einer Reihe von Berichten zu einer Website. Für eine Website muss mindestens ein Profil konfiguriert werden. Häufig ist es wünschenswert, mehr als ein Profil pro Website einzurichten, doch besteht im Allgemeinen ein 1:1-Verhältnis zwischen Profilen und Websites.

Technisch gesehen stellt ein Profil eine Reihe von Regeln dar, mit denen die Verarbeitung von Protokolldateien und das Erstellen von Berichten für eine Website oder Teile einer Website gesteuert wird.

Da Profile das wichtigste Verwaltungstool des Urchin-Systems sind, ist es wichtig, ihre Funktionsweise zu verstehen.

Lizenzinformationen

Die Basislizenz für Urchin 6 umfasst 1000 Profile. Jedes dieser Pofile kann eine Protokollquelle aufweisen oder mehrere Profile können eine Protokollquelle gemeinsam verwenden. Letzteres ist nützlich, wenn ein Profil als "Master" fungiert und auf mindestens ein anderes Profil Filter angewendet werden.

Jeder Website muss mindestens ein Profil zugeordnet sein, damit Berichte erstellt werden können. Normalerweise reicht ein Profil aus. Für spezielle Analysen einer einzelnen Web-Eigenschaft können Sie jedoch beliebig viele Profile erstellen.

Verwenden von Profilen

Melden Sie sich zum Erstellen eines Profils in der Verwaltungsoberfläche von Urchin als Urchin-Administrator an und klicken Sie auf den Link "Konfiguration" oben rechts auf der Seite. Der Profilassistent wird aufgerufen. In einer einfachen Reihe von Schritten können Sie im Handumdrehen ein einfaches Profil erstellen. Jede Seite des Assistenten bietet ausführliche Hilfeinformationen.

Empfehlungen

  • Aktivieren Sie unbedingt Urchin Traffic Monitor (UTM), um eine optimale Qualität der Berichte sicherzustellen, insbesondere, wenn Sie Informationen zu eindeutigen Besuchern erhalten möchten. Die Verwendung von UTM umfasst drei grundlegende Schritte:
    1. Wählen Sie auf der Registerkarte "Einstellungen" eines Profils "Urchin Traffic Monitor (UTM)" als Besucher-Tracking-Methode aus.
    2. Installieren Sie den UTM-Sensor auf jeder Seite, die Sie mit höchster Genauigkeit protokollieren möchten. (Ohne UTM werden Seiten zwar protokolliert, doch werden einige Aktionen wie erneute Seitenladevorgänge oder die Verwendung der Schaltfläche "Zurück" nicht erfasst.)
    3. Ändern Sie das Protokollierverhalten Ihrer Website zur Aktivierung von Cookies.
  • Wenn Sie einen oder mehrere Filter mit Ihrem Profil einrichten möchten, ist es zweckmäßig, dieser Website oder Teilen der Website mehrere Profile zuzuordnen. Wir empfehlen ein Profil als "Master" einzurichten - darin ist alles enthalten. Sollen z. B. Spider oder Roboter herausgefiltert werden, nehmen Sie diese Filter am besten in ein zweites Profil auf. So können Sie die Ergebnisse des gefilterten Datenprofils mit dem Master-Profil vergleichen.
