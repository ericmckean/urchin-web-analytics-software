Erfassen von externen Links und Werbebannern

Urchin bietet eine einfache Möglichkeit, um Klicks auf Werbebanner oder externe Links zu erfassen. Da diese Links nicht auf eine Seite Ihrer Website führen, die das UTM-JavaScript enthält, müssen Sie den Link selbst mit dem JavaScript urchinTracker mit Tags versehen, wenn Sie diese Klicks erfassen möchten. Durch dieses JavaScript wird jedem Klick auf einen Link ein Seitenzugriff zugewiesen. Der Seitenzugriff ist dem von Ihnen angegebenen Dateinamen zugeordnet.

Bei einem animierten GIF-Banner oder einer anderen Art von statischem Werbebanner fügen Sie das JavaScript urchinTracker zum <a>-Tag Ihres Banners hinzu: 

    <a href="http://www.beispiel.de/ads/banner1.gif" onClick="javascript:urchinTracker ('/ads/adv_name/bannername');">
