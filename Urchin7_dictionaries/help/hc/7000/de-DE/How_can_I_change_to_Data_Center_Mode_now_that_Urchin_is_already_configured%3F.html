Wie kann ich in den Datacenter-Modus wechseln, wenn Urchin bereits konfiguriert ist?

  • Dieser Artikel wurde von ROI Revolution erstellt, einem autorisierten Berater für Urchin-Software.

Im Datacenter-Modus können Sie einem oder mehreren Profilen in einer Gruppe Administratorrechte hinzufügen. Falls Sie die Erstkonfiguration bereits abgeschlossen haben und den Datacenter-Modus aktivieren möchten, können Sie dies einfach mit MySQL oder PostgreSQL tun.

Für MySQL: 

mysql> USE urchin;
mysql> UPDATE uglobals SET ubgl_enable_accounts = 1;

Für PostgreSQL: 

postgres=# \connect urchin; 
postgres=# UPDATE uglobals SET ubgl_enable_accounts = 1;
