Tracking von Transaktionen eines sicheren Shops

Dieser Artikel wurde von ivantage Limited erstellt, einem autorisierten Berater für Urchin-Software.

Urchin kann Daten aus einem Warenkorb in Ihrer eigenen Domain erfassen sowie - durch Hinzufügen von Code - auch in anderen Domains. Informationen dazu, wie Sie Ihre Seiten einrichten, um Transaktionsdaten an Urchin berichten zu können, erhalten sie unter Wie kann ich E-Commerce-Transaktionen erfassen?

Befindet sich Ihr Online-Shop auf einer anderen Sub-Domain oder Domain als Ihre Hauptwebsite, befolgen Sie bitte zum Konfigurieren Ihrer Seiten die unten aufgeführten Anleitungen.

Wenn Ihre Website einen Kaufvorgang auf einer anderen Sub-Domain auslöst (z. B. wenn Sie Kunden von www.meinshop.de zu warenkorb.de senden), gehen Sie wie folgt vor:

  • Fügen Sie Ihrem Tracking-Code auf beiden Seiten, d. h. auf der Online-Shop-Seite und Ihrer Warenkorb-Seite, die folgenden Zeilen (in Fettschrift) hinzu: 

   <script src="/urchin.js" type="text/javascript">
   </script>
   <script type="text/javascript">
    _udn="mystore.com";
   </script>

Wenn Ihre Website einen Kaufvorgang auf einer separaten Online-Shop-Website auslöst (z. B. wenn Sie Kunden von www.meinshop.de an www.sichererwarenkorb.de weiterleiten), gehen Sie wie folgt vor:

  • Fügen Sie Ihrem Tracking-Code auf beiden Seiten, d. h. auf der Online-Shop-Seite und Ihrer Warenkorb-Seite, die folgenden Zeilen (in Fettschrift) hinzu: 

     <script src="/urchin.js" type="text/javascript">
     </script>
     <script type="text/javascript">
       _udn="none";
       _ulink=1;
     </script>

  • Soll utmLinker verwendet werden, gehen Sie beim Ändern der Links von der Hauptwebsite zur gesicherten Website folgendermaßen vor. Wenn Ihre aktuellen Links wie folgt aussehen: 

     <a href="https://www.securecart.com/?store=parameters">Jetzt kaufen</a>
  • ändern Sie diese wie folgt: 

   <script type="text/javascript">
   document.write('<a href="javascript:__utmLinker(\'https://www.securecart.com/?store=parameters\');">Jetzt kaufen</a>');
   </script>
   <noscript>
   <a href="https://www.securecart.com/?store=parameters">Jetzt kaufen</a>
   </noscript>

  • Der oben angegebene Code bietet Links für Nutzer mit aktiviertem oder deaktiviertem JavaScript. Beachten Sie, dass einem Apostroph im Link oder im Linktext ein umgekehrter Schrägstrich vorangestellt werden muss. Wichtig: Enthalten Ihre Seiten einen Aufruf an urchinTracker(), utmLinker(), utmSetTrans() oder utmLinkPost(), muss der Analytics-Tracking-Code in Ihrem HTML-Code über diesen Aufrufen platziert werden. In diesen Fällen kann der Tracking-Code an beliebiger Stelle zwischen dem öffnenden Tag <body> und dem JavaScript-Aufruf platziert werden.

Wenn Sie mithilfe von Formularen Informationen an Ihren Warenkorb senden, verwenden Sie die Funktion utmLinkPost(). Mithilfe dieser Funktion werden Cookiedaten an die URL des bereitgestellten Formulars angehängt. 

   <form action="http://newdomain.com/form.cgi" onSubmit="javascript:__utmLinkPost(this)">
  • Dies funktioniert sogar bei Formularen mit method="GET"

In diesen Fällen kann der Tracking-Code an beliebiger Stelle zwischen dem öffnenden Tag <body> und dem JavaScript-Aufruf platziert werden.
