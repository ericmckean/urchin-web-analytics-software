Wie kann ich Urchin so konfigurieren, dass bestimmte Keywords oder URLs als direkte Verweise statt als Keywords behandelt werden?

Sie können bestimmte Keywords oder URLs als direkte Verweise statt als Keywords behandeln. Konfigurieren Sie dazu die Datei "urchin.js" so, dass Urchin die Keywords als direkte Verweise in Ihren Berichten protokolliert.

  1. Öffnen Sie die Datei "urchin.js" in einem Texteditor.
  2. Suchen Sie die folgende Überschrift: 

          //-- Auto/Organic Keywords to Ignore
  1. Fügen Sie ein weiteres indexbasiertes Keyword, das ignoriert werden soll, am Ende der Liste in folgender Form ein: 

          _uOno[#]="keyword";
  • ..wie im folgenden Beispiel dargestellt: 

        //-- Auto/Organic Keywords to Ignore
        var _uOno=new Array();
        _uOno[0]="urchin";
        _uOno[1]="urchin.com";
        _uOno[2]="www.urchin.com";

Warum verursacht mein Browser beim Anzeigen meiner Berichte eine Zeitüberschreitung?

Die Anwendung zum Abfragen der Urchin-Datenbank und Erstellen der einzelnen Berichte verursacht eine Zeitüberschreitung, wenn die Daten nicht innerhalb von 45 Sekunden zurückgegeben werden. Je nach Größe der Datenbank reichen 45 Sekunden möglicherweise nicht aus, um alle angeforderten Berichtdaten abzurufen. Somit wird in der Regel eine Zeitüberschreitung verursacht und ein Fehler wird angezeigt.

Sie können den Zeitüberschreitungswert erhöhen, wenn Sie über die entsprechenden Berechtigungen für den Zugriff auf den Server verfügen, auf dem Urchin ausgeführt wird.

Erhöhen des Zeitüberschreitungswerts für die Anwendung

  • Stellen Sie mithilfe einer Befehls-Shell eine Verbindung mit dem Urchin-Server her.
  • Wechseln Sie in das Verzeichnis ".../[urchin-install-dir]/etc/".
  • Öffnen Sie die Datei "session.conf" in einem Texteditor.
  • Erhöhen Sie den Wert von "APPLICATION_TIMEOUT:" (in Sekunden).
  • Speichern Sie die Datei. Die Änderungen werden dann sofort übernommen.
