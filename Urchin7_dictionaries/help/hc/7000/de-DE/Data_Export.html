Datenexport

Die Schaltflächen zum Datenexport in Urchin ermöglichen ein einfaches Extrahieren von Daten aus jedem Urchin-Bericht . Mit dieser Funktion können Berichtdaten bequem in eine Tabelle, ein Textverarbeitungsprogramm oder eine Datenbank zur weiteren Analyse exportiert werden.

Wählen Sie zum Exportieren von Daten aus einem Bericht den entsprechenden Exporttyp aus. Dieser Typ hängt von der Anwendung ab, mit der Sie die Daten bearbeiten möchten. Verwenden Sie für einen allgemeinen Datenbankimport das Format mit tabulatorgetrennten Werten.

  • Tabulator: Klicken Sie auf die Schaltfläche "T", um Daten im Format mit tabulatorgetrennten Werten zu exportieren.
  • Word: Klicken Sie auf das Word-Symbol, um Daten im Microsoft Word-Format zu exportieren.
  • Excel: Klicken Sie auf das Excel-Symbol, um Daten im Microsoft Excel-Format zu exportieren.

Drucken: Klicken Sie auf das Druckersymbol, um eine Druckversion der Daten anzuzeigen. Klicken Sie zum Drucken des Berichts auf der entsprechenden Seite auf den Link "Seite drucken".
