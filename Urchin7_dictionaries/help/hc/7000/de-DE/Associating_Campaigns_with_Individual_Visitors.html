Verbinden von Kampagnen mit einzelnen Besuchern

Urchin 6 bietet Berichte, die detaillierte Informationen zu Kampagnen auf Ihrer Website enthalten. Sie finden diese Berichte unter "Berichte" -> "Marketingoptimierung" -> "Ergebnisse der Marketingkampagnen". So zeigen Sie eine Liste der Besucher an, die über eine bestimmte Kampagne auf die Website gelangten:

  • Wechseln Sie zu "Alle Berichte" -> "Marketingoptimierung" -> "Ergebnisse der Marketingkampagnen" -> "Kampagnen-Conversions" (oder "Quellen-Conversions", "Medium-Conversions", "Verweis-Conversions").

  • Klicken Sie auf das Zeichen "+" neben einer Kampagne (z. B "indexbasiert").

  • Klicken Sie auf die Schaltfläche "Analyseoptionen" () neben der Schaltfläche der Suchmaschinen (z. B. "google[indexbasiert]") und wählen Sie die Option "Besucherprotokollaufschlüsselung".

  • Klicken Sie auf eine Besucher-ID.

Verweis-/Kampagneninformationen werden zusammen mit einer Liste spezieller Zugriffe angezeigt.
