Zeitraum

Übersicht

Mit der Zeitraumfunktion von Urchin können Sie Berichtdaten für jeden beliebigen Zeitrahmen anzeigen, von einem einzelnen Tag bis zu einer partiellen oder der gesamten Zeitspanne, für die Daten vorhanden sind. Klicken Sie zur Auswahl des Zeitraums auf ein Jahr, einen Monat, eine Woche oder einen Tag oder geben Sie einen benutzerdefinierten Zeitrahmen ein (z. B. 3. März bis 14. Mai 2002). Sie können auch die Daten zweier Zeiträume vergleichen. Der gewählte Zeitraum und der Vergleichszeitraum bleiben aktiv, bis Sie die Angaben ändern oder die Benutzeroberfläche für Berichte schließen.

Verwendung der Zeitraumfunktion

  • Standardzeitraum: Klicken Sie auf den gewünschten Zeitraum auf dem Kalender, um die Daten in einem Standardzeitrahmen wie Tag, Woche, Monat oder Jahr anzuzeigen. Die Daten im Bericht ändern sich dann dementsprechend.
  • Benutzerdefinierter Zeitraum: Klicken Sie auf das einzelne Kalendersymbol. Ein Feld mit zwei Kalendern wird angezeigt. Wählen Sie im ersten Kalender das Startdatum und im zweiten Kalender das Enddatum aus. Klicken Sie dann auf "Bereich anwenden". Die Daten im aktuellen Bericht ändern sich entsprechend dem ausgewählten Zeitraum.
  • Vergleichszeitraum: Klicken Sie auf das doppelte Kalendersymbol. Unter dem ersten Kalender wird ein zweiter Kalender angezeigt. Legen Sie den Zeitraum für den Vergleich im zweiten Kalender fest.

Empfehlungen

  • Legen Sie für die Überprüfung einer Website mit geringen Zugriffszahlen einen längeren Zeitrahmen fest, um aussagekräftige Daten zu erhalten.
  • Sie können Zugriffstrends für Ihre Website (und deren Daten) ermitteln, indem Sie dazu die Daten eines Jahres oder länger analysieren. Urchin passt die Größe der Diagrammelemente Ihrer Auswahl entsprechend an.
