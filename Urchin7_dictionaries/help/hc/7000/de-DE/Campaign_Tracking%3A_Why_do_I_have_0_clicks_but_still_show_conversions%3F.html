Kampagnen-Tracking: Warum habe ich null Klicks trotz angezeigter Conversions?

Falls für eine Ihrer Kampagnen null Klicks angezeigt werden, obwohl eine gewisse Anzahl vn Cnversions vorhanden ist, ist Ihr Datenbanklimit möglicherweise nicht hoch genug festgelegt. So ändern Sie das Datenbanklimit:

  1. Klicken Sie auf "Konfiguration" (auf der Benutzeroberfläche oben rechts).
  2. Bearbeiten Sie das Profil (klicken Sie auf den Bearbeitungsschlüssel).
  3. Klicken Sie auf die Registerkarte "Speicherkapazität/Datenbank".
  4. Erhöhen Sie im Dropdown-Menü "Datenbanktabellenlimit" die Anzahl der Datensätze. (Der Standardwert lautet "100.000".)
  5. Klicken Sie auf "Aktualisieren".
  6. Klicken Sie auf "Alle Daten löschen".
  7. Klicken Sie auf die Registerkarte "Ausführen/Planen".
  8. Klicken Sie auf "Jetzt ausführen".

Falls das Problem weiterhin besteht, wiederholen Sie die obigen Schritte und geben Sie eine höhere Anzahl Datensätze an.
