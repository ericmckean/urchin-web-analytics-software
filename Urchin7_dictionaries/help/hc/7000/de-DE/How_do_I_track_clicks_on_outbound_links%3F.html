Wie kann ich Klicks auf externe Links erfassen?

Wenn Ihre Website externe Links enthält, können Sie leicht erfassen, auf welche Banner die Besucher klicken, um Ihre Website zu verlassen.

Fügen Sie zum Erfassen ausgehender Klicks den folgenden Code innerhalb des Tags <a> hinzu: 

    <a href="http://www.site.com" onClick="javascript:urchinTracker('/outboundlinks/sitename/pagename');">

Mithilfe dieses Codes wird jeder Klick auf den externen Link wie ein Seitenzugriff unter dem Namen /outboundlinks/sitename/pagename protokolliert. Es empfiehlt sich, alle externen Links, wie im Beispiel dargestellt, in einer logischen Verzeichnisstruktur zu protokollieren. Dadurch können Sie problemlos ermitteln, auf welche Seiten Besucher geklickt haben, um Ihre Website zu verlassen.
