Erstellen von benutzerdefinierten Berichten

So erstellen Sie einen benutzerdefinierten Bericht:

  1. Wählen Sie eine Datenzuordnungstabelle aus oder definieren Sie eine neue. Informationen hierzu erhalten Sie unter Benutzerdefinierte Datenzuordnungen.

  2. Kopieren Sie <Urchin>/lib/custom/profiletypes/Default/default.rs2.sample, um eine neue Berichtdatei zu erstellen (".rs2").

  3. Definieren Sie einen neuen Bericht, indem Sie der Berichtdatei eine neue Zeile hinzufügen.

Die Berichtdatei hat folgendes Format:

  • ID - eindeutige ID des Berichts:
    • C = Kategorie
    • D = Daten
    • M = Menü
    • R = Bericht
    • S = Unterbericht
    • P = Seitenbericht
    • B = Dashboard
  • RS - Dashboard/Berichtabschnitte:
    • 1 = Geschäftsführung
    • 2 = Vermarkter
    • 4 = Webmaster
  • RR - Berichtanforderungen:
    • 1 = Software
    • 2 = Erweitert
    • 4 = E-Commerce
    • 8 = UTM
  • NTITLE - Navigationstitel (Wörterbucheintragsnummer)
  • ONCOLOR - Farbe bei Aktivierung
  • OFFCOLOR - Farbe bei Deaktivierung
  • HELP - Hilfebericht (Wörterbucheintragsnummer)
  • NDATA - Anzahl der Datenelemente. Die Einträge müssen sofort nachfolgen.
    • Auf dieses Feld folgen möglicherweise zusätzliche Unterberichte mit Typ D (Daten).
  • CTITLES - Spaltenüberschriften (Wörterbucheintragsnummer)
  • CTYPES - Spaltentyp zu Formatierungszwecken
    • 10 = Zeichenfolge
    • 11 = Datum/Jahr/Monat
    • 12 = Datum
    • 20 = Nummer
    • 21 = Umsatz
    • 22 = Byte
    • 23 = Zeitverhältnis
    • 24 = Verhältnis
    • 25 = Prozent
    • 26 = Zeit
    • 30 = Prozent insgesamt
    • 40 = Umsatz
    • 41 = Umsatz/Anzahl
    • 42 = Anzahl/Umsatz
  • TABLE - Datentabellennummer:
    • T = Gesamt
    • TR = Transaktion
    • H = Histogramm
    • V = Besucher
    • 1-n-Tabelle
  • DXFORMAT - Datenextraktionsformat
  • RF - Erforderliches Feld
  • RT - Berichtart:
    • 1 = Gesamt - Grafik
    • 2 = Histogramm
    • 3 = Stapel
    • 4 = Beliebteste
    • 5 = Struktur
    • 6 = Verzeichnis
    • 7 = Eingeschränkt - Grafik
    • 8 = Punkt
    • 9 = Klickrate
    • 10 = Definierter Trichter
    • 11 = Als Startseite festlegen
    • 12 = Karten-Overlay
    • 13 = Zielüberprüfung
    • 14 = Zielpfad
    • 20 = KPI-Datumbericht
    • 21 = Top-Ten-KPIs
    • 22 = Zusammenfassung der Trends
    • 23 = Conversion-Zusammenfassung
    • 24 = Dashboard-Übersicht
  • GTYPES - Typen von Grafiken:
    • 1 = Stab
    • 2 = Linie
    • 3 = Kreis
    • 4 = Stab - horizontal
    • 5 = Trichter
    • 6 = Kurven
  • FSEP - Feldtrennzeichen
  • SP - Besondere Optionen (Bitmapnummern hinzufügen):
    • 1 = Gesamt
    • 2 = Durchschnittswerte
    • 4 = Zeitraum vergleichen
    • 8 = In Kleinbuchstaben
    • 16 = Daten im Laufe der Zeit
    • 32 = Overlay
    • 64 = Sprache suchen
    • 128 = Segment
    • 256 = Lifetime-Wert
    • 512 = Transaktionsdetail
    • 1024 = Analysen zu Besuchern
    • 2048 = Begriff suchen
  • SR - Kennzeichnung spezieller Berichte:
    • 1 = Auswahl von Keywords
    • 2 = Klickrate
    • 3 = Aufschlüsselung nach Tagesabschnitten
    • 4 = Besucherliste
    • 5 = Karten-Overlay
    • 6 = Benutzerdefiniertes Wörterbuch
    • 7 = Datumsvergleich
  • FNT - Filterartnummern-Filter:
    • 1: =

    • 2: >=

    • 3: <=

    • 4: >

    • 5: <

  • FNF - Nummernfeldfilter
  • FNV - Wertfilter
  • FST - Zeichenfolgenart-Filter
    • 1 = Ausschließen
    • 2 = Einschließen
  • FSF - Zeichenfolgenfeld-Filter
  • FSX - Reguläre Ausdruckszeichenfolge
