Wie kann ich den vollständigen URI einschließlich der Suchbegriffe in einem Bericht anzeigen?

In Urchin 6 werden Abfragemuster in einer speziellen Aufschlüsselung angezeigt: "Content-Optimierung -> Content-Leistung -> Seitensuchbegriffe". Wenn Sie die gesamte Seite und die Suchanfrage im Bericht "Beliebtester Content" anzeigen möchten, können Sie den unten aufgeführten Filter verwenden, um die Art der Anzeige dieser Daten zu ändern.

Hinweis: Dadurch erhöht sich die Verarbeitungszeit für Profile, auf die der Filter angewendet wird. Außerdem wird möglicherweise aufgrund der höheren Anzahl eindeutiger Seitennamen das Datenbanklimit überschritten.

Erstellen Sie einen neuen "Profilfilter" mit den folgenden Einstellungen:

  • Filterart: Erweitert
  • Filter A: request_stem (AUTO)- A extrahieren (.*)
  • Filter B: request_query (AUTO) - B extrahieren (.*)
  • Ausgabe in: request_stem (AUTO) - Konstruktor $A1/$B1
  • Ausgabefeld überschreiben: "Ja"
  • Erforderliche Felder: "Beide Felder"
  • Groß-/Kleinschreibung beachten: "Nein"
